Luca Mauri’s Summary

I started my career with the Microsoft technologies in early 2000, getting involved in various client/server projects developed with Visual Basic/COM components and classic ASP web sites.

From the first release of the .NET framework I had been focused on the design and development of web applications using ASP.NET, C# and Sql Server and object-oriented and layered architectures.
The web applications ranges from intranet portals and e-commerce sites to advanced e-procurement solutions.

I have designed and developed Business Solution projects using Sql Server 2005 Integration Services and Reporting Services.

Since the beginning of 2007 I'm getting deeper in the Sharepoint product and application framework, and I actually design, develop and promote that platform.

Actually I'm a software architect, developer and team leader and I'm usually what it could be said a "technical PM".

I'm also responsible for the new tecnologies embracement and I do technical interview to new potentially employees.

I try to put in practice some extrem programming principles, such as continuous feedback and pair programming.

I like to share technical materials and what I learn and work with.
I believe in the added-value of technical communities.
