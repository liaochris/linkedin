Francesco D'Amore’s Summary

After achieving my IT engineering MSc degree (specialized in automatic
systems), I became familiar with developing functional applications for Workflow Systems. I worked also on an ERP system used for a multinational enterprise in order to manage and improve the business. At moment I working in a GIS enabled system, where the spatial data are stored in spatial database.
I like designing and developing applications with OO approach in C# and Java. I am interested in technologies, new tools and techniques appearing on the market. Especially in middleware and frameworks that can speed up delivery project time.
I like designing software system architecture and contributing to implementation phase using the design patterns as guideline for the well known problems.
I like work in team and I like also communicating directly with customers.

    January 2008 — December 2008 (1 year)

  • Functional and Platform Developer

    Pitagora S.p.A

    (Privately Held; 51-200 employees; Information Technology and Services industry)

    March 2005 — December 2007 (2 years 10 months)

    I have contributed to design & implementation of PANDORA (Parallel
    Activity Networks Designed On Real Application). The PANDORA
    enterprise application is a Workflow application designed as a client
    server application. The server reads the process that is described
    through XML. After validation, the server is ready to run that process.
    Each client requests the tasks at the server, according to the process, and performs it. Communication between client and server are carried out with a TCP channel or, alternatively, a SOAP communication. PANDORA was developed with Visual Studio .NET as IDE, C# and Oracle and with Agile methodologies.
