Technofreak, is the term i like to use instead of nerd or geek to describe my interest in keeping up with the latest IT and Software Technology. IT Consulting, ERP, Open Source, Java, SOA, SaaS, Semantic Web and Enterprise Integration are some of the topics i like to read the most. I am still too foolish for stopping to learn, adapt and implement new technology.

Nature calls in unfashioned way and can never be standardized as XMLRPC, REST or Hessian/Burlap. :)

I think this will be enough to describe me. My deepest gratittude for reading my writing.
