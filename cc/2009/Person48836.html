Johan Pelgrim’s Summary

I'm an enthusiastic software engineer with a preference towards Java Enterprise applications. We all know a single programming language will not suffice in your day-to-day job so I do an occasional shell-script, Oracle PL/SQL stored procedure, Perl script, Groovy script as well. I've also got many years of team lead experience and simply love to drive or work in a team.

I'm a Sun Certified Java Programmer (SCJP) and Sun Certified Web Component Developer (SCWCD). I'm currently working on getting the Spring Framework and Sun Certified Business Component Developer (SCBCD) certifications.

My main interests (and expertise) are: Enterprise Java (EJB 2.1 / EJB 3.0), Spring (2.0 / 2.5), Hibernate (3) & JPA, Ant / Maven (2), CVS / Subversion, XML / XSD, Eclipse, etc.

My goal is simply to have fun in my role as software engineer, doing a good job and continue to expand my knowledge in whatever way I can.
