Marco Mornati’s Summary

Marco Mornati, members of Byte-Code Leet Group, who works primarly on OpenSource and Emerging Technologies, is an OpenSource Developer and Software Engineer for Byte-Code since March 2007.
Actually he is actively working with Groovy (the agile and dynamic language for the Java Virtual Machine) and Grails (an innovative and versatile web application framework) projects. With these technology he has founded and developed, with others ByteCode Leet Members, a new open source project Symbolic (www.opensymbolic.org): a web application which aim to simplify the management within complex enterprise infrastructures.
