Beau Frusetta’s Summary

I've spent over a decade pursuing the quest of building, breaking, and subsequently "fixing" things. I started off my career building custom MS Access based database applications using Visual Basic for Applications. My mentor at the time (total Microsoft nerd) taught me the basics of building software using Visual Basic with a basic database back-end. I then moved on to SQL Server 6.5, then 7.0, and then settled with 2000. I worked primarily within the financial industry building reporting applications and customizing automated software processes for easy access to "the good stuff". I then moved away from the Microsoft way of life and dove in to the "dark side" -> L.A.M.P. (Linux, Apache, MySQL, & PHP). I've spent the last 4 years of my life dedicated to web development, but still doing some crazy database architecture when the project comes up.

I feel I'm a very well rounded web developer now, with not only extensive back-end experience and "making things tick", but also experience in developing the best in front-end design using XHTML & CSS.
