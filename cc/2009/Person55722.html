Kingsley Starling’s Summary

I specialise in infrastructure consultancy using Microsoft technologies, designing Microsoft Active Directory and Group Policy implementations as well as automated deployment of Windows operating systems.

Konsultancy is certified by Microsoft for delivering Desktop Deployment Planning Services (DDPS) to Microsoft Software Assurance customers. Get in touch to find out how customers could get my services for FREE.

I also carry out audits of customers Active Directory implementations to help highlight how Active Directory should be designed, secured and optimised to make better use of the investment made in Microsoft technologies.
