Stefan Cruysberghs’s Summary

I’m a passionated software developer and I've been developing on the Windows platform for almost 15 years. I have a lot of experience with OOP, UI design, Borland/Codegear Delphi, Microsoft Visual Studio, C# & .NET and databases like SQL Server and Oracle.

My personal website (http://www.scip.be) has been developed with PHP and MySQL. This website is being used to publish (travel) photos, components and articles. These articles are about the technical challenges I face on a day to day basis and the research/study I am doing in my free time.

My current interests are learning as much as possible about .NET 3.5 and 4.0, WCF, LINQ, WPF, Silverlight, RIA Services, Enterprise Library, ADO.NET Entity Framework, Visual Studio Team System, Live platform, Dependency Injection & Mock frameworks, Design patterns, …
