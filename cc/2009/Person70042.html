Monty Kamath’s Summary

Used VisualWorks 7.4.1d to build the 5.0 and 5.1 releases of Netcool/Proviso – a distributed network monitoring system implemented in an event stream processing architecture. These releases featured JavaScript formulas that execute within the Smalltalk vm and allowed customers to write JS formulas that perform SLA computations in a massive stream processing system. The JS syntax provided for time semantics and buffering requirements (formula windows) as well as data joins via input arguments.

Created and taught GO Training for JavaScript formulas class.

Implemented the Plan Builder component to plan the entire graph of CMEs and computations that take place.
