Micah Burnett’s Summary

Over 8 years experience in the information technology industry with extensive experience in design and development of applications for the web, and for Windows applications. Over 6 years experience with SQL Server in designing, creating, maintaining, databases from user requirements. Have experience using cursors, writing stored procedures, triggers, views, T-SQL scripting data to be published

Developed object-oriented front-end components which connected to middleware services, which gave front-end developers easier access to the services. Maintained and published several concurrent versions of these objects which allowed multiple parties, including front-end, offshore and other middleware developers to coordinate development efforts. Worked cooperatively with other departments in deploying code, debugging problems with front-end code, middleware and back-end systems. Mentored other developers in using and to enable them to take over these components.

Responsible for design and development and documentation of contract-first WCF and Web Services on the company ESB. Defined XSD contracts (using schema inheritance) to validate serialized XML instances. Interfaced with several company systems including Siebel, Oracle, MySql, Active Directory, MSMQ and SQL Server to create a unified method of accessing business entities. Coordinated the use of these services between middleware and front-end / offshore groups. Responsible for the deployment of middleware services and front-end objects to DEV, QA and PROD servers, and analyzing the impact and implications of these moves before they were made.
