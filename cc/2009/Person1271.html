Adam Wolf’s Summary

Blue Cognition's main focus is developing custom applications delivered on the Microsoft platform. We use a modern agile development and project management methodology to deliver complete and functional software that meets and exceeds our client's expectations. The main points of our agile practice include:

We will deliver working software within the first week of our project to facilitate an open and transparent view into the project progress.
Usable software. The project will always be in a release state so our project sponsors can use the developed software at any stage of development.

Our client control the development process by setting the priority of all features. Our clients can change and add items to the development project by adding new features and reorganizing other features. We will always work on the next highest priority item.

The most important conversation will be between the business feature sponsor and the developer right before the developer codes the feature. We believe in using open communication, not a specification document, to deliver feature requirements to the developer.

We use automated testing tools and automated build system on every project to enable us to always deliver working software at every step of the development lifecycle.
