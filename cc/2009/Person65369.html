Matt Owen’s Summary

Experienced (18 years+) IT Professional, specialising in infrastructure architecture and design. Also a qualified Prince2 Project Manager, with several successful projects completed. These include:

- Design and implementation of Large Active Directory installations for several blue-chip companies.

- Design and implementation of several corporate desktop systems, including Group Policy management and remote configuration solutions.

- Design and implementation of a large scale unattended file re-permissioning model for a global Corporate Banking customer.

- Managed successful Europe-wide desktop infrastructure rollout of Windows NT to over 2,500 corporate users.

- Managed a global banking call-centre implementation project, from concept to completion.

- Created reference design documentation for various large scale projects.

- Engineered remote low-bandwidth application deployment system for 1,500 remote laptop users.
