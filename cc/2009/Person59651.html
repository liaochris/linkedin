David Little’s Summary

A creative and enthusiastic web professional with over nine years’ experience. I’m passionate about designing and implementing web interfaces that are easy to use and meet users’ needs and expectations.

I'm currently seeking to refine my career more in the direction of my key interests in web user interface / interaction design and usability.

Please note I am not looking for front-end developer roles, unless they contain a substantial UI design / usability element.

Most recently I’ve been working on UI redesign projects and user testing and have recently completed a postgraduate-level course on User Interface Design and Evaluation with the Open University (UK), for which I gained a distinction. In addition to these skills and knowledge I have strong web development skills, with front-end technologies such as HTML, CSS and Javascript and back-end coding with Python for the Plone CMS.

I'm involved with the Out of the Box Plone Themes group, producing alternate visual designs for the open source Plone content management system.

I've recently completed my third year of German (to first undergraduate level).

David Little’s Specialties:

User experience design (inc. site audits, use cases and user scenarios and documenting specifications).

User interface design and evaluation, including wireframing, prototyping and user observation and testing.

Creation of content management system (CMS)-based sites with the open source Plone CMS (versions 2.5 and 3x).

Excellent front-end web design and development skills, including HTML, CSS and Javascript.

First class communication and writing skills.
