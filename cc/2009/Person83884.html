Rich Smrcina’s Summary

Over 25 years experience in the IT industry with an emphasis on System z infrastructure and support.

Installation and implementation of z/VM and Linux for System z, including but not limited to internal networking infrastructure of Virtual Switches, Hipersockets and Guest LANs. Requirements for good performance, including I/O generation and system configuration options.

z/VSE installation, implementation and maintenance. Also IBM and ISV software available for z/VSE. Very familiar with the IBM VSE Connectors software (VSE/VTAPE, VSE Redirector, VSE Web Services).

Very comfortable with a large number of open source software packages on Linux; Including but not limited to: Apache, MySQL, Xymon, SugarCRM, OpenCOBOL.

Programming experience in several languages, COBOL and S/390 Assembler and multi-platform REXX. Some experience with C and Java, a little Perl and PHP.

Wrote the following articles for zJournal, a System z oriented publication, http://www.zjournal.com:

Configuring Linux to Authenticate to the z/VM LDAP Server - Apr 2009
REXX for CICS - Jun 2007
Web Services and VSE: Providing SOAP Services - Aug 2005
Web Services & VSE: Fit Into a Modern Systems & Applications Infrastructure - Jun 2005
Using the CICS Transaction Gateway - Aug 2004
The VSE Redirector - Jun 2004
Linux for zSeries: Expectations vs Reality - Dec 2003

Coming Soon: COBOL Modernization and Development Under Linux

Maintainer of the System z clients for the Xymon network services monitor (formerly Hobbit) http://hobbitmon.sourceforge.net/.

Rich Smrcina’s Specialties:
