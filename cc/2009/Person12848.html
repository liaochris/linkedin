Brad Turner’s Summary

Active Directory 2003 Migration w/MIIS 2003 - USAirways, Architect in charge of MIIS utility server to aide in migration from Lotus Notes 5.09/NT4, Exchange 5.5/AD, to new AD forest w/Exchange 2003.

Microsoft Identity Integration Server 2003/Version 3 SSO Deployment – Casino Arizona, Architect in charge of Assessment, Design, Build, Testing, and Implementation of an identity management implementation involving Active Directory, Exchange 2003, Version 3 Simple Sign-On, and a AS/400 DB2 based HR system (Infinium). The solution required the development of a Delayed Events system as well as Reporting, Audit and Entitlement checking for all positions.

Version 3 Simple Sign-On Deployment – Casino Arizona, Architect in charge of Assessment, Design, Build, Testing, and Implementation of a single sign-on solution utilizing Version 3’s SSO. AS400 Client Access, Casino Marketplace, and Kronos Workforce Central localized accounts were interfaced with the SSO Client and integrated with Active Directory identities.

Active Directory 2003 Migration w/MIIS 2003 – DHL Worldwide (Prague, CZ), Architect in charge in Designing, Building, and Testing an MIIS 2003 based discovery system for the worldwide migration of multiple legacy (NT4 & AD) domains throughout the European region. MIIS was integrated with NetIQ to perform the migrations.This project netted Ensynch a 2006 Partner of the Year Award in Best Practices/Competency.
