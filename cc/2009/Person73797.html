Nicholas Kellett’s Summary

Web application developer, solution architect, speaker, instructor, and blogger, with over a decade of international experience in New York, London, Ottawa, and Sydney. Currently focusing on Web 2.0 trends and technologies and Microsoft Solutions, in particular C# development and SharePoint 2007 (Microsoft Office SharePoint Server 2007 (Certified) and Windows SharePoint Services).

I am finalizing the open source release of a custom C# 3.0 web application integration framework. This framework, evolving since 2003, makes it easy to create multilingual n-tier web applications. Its flexible architecture supports run-time plug-and-play components via use of the .NET Provider Model, Generics, and object oriented best practices. More details and support website available shortly.

Follow me on Twitter at http://twitter.com/nickkellett
