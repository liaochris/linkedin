Frans van Berendonk’s Summary

As a Flash developer I specialize in dynamic Flash AS3 websites, RIA's and desktop applications. My main expertises are interface design, application logic, visual programming and Flash remoting. Main interests are game development, data binding, design patterns and persistant user-driven webapplications. I have a serious attitude towards code, a lenient one towards people. I tend to have a DRY / KISS style of OOP programming.

Frans van Berendonk’s Specialties:

Object Oriented Programming, Actionscript 3.0 (Flash), XML, AMF, PHP, MySQL, (X)HTML, CSS, PhotoShop, Cinema4D. Open Source (MochiAd API, PaperVision3D, APE, Joomla, Drupal, Wordpress). I have recently begun to explore the Collada format and have a strong interest in 3D graphics for the web. My main fields of expertise are OOP, Flash remoting and UI development.
