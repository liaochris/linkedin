Time for a rewrite of this section.
Now have a job in Amsterdam so all good. Back to work with C# and ASP.NET building SOAP based web services, good fun. hope I will get a chance to introduce my new job to python at some point, i think they like it.
Have also introduced myself to Stripes, a nifty Java-based web framework that's far nicer than Spring imho. Shame int's mindshare isn't bigger, but who knows where it's heading?

So who am I?? Application Architect and Developer, Been working with a variety of platform, started out with c++ and java application (J2SE) and then moved on to J2EE (Weblogic and JBoss), then over to .NET.

Lately I been in c#, php, python and javascript land, doing web applications using AJAX via the AJAX.NET framework (for .NET stuff, quite nice) and the PHP based XAJAX framework (neat). Also started to use more 'modern' web application stacks such as CodeIgniter, Django and Turbogears .

I have a in-depth understanding in mobile phone application technologies such as WAP, i-mode and SMS communication, although I suppose I'm starting to get a bit rusty!

Am a big fan on Agile methodologies, it makes sense.
