Karin Dirkx’s Summary

My goal is designing attractive, intuitive and fun to use interfaces.

I've been a sr. web designer / interaction designer at Partout Interactive (www.partout-interactive.nl) for 5 years. And tried to be a game designer at Fantazm (www.fantazm.nl) for about 1 year. After that, back to web design as a sr. web designer at Integrace (www.integrace.nl).

Currently I'm a multi media designer / project manager at ISAAC Eindhoven (www.isaac.nl) and I'm owner of Medialoog (www.medialoog.nl) which is a design company, on- and offline.
