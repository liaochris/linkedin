Craig McDonald’s Summary

Web developer/designer with 10 years experience using Adobe technologies to create compelling web solutions.

With extensive experience using ColdFusion (versions 4 -> 8), integration of large legacy systems to interface with the web, database platforms such as Oracle, MSSQL, MySQL and more.

A keen follower of latest technology trends and best practices with exposure to ColdBox, Reactor, ModelGlue, Fusebox and others.

Strong experience not only in coding but also graphic design using Adobe Fireworks and implementation in CSS, creating interactive websites using JQuery, Ajax, Flash and video encoding technologies such as Flix On2.

A quick learner who is always looking to develop compelling web experiences using the latest and current technologies, with a distinct focus on usability and an eye for detail.
