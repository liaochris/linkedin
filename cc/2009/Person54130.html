Keiron Waites’s Summary

I am a web application developer with five years of experience. I work for a successful design agency with many popular-brand clients including Tesco, Asda and Nationwide.

I also oversee development of a web 2.0-esque online wish list service at www.wishlings.com.

I intend to continue to progress within my field, expanding my skill set and offerings within my current employer. I am also focussed on growing the wishlings.com community and pushing the service towards unobtrusive profitability.

Keiron Waites’s Specialties:

Core skills: Object oriented web application development in PHP 4, PHP 5 and Ruby on Rails, Facebook integration using FBML, FBJS, Mock JS and the Rails Facebooker library, SQL mainly through mySQL.

The usual web development suspects: XHTML, CSS, XML, Photoshop, Javascript and various ajax libraries including jQuery, Ext and YUI.

I also maintain Debian based linux servers and do the odd bit of bash scripting.
