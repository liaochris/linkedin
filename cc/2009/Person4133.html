    (Information Technology and Services industry)

    August 2005 — Present (4 years)

    The HddStatus project uses millions of reports from real users with real hard disks to create a statistical model of the S.M.A.R.T. status of each hard disk model. This way the software can identify hard disks with S.M.A.R.T. attributes that deviate from normality.
    The online analysis of the S.M.A.R.T. attributes of an hard disk include additional checks and messages that inform the end user as if an expert had a look at those attributes. The power of a human expert and the numerical knowledge of the computer are used to evaluate the fitness of the hardware to allow for early detection of failing hard disks. This project is even more interesting if you consider that S.M.A.R.T. has little documentation and that no manufacturer was willing to describe his specific implementation. Nonetheless, the model generated out of the reports is able to early identify several issues.
