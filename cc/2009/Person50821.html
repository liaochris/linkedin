Joe McCann’s Summary

UI/UX Engineer & Creative Director w/8+ years of experience w/heavy focus on the future of web-based applications involving AJAX, jQuery, Object-Oriented JavaScript, PHP, XML, XSLT, and MySQL. Well versed in XHTML 1.0 Strict and CSS 2.1 for web design.

Heavy experience w/many web services specifically Google APIs including the Google Maps, Google AJAX Search, Google Visualization, and Google Language Translation APIs.

Accessibility, & ultimately universal design, is a major focus when developing a site or application. All projects are cross-browser compatiable (Internet Explorer 6-8, Firefeox 2+, Opera 9+, & Safari 3.2+ at a minimum) along w/whether or not the user has Javascript turned on or off. I implement the WAI-ARIA (Web Accessibility Initiative - Accessible Rich Internet Application) specification into my rich internet applications as ARIA is quickly becoming the standard for making applications truly accessible.

I integrate performance & optimization techniques such as pre-loading/caching images & files & even concatenating multiple JavaScript files, compressing them, & outputting a single file during build/deployment, thus reducing page weight & the overall number of JS file requests. These cost-effective measures save money on bandwidth & provide the user w/an overall improved experience w/faster page loads.

Writing documentation is a specialty of mine, whether it be generating wireframes, userflows, or design documentation. At BearingPoint, I created a template to be used for all projects going forward called the User Experience Design Document, which is an elaborate "proposal" for the client complete w/design compositions & thorough descriptions of high-level concepts & even granular ones such as interactive elements.

Loads of direct correspondence w/high-profile clients. Moreover, I have had tight relationships w/many project managers & requirements analysts to help shape, define, & determine use cases & even core requirements.

Joe McCann’s Specialties:

The scope of my expertise lies on the developement AND design side. User Interface/User Experience development is where my passion lies in regards to the web. I feel that the interaction between the application layer and the presentation layer can be unified to create truly rich internet applications. I tend to focus on the programming aspect first for any web application and then apply the INTERACTION to how the User Interface is going to be presented.
