Dan Johnson’s Summary

Over 28 years of experience in the VLSI Design Automation field primarily focused on mixed-signal design environments. Would be interested in networking opportunities with companies/individuals that are developing deep submicron process technologies or EDA tool vendors focused on deep submicron/mixed-signal design issues. I have extensive experience porting open source EDA tools to Mac OS-X and various Linux platforms. Broad exposure to operating systems (Mac OS-X, Fedora 10 Linux, Ubuntu 8.4, Solaris 10, Windows XP) and software development using C, C++, Objective-C, Java, Qt, and some TCL/TK. Recently became a registered iPhone developer.

Embedded Java multimedia applications are also of interest especially those which use QuickTime, MPEG-4, OpenAL, and advanced 3D modeling/rendering technologies. Java technologies of interest include J2SE, J2ME, Java3D, JINI, JMF, Android, & JavaFX. Using Xcode, Eclipse, and Netbeans for Java development projects. Also develop virtual world content for Second Life and OpenSim.
