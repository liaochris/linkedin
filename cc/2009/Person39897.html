Hobi HaQue’s Summary

I am a Software Developer with six years of computer experience writing applications for Unix, the web, and Windows using Java, HTML, XHTML Servlet/JSP, JSF and Hibernate. I am skilled in developing, building and deploying enterprise level web-based GUI applications using JavaServer Faces, Ajax api like JQuery, CSS with Object/Relational Mapping Framework (Hibernate) in Oracle 9i, and Web Services. I received my Bachelor of Science, Computing Science with Management Information System concentration from the University of Central Oklahoma in 2002.
As a Computer Programmer (AJW-141, Contractor, Arctic Slope), I maintain, develop, test, deploy, and refactor web applications. Automation is a fundamental aspect of my job whereby I use JEE framework to make it happen. My goal is to use component based web applications which are similar to Swing in a desktop world. I am currently working to migrate to Web-enabled, server-centric, and component-based enterprise Data Management System (DMS) from a large legacy Power Builder application. I am utilizing JSF and Ajax, Seam and Hibernate with SOA web services in JEE framework, Sun Java Application server 9.2, and am developing portlet for Portal server using Liferay. For further information, please visit my blog: http://hobione.wordpress.com
