Arul Selvan’s Summary

Dynamic software professional with 19 years of design/implementation experience in leading-edge technologies including network management/security, internet/electronic commerce, distributed client/server, image processing/display systems using C/C++ and JAVA programming languages under UNIX (LINUX,SOLARIS,AIX).
