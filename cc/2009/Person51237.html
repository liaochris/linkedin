Jos Kirps’s Summary

I've created a certain number of applications and platforms, including the Galaxiki community site, which is a wiki based editable galaxy with millions of stars, planets and moons (www.galaxiki.org).

Other projects are CorneliOS, which is a webOS and web 2.0 application framework, and the Joopita Enterprise Appliance which is a CRM for companies using agile development, management and marketing techniques.

I'm also the creator the OLEFA web platform for schools (www.olefa.com) which has become the market leader in Luxembourg, Europe, followed by the AFELO image wizard (www.afelo.com) in 2006. But I'm now no longer involved in this project.

    The Galaxiki Project (Sole Proprietorship)

    (Sole Proprietorship; Myself Only; Internet industry)

    February 2006 — Present (3 years 6 months)

    Galaxiki is a a community/web2.0 website, it's a virtual galaxy with over a million stars and solar systems that can be explored using a 2-dimensional map. Each star, each planet and each moon represents one wiki page, the idea behind the project is that Galaxiki site members can name edit those pages (stars, planets & moons) and write fictional histories about them. As a website member you can optionally "purchase" a virtual solar system, so that only you (and not other community members) can name and edit it.
