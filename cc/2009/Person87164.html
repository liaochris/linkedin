Ruud Hirschmann’s Summary

Ruud Hirschmann is an experienced Project Manager regarding applications and outsourcing.

As a calm and focused Project Manager he is able to fully motivate the project team. His last projects were data centre migration and application implementation projects. These implementation projects were related to the products MS Sharepoint and SAP Business Warehousing.

He provides optimal performance in hectic (international) environments under pressure and strict timelines. Ruud satisfactory managed the expectations and activities as set by the customers’ requirements.

Ruud managed complex application and infrastructural projects at large international companies. Often these projects concerned the introduction of entirely new applications. The acceptance of the application has been achieved by successful user acceptance tests (UAT). Ruud is able to communicate with customers as wel as (project) parties involved. His strength lays in his calm and keeping the overview of the complex matter and environment. Prince2 project standard were applied where possible.

Beside implementation projects, Ruud operated as Project Manager within several outsourcing programmes. As a transformation manager, he realised the politically sensitive transformation phase of these outsourcing deals. He completed these programmes/projects successfully, while safekeeping the business continuity.
