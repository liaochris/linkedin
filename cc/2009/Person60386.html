  • Developer

    TikiWiki CMS/Groupware

    (Internet industry)

    2004 — Present (5 years)

    TikiWiki is a Free Software to which I contributed multiple times over the years. Among others, my contributions to the project includes:

    * TikiSheet, which was probably the first JavaScript-based spreadsheet editor on the web to support standard features like calculations and import/export from common formats.
    * GraphEngine, a companion library allowing to generate graphs from spreadsheets to different output formats (Image, PDF, PostScript).
    * I worked on improving the support for multilingual websites by adding features to enable collaborative translation. These efforts lead to a published article. (See http://www.wiki-translation.com)
    * In order to simplify the release process, I improved the release automation scripts and database update tools.
    * Performed various optimizations to remove bottlenecks, improved security filters, performed refactoring on many aspects including plugin management.
