Toby Hilden’s Summary

I started in web production about 10 years ago. I learned HTML and built my first web pages by reverse engineering Gillian Anderson fan sites. I learned production in the pre DOM "do whatever it takes to make it look good in netscape 4" era. When dot com websites were made with complex tables, spacer gifs and crazy glue.

I moved on to ad agencies where projects were built entirely in photoshop, cut up and jig sawed together with dreamweaver. Along the way- I've mastered photoshop, dabbled in 3-d art and incorporated Flash into my toolbox. I'm currently embracing the world of pure CSS layout.

Toby Hilden’s Specialties:

Hand Code Production HTML, Graphics for the web. Building web templates. Search.
