Alex Stanford's Projects

  • SiteSense

    • August 2009 to Present
    Team Members: Alex Stanford

    SiteSense is a simple but powerful, secure and high-performance web platform. At the foundation of the platform is a content management and blog publishing system. SiteSense is easily extendable making it capable of being the starting point for any web application. The framework is constantly scrutinized for security purposes and is written in PHP & MySQL utilizing the latest best practices.

  • BoxSir

    • April 2011 to Present
    Team Members: Alex Stanford

    BoxSir is an innovative, fully customizable, standalone and lightweight web-based image viewer that's written in JavaScript. BoxSir sets itself apart from the crowd by forgoing fancy animations in favor of optimization, reliability and functionality. Using BoxSir, web developers can showcase images in all major browsers without navigating users away from the current page.
