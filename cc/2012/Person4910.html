Mathieu Martin's Summary

Currently I create assets for the online massively multiplayer role playing game Second Life created by Linden Labs, and run an in-world shop that sells the displacement maps and textures required to create objects. This experience has taught me a great deal about working with sculpt maps and the tools needed to export models from 3DS Max into Second Life, as well as the in-game tools create to create and texture a wide range of objects.

Prior to this I worked at Absolutely New Inc, a place that helped hone my skills in 3DS Max, Photoshop, Illustrator, and After Effects as well as acquire basic skills for assembling and wrangling render farms using multiple platforms.

I am currently looking for full time employment.

Specialties

Object and hard surface modeling, unwrapping, texturing, rigging, quickly learning new software and productivity techniques including proprietary software tools, respect for deadlines, working as part of a team, startup company experience, and render farm wrangling.
