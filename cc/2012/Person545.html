Andrew Corkery's Summary

I work as a web developer in Dublin. Most of my professional experience to-date has involved building web & server-based applications on the Microsoft .NET platform. I’m passionate about software development and like to keep up with what’s going on in the industry.

Though my day job is almost excusively ASP.NET / SQL Server development, I’m comfortable working with other, more open-source, development platforms like PHP & Ruby on Rails. At the moment I’m focusing on honing my Javascript skills and keeping a keen eye on where Node.js is headed.

I started working as a web developer at Newmedia (now Framework Design) in 2006, where I learned heaps. In 2007 I did some travelling which included a stint at the fast-paced digital agency Amnesia Razorfish in Sydney.

After returning to Ireland, I took up a position with X Communications where I am happily working today.
