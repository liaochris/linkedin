Jonas Bonér's Summary

Hacker and entrepreneur.

Currently the Co-founder and CTO of Typesafe. Have previously worked at Terracotta and the JRockit JVM at BEA Systems.

Active contributor to the Open Source community; most notably created the Akka Project, the AspectWerkz Aspect-Oriented Programming (AOP) framework, committer to the Terracotta JVM clustering technology and been part of the Eclipse AspectJ team. In 2011 he became a Java Champion.

For more info see: http://jonasboner.com/

Typesafe

Privately Held; 11-50 employees; Computer Software industry
