Cesare Rocchi's Summary

Cesare Rocchi is a speaker, book author, UX designer and developer specializing in web and mobile applications. He began working on interactive applications fifteen years ago, while he was a researcher in the academia. He runs Studio Magnolia (http://www.studiomagnolia.com), an interactive studio that creates compelling web and mobile applications. You can find him on Twitter (http://twitter.com/_funkyboy).
When off duty he enjoys snowboard and beach tennis.

Specialties

Mobile Applications Development, Rapid prototyping, User Centered Design, UX design.
Objective-C, iOS SDK, Lisp, Python, Ruby, (X)HTML, CSS, JavaScript, Sinatra, Wordpress, Flash Platform, Java
