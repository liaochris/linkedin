Background

Summary

I am a senior software engineer with over 14 years of experience in software development with Java and building Web applications. I developed dozens of enterprise systems carrying out activities of gathering requirements, UML modeling, definition of software architecture, implementation, testing and optimization of applications.

In parallel with the activities of software development, I was a teacher of courses in Java technologies from basic to advanced level covering concepts of object-orientation, Java programming, development of enterprise applications with Java EE, Spring Framework, HTML and Javascript.

I gave several lectures and short-courses in events in Brazil and the USA. As a teacher and lecturer I honed the ability to transfer knowledge and transforming what is complex in simple for students and other programmers.

The ability to prepare technical presentations for technical classes and large audiences in technical events allowed me to interact with sales teams to assist them during presales and technical presentations to customers driving them to the adoption of Java related technology. That ability also allowed me coaching other programmers in the best way to use the Java technology.

As a software engineer, I have developed software using techniques of object-orientation, design patterns, algorithms and data structures aiming to build applications that are efficient, optimized, easy to maintain, easy to read and always meeting the deadlines.

My strengths include reliability, honesty, flexibility, commitment and teamwork.

Specialties: Software design and programming with Java, Spring Framework, object-orientation, HTML, Javascript and other related Java technologies. Presentation in lectures and classes for small and large audiences.
