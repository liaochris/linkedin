Publications

BLOB + Stream = BlobStream(Link)

Microsoft .NET Magazine
March 2006

This article tries to provide a solution to a common problem: how to handle blob data in your applications efficiently without leaving your code in difficult to read state of spaghetti. In .NET there is a fairly simple solution: combine the T-SQL-commands that SQL Server supply for working with blobs (Binary Large OBjects ) with the capabilities of System.IO.Stream.

This article is written in Dutch. For a translation in English and a slight expansion to Oracle Lob storage by Peter De Jonghe, I refer you to http://www.codeproject.com/KB/database/LobStreamFrameworkPart1.aspx

Computer aided generalization of road network maps(Link)

Department of Computer Science, Utrecht University, INF/SCR-97-15
December 1997

This article presents an algorithm for generalizing road networks that is created with the generalization framework proposed by Shea and McMaster [SM89] in mind, and that will take the geometrical properties of the individual roads into consideration. Both a specification and an implementation of the algorithm (in Java, and available on the world wide web) will be presented. Unfortunately the implementation could not be completely finished in time to include some results and some conclusions about the practical usefulness of this algorithm. (The link is to the presentation of this work at GeoComputation '98 in Bristol, as the original thesis and Java applet are now no longer online.)
