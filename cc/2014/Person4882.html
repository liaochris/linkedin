Background

Summary

Ryan R. Smith specializes in web standards compliant web design, usability, interaction design, information architecture, search engine optimization, graphic design, and rich user experiences.

Ryan's specialties: Building user interfaces for web sites and applications with Drupal (an open source software package that functions equally well as a PHP and JavaScript-based application framework as well as a content management system - see also: http://drupal.org), user-centered design, user experience, print and web graphic design, interaction design, communicating with screen-based prototype sites, wireframe and page flow diagrams, whiteboarding, information architecture, web standards, CSS, HTML, usability, search engine optimization (SEO), rich user experiences (RIA systems), centralized and distributed team environments, agile project management, open source methodologies (tools, applications and community), GoogleMaps and web-based cartography, web-based software-as-a-solution (SaaS) systems and mapping return on investment (ROI) with web-based analytics.

Ryan is also always staying abreast of new and upcoming web technologies. These days it seems to be all about educating stakeholders about exactly what "responsive web design" means and how having a "responsive website" does not mean having a "mobile ready website" nor a "mobile friendly website" nor even a "usable website on a mobile device". Ryan is looking forward to fully embracing the mobile-first aspect of designing & developing online services & systems for real people.
