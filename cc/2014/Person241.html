Publications

"A Study of the Effect of Using Kademlia as an Alternative to Centralized User Location Servers in SIP-based IP Telephony Systems"(Link)

Australasian Telecommunication Networks And Applications Conference - ATNAC
December 2008

Some proposals are found on the literature for decentralizing SIP-based Voice over IP (VoIP) systems, replacing the client-server architecture for a P2P overlay network but none of the proposed solutions deploying a P2P network had studied the impacting of decentralization on call setup delay. We proposed a system using Kademlia instead of a centralized user location approach. Kademlia offers concurrent searches and this improves the latency in user location and thus provide a faster call establishment when the network experiences different delays, as showed in this study. In this paper it is also described how the system using Kademlia works as well as the tests performed in order to measure the impact on call establishment time when decentralizing a user location service on an IP telephony system.

A Self-managing SIP-Based IP Telephony System Based on a P2P Approach Using Kademlia.(Link)

Asia-Pacific Network Operations and Management Symposium (APNOMS)
November 2007

Most of Voice over IP(VoIP) systems are deployed using SIP protocol and based in a client-server architecture. A lot of people and companies could be benefited if the system did not require the configuration and maintenance of central servers. Some proposals are found in the literature. In order to improve the efficiency of such approaches and to provide a complete self-configuring service of IP telephony, we suggest a new solution based on the P2P algorithm Kademlia. According with the results of our experiments, our approach has a better performance when network experiences di?fferent kind of delays due to the fact that Kademlia supports concurrent searches, improving the latency in user location, and providing a faster call establishment.

Descentralizando Sistemas de Telefonia IP utilizando o algoritmo Kademlia(Link)

XXV Simpósio Brasileiro de Telecomunicações - SBrT
September 2007

Most of Voice over IP(VoIP) systems are deployed using SIP protocol and based in a client-server architecture. A lot of people and companies could be benefited if the system did not require the configuration and maintenance of central servers. Some proposals are found in the literature, and based on these we suggest a new solution based on the P2P algorithm Kademlia, with the intuition to improve the efficiency of such approaches.
