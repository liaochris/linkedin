Projects

Automated migration from Windows XP to Windows 7

April 2013 – Present

For a mid-sized company (3500+ seats) based in the Netherlands, we provided an automated solution, to migrate from Windows XP to Windows 7. We provided this solution based on the customers' SCCM 2007 environment and in close cooperation with the customers' on-site serviceprovider.
We demonstrated the solution in a Proof of Concept, after which the customer received the documentation to implement the solution in their own environment. Key in this migration was securing the data from the old situation to the new Operating System. This goal was met through the use of custom built scripts, which took care of the migration of data and the compliance with the new naming convention of the company.
Desktop-hard-drives were unencrypted and no problem whatsoever. The hard-drives of the customers' mobile devices (notebooks) were encrypted with an older version of Safeguard Easy. With support of the local on-site party we were able to disable pre-boot authentication through the command line and migrate the laptops according to regulations.
