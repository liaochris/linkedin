Wilhelm Svenselius' Summary

Software developer and architect with broad experience in many fields of development; particularly web services and mobile apps but I have done extensive application development work for Windows as well. I enjoy solving complex problems and coming up with effective designs for software, no matter the scale. I also greatly enjoy working in pre-sales and other roles where I can interface directly with customers, partners and third-party developers. Aside from development and systems architecture, I have significant experience in technical writing as well as general server/database/network administration, particularly with Microsoft products. I have been working full-time in the IT field for several years while trying (with limited success) to finish up my Master's thesis for an M.Sc in Information Technology.

Specialties: Mobile device software development (Windows Phone 7/8, Windows Mobile, Android, C#, Java), web front and back-end development (HTML/CSS/JS, PHP, ASP.NET, C#), desktop application development (Windows, Windows 8/RT, C/C++, C#, VB, Java). Agile practices.

Software architecture and design, database design and database server administration, Windows server administration, database server administration, computer hardware, computer networks and network protocols, information security, web/print graphics production.
