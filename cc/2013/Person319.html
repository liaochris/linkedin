John Madrak's Summary

I am passionate about developing web applications for higher education and similar environments. I enjoy working in these environments because I believe that there is a large opportunity for growth and better utilization of technology for universities.

My primary skills and talents lie in the area of web application development, deployment and design. I have had numerous opportunities to use these skills in various development environments as well as leading and assisting other developers and designers towards these same goals on many projects.

Working with HTML5, CSS3, jQuery, PHP 5 and MySQL I strive to create powerful web applications that will empower universities to meet their needs. So far, I've developed several large scale applications including a custom Board of Trustees Portal (based entirely on my own custom MVC framework), a complete targeted content management system and a QR traffic tracking system as well as several other smaller projects.

I have also had the opportunity to work with PL/SQL while modifying and updating various packages using Banner as well as working with the BlackBoard Building Blocks API (Java/JSP) to create several tools for our university including one published on the oscelot community.

Finally, I also have experience in managing linux web servers and various types of windows servers. I have worked with Debian, Ubuntu, RHEL, CentOS, Windows Server 2003/2008 and Windows Vista/7 in various different circumstances. I am most comfortable with CentOS, focusing on managing and maintaing web application servers. In the past, I have managed several servers for individual client's applications using Apache, MySQL Server, PHP-FPM, IPTables, and DenyHosts. I have also grown quite familiar with Proxmox Virtualization Platform and have managed several different virtual hosts for my own use as well as professionally for my clients.
