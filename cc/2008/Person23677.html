Evgenios Skitsanos’s Summary

IT dedicated experiece since 1992, since 1996 in web development and web project management. From 1999 working for USA, Israeli, Cyprus, Norway and UK companies remotely and trough relocating to their offices.

IT activity mostly in applied programming with ASP/VB/VB.NET/Rapid-Q and other Basic-like syntax languages, include own compiler BLOC, XML/XSLT/XQuery, SQL, ODBC, ADO, MySQL, XMLDSO. I’m an expert in XML-driven web development with using "WDK for ASP/ASP.NET” (webware development kit for extremely fast webware deployment). Great with Adobe Flex/ActionScript and other RIA Frameworks including AJAX-ones, like Adobe Spry, YUI, Backbase and others.

I have good experience in work with Microsoft .NET Frameworks v.1.1 and 2.0 both, implemented first MembershipProvider service for .NET v.1.1 that announced by Microsoft to be included only with .NET 2.0.

Have completed few projects for serial communications and TCP/IP for Win32, implemented ActiveX/COM components for developers of VBA based applications within Microsoft Office suite.

I’m also implemented own electronic data exchange protocol based on XML for few American corporations for use in financial data transmission and synchronization.

Done many Windows Service applications, include own implementations of web server engine with scripting support, IRC server, SMTP/POP3 server, XML Data storage server.

For past 2 years completed over 200 software and webware projects with programming and design (GUI design, web design, usability and accessibility issues).
