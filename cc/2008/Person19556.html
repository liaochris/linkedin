Pierre Andrews’s Summary

I graduated as an engineer in computer science and software engineering, I have already developed Java applications in professional environments but I also worked for different open source projects in other programming languages, in particular for web based applications.

After obtaining my engineering degree (Master level) in Switzerland I started a Ph.D. in Computer Science at the University of York (UK). I am currently doing research on Human Computer Communication and Dialogue.

Web communities and open source development is amongst my personal interests and I belief they will have a great importance in a short future. Therefore I try to find time to develop skills in that field by:
- contributing to the SPIP content management system open source project. In the context of this open source community, I collaborate by providing user support, writing/translating documentation (some of which was published by Eyrolles Publications) and developing plugins (in PHP, Javascript and XHTML).
- developing Greasemonkey (Javascript) extensions to popular websites like Flickr.com and Ebay.
