Lawrence Oluyede’s Summary

I've an advanced knowledge of the Python language, a good knowledge of Java and C#. I also wrote a chapter of the first Italian Ruby Book. I developed the first open source Atom library (atomnet) sold to a private company in 2005. I've solid bases of object orientation and good acquantaince with functional programming.

I reviewed the first "Ruby per applicazioni web", the first Ruby book written in Italian.
I am currently reviewing a book on Ruby On Rails for .NET developers.

I am interested in concurrency and large scale development approaches.

I am also author and editor of an an Italian technology aperiodic webzine, Stacktrace (http://stacktrace.it).

I contributed to various opensource projects. I also contributed as a student in the Summer Of Code 2006 to the PyPy project writing extension modules; I created a library to interface with Campfirenow.com (http://dev.oluyede.org/pinder/); I created the first .NET library to handle the Atom format (sold to ToolButton Inc. in 2005) now incorporated in Web 2.0 tools for .NET (http://www.web20tools.net/).
