Jesse Middleton’s Summary

* Created a global plan for business continuity through network connectivity using Cisco VPN (PIX/ASA/Concentrators), Cisco routers and EIGRP, BGP and static routing protocols.

* Ran a security audit with an outside team to verify external and internal risks to the business as well as created a plan to mitigate those risks.

* Migrated over 1500 endpoint devices to the new ePolicy Orchestrator 4 server and the 3.6 agent to protect all of the computer systems from viruses and spyware using McAfee's Anti-virus and in the future Anti-spyware software.

* Implemented a global standard and deployed Cisco's wireless LAN controllers and Wireless Control System for secure wireless access for both employees and outside contractors/visitors with one set of SSIDs, encryption technology and a monitoring/logging solution.

* Led the network group through a series of evaluations, tests and implementation of a global WAN acceleration product. Using Riverbed's Steelhead appliances and mobile clients, the company was able to increase the available bandwidth by 3.5 times to help alleviate the need for added MRCs and network equipment.

* Designed a plan and implemented a product and process to handle secure file transfers using Accellion's Secure File Transfer Appliance for use in Engineering and Marketing.

* Evaluated and deployed a process and system to filter all web traffic globally using Secure Computing's SmartFilter. This product saved on bandwidth and increased productivity by limiting the types of web sites that could be accessed by each individual user.
