Dan Meineck’s Summary

I'm a Senior Web Software Developer with substantial experience in Microsoft .NET technologies, having significantly contributed to building the UK's leading .NET CMS product, and previously supporting Visual Studio .NET working for Microsoft UK.

My most significant achievement to date is the completion of Immediacy SharePoint Connector v1.0 and 2.0, a product recently described by Forrester Research as the best in the market. I was responsible for the majority of the architecture design, UI design, and actual code implementation that went into the project.

I'm currently working as a Scrum Master and developer in a team working with SharePoint to develop the Immediacy SharePoint connector solution, focusing on WCF, ASP.NET Dynamic Data and .NET architecture.

I enjoy developing, consulting and selling software.
