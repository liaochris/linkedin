Lisa McMillan’s Summary

Academic website development is a very specific endeavor. Data migration with incomplete data, multiple and diverse audiences, and antiquated development practices are just some of the challenges an academic web developer faces. I intend to change the way academic institutions build their web presences, with more focus on usability, dynamic content, design and content development.

Specifically interested in web standards development and application development - joining standards based websites and data instead of producing static, ugly, slow, boring web pages is a particular passion.
