Andrew Mitchell’s Experience

  • Owner

    ProductiveFirefox.com (Sole Proprietorship)

    (Sole Proprietorship; Myself Only; Internet industry)

    July 2007 — Present (11 months)

    ProductiveFirefox is an umbrella brand for multiple productivity addons for Firefox. It includes GTDInbox (formerly GTDGmail) and MeeTimer.

    GTDInbox - started in August 2006 - is a Firefox extension that tightly incorporates GTD (Getting Things Done - a productivity principle) into Gmail.
    It garnered favourable mention on About.com, Lifehack.org, Lifehacker.com and Digg.com; amongst others.
    User feedback and participation is intense and very positive. It has been adopted inside Google itself.

    Other Info:
    Productive Firefox is the end point for a journey beginning with BumbleSearch in mid 2005; an extension that enabled users to mash up the web for more efficient browsing and information discovery. Despite its short life, through word of mouth it reached 30,000 users and received good feedback.
    An offshoot of a ProductiveFirefox extension that was never released (WebCards) ended up being contributed to the Microformats code in Firefox 3.
