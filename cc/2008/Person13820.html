Jorge Escobar’s Summary

Jorge Escobar has worked for several high profile companies as well as an entrepreneur for a number of projects.

He served as Product Manager for Yahoo! North Latin America, coordinating the effort of Content, Marketing, Sales and Engineering for Yahoo! En Espanol and Yahoo! Mexico.

He also served as Chief Web Technology Officer for Hispanic Broadcasting Corporation (now Univision Radio) developing and managing their Hispanic local and radio station portal, Netmio.com.

Previous to the Internet revolution, Jorge worked as an Image and 3D Graphic designer for Telesistema TV Network in Ecuador and as a developer for interactive multimedia applications at the Caracas Museum of Natural History.

Jorge Escobar’s Specialties:

Web development, digital video encoding, storage and distribution, LAMP Stack, Linux / Apache / MySQL / PHP / Perl specialist, CSS, user experience architect, user generated content (UGC) specialist, server deployment and monitoring, server architecture, product management, project management
