Nicolò Volpato’s Summary

In web design field since 1999 as passionate hobbyist during high-school.
Then as a professional since 2004, after attending University and specilised training.

Specialised in website design using most modern techniques and standards, creating hand-coded, valid and semantic XHTML and CSS layouts.

Focused on design of visually engaging websites, thanks to advanced Photoshop skills to produce beatiful graphics and Flash skills to produce animated and interactive websites. Good knowledge of ActionScript.

Expertise in working with WordPress as blog platform and CMS, with in-depth knowledge of its templating structure. Providing custom design of WordPress themes.

Currently learning more on client-side programming using JavaScript and AJAX and server-side programming using PHP and MySQL.

Basic knowledge of server-side programing using ASP, ASP.NET (vb) and Ms Access, thanks to previous work experiences using these technologies.

Nicolò Volpato’s Specialties:

Advanced knowledge of Photoshop, XHTML, CSS, Flash and ActionScript, WordPress.
Basic to intermediate knowledge of JavaScript , AJAX, PHP, MySQL, server administration (both Windows and Linux).
Understanding of ASP, ASP.NET.
