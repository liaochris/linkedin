Federico Fissore's Projects

  • JRecordBind

    • January 2008 to Present
    Team Members: Federico Fissore

    JRecordBind is a a tiny and super fast library that maps a fixed or variable length text file to bean instances, ready to be chewed by an import procedure (Unmarshaller) and export record beans into a fixed or variable length text file (Marshaller)

  • Agent Smith

    Team Members: Federico Fissore

    Agent Smith is a java agent that scans your classes for modification and instrument them as you change the bytecode, while the application is running, making java a little more similar to a scripting language. Think of it as "hot swap without Eclipse"

  • Parallel Bitset Operations (pet project)

    Team Members: Federico Fissore

    Boosts Lucene searchers that heavily use bitsets

  • Pipes (pet project)

    Team Members: Federico Fissore

    Tiny and easy implementation of the producers-consumer pattern.

  • Presentz.js

    Team Members: Federico Fissore

    A javascript library to show and sync video+slides presentations. Currently powering Presentz.org http://presentz.org/

  • Node OrientDB

    • March 2012 to Present
    Team Members: Federico Fissore, Gabriel Petrovay

    Node.js driver for OrientDB, using the binary protocol
