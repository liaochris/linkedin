Technical Artist

Aquiris Game Studio

Partnership; 11-50 employees; Entertainment industry

January 2010 – August 2013 (3 years 8 months)

For more than three and a half years inside the studio, I had the opportunity to do different tasks such as modeling, texturing, animation and programming. By doing that I was able to understand the main steps in the game development process for standalone devices like PC/Mac and mobile, iOS/Android.

I used Unity 3D as the game engine, being able to create and set up effects, particles, lightmaps, shaders and scripts. For modeling and animation I used 3DSMax, and in some cases ZBrush for organic shapes. To optimize process I have developed editor tools for Unity using C# and for 3DSMax using MaxScript. I have also created customized shaders for Unity using SurfaceShader and CG (mostly for mobile).
