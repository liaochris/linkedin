  • Chasing zero

    • 2010 to 2012
    Team Members: Dennis Fleurbaaij

    Privately bought nearly every network card in existence (Intel, Broadcom, Mellanox, Myricom, Solarflare, Emulex...) and measured performance with all possible interfaces (posix, epoll, iovec, aio, spinning, openonload, winsock, myricom, Linux kernel module, c#) on various platforms (Intel, AMD, Sun) running on tuned operating systems (linux 2.6, windows 7) and platforms. Prototyped auto-generated, exceptionally low latency software systems.

    Findings: OS is irrelevant, decent code structure prevents even lower latency: so don't do that, buying faster line order of magnitude faster than system optimizations. Predicted that lower bound of software system response (early 2012) is about 6.2us over controllable stack.
