    • Network World

    My share in this project is the main idea, methodology development, project management (process control, consulting, executing).

  • Using NoSQL Databases for Interactive Applications

    • December 18, 2012
    Authors: Kirill Grigorchuk, Diomin Alexey

    Designed to replace the traditional RDBMS, NoSQL databases provide near-endless scalability and great performance for data-intensive use cases. However, with so many different options around, choosing the right NoSQL database for your interactive Web application can be tricky. This whitepaper provides an overview of three popular NoSQL solutions: Cassandra, MongoDB, and Couchbase, as well as a vendor-independent performance comparison of these products that can be used as a guide when choosing a NoSQL database.

  • Building Stock Trading Strategies: 20% Faster with Hadoop

    • Hortonworks Blog
    • August 16, 2013
    Authors: Kirill Grigorchuk, Sofya Parfenovich, Alena Vasilenko

    Based on complex mathematical algorithms, automated stock trading solutions take into account hundreds of factors and suggest the right time for placing buy/sell orders. However, if an algorithm omits essential market parameters, this may bring significant loss. This article provides an insight on how Hadoop and improved data clustering speeded up stock trading system’s performance by 20% and increased a customer’s revenues by 12%.

  • Hadoop + GPU: Boost performance of your big data project by 50x-200x?

    • Network World
    • June 24, 2013
    Authors: Kirill Grigorchuk, Alena Vasilenko, Vladimir Starostenkov

    Hadoop has changed the way we deal with big data. Parallel processing with this set of tools can improve performance several times over. The question is, can we make it work even faster? This article analyses cases when offloading calculations from a CPU to a graphics processing unit (GPU) can help to speed up system’s performance by 50–200 times.

  • Hadoop on Windows Azure: Hive vs. JavaScript for Processing Big Data

    • Network World
    • December 6, 2012
    Authors: Kirill Grigorchuk, Alena Vasilenko, Sergey Klimov, Andrei Paleyes

    For some time Microsoft didn’t offer a solution for processing big data in cloud environments. With the launch of the Hadoop on Windows Azure service, .NET developers got a possibility to distribute the load and speed up big data computations. The article overviews two out-of-the-box ways of processing big data with Hadoop on Windows Azure and compares their performance.

  • Cloud platform comparison: CloudStack, Eucalyptus, vCloud Director and OpenStack

    • Network World
    • July 16, 2012
    Authors: Kirill Grigorchuk, Alena Vasilenko, Vadim Truksha

    The variety of cloud tools makes it difficult to choose an appropriate cloud platform even for professionals. Even though the decision is made, some hidden issues may arise and still require custom bug fixing. This investigation covers functionality implemented in free and commercial versions, price, documentation, and community matureness, as well as assesses the possibility of deploying a platform without having proper technical background. It also includes information on possible errors and provides detailed instructions on bug fixing.
