Jon Loyens' Summary

Currently Principal Architect for User Interfaces at HomeAway. Responsible for client side software architecture for all of HomeAway's owner and traveler websites. I ensure that HomeAway's consumers have an outstanding user experience while the engineers at HomeAway have the tools and technologies necessary to efficiently build scalable, maintainable and modern user interfaces for both web and mobile.

Additionally, I advise a limited number of technology startups. I'm currently officially working with Plato Labs, Swapt.com and Res-Q-Threads in this capacity.

Was Chief Technology Officer at Marketvine. Marketvine was a Software as a Service company focused on e-commerce merchandising. As CTO, I was responsible for the product management, engineering and technical implementation of our solutions for our customers.

Previously held multiple roles at the public SaaS company, Bazaarvoice, including Vice President of Engineering for Products and Director of Bazaarvoice Labs. Was responsible for the engineering and delivery of Bazaarvoice's Software as a Service User Generated Content products. Additionally, I founded the Labs group at Bazaarvoice, which was charged with creating innovative new pilot products and beta technologies in the user generated content and social media space.

Before Bazaarvoice, built a world class User Interface technology group and grew revenue in the implementation space as CTO of Thinktiv, a product management and design consultancy in Austin, TX.

Prior to Thinktiv, I worked at Trilogy as a Program Manager in charge of millions of dollars of auto dealership inventory, a UI designer on a major auto OEM enterprise system, Development Manager, UI Designer and Developer on various other projects.

Specialties: HTML, CSS, JavaScript and Java Script Libraries, Python, Django, Ruby, Rails, Java, JSP, various User Interface Technologies, Interaction Design, Software Architecture, Software Requirements and Specification
