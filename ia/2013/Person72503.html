Sethuraman Vinaitheerthan's Projects

  • Staging Environment and Extraction of Data (SEED)

    • January 2010 to July 2012
    Team Members: Sethuraman Vinaitheerthan, Shishir Prasad, Archana Jegatheesan

    Designed and implemented SEED (Staging Environment and Extraction of Data), an ETL application, that fetches the transactional data from multiple sources, enriches it and then persists to the destination database to serve as the central data repository for the entire firm.

    The main challenges were to add bitemporal enrichment to the data, supporting both batch and real time ETL, dynamically pivoting vertical data to flattened representation in-memory using Java reflection, caching to optimise response times and database optimization to handle large reads/writes at the destination.

    Technologies: Java, T-SQL, iBatis, Spring, SQL Server, Sybase, JMS, Drools.
