SELECTED PROJECTS
● Created supervised machine learning algorithms for pattern recognition of physical objects in real-world radar data and implemented these in both MatLab and C++. Cleansed data by applying a power transform and segmenting the signal from ambient noise.

● Using statistical modeling and signal processing techniques to detect low-observable objects in heavy noise, dynamic environments. To cleanse the satellite infrared imagery an optimal transform using information theory to minimize registration errors was used. ● Developed machine learning models for the approximate solution of POMDPs (Partially Observable Markov Decision Processes). Applied Stochastic Markov chain Monte Carlo methods to solve the problem using an interruptible algorithm so user could trade off accuracy and runtime in arriving at optimal decisions based on error-laden data.

● Used probabilistic graphical models to fuse weak information from massive sensor data sets to infer identity and intent. Used k-means to form cluster objects and then developed Bayesian networks to model the aggregate.
