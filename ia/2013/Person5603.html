Armond Mehrabian's Summary

Armond Mehrabian provides strategic guidance on the implementation of the Scaled Agile Framework (SAFe) through training and coaching of leaders and their teams.

Armond is an Enterprise Agile coach and trainer (PMP, CSP, SPC). He helps software intensive companies adopt and improve their use of agile processes and techniques by bringing his energy and passion for continuous improvement and innovation to every client engagement. Armond has been a change agent to product development and IT organizations by applying lean principles and agile methods at the enterprise level.

Clients include Oracle, Boeing, Honeywell, SEI Investments, Invensys, Cigna, Encore Capital and Mitchell International.

Specialties:
• PMP, CSM, CSP, SPC
• Conference speaker: PMI Annual Conference 2010, 2013, InformationWeek, The Product Management View (PMV) Webinar series, AgilistaPM
• Agile product development instructor and coach
• Scaled Agile Framework (SAFe) with Scrum, XP and Kanban
• Coaching and mentoring distributed Agile programs

Armond Mehrabian's Patents

  • SYSTEM, METHOD, AND COMPUTER PROGRAM PRODUCT FOR SCANNING

    • United States Patent 12,163,822
    • Issued June 27, 2008
    Inventors: Armond Mehrabian,

    The ability to mount a scanned image datastore as a virtual file system so that scanned images can be directly accessed and attached to documents from within applications. For example, a web based email application will be able to attach a document that is about to be scanned. Prior to this invention many steps had to be taken as stated in the excerpt of the filed patent below:

    1) User must open a scanning application
    2) Set one or more parameters of the application
    3) Select an output location
    4) Scan the document
    5) Switch to the email authoring application and select the scanned file from the file system.
