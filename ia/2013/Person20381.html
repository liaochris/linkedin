Caroline Amaba's Publications

  • User Interface & User Experience Synergy Through Design & Future Interfaces

    • Stevens Institute of Technology
    • May 2012
    Authors: Caroline Amaba

    Senior Thesis,l Art & Technology, B.A.

    ABSTRACT
    An interface is a medium, a point of interaction between a user and an objective. Designers have long known that their interface designs shape the aesthetic and functional aspects of user design. Because the digital medium is non-physical, designers are becoming more aware of the ways that user interactions can create holistic experiences for the user and influence new iterations of the design process. Therefore, in order to design user interfaces (UI) that are successful, designers should
    take into account aspects of user experience (UX). The way users interact with an interface influences its design; additionally, the UI design - both in aesthetic and functional aspects - impacts user activity and experience. This study will explore who exactly are the “users,” what an “experience” is, design principles of tangible interfaces applied to a digital space, aesthetic and functional synergy of interfaces that use the aforementioned principles, other influences on UX (such as the interface hardware being used), and examples that illustrate applications of these concepts. Furthermore, this
    study will synthesize existing literature on UX and UI, and define with new clarity the synergistic relationships that exist between UI and UX. The synergy that exists is that the design of a UI affects the UX, and that the feedback from the UX will influence the future design and functionality of an interface.

  • re:touch, a small exploration in user interface design

    • self-published, lulu.com
    Authors: Caroline Amaba

    re:touch is a small exploration in user interface design that asks: “if [insert name of thing here] had a touch screen, what would the touch screen look like? What would it need to have?” re:touch focuses on five different aspects of interface design: wireframing, designing more than once, faking tactile feedback, designing for gestures, and creating functional design and aesthetic synergy. Each topic has an example of an object that has been given a touch screen. Discover the basics of user interface design as you explore a small portion of it in re:touch.
