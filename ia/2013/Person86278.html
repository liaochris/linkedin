Mark Aherne's Projects

  • UNION Spaceship Simulator

    • June 2011 to Present
    Team Members: Mark Aherne

    I am designing and developing a game, currently titled "UNION Spaceship Simulator". It is a multiplayer co-operative spaceship simulator where one to five players work together to pilot a spaceship (each taking one or more stations on the ship).

    It is being built using C# (Mono Framework) and the Unity 3D game engine.

    The game is making excellent progress, the game's foundation is complete (networking framework, multiplayer physics, crew management etc.) and development of the actual gameplay in under way.

    Work in progress images can be found here: http://union.imgur.com/

  • Space Station 13 - UI Overhaul

    • March 2012 to Present
    Team Members: Mark Aherne

    Space Station 13 (SS13) is an open source online sandbox space station simulator.

    The graphics are primitive, but behind that is a fully simulated space station with atmospherics, power networks, machinery and computers.

    Within this sandbox players take on a role, such as Engineer, Atmospheric Technician, Security or Doctor. They work together to protect the station, and themselves, from random events including meteor showers, alien invasions and traitors.

    SS13 has long suffered from having a very poor UI. I have built a new UI framework within the game which brings many major new features:

    * Improvement and standardization of UI styling.
    * Smooth updating with lag compensation (SS13's old UIs had issues updating, and if the server was lagging clicks would be lost).
    * Separation of game logic from the "view". Previously all display code was embedded within the game logic, making updating and managing UIs very difficult. The new UI framework moves display code into templates, with a markup syntax for formatting data.

    Using this new UI framework I have been rebuilding the game's UIs.
