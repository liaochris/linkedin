David Sandoz's Projects

  • Disclosing Sinks and Architectures Behind Android Apps Private Data Collection

    • September 2011 to January 2012
    Team Members: David Sandoz, Alexis Kessel

    Bachelor project in the Laboratory for Communications and Applications of Professor Jean-Yves Le Boudec under the supervision of Nikodin Ristanovic.

    Keywords:
    Location Privacy; Mobile Data Privacy; Android; Location Data Mining; App Analytics; Mobile Advertisers; Content Delivery Networks; Wireshark; Packets captures, IMEI, Android ID, Geolocation.

    Abstract:
    Several recent studies have shown that sensitive private information can be found in the traffic exchanged between mobile applications and various destinations on the Internet. However, these studies concentrate on the behavior of individual applications and the type of private data leaked from the phones. Unlike them, we focus on the third parties that collect private information and the sinks where such information is collected. We are particularly interested in the destinations that are common for a number of different mobile applications, as they can collect sensitive user information (such as location or phone identifiers), whenever one of the application that communicates with them is used.
    In order to intercept leaked private information and identify the parties that collect it, we develop a desktop applications that can filter and record the data packets sent by mobiles via their Wi-Fi interfaces. We find this approach to be more practical than developing a smartphone app with the same purpose, as such an app always requires root access. Using this application we analyze 60 mobile applications and we show that several advertisers, app analytic companies and Content Distribution Networks (CDNs) are found behind a large fraction of these applications, meaning that they can collect sensitive user information irrespective of the application used.
