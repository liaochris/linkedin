Mitchell Lutzke's Projects

  • Bucky Platformer

    • October 2012 to Present
    Team Members: Mitchell Lutzke

    Bucky Platformer is a very generically named platformer that I've been working on for a while now. It started with a recent interest in the new HTML 5 / JavaScript functionality recently implemented in most recent browser releases.

    I quickly took a large interest in this new-found technlology and started to work on my own game. The first attempt/version of the game I worked on turned into a large mess, as I was fairly new to trying to structure JavaScript in an OOP manner (It's OOP concepts aren't nearly as nice as other languages), so I've been rewriting the game in a way that it could turn into an engine that would work for other games.

    It is still periodically under development; It's github repository can be viewed at https://github.com/melutzke/BuckyPlatformer

    The game can be played here: http://heymitch.com/projects/BuckyPlatformer/bucky_rewrite.html

  • PaceMaker

    • January 2013 to Present
    Team Members: Mitchell Lutzke, Nick Heindl, Alex Sohail, Jason Altekruse, Jesse Krizenesky

    With our tiny 5 man team, we conceptualized, implemented and brought a game to life within a weekend. We used HTML5, Canvas and Javascript with the addition of a sound and gamepad library. PaceMaker features both keyboard and xbox controller integration and a dynamic audio system. We also adapted a previous code base to implement a map creation system.
    Original code can be found here: https://github.com/melutzke/Pacemaker/tree/GameJam2013
    All updates after the Jam can be found here: https://github.com/melutzke/Pacemaker

  • Robotics

    • August 2010 to Present
    Team Members: Mitchell Lutzke

    I've always had an interest in robotics. While in high school, there was no one I could turn to that could help me learn about this field, so I took it upon myself to learn what I could.

    I started out with a basic circuits kit from Radioshack and worked my way up. Soon, I had ordered my first PIC Microcontroller and created my own wirelessly-driven robot.

    This lead me to wanting to make bigger and better robots, so I started with an R/C Car chassis, gutted all of the electronics, and built the control system from the ground up my self with the help of my trusty PIC.

    My current long-term goal is to build a robot with a wireless video feed capable of being driven up to a half-mile away.
