Malcolm Brook's Summary

5 years experience with eCommerce web sites (mainly Actinic), SEO, and writing PHP scripts, also web hosting and plug ins for Actinic using javascript, PHP and MySQL.

30 years experience in design of control systems for custom machinery using PLC's and PC's. Largest project was the complete PLC control system (over 40 networked PLC's) for the Pirelli Cable Factory in Aberdare, Wales.

20 years working with marking equipment and designing traceability systems based on pin stamp and laser marking systems using various database systems with PC and web based designs, incorporating vision systems for reading 1D and 2D codes.

Latest designs involved a multi axis drive, label material feed mechanism for Laser Marking Machine using Beckhoff drive Technology. Machine includes, material feed, material cutter and laser head movement allowing a single marking unit to produce laser marked flexible labels as well as marking stainless steel plates on the same machine by means of moving the laser marking head automatically as part of the machine cycle.

Developing m2m applications for second tier meter reading via the internet, using IEC 62056 and modbus communications via GPRS and Broadband systems.

9 years running a multiserver windows 2000 domain with exchange server and 25 workstations.

Specialties: Programming in C, C++, Visual Basic, Sculptor 4GL, PHP, HTML, CSS, JavaScript, Perl, also Forth, Assembler languages, Fortan and Algol.

Programming of Siemens, Group Schneider, AB, GEM80, Square D, Modicon and Mitsubishi PLC's.

Programming of Beckhoff I/O systems linking a variety of Beckhoff couplers, I/O and drive terminals to PC applications.

Designing and building ecommerce web sites using Actinic ecommerce software, including search engine optimisation and plug ins for Actinic.
