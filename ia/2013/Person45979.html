Frederick Bobrow's Patents

  • SYSTEM AND METHOD FOR PRESENTING COPY PROTECTED CONTENT TO A USER

    • United States Patent WO/2005/119962
    • Issued December 15, 2005
    Inventors: Frederick Bobrow

    A device for presenting content to a user is utilized in combination with a general purpose computer. The computer [figure 1, 110] has a processor [figure 1, 111], a computer communications interface, a computer memory [figure 1, 112] and an operating system [figure 1, 118]. The operating system has one or more file management tools. The device [figure 1, 100] has a housing, a device communications interface and a device memory. The device memory [figure 2, 102]comprises a boot partition [figure 2, 201]which includes boot software [figure 2, 202]. The boot software is copied from the device into the computer memory and is executed from the computer memory by the processor. The device memory also comprises a secure partition [figure 2, 210] inaccessible by the file management tools and having content stored thereon. The device memory also comprises content delivery software, which is copied to the computer memory, and when executed by the processor from the computer memory can access the content from the secure partition and present it to the user as sensory data.
