My specialties lie in ads & analytics, but I also tend to focus most of my work on Flash, backend technologies (loving Node.js at the moment) and javascript for front-end (working with angular.js at the moment).   • TVNZ

    Team Members: Brandon Aaskov, Brian Franklin

    TVNZ is essentially Comcast of New Zealand. They have a huge library of content that needed to be migrated over to Brightcove. I worked with them from presales until project launch setting up the structure, guiding them on best practices as well as developing the ad integration as well as the custom player.

  • Open Source Omniture Integration

    • March 2011 to Present
    Team Members: Brandon Aaskov

    Frequently, Brightcove customers need a straightforward Omniture integration and at the time this was published, Adobe's plugin for Brightcove left a few things to be desired. This was created to offer a simple way for customers to get up and running and leverage Omniture in a powerful way. By open sourcing the project, we also give developers a fantastic starting point saving days if not weeks of development.

  • Swayze

    • June 2012 to June 2012
    Team Members: Brandon Aaskov

    Swayze was an Brightcove internal project that we created as part of a 24 hour hackathon using the new Analytics APIs. It was a Node.js and Backbone.js project that not only created an API wrapper to make a more robust API for end users, but also leveraged Socket.io to get true, real-time analytics displayed directly in a dashboard interface.

  • Open Source Google Analytics Plugin

    Team Members: Brandon Aaskov

    Google Analytics is free, so the integrations should be too. Most people don't realize that Google Analytics supports event tracking in Flash, which allows this plugin to work anywhere it's published, including javascript-blocking sites like Facebook. This Brightcove plugin gives customers a turn-key solution to integrate with Google Analytics.

  • State Department iPad App

    Team Members: Brandon Aaskov

    Using Brightcove App Cloud, we created a custom State Department iPad app integrating their feeds as well as a custom Google Maps tie-in.

  • VEVO

    Team Members: Brandon Aaskov

    VEVO used Brightcove to syndicate their content (YouTube is their primary vendor), but needed a custom player that integrated with Last.fm, Tradescape (analytics) and Freewheel as well as a custom look and feel for their Flash player.

  • Access Hollywood

    Team Members: Brandon Aaskov

    Access Hollywood switched from their internal solution to using Brightcove. We guided them on a migration plan, as well as developing a flexible video portal that offers multiple caching options, Facebook integration, Google Analytics for page tracking and the ability to flip a switch for site-wide AJAX features.

  • Open Source comScore Brightcove Plugin

    Team Members: Brandon Aaskov

    comScore is a data aggregation company (similar to Nielsen) that requires a specific, albeit relatively simple integration. The process was drastically simplified for Brightcove customers by introducing this plugin, allowing them to modify only two lines in an XML file to be completely integrated with comScore's analytics.

  • Open Source Webtrends Brightcove Plugin

    Team Members: Brandon Aaskov

    Webtrends is another popular analytics tool, and this open source project allows customers to use the plugin out-of-the-box to capture information from their Brightcove players, or as a starting off point to customize the code to their needs.

  • Open Source Twitter Feed Plugin

    Team Members: Brandon Aaskov

    Created originally for Joey Kramer (drummer or Aerosmith), this Twitter Feed plugin ads a visual element to a Brightcove player. By specifying a hash tag or user's handle, the plugin will scroll through the several tweets that come back from the Twitter API request. The end user can scroll through manually, or let it auto-animate.
