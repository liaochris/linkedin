Greg Wang's Projects

  • doGRT for iOS

    • January 2012 to Present
    Team Members: Greg Wang

    iOS App that help user to look up Grand River Transit's bus schedule. It is designed with SIMPLE user interaction, yet STRONG functionality. The schedule data is provided by GRT.

    Technologies Applied:
    - SQLite Database, offline schedule data
    - Core Data Storage, store user favorites
    - MapKit, display bus stops and route paths
    - Http Request, auto schedule data update

    This project is open sourced on Github:
    https://github.com/doLast/doGRT-iOS

  • Moe FM for iOS

    • April 2012 to Present
    Team Members: Greg Wang

    ACG Music Radio for iOS. Music service is based on moe.fm's API.

    Technologies Applied:
    - Self defined Objective-C wrapper for RESTful API
    - Audio Streaming and Background Playback
    - A-synchronized image downloading and caching
    - OAuth authentication for interactive online service

    This project is open sourced on Github:
    https://github.com/doLast/Moe-FM

  • Realtime Embedded Kernel with Train Controller

    • January 2013 to April 2013
    Team Members: Greg Wang, Tony Zhao

    - Performs fast Context-switch in ARM assembly instructions
    - Handles Software & Hardware Interrupts for system calls, timer and I/O events
    - Delivers inter-tasks messages with self-defined RPC protocol
    - Interacts with the Train Set through UART and serial port
    - Directs trains base on the sensor data and prevents collisions

  • Joos (Java) Compiler

    • January 2013 to April 2013
    Team Members: Greg Wang, Man Wenzhu

    The compiler was implemented in Java. It follows the modern compiler structure, which includes DFA Scanner, LR1 Parser, AST Constructor, Symbol Table Generator, Type Checker, Static Analyzer and Code Generator.

    Techniques Applied:
    - Visitor Pattern, integrated in the Abstract Syntax Tree (AST) data structure, is widely used for tree traversing.
    - A HashMap based Symbol Table offers two-way direct access between an AST Node and a Symbol (i.e., method names).
    - JUnit test, automatically tests the compiler against hundreds testcases.

  • CampusNav

    • April 2012 to August 2012
    Team Members: Greg Wang, Jayson Zhu, Junya (Collin) Zhang

    An iOS App that help students to find their way in the Campus and Building. It was built as the Software Architecture course project.

    Technologies Applied:
    - iOS Framework and Core Data Storage
    - Graph Theory, modeling the campus and building environment
    - Dijkstra's Algorithm, finding route to a certain destination.

    This project is open sourced on Github:
    https://github.com/doLast/CampusNav
