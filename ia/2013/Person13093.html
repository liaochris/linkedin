Ben Fox's Projects

  • "Our Thinking" Flex Application

    • January 2010 to February 2010
    Team Members: Ben Fox, Ron Yeager, Liz Molitor

    A Flex application to showcase core competencies of Allyis

    Details:
    An integral part of the redesigned Allyis.com site, the 'Our Thinking' section was intended to highlight Allyis' ideas, applications, and insights on the company's technologies and business processes. To support this goal the design included an interactive application featured on the 'Our Thinking' homepage. The application needed to showcase individual topics with introductory information and links to additional resources as well as support video playback within the application.

    Responsibilities:
    As the sole developer on this project, Ben worked within the Flex framework to create both the User Interface from the designer's mock ups and the application functionality based on the the functional specifications. During the initial stages of development Ben worked closely with the designer to flesh out nuances to the interface and user interactions. To create the elements of the UI he borrowed graphics from the mock ups in addition to creating other elements from scratch, such as vector graphics, in Adobe's Flex and Flash programs.

    Working on the application programming, Ben succeeded in making the code organized while also including enough documentation and comments to allow individuals to easily pick up and modify any part of the application. It was especially important that non-technical personnel be able to make changes to the content without their needing a great deal of application specific knowledge.

    Ben architected the first version of the application so that content changes were accomplished via a simple XML layout containing the information for each topic. In version two of the application Ben leveraged the SharePoint web services to pull the data for each topic from an externally managed SharePoint list, allowing the content to be edited through a web interface which was faster, simpler, and easier to manage.

    Technologies:
    Flex, ActionScript, Flash

  • Microsoft ISV Connector

    • April 2009 to April 2009
    Team Members: Ben Fox, Sean Phelan

    A web portal to gather in one place relevant information for Independent Software Vendors.

    Details:
    The Microsoft Independant Software Vendor (ISV) Connector was developed as a client-facing site aimed at bringing together in one place the huge amount of information on the web that is of interest to Microsoft ISVs. The client identified four main categories to be used as the information portals, and had one homepage to direct users to the appropriate location. Each of the four individual portals contains news, articles, blogs, upcoming events, and key resources for ISVs.

    Responsibilities:
    With only graphic mockups of the site, Ben was given full control to break up the pages as he saw fit. In order to make the site as flexible and expandable as possible he used advanced HTML and CSS based layouts. This allowed not only for easy editing and updating of the content, but also did not limit the amount of content any column could hold. As it was a client facing site, this layout also gave 100% HTML and CSS validation by W3C standards and maximized cross-browser compatibility and search-engine crawlability.
    Ben also implemented a custom CSS-based popup with a JavaScript helper to add teaser-text to select links. The helper was used to manage popup z-indexes which allowed a popup anywhere on the page without worrying about the popup running off the page or falling underneath other text.

    Technologies:
    HTML, CSS, JavaScript, Adobe Photoshop

  • El Centro De La Raza

    • February 2009 to October 2009
    Team Members: Ben Fox, John Martin

    Web Site Design and Development From Scratch through The Taproot Foundation

    Project Details:
    The Taproot Foundation awards service grants to non-profit organizations in a number of areas. El Centro De La Raza, a Seattle non-profit, was awarded a grant for a 15 page basic website. The project team of Taproot volunteers consisted of six individuals, each of whom brought with them a specific area of expertise. Ben filled the role of Web Developer.

    The team worked closely with several individuals at El Centro De La Raza, creating specifications and conducting interviews to determine how best to update both the design and architecture of the website.

    Personal Responsibilities:
    Ben's main responsibilities on this project included advising on technical issues, determining technologies to use, and designing and writing the code architecture.

    A "basic website" grant from Taproot does not allow for any server-side code, and one of the biggest challenges Ben encountered was finding ways to allow for easy content management of frequently updated information including news articles, blog posts, image banners, and quotations.

    To create news and blog syndication, Ben took RSS feeds from both Delicious and Wordpress, two platforms El Centro was already familiar with, and parsed the XML to HTML via an intermediary site. This allowed the content to be styled and integrated with the site, while allowing updates to be made without having to re-push pages.

    Ben also created several custom JavaScript systems for the El Centro employees to update the banners and quotes by replacing or adding simple strings without needing advanced knowledge of the underlying code used to serve the content.

    Technologies Used:
    HTML, CSS, JavaScript
