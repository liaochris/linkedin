  • Virtual Walking the Pens v2.0

    • February 2012 to March 2013
    Team Members: Daniel Kane

    (URL is for 1.0; 2.0 is not yet publicly available)

    The interactive 3D simulation allows users to do a virtual barn-walkthrough of a wean-to-finish swine farm, identifying and treating illness in the barn.

    My responsibilities included 2d UI elements using the XAMLite library, non-pig 3d movement such as fans or doors, manually blended textures at runtime with DXT compression, testing & QA, client feature demos, publishing stable weekly builds for clients, and coordinating with artists for required 2d and 3d assets.

    SunBurn Game Engine (XNA-based) with XAMLite (XNA) 2d UI elements.

  • State Compensation Insurance Fund HazOps

    • February 2011 to February 2012
    Team Members: Daniel Kane

    The interactive 3D safety training game allows players to learn about the hazards they may face in the work place, and experience the potential consequences of mistakes, in a fun and safe virtual environment.

    My responsibilities included systems design, publishing client builds, providing technical input in client meetings, mouse picking, keyboard input, Windows Forms UI elements, and working with the artist to create and maintain scene markers for cutscene blocking & object spawning.

    Custom XNA-based engine with Windows Forms 2d UI elements.

  • McCurdy's Armor™, Military Training Simulator

    • August 2010 to January 2011
    Team Members: Daniel Kane

    The interactive 3D assembly training simulator allows soldiers to prepare for the assembly of armor, a mission critical combat component, in a safe and risk-free virtual environment.

    Responsibilities included cameras, ground shaders, font handling, 3d mouse picking, and support of very low-end graphics cards, as well as positioning, sequence, and rigid mesh animation for guard post components.

    Custom XNA-based engine with Windows Forms 2d UI elements.
