  • TeamScor.es - Team Scoring Made Stupid Simple

    • September 2013 to Present
    Team Members: Jeremy Fox, Dillo Raju

    TeamScor.es is a Ruby 2.0 / Rails 4.0 application that was built the a single goal in mind, make it easier for remote development teams to score stories together. TeamScor.es also features a sprint monitoring tool which show things like...

    * A dashboard which shows the sprint #, total points in the sprint, completed points, the number os stories in all possible states (unstarted, started, finished, delivered, accepted and rejected), the number of each type of story (Feature, Bug and Chore).
    * An activity log which shows all activity in the current project.
    * All stories in the current sprint with an icon to represent the type of story, the story estimation, how many hours a story has been Started or how many hours it took for the story to go from Started to Accepted, the labels assigned to the story, the story title, who owns the story and the current state of the story.
    * This page updates automatically every 10 minutes.
