  • North Texas Allergy and Asthma Associates Website

    • December 2011 to February 2012
    Team Members: Calvin M. Cole, Robert Dudley, Sandeep Gupta

    This website was partially redesigned but completely redeveloped from an old school table based layout to a pure CSS page layout with other modern technologies along with the introduction of new information architecture to keep everything nice and organized for both the site users and site owners. HTML 5 for markup, jQuery for interactivities and Php/MySQL for dynamic data retrieval. No content management system was employed, so all file structures had to be well documented along with a lengthy README file for explanations, details and usage samples of templates, CSS classes, JavaScripts and Php functions. Additional old school technologies like Flash were also replaced by modern counterparts. Basic use of the LinkedIn API was called to action for certain staff members of NTAAA, for a friendlier, social feel in conjunction with links to the company Facebook page. The new website provides a far more enhanced user experience.

  • Gideon Math and Reading Website

    • March 2012 to September 2012
    Team Members: Calvin M. Cole, Stephanie Coppedge, Boston Coppedge

    The Gideon Math and Reading website was in need of modernization. The site was reliant on antiquated technology for major parts of the architecture and design. The solution provided was based on current web standards HTML5 and CSS3. Also utilized was, JavaScript and the library jQuery. For the first time the site owners would no longer be reliant on any developer to make contextual changes and content updates as the upgraded website included the blogging platform WordPress as a means to manage content as well as import their preexisting weblog history.
