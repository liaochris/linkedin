Erik Herlaar's Projects

  • Natural Control

    • February 2012 to April 2012
    Team Members: Erik Herlaar, Bouwe Krol, Gonnie Slootmaker, Hans Moonen

    At CGI Groningen, The Netherlands, we realised controlling an application using gesture recognition for a client. The application presented some analysis results in a 3D environment on cubes.

    Features:
    - Four commands are recognised: Open, Back, Turn left and Turn right.
    - Visible and audible feedback
    - Automatic vertical elevation adjustment
    - Ignoring bystanders
    - Passing control to next user
