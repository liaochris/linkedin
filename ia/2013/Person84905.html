Marc Eley's Summary

I am a UI solution specialist with a passion for great design. My main focus is frontend development and for 5 years I've furthermore been managing a team of 6 frontend developers.

I have implemented the UI part of 30+ solutions on the following platforms:
* SiteCore (mainly public facing websites)
* Custom .NET solutions (mainly public facing websites and mobile solutions)
* Sharepoint 2013, 2010 and 2007 (both public facing and intra-/extranet solutions including E-commerce)
* WordPress (small private projects)

I focus on implementing modern frontend technologies and solutions on the platforms mentioned above. In particular using:
* JavaScript: jQuery, Modernizr, Backbone.js, Requre.js, Grunt.js, Yeoman
* HTML5
* CSS 3, SASS/Compass, SASS Sourcemaps, Bootstrap
* Templating with Mustache, Handlebars, HAML
* Adaptive/responsive Web design
* Implementation of WCAG and ARIA guidelines

Backend:
I have a solid knowledge about and practical experience using C# and the .NET framework as a whole from working in a Microsoft environment for 7 years. My frontend development work has always been tightly coupled with backend development/developers.
I've used different Source Control systems over time like Subversion and TFS.
I know my way around the LAMP stack with old-school PHP and am currently looking into Ruby by way of Sinatra/HAML.

Teammanagement:
Have been acting as team manager for 2 to 6 frontend developers for 5 years.

Writing:
Experience writing the UI part of public and private sector contracts (tenders).

Project development:
* For wireframing, mockups or proof-of-concepts I've been using Axure, Balsamiq or creating mockups directly in the the browser.
* As the main responsible UI developer I've been planning and estimating UI development for enterprise size projects for many years.

Graphical design:
12 years working experience with Adobes Creative Suite applications (mainly Photoshop, Illustrator and InDesign) for both web and print.
