Lance Vick's Summary

Software engineer, open source software advocate, and entrepreneur. Between building startups, and providing custom technology solutions for many industry leading companies (including 3M, Amway Arena, Mayo Clinic, Citigroup, and Atlantis Resorts) I have built experience in a variety of areas. In a given project I am often responsible for any of a range of things such as software engineering, systems architecture, network security, and facing start-up challenges head-on. A few of my projects include a mobile web based showroom control system, a web based QR code authentication platform, a 360 degree projection gaming system, a markdown driven static blogging library, and a musician scouting website for bands.

I thrive on solving unique problems, learning, teaching, and trying to improve my community.
