Chris Murray's Summary

14 years of software application development and architecture experience, including all levels of technical design and implementation as well as user experience design.
Since 1998 I have been designing, building, and maintaining web applications and systems. First, using HTML and Perl, then Java for two years. The last 9 years have been primarily LAMP (Linux, Apache, MySLQ, PHP) development, including two years at an ISP (Concentric), building customer-facing administration web applications. Then two years at VMware building a Test Case Management System for the QA department. Then 3 years at Webroot as a database-focused developer making performance enhancements in the database and core application code, and slowly transitioned the Web Security SaaS product to a modern web application architecture based on the latest Zend Framework. During this time I honed my architectural skills, building rule-based relationships between the user interface elements, the software model, the ORM, and the database schema.
My current position at Apple comprises what would normally be considered a director of IT role within a data-heavy department, managing numerous contractors, applications, and a highly automated IT infrastructure. Additionally, with the help of a talented developer, we have built a code-generation system (built around Zend Framework) that has allowed us to continuously design and iteratively build a highly complex test management system with 75 database tables and hundreds of individual web pages with only 2 full-time developers.
