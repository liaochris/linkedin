Erik Luchauer's Summary

Currently employed as a web developer with Gannett Digital, a Fortune 500 media corporation. I work on their suite of websites to create and update HTML, CSS and JavaScript.

Previously, I have worked as an interactive project manager in a boutique design shop as well as a separate national media organization. In each position, I was in charge of multiple projects across differing organizations, bringing each project from conception through implementation and then to support. My past experience as a project manager helps to bridge the gap between the development and business side of a company.

Specialties

Web development, HTML, CSS, JavaScript, user experience, usability, project management, implementation planning, scope definition, requirements gathering and documentation, communication and interpretation between technical and business minded employees, organization, documentation, content management systems.
