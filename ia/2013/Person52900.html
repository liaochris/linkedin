Hernan Beati's Summary

-Me especializo en Capacitar a diseñadores web y programadores web, guiándolos en la aplicación de estándares, usabilidad y accesibilidad para la web, mediante la organización de cursos presenciales, seminarios in company, y formación a distancia desde mi campus virtual.

-Complementariamente, coordino desarrollos de sitios y aplicaciones web, en especial en los últimos tiempos, enfocándome en rediseños de versiones móviles de sitios y aplicaciones, para implementar Responsive Web Design + RESS, HTML5 y CSS3, y lograr que puedan visualizarse sin problemas de usabilidad ni de accesibilidad en todo tipo de celulares y tabletas.

Estoy disponible para dictar Capacitaciones in company o a distancia en temas de:

-Responsive Web Design (diseño adaptable a celulares y tabletas) incluyendo detección del lado del servidor

-Diseño Web con marcado semántico HTML5 + Microdata

-Maquetación y diseño con CSS3

-Programación Web con PHP y MySQL

-Metodología de Diseño Centrado en el Usuario (DCU), Usabilidad y Accesibilidad Web

-Optimización de Performance Web (WPO)
