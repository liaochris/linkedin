E-dan Bloch's Projects

  • YouTube SmartSearch

    • October 2010 to Present
    Team Members: E-dan Bloch

    JavaScript & HTML5.
    YouTube Instant Search extension for Google Chrome, Maxthon and Firefox (Greasemonkey).

    Search results and video preview displayed elegantly within the same YouTube page you're currently on. Allows you to watch, queue or add a video result to a playlist with a single click.

  • JCC JavaScript Library

    • October 2010 to August 2011
    Team Members: E-dan Bloch, David Yona

    JavaScript, jQuery & HTML5.

    The JCC JavaScript library provides users with a way of creating rich & dynamic web applications with limited JavaScript knowledge, or even no knowledge at all. The idea behind the project is to “teach HTML new tricks” and allow creating lightweight & flexible templates or widgets that may be used across a website with dynamically changing data sources. The JCC library brings a whole new way of working with external data sources, local-storage and continuously updating streams (social networks, news etc.).

  • Colorful

    • February 2011 to Present
    Team Members: E-dan Bloch, Achia Shaag, Maayan Negbi

    C#, Silverlight, Windows Phone 7.

    Colorful for mobile devices is an application that is intended to help color-blind persons to distinguish between different colors and recognize colors in photos or through a live video-camera feed. Colorful allows the user to get information about the different colors seen on the screen. Colorful includes assistive technologies that can change the color spectrum according to the user’s color-blindness type for better perception of the environment.
