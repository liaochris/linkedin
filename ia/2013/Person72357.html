Karla Fisher's Projects

  • Profile System 2.0

    • April 2004 to Present
    Team Members: Karla Fisher, , Duy Nguyen, Ninad Malvankar, Mahesh Gupta

    Profile 2.0 is the next generation of the legacy Profile System at UT Arlington which served as the information repository for faculty, students, and other entities within the university. Profile 2.0 supports defining templates (set of manageable categories and sections) for different type of user/unit such as a profile template for Faculty or Student, a profile template specific for Department, etc.

    Profile 2.0 serves as a central hub of information to pull and push data from/to many authoritative sources or databases securely and efficiently. With the help event-based implementation, user always gets notified about the changes via email, and internal messaging system to keep their profiles up to date.

    Profile 2.0 takes dashboard approach with to-do item and feed updates to keep user informed about events happened. Every action within the system is timestamped for future auditing.

    Profile 2.0 mainly built on LAMP with ZendFramework. Frontend is a fat client developed on top of jQuery, Underscore, BackboneJS, RequireJS, LESSCSS, and Twitter Bootstrap.

    Learn more about Profile 2.0 at https://www.uta.edu/mentis/public/#profile/index/index
