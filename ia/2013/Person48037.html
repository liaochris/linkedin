    Authors: George Chang

    A paper and a presentation describing the use of Hadoop and Amazon EC2 to perform on-demand processing of gigabyte-sized lunar images.

  • Automated Performance Testing Using Cloud Computing Architecture

    • ACM Conference on Software Engineering
    • May 2011
    Authors: George Chang

    Demonstrates the use of using cloud computing, specifically Amazon EC2 to perform a worldwide distributed load testing framework for LMMP (Lunar Mapping and Modeling Project) web portal.

  • Demonstration of LMMP (Lunar Mapping and Modeling) Using Amazon’s Elastic Compute Cloud

    • ACM Conference on Software Engineering
    • May 2011
    Authors: George Chang

    The Lunar Mapping and Modeling Project (LMMP) is currently being built by NASA. The goal is to provide a single point of access to the best state of knowledge of the moon’s terrain, rock and crater fields, resource maps, lighting conditions and thermal conditions. The architecture and design employ a variety of technologies, allowing for execution of complex models, the processing of large data sets and the distribution of the information, over the internet, to both authenticated users and the general public. The architecture supports a variety of light-weight clients including a Flash based display, an iPad/iPhone interface and a set of programmatic APIs that allow rich clients to interact with the LMMP system.

  • A Novel Approach to Automated, Secure, Reliable, & Distributed Backup of MER Tactical Data on Clouds

    • IEEE Aerospace Conference
    • 2012
    Authors: George Chang

    Describes an automated process to archive, compress, encrypt, and backup science data to Amazon's cloud storage. By migrating to the S3, the MER (Mars Exploration Rover) is able to save tens of thousands of dollars a year over their existing local backup solutions.
