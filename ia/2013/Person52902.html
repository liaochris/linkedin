Hernán Rodríguez Colmeiro's Summary

Javascript and Open Source lover, my passion is to push the web a step forward. I highly enjoy open source and geo-distributed working environments, where collaboration and good will is the key to achieve a good performance.

On my spare time I contribute to the Mozilla project, being member of Mozilla Argentina and a Mozilla Jetpack Ambassador (addons tech evangelist). I've led and now help run the Mozilla Hispano Labs section.

My collaborations to the Mozilla project are mainly code contributions, mostly to the Addon SDK and Firefox's Addons Manager, but in the past I spent a good time contributing to Firebug too.
