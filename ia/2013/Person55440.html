Dan Pratt's Experience

Founder, Full-Stack Web Developer and DevOps

Innov8d!

Partnership; 1-10 employees; Internet industry

March 2012 – Present (1 year 8 months) Biddeford, ME

I provide web application development and DevOps services using Ruby on Rails on the Amazon Cloud. Innov8d! is one of the first AWS Consulting Partner organizations and I am pursing the AWS Architect certification currently.

I am building a development ticketing and colaboration system called muLA (http://mu.la), which I use with my clients; a web application monitoring platform called AppStats, which I also use with my clients; and a closed-network photo sharing site called Fotopio.

I am the development parter for both The Webby Awards and The Lovie Awards. Major projects have been refactoring both systems to share a core code base, refactoring the database access code out into a separate Goliath-based web service API that can be scaled to handle load independently of the Rails-based front-end, and rebuilt the Call For Entry web application utilizing KnockoutJS significantly improving the entry process.

I also am working to bring the web development technical community in Southern Maine together with the hopes of helping to improve Maine's technical reputation. Smart people can and do live in Maine and we're doing some cool stuff, all while living in a state that has four seasons.
