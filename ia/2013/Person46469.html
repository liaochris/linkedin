Francis Williams' Projects

  • Lunar lander imaging prototype

    • January 2012 to Present
    Team Members: Francis Williams, Mustafa Faraj

    An imaging subsystem to be mounted on a lunar lander. The system uses a Gumstix Overo board with a camera extension and a mounted laser and diffraction gradient, to extrapolate 3D terrain data from a surface. This data will be used to perform hazard avoidance during landing.

  • Terrengine

    • December 2011 to Present
    Team Members: Francis Williams

    Terrengine is a small procedural terrain generator I wrote with the hope of eventually making a game. It is currently capable of generating undergroud cave like terrain. The engine features a camera with pan and zoom as well as JSON level storage. I intend to eventually clean up the code (it really needs it) and integrate physics using Box2D.

  • RPG Node

    • August 2011 to August 2011

    An entry for the node knocout competition, RPG Node is a real time RPG simulation using websockets to send state to a small node.js gameserver.
