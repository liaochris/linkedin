Daniel Londero's Summary

Web application developer specialized in PHP (since 2005) & symfony framework (since 2007). Now in love with Symfony2 and TDD.

Very good knowledge of Linux system administration (LAMP, Ubuntu, Debian, CentOs). Basic knowledge on cluster computing systems, shared storage, fault tolerance and high availability technologies.

I do my best for the symfony community translating official documentation (to italian) and trying to close some bugs.

I'm a co-founder of PUG Friuli (http://friuli.grusp.org) and member of the GrUSP (The Italian PHP User Group) board committee.

I like to attend and to speak in PHP and IT conferences in Italy and other European countries. I was a speaker at:
- phpDay 2011 (Verona) talking about "symfony and big numbers"
- phpDay 2012 (Verona) talking about using in memory tools vs database queries (title "Io cache, tu database", italian).
- BetterSoftware 2012 (Florence) talking about telecommuting
- SymfonyDay Italy 2012 (Turin) talking about Symfony2 and Doctrine ODM and ORM

Specialties: PHP, symfony framework, doctrine, documentation, agile, linux servers, XHTML & CSS, ubuntu, debian, centos, phpunit, tdd
