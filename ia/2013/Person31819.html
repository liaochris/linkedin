David Fox's Summary

Contract London-based iOS developer. Previously a senior Flash dev. 8 years in the industry, 5 of which as a contractor. Interested in lively offices with good coding standards and large-scale projects.

I'm competent in all-round iOS development including but not limited to...
--Location based (CoreLocation and MapKit) development
--Data persistance using CoreData/SQLLite
--Implementation of push notifications using Apple's APNS and Urban Airship
--App integration with restful web services
--Core Animation
--Building universal binaries (iPad and iPhone targeted apps)
--Performance tuning using Instruments
--Source control using Git and SVN
--App store submission and device provisioning
--Working in Agile environments (developing to user journeys, peer code reviewing, daily scrums, sprint planning etc)
--Test-driven development (TDD) using OCUnit (SenTestingFramework) and Jenkins for continuous integration

Outside of the geeky stuff, personal interests include the Japanese language and anything Larry David's had a hand in.
