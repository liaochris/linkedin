Mike Fullerton's Projects

  • FishLamp

    • May 2009 to Present
    Team Members: Mike Fullerton

    FishLamp (http://fishlamp.com) is an objective-c framework for building apps, tools, and APIs on iOS and Mac OS X. FishLamp abstracts and solves many difficult problems with high performance, flexible, and powerful solutions. FishLamp3 is under active development. Source code and tools released under the MIT open source license (http://fishlamp.com/license).

    FishLamp was designed and implemented using established design patterns such as bridges, strategies, factories, encapsulation, lazy initialization, singletons (sparingly), MVC, data providers, delegates, etc. These patterns were chosen, used, and modified carefully to improve usability, performance, and reliability and to reduce coupling and other unwanted interdependencies. FishLamp's design also uses hard won design principles: avoid clever base classes (don't try to predict how it will be used), use double dispatching to make decisions, aggregate over subclassing, and most importantly, always do one thing well.

  • PackMule (and Whittle)

    • January 2009 to Present
    Team Members: Mike Fullerton

    PackMule is a Mac OS X developer tool (a document based Cocoa app) which generates objective-c code from XML, JSON, or WSDL input. Generated code is integrated with FishLamp’s Model Object abstraction which provides caching, streaming, and other framework features. PackMule automates the lengthy and tedious task of abstracting data models, primarily those published by third party web-services or an app specific data model. Whittle is the command line version of PackMule's code generator, which allows for flexible automation.

  • myZenfolio

    • May 2009 to December 2011
    Team Members: Mike Fullerton

    Developed under contract for Zenfoilo, Inc.

    Shipped three major releases with several incremental updates. Designed much of the app personally. Additional design done in collaboration with Zenfolio team. Built with FishLamp. App offers many powerful features targeted at Zenfolio's customers who are primarily professional and serious photographers. App elegantly solves many specific user scenarios: e.g. "in studio" presentation mode, photo uploading and downloading capability, mobile gallery and photo management, social media sharing (before support was available in iOS SDK), and full gallery synchronization for offline support ("To Go" mode), etc. App achieves high perceived performance with tuned caching and strategic use of contextual prioritizing. Uses Zenfolio's complex web API - the Model Objects were generated with PackMule using the WSDL defined on Zenfolio's website (http://www.zenfolio.com/zf/tools/api.aspx).
