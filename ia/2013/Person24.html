Victor Brito's Summary

Travaillant dans le Web depuis 2006, en freelance depuis janvier 2013, j'interviens dans les domaines suivants :
- intégration (X)HTML / CSS (dans le respect des standards du W3C),
- accessibilité du Web (audits d'accessibilité de sites Web, mise en accessibilité de sites Web, balisage de documents PDF).

Je suis expert Accessiweb en évaluation et membre actif du groupe de travail Accessiweb (voir mon profil sur www.accessiweb.org/index.php/fiche_gta/items/site_victor_brito.html).

Pour en savoir plus sur moi : www.victor-brito.fr (mon site pro).

Spécialités : HTML 4, HTML 5, XHTML, CSS 2.1, CSS 3, accessibilité, intégrateur XHTML CSS, expert Accessiweb
