Bruce Taylor's Summary

I started my career in the web industry at the end of the 90's when I worked for Boo.com, (read more here: bit.ly/boo-dot-com), as a content manager. It was the first global internet-based high-end sport fashion company.

Soon after that I started freelancing as a web designer for some years before being employed permanently by Airmiles in 2004 (now Avios). I am currently employed as their Senior Front-end Developer / Designer which involves creating new digital functions, using the latest web technologies, to help increase revenue and customer engagement on Avios.com

Most things I work on these days adopt the “Mobile First” approach and use methods of development such as Responsive Web Design (RWD) and more specifically RESS (Responsive Web Design + Server Side Components). My team and I are currently exploring multi-device layout patterns that provide the best possible user experience on mobile devices for our particular offering - without neglecting the desktop.
