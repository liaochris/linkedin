Bruno Azisaka's Summary

My main programming language is Ruby for about 8 years, with it, Object Oriented Programming became a big part of my studies and work, which now I improve applying the S.O.L.I.D. principles.

TDD came with the Ruby knowledge and using tools like Capybara and RSpec makes me feel safer on a project. It’s part of my daily work, a very important one.

As a web developer, I managed to master HTML, CSS and Javascript. I prefer HAML, SASS and Coffeescript to write them. For Javascript I’m used to apply Backbone.js and JQuery into the projects I work on.

Relational databases like PostgreSQL and MySQL are my main choices for data storage, but MongoDB and Redis are very used in my projects too. It depends on the goals.

I am used to agile development using Scrum with a little help from Pivotal Tracker. DevOps is a role a do with ease when I can use tools like Chef. To improve server configurations is a kind of hobby to me.
