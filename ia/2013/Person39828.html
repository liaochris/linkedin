Edward Beuchert's Experience

iPad and iPhone Application Developer

JamWithIt.com

January 2009 – Present (5 years)

Version 2.0 of my WMFO iPhone application is now available for free from the Apple iTunes App Store. It's an Internet listener app for WMFO (91.5 FM in Medford) that allows users to listen to the web stream, browse the complete live playlist, access the Twitter feed, and view WMFO YouTube videos. To download it, just go to iTunes and search for WMFO (or click the link next to "Websites" up above *8^!)...

I’m currently working on a commercial application called “Jam With It!” which is a visual audio mixer that will let users mashup music live on their iPad, iPhone and iPod Touch devices.

I'm actively enrolled in the Apple iPhone Developer program and I've been working through the various SDK examples and documentation to learn all aspects of iPhone programming including Objective-C, the Cocoa Touch, Media, Core Services and Core OS APIs.

Chief Software Architect

WebEx / Cisco

Public Company; 10,001+ employees; csco; Telecommunications industry

April 2006 – May 2008 (2 years 2 months)

Originally hired as the Chief Software Architect for the WebOffice online collaboration suite, a C# and XSLT .NET-based “Software as a Service” product with 325,000 users.

As a key member of the WebOffice Vice President of Engineering’s Management Team, I helped deal with many of the technology integration issues between WebOffice and the WebEx Meeting Center family of products, including the challenges of developing software across a vastly geographically distributed team. Designed XML Web Service APIs, wrote and reviewed specifications, produced presentations and helped plan product strategies.

After WebEx was acquired by Cisco, I worked as an Architect on the Connect project, designing and writing AJAX, JavaScript and Java code on a platform for developing collaborative Web 2.0 applications.

Consulting ASP.NET Programmer

Creative Data

Privately Held; 11-50 employees; Information Technology and Services industry

2003 – 2004 (1 year)

Designed and implemented an integrated ASP.NET-based Project, Budget and Timesheet tracking application based on a suite of three legacy FoxPro applications. Wrote ASP.NET and VB.NET code that integrated with a SQL Server 2000 database.

The architecture was heavily based on User Controls and customized DataGrid controls to display, edit and drilldown through data. Primarily responsible for the User Interface, which communicated with the database through stored procedures.

Systems Architect

LifeFX

2000 – 2001 (1 year)

Systems Architect for the LifeFX Player, an ActiveX/COM/ATL-based facial animation system that renders “digital human beings” for Internet email (FaceMail) and CRM applications.

LifeFX developed technology that Ray Kurzweil demonstrated as his Ramona female alter ego at the 2001 TED Conference -- Ray also served on the LifeFX Board of Directors.

I designed the overall product architecture and implemented it with a team of other engineers. Took prototype algorithms designed by researchers at the University of Auckland and developed them into the production version of the LifeFX animation engine. Built the first version of the installation utility. Set up the source control system (initially SourceSafe and then CVS) and worked with another engineer to implement an automatic nightly build process and smoke test for the product. Managed Release Engineering. Worked with Sales, Marketing and customers to determine their needs and deploy solutions that met their requirements. Responsible for development of the LifeFX Software Development Kit (including documentation and JavaScript code samples).

Primarily Visual C++ programming under Windows 98/NT/2000.

Contract Systems Architect

Lotus Development

Public Company; 1001-5000 employees; IBM; Computer Software industry

1995 – 2000 (5 years)

Contract Systems Architect on the 1-2-3 spreadsheet for the first 32-bit release (1-2-3 97), and the subsequent follow-on versions (9.0, 9.1 and 9.5).

I was originally hired to work on the 1-2-3 “Framework,” which serves as the core object-oriented windowing and C++ class library for the product. Later served as technical lead for the system. Worked across virtually all areas of the product including low level Windows graphics rendering, event handling, file system, OLE, DDE, LotusScript language interpreter, memory management and error and exception handling. Designed and coded algorithms that increased LotusScript and macro execution performance by 600%. Produced a prototype port of 1-2-3 running under Red Hat LINUX. Helped solve problems in the SmartSuite Shared Components and sister products such as Notes, Approach, and Freelance.

Extensive use of the Win32 API, including multi-threading. Visual C/C++ programming under Windows 95/98/NT/2000, with significant use of UNIX tools.

Principal Engineer

Project Leader

Intersolv

Privately Held; 501-1000 employees; Computer Software industry

Originally hired to implement the print and font subsystems of the Index Technology “Virtual Terminal,” a platform independent windowing system and WYSIWYG graphics programming environment. Subsequently promoted to Project Leader of the six engineer team.

The VT provided portable “MacApp”-like functionality to twenty-two client subsystems which comprised the Excelerator Series II CASE tool. Wrote specs, coded, worked with internal clients to solve their issues and scheduled tasks for team members. Single handedly completed feasibility port of the VT to Windows 3.0.

Object-oriented C programming under UNIX/X Windows and OS/2-Presentation Manager.

Lead Engineer

Grumman Aerospace

Public Company; 10,001+ employees; Defense & Space industry

1985 – 1988 (3 years)

Lead Engineer for the COSM project, a prototype system to Check-Out, Service and Maintain the space suits and Extra-Vehicular Mobility Units for NASA’s space station. This involved a network of three Sun workstations, one that simulated a space suit, another the EMU and a third that functioned to diagnose and assist astronauts maintain each of the other two components. Responsible for large portions of the overall software system design, development of network communication routines and user interfaces. C and POSTSCRIPT/NeWS programming under UNIX.

Programmer for TENA, the Test ENgineer’s Assistant, a Grumman productivity tool to automatically generate diagnostic strategies for analog avionics. Artificial Intelligence and frame-based modeling techniques in C and LISP.
