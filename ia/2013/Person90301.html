Matthew Brewer's Summary

I'm an experienced Ruby on Rails and iOS engineer who enjoys challenging projects and working with a talented group of folks.

My Rails experienced covers 3.x and 4.x versions with experience in rspec, cucumber, and FactoryGirl. Have written and consumed many APIs and I'm an active repo contributor to the ActiveAdmin gem.

My iOS experience covers several iPhone and iPad applications written for iOS 4, 5 and 6 over the last several years. I'm very familiar with Xcode and CoreData, but have also written 3 applications to date in RubyMotion and love the test framework and workflow that it provides. I have a couple gems for RubyMotion projects that are publicly available, and also actively contribute to the great ProMotion gem for RubyMotion projects.

I'm also comfortable, but not an expert, in server management via Ubuntu or RedHat derivatives. Have configured a server from scratch to server multiple Rails application with MySQL or PostgreSQL, Apache/Passenger and Nginx/Puma for production environments. I have used Heroku for deployments quite extensively, and I'm familiar with EngineYard as well.

  • API "Cacher"

    • November 2012 to Present
    Team Members: Matthew Brewer

    Approached by a vendor developing native mobile applications needing access to protected data within the organization. Data is provided as a series of JSON responses, but not available publicly and had rate limiting that would not allow the large mobile user base to retrieve the information.

    Created a Rails 3 application that provides logging and monitoring of the service, which was implemented as a class exposed via several rake tasks that insert jobs into a Delayed::Job queue, via cron. For tasks that needed more frequent, reliable (every few seconds) updating, implemented a daemon that runs & integrated with capistrano.

    JSON responses are parsed & queued up, so that all possible information a mobile user could ask for is made available, resulting in > 5k API calls. Results are pushed to a CDN for delivery to mobile clients.
