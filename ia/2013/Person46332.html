Fernando Salom's Projects

  • MORE - Movilidad Orientación RFID en Edificios (Movility Orientation RFID inside Buildings)

    • October 2008 to July 2009
    Team Members: Fernando Salom, Victor Marco

    Highest mark: 9.5

    This project arose from the idea of ​​helping the mobility of visual impairment people inside buildings using RFID technology. For this reason I created an application for mobile phones (NOKIA) that make use of a Bluetooth RFID receiver, RFID tags and a XML file, to guide them inside buildings with a greater freedom.

    The technologies used in this project were:
    Java/J2EE and XML for the mobile phone APP
    PHP, MySQL, HTML and CSS for the web platform

  • 2CAM

    • October 2007 to June 2008
    Team Members: Fernando Salom, Victor Marco

    Highest mark: 10

    This was a research project to add stereoscopy (or a 3D perception) in a videoconference with the use of 2 High Definition cameras. This project was a challenge, because it was very complex and there were not too many information about how to extract frames from 2 USB cameras at the same time.

    The technologies used in this project were:
    C++, OpenCV Library and OpenGL
