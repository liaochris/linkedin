Immediately upon joining the company I was handed an ‘out of control’ development project for a core product. Personally built the first set of automated acceptance tests in Ruby, RSpec, and Selenium for the StrikeTracker application. I also personally setup and configured a multi-node test environment to support automated testing across all major web browsers using the continuous integration server, Jenkins.
