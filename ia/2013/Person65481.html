Joel Kraut's Summary

Software engineer focusing on graphics, animation, user interface, and data visualization on the Mac, iPhone, and iPad. I'm about using cutting-edge graphics and animation technologies to help users understand what's going on, why, and whether they should care.

Specialties: Core Graphics, Core Animation, Cocoa, iPhone development, UIKit, Core Data

Joel Kraut's Patents

  • Methods and Apparatuses to Arbitrarily Transform Windows

    • United States Patent US8379058 B2
    • Issued February 19, 2013
    Inventors: Joel Kraut

    Complex mesh transforms for windows fixed to other windows--shadows, drawers, alerts, and so on. Used for window shadows in OS X.

  • Ambient Noise Based Augmentation of Media Playback

    • United States Patent Application 12330335
    • Filed December 8, 2008
    Inventors: Joel Kraut

    Intelligent responses to noisy environments on mobile devices.

  • Displaying Multi-Dimensional Data Using a Rotatable Object

    • United States Patent Application 12478752
    • Filed June 4, 2009
    Inventors: Joel Kraut, Camilo Mattos, Santiago Becerra, Jaime Zuluaga, David Becerra, Patrick Cheng, Quinton Alsbury

    Viewing large-scale multi-dimensional data by rotating along one or more dimensional axes. Used in the Cardex view in Roambi.

  • Predictive Scrolling

    • United States Patent US 8271898 B1
    • Issued September 18, 2012
    Inventors: Joel Kraut, Santiago Becerra, Patrick Cheng, Camilo Mattos, Alex Schaefer, Jaime Zuluaga

    Pre-fetching data based on a prediction of where a user-intiated scroll will stop. Used in the Cardex in Roambi.

  • User Interface for Data Comparison

    • United States Patent Application 13/415,651
    • Filed March 8, 2012
    Inventors: Joel Kraut

    Syncing multiple charts one set of user input to create an ad hoc dashboard / comparison interface. Used in Roambi's Cardex on iPad.
