Henrik Nilsson [MVP]'s Publications

  • FIM Ultimate File Connector (Open source project)

    • http://fimfileconnector.codeplex.com/
    • May 15, 2012
    Authors: Henrik Nilsson [MVP]

    Text file connector with the additional functionality of supporting full exports, management of files on FTP, SFTP, FTPS, SCP, and file systems, compression and encryption of files and substitution of date time values in file names.

  • ADFS 2.0 Attribute Store for FIM (Open source project)

    • http://fimattributestore.codeplex.com/
    • December 5, 2010
    Authors: Henrik Nilsson [MVP], Mattias Fagerlund

    ADFS 2.0 AttributeStore that allows you to fetch attribute values from Forefront Identity Manager and issue them as claims.
    For example:
    *Attributes on the Person resource can be fetched and issued as claims.
    *Groups, Sets and Roles in FIM can be issued as group or role claims.

Henrik Nilsson [MVP]'s Projects

  • Swedish School of Economics - Upgrade of Live@Edu to Office 365 for Education

    • December 2012 to Present
    Team Members: Henrik Nilsson [MVP], Bodén Fredrik

    Project for upgrading Live@Edu to Office 365 for Education on Swedish School of Economics. The use of tools are FIM for identity management and ADFS for federation of identities.

  • Sveriges Television MIIS to FIM replacement

    • February 2012 to January 2013
    Team Members: Henrik Nilsson [MVP], Joakim Wall, Martin Ericsson, Cecilia Forss, Hoshang Asadi, Ulrika Hill

    Total rebuild of Microsoft Identity Integration Server 2003 solution to Microsoft Forefront Identity Manager. Connected systems are SQL Server, Active Directory, Lotus Domino, Oracle and text files.

  • The Absolute Company, ILM to FIM Rebuild

    • April 2011 to September 2011
    Team Members: Henrik Nilsson [MVP], Christian Dommersnes, Ingvar Johansson

    Upgrade from Microsoft Identity Lifecycle Manager 2007 to Forefront identity Manager 2010. Amongst the connected systems were AD+Exchange+OCS, Postini, SQL Server.

  • Bonnier BonCom

    • February 2006 to June 2007
    Team Members: Henrik Nilsson [MVP], Fredrik Gunne, Patrik Lassander

    The Bonnier Group is widely spread and loosely coupled and the aim for the project was to simplify communication within the group. I worked together with MSC to develop a solution to connect ~45 connected systems for fetching contact details to employees withing the Bonnier Group. The solution also included a portal for searching employees and company information withing the Bonnier Group. I still work with this solution (February 2013).

  • Bonnier BonID

    • 2009 to 2012
    Team Members: Henrik Nilsson [MVP]

    The Bonnier Group needed a good authentication solution for their new group wide intranet and my idea that became BonID was built on Active Directory Federation Server 2.0 (formerly Geneva) and information from the BonCom system that was used as source of identities. Still manage this application (February 2013).
