Kristofer Schlachter's Summary

I am a graphics and simulation programmer with years of experience in programming games and 3D graphics software for PC's, gaming consoles and mobile devices. I have created various game and graphics engines using custom software renderers, OpenGL / OpenGLES and DirectX. I have written multi-threaded applications that take advantage of the cores on the CPU and the GPU using pthreads, atomics and OpenCL.

One of my specializations is in fluid dynamics for games. My fluid simulator technology is scalable all the way from mobile devices to game consoles and multi-core PC's. There is a toy example at https://github.com/kristofe/OpenCLFluids.
