My main project was to develop a specification sheet for analog video measurement test system VideoMASTER. This consisted of going through LabVIEW source code and dissecting all the measurement algorithms and see how they correlated with ITU and IEEE standards. I also had to develop an extensive set of test video waveforms with known errors in order to analyze performance. The hardware front end of videoMASTER also had to be specified so signal paths had to be analyzed including impedance matching options and SNR of digitizers and the frequency response of both digitizer and arbitrary waveform generator was investigated..

Another task I had, was to perform verification and validation on the audio measurement suite AudioMASTER this also entailed learning how to use TestStand (NI’s test bench sequencer).
