Mathias Nitzsche's Publications

  • Implementation of a browser based user interface to create PACT-programs

    • Master Thesis at Humboldt University Berlin
    • 2011
    Authors: Mathias Nitzsche

    Within the Stratosphere Project PACT-programs are used to process large amounts of data in parallel on a distributed cluster computer architecture (a Cloud). The thesis describes the implementation of the PACT-GUI web application, which for the first time allows to graphically model such PACT-Programs.

  • Privately Waiting – A Usability Analysis of the Tor Anonymity Network

    • Americas Conference on Information Systems
    • 2010
    Authors: Mathias Nitzsche

    This paper quantifies the loss of usability by measuring the additional latency of the Tor software and combine the measurements with metrics of the existing Web usability and performance literature. The findings indicate that there is still a major usability gap induced by Tor, leading to its possible disuse accompanied by a higher risk exposure of Internet users.
