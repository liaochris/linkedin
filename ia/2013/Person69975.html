Jonathan Shoaf's Summary

I started web development back in the days (1995) of Netscape, C++ and Perl for CGI, table layouts, animated gifs, java applets, early javascript, and the beginnings of IE. Loved every moment of the metamorphosis to today's web technologies. HTML5, CSS3, jQuery, web services, ASP.Net, and Flash are more likely to be my tools of choice today. Much has changed but much has stayed the same.

I've been with javascript from the beginning and have continued with it through out the years. I understand industry issues like browser compatibility, responsive design, mobile, user experience, and more. I'm very keen on creating the best user experience possible through an intuitive UI and appropriate data presentation.

E-learning has been an interest of mine the last few years. It allows me to use a lot of my web development skills combined with my interest in user experience and love for learning. My experience with the underlying technologies of standards like SCORM allows me to see things in a unique way.

I spent a few years doing Windows application development as well using Visual Studio, Visual Basic, C++, DLLs, and more.
