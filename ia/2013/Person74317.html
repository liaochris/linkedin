Graduate Intern

Intel Corporation

Public Company; 10,001+ employees; INTC; Semiconductors industry

May 2004 – August 2004 (4 months) Hudson, MA

Parallelized the ASIM simulator with pthreads. ASIM is a modular performance model framework in
which parts of a computer system are represented by reusable modules connected by ports. By insisting that inter-module communication occur only through modified ports and allowing modules to be invoked in distinct threads, the new parallel framework allows each simulated CPU to execute on a distinct host processor. This organization exploits inherent parallelism to accelerate simulation of multi-core models.
