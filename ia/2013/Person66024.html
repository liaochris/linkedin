John Ankcorn's Experience

Consulting Member of the Technical Staff

Quanta Research Cambridge, Inc.

2013 – Present (less than a year) Cambridge, MA

Distinguished Researcher and Project Chief Architect. Former Member of CEO Technology Council.

Nokia Research Center

Public Company; 10,001+ employees; NOK; Telecommunications industry

2005 – 2013 (8 years) Cambridge, MA, USA and Ulm, Germany

Operating system innovation for volume phone products.
Learned a lot about realities of disruptive innovation in large companies.

- 2006 • Performed software needs analysis for Nokia, identifying the need for adding 3rd party applications to feature phones as an important growth business segment (Now called "smart feature phones"). Proposed replacing RTOS in these products with Linux to provide needed isolation of application behavior from core platform.
- 2007 • Demonstrated ability to run Linux with existing real time modem code with UI on a single processor, without requiring virtualization. This ran on existing high volume Nokia production products with full ARM TrustZone security and removed all technical restrictions to productization. This was done my creating a "host firmware" API, allowing execution of binary images inside the Linux kernel using a kernel pthreads-like managed API (the firmware was independent of Linux and not compiled/linked with any kernel modules).
- 2008 • Demonstrated existing S40 modem and UI code running at full speed on existing S40 production hardware running applications with X Windows.
- 2009 • Demonstrated multiple application frameworks on production devices using optimized window manager: Qt, Html (Common Web Runtime), QML.
- 2010 • Integrated Linux support into S40 production code trees, allowing it as a standard product configuration option. Software running on 14 different production devices.
• Pushed business proposal up to CEO, receiving approval for new development unit.
- 2011 • Selected technical staff and made key technical decisions for new development unit.
- 2012 • Lead platform-wide optimization/stabilization project, preparing for volume product delivery.
• After project was cancelled, helped project members find jobs outside Nokia.

Principal Research Scientist

HP Labs

Public Company; 10,001+ employees; HPQ; Information Technology and Services industry

2001 – 2005 (4 years) Palo Alto, CA

Development of streaming video server for 3G networks, supporting file streaming, live feed splitting, transcoding, recording. Became a key product of Open Call Business Unit and is listed as one of "40 Innovations that counted" in HP Labs 40th Anniversary.
Developed "wrist mounted devices" as the next area of innovation in mobile computing (after cellphones saturate).

Research Scientist

Added video signal processing support to Spectrumware signal processing system, triggering system-wide refactoring to increase processing speed ~80x.
Technical Coordinator for the Project Oxygen consortium, across the Laboratory for Computer Science, Artificial Intelligence Laboratory, DARPA and 6 corporate sponsors.

President and Founder

Intersync

1992 – 1998 (6 years) Kawasaki, Japan

Created protocol software for X.400, X.500, FTAM, CMIP and ISDN. Built ISDN to TCP transport bridge products, allowing a wide range of computer products to directly support ISDN communications via ethernet interfaces. Built a Multi-point Control Unit for ISDN video conferencing, both FPGA logic and realtime protocol software.

VP, Engineering and Corporate Director

Custom Technology Corporation
