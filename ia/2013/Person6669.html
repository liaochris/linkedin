André Krämer's Summary

Mit mehr als15 Jahren Berufserfahrung als Softwareentwickler und Consultant verfügt André Krämer über ein eingehendes Verständnis sowohl der technischen, als auch der betriebswirtschaftlichen Seite der Informationstechnologie. Neben der tiefen technischen Ausrichtung liegen seine Stärken in einer hohen Kundenorientierung, Teamfähigkeit und der Fähigkeit Projekte innerhalb von Zeit und Budget erfolgreich abzuschließen. Während seiner beruflichen Laufbahn brachte Herr Krämer zahlreiche Projekte in Europa und den USA zu einem erfolgreichen Ende.

Sein Fokus liegt auf der Microsoft .NET Plattform mit folgenden Schwerpunkten:

- Tiefgehende Erfahrung in mehreren Programmiersprachen und Technologien wie Beispielsweise: Windows 8 Apps, C#, Visual Basic.Net, ASP.NET, JavaScript, Windows Forms, SharePoint, .NET Framework, TX Text Control, Infragistics NetAdvantage, SAP .NET Connector (NCO), Team Foundation Server, VB6, VBA
- Erfahrung mit dem Microsoft SQL Server und Oracle
- Mentor für .NET Software Entwickler
- Mitbegründer und Leiter einer deutschen .NET User Group
- Autor mehrerer technischer Artikel in diversen deutschen .NET Fachmagazinen
- Sprecher auf deutschen Entwicklerkonferenzen

Neben seiner akademischen Ausbildung als Diplom-Informatiker (FH) hält André Krämer mehrere Microsoft Zertifikate, unter anderem den Titel des Microsoft Certified Solution Developers und des Microsoft Certified Professional Developers. Als Anerkennung für sein Engagement in der Microsoft .NET Entwickler Gemeinschaft wurde er im Februar 2009 von Microsoft in das Microsoft Community Leader/Insider Program (CLIP) aufgenommen. Im März 2009 wurde er von der Firma Infragistics als Infragistics Most Valuable Professional (MVP) ausgezeichnet. Seit 2013 ist er außerdem Microsoft MVP und TX Text Control MVP

Spezialgebiete: .NET, C#, TFS, Sharepoint, ASP.NET
