Jakub Holý's Summary

*DO NOT SEND ME JOB OFFERS OR CALL ME, THANK YOU* (Unless your are ThoughtWorks opening an office in Oslo and starting a Clojure project there ;-))

I’m Java[EE] developer since 2005. I’m highly interested in code quality, developer productivity (and tools like Maven and AOP/AspectJ), testing, performance, and web frameworks. I’m also very fond of the lean thinking and agile principles, especially feedback-based development, focus on creating true value ASAP, and respect for people.

I've worked on large and long-term international projects as well as small local ones and both on new and maintenance projects in various positions. I enjoy working with people, sharing knowledge, being creative and exploring new stuff. I don't like tedious, repetitive work (though it really is sometimes necessary) and am thus a big fan of automation and tools that can take it over. Having maintained a lot of old code, I really appreciate high-quality, well-written code and good and automated unit tests.

Lately I also gained interest in functional programming, which I'm exploring (Clojure, Scala).

Specialties: Java web application development, automated unit/integration/performance testing, continuous delivery, code quality, lean thinking.
