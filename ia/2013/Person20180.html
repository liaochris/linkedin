Carl S. Gutekunst's Summary

Software engineer with over 30 years of professional experience, primarily in data communications, electronic messaging, low-level protocol interfaces, and controller firmware. I have developed software for a broad range of platforms, from micro-controllers to mainframe computer systems, from end-user boxed software to enterprise-grade hosted services.

Specialties

E-mail (RFC 822, MIME, SMTP, POP3, IMAP); Transport Security and PKI (SSL/TLS/X.509); Internet stack (UDP, TCP, IP); Domain Name Service (DNS); Web Services (HTML, CSS, HTTP, XML); other Internet applications protocols (FTP, IRC, SIP/RTP, Telnet); Directory Services (X.500, LDAP); Languages (C++, OO Perl, C, Java); Operating Systems (Linux, UNIX, Windows, various real time systems).
