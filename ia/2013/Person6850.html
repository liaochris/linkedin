Andrew Wylie's Experience

Senior Systems Analyst

NAV CANADA

Public Company; 1001-5000 employees; Airlines/Aviation industry

July 2012 – June 2013 (1 year) Ottawa, Canada Area

This time at Nav Canada I was hired on a one year term to work with a larger team (~30) on the Hong Kong Air Traffic Services Data Management System. During my employment I was involved with build management, software integration testing, system environment setup (think base os configuration for custom software, but with a couple hundred machines), system deployment, project support, and some python-django work for a management interface to a subsystem.

Coming into the project near the end of development/start of testing (seemed like waterfall or v-model) I was tasked with responsibility over the jenkins and trac servers. After figuring out a nice way to package our software the jenkins jobs were re-configured, and an ant script was made to replace a custom in-house java-ant extension (3Kloc → < 1Kloc).

In addition to the build management responsibilities I was then tasked with coming up with requirements for the base system configuration and a way to test it. A simple versioned spreadsheet was useful to keep track of software dependencies, basic images were captured (& deployed) with the HP Client Automation tool, and then tested with python scripts.

Lastly was a custom authentication backend written in python-django. This backend provided a web interface along with an api, and would be hooked into by other django apps on the same server as well as java applications & other web applications running within the system. It used token-based authentication, with the blowfish algorithm (via py-bcrypt) as the password hashing function.

Software Developer

NAV CANADA

Public Company; 1001-5000 employees; Airlines/Aviation industry

May 2010 – August 2010 (4 months) Ottawa, Ontario

For this placement (Carleton University Co-op) I began work with a small group of developers (~6) on an online collaborative flight planning system (CFPS) at the beginning of its implementation stage.
