Daniel Wellman's Publications

  • From One Expert to Another: Dan Wellman on Test-Driven Development

    • StickyMinds.com
    • December 31, 2012
    Authors: Daniel Wellman, Steve Berczuk

    I was interviewed by Steve Berczuk about my experiences practicing and teaching Test-Driven Development (TDD). We discussed how TDD can improve the quality of code and help an agile team deliver software, some contexts where TDD might or might not help a team, and the technical and cultural challenges of adopting TDD.

  • Alternative JVM Languages for Java Projects

    • Better Software Magazine
    • September 1, 2011
    Authors: Daniel Wellman

    Groovy, JRuby, Scala, and Clojure are some of the most widely used alternative Java Virtual Machine (JVM) languages today. I spoke with several developers who adopted one of these languages in a Java-based project. This article tells their stories - which language they picked, why they picked it, and how they integrated it with their existing Java code base or infrastructure.

  • Introduction to Scala

    • Better Software Magazine
    • November 1, 2010
    Authors: Daniel Wellman

    Scala is a programming language that blends functional and object-oriented language features. Scala programs run on the Java Virtual Machine and can easily interact with Java code. Learn how Scala can yield concise, safe, and compatible code, and how you can start learning Scala on your own

  • What to Expect When You're Automating Testing

    • Better Software Magazine
    • May 1, 2009
    Authors: Daniel Wellman

    After learning the basics of testing frameworks like xUnit, starting to write tests for your existing codebase can be a daunting challenge. How do you know where to start testing, and what kind of tests will be the most effective to start with? Learn strategies to kick-start your testing and some solutions to problems teams frequently encounter.

  • Google Web Toolkit: Writing Ajax Applications Test First

    • Better Software Magazine
    • November 1, 2008
    Authors: Daniel Wellman

    This article introduces Google Web Toolkit's testing infrastructure and demonstrates how to build an Ajax application test first. The Model View Presenter design pattern is introduced as a way to build easily testable user interfaces.

    This article was included in the GWT project documentation in the Articles section as "Testing Methodologies Using Google Web Toolkit".

  • Google Web Toolkit: Your Shortcut to Ajax Applications

    • Better Software Magazine
    • October 1, 2008
    Authors: Daniel Wellman

    Many an agile developer has gotten stuck in the AJAX mud when trying to develop Web 2.0 applications. Google Web Toolkit (GWT) applications are written in Java and compiled to JavaScript. This helps developers stay unstuck by leveraging the power of well-established Java tools and techniques. Learn how GWT leverages the maturity of Java tools, supports Test-Driven Development with JUnit, and handles all the cross-browser JavaScript for you.
