Public Company; 5001-10,000 employees; DNLM; Retail industry

February 2010 – May 2011 (1 year 4 months)

Providing 3rd Line Technical Support for Dunelm's EPOS estate across more than 115 locations and over 1,000 devices. Responsible for the support and capacity of EPOS Hardware and Software architecture, as well as for the custom development of enhancements to what is effectively a closed source product.

Responsible for the development of an in-house replacement to the Experian QAS (QuickAddress) product which was integrated with the EPOS system, resulting in significant annual cost savings with no loss in functionality.

Heavily involved in problem analysis and debugging of 3rd party code as part of ongoing efforts to reduce support incidents and eliminate software bugs and performance bottlenecks.

Also responsible for the specification, development, testing and deployment of software releases and configuration changes into the production EPOS estate, following the ITIL framework.

Designed and single-handedly developed an in-house web based system for capturing the working hours of over 5000 employees across the company. This system was used daily by over 115 retail stores in order to determine the applicable monthly wages payment for each employee. The system was developed using Silverlight 4, Windows Communication Foundation and SQL Server 2008 using c#.net, and was directly integrated with a SAP HR/Payroll implementation, as well as an Enterprise Business Objects reporting system to allow prompt management reporting. The implementation of this system resulted in a reduction in 2 days workload for a single member of staff each week over the legacy excel based system, whilst allowing the automated production of reports significantly earlier and reducing the margin for data entry errors.
