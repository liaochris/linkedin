Application Development Contractor

Innotek New Zealand

January 2004 – September 2011 (7 years 9 months)

The Innotek New Zealand Web site is on online Shopping cart application.

The solution consists of

1. A web application which ranks high across many search engines and provides an interface to browse and shop for products. The shoppers can add products to their shopping cart and move on to the checkout process when they are finished shopping.

2. A windows client application that monitors web site orders. On receipt of an order the application ensures the credit card payment was successfully received and the prints off a picking list and delivery docket. Once the order has been despatched the same application prepares the Customers Invoice.
This same application provides an interface for adding new stock and maintaining existing stock status. All changes immediately update the web site.

3. Both application described above communicate to a back-end application that provides the business logic and data store. Having the two application talk to the same back-end ensures there is no duplication in coding on any logic shared is both applications. The windows client application communicates to the back-end using WCF technologies. The communication is secured and data encrypted using a SSL certificate. This same certificate is used in the web application to provide security during the check-out phase of purchasing.


The technologies applied in the solution are:
