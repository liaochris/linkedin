Charles Moyes' Projects

  • Brain-Computer Interface

    • January 2012 to May 2012
    Team Members: Charles Moyes, Mengxiang Jiang

    We built a robust Brain-Computer Interface (BCI) using single-channel electroencephalography (EEG) with an AVR microcontroller, and we were able to play Pong using our brain waves.

    Featured by Engadget, New York Times, Popular Science Magazine, CNet, CBC News, Slate, PC World, Mashable, German National TV (ZDF kulturpalast), TechFest 2013 conference at Indian Institute of Technology, Bombay campus

  • Codentical.com

    Team Members: Charles Moyes, Tony Diepenbrock IV

    Built a source code plagiarism detection web service. Codentical's algorithm compares subtrees in the program Abstract Syntax Trees by hashing each one through a bottom-up traversal in a high-dimensional metric space. Upon kNN-querying this space with a test subtree, proximate subtree matches in hash space are filtered and further compared using a dynamic programming edit distance algorithm. High-threshold matches are flagged as potential plagiarism. Certain tasks such as spatial data structure construction and matrix SVD are performed using parallel algorithms. By using a spatial data structure and locality-sensitive hashing, Codentical can scale to millions of documents.

  • GraphMuse.com

    • August 2011 to August 2012
    Team Members: Charles Moyes, Tony Diepenbrock IV, Ryan Shea

    Developed a set of fast algorithms that run on the GPU using NVIDIA CUDA to process Facebook social graphs and determine user's social groups, group interests and identities, and influential users. Created computationally-assisted viral marketing technology. UPenn Innovation Fund recipient, YCombinator interviewee, MIT 100k entrepreneurship competition entrant, and patent pending. Currently in use by the sixth largest social gaming company.

  • Greekdex.com

    • August 2011 to August 2012
    Team Members: Charles Moyes, Tony Diepenbrock IV

    Social networking site for fraternities and sororities at the University of Pennsylvania with 500 users. Developed Flask web application and CUDA GPU computing graph clustering algorithm to determine all of the members of a user's fraternity or sorority in 0.2 seconds with 90% accuracy from their Facebook social graph. Featured in The Daily Pennsylvanian, BroBible.com, and Under the Button.

  • Rigid Body Sound Synthesis Engine

    • January 2011 to May 2011
    Team Members: Charles Moyes, Shentong Wang

    Final project for CS 6650: Computational Motion with Doug James. Wrote a rigid body sound synthesis engine using modal synthesis and contact force projection with a tetrahedral finite element model in C++. Developed a rigid body dynamics engine using the projected Gauss-Seidel method to iteratively resolve contact forces. Was able to model the sound of objects hitting glass, along with striking resonating bells in real-time.

  • OpenGL 3D Flight Simulator with Realistic Physics

    • June 2006 to Present
    Team Members: Charles Moyes

    Created a realistic software-based flight simulation of a Cessna aircraft using Newtonian rigid body dynamics, a physically-accurate flight model evaluating lift and drag force equations over a finite element model (FEM), and my own 3D engine. Atmospheric scattering sky color calculations, quad-tree LOD terrain rendering, and cloud rendering lend graphical realism to the project. WIP WebGL in-browser demo with accelerometer controls.
