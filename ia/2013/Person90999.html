Software Engineer

L-3 Communications

Public Company; 10,001+ employees; LLL; Defense & Space industry

March 2006 – May 2008 (2 years 3 months)

Development of client-server model for an EGL/OpenGL system to be used at F-35 military aircraft.

JSF (Joint Strike Fighter – F-35) – Panoramic Cockpit Display System
. Develop the architecture of a SEAL-1 (DO-178B, Level A) certifiable Client-Server interface with underlying OpenGL-EGL Windowing System supporting requirements of JSF SDD Program. The system architecture has a client with GreenHills Integrity RTOS based multi-threaded, multi address space system and a server with LynxOS based multi-threaded system. The system preserves the atomicity of commands, the order of commands among different tasks with different priorities and is non-blocking even for the functions with return values. It runs supporting at least 30 Hz. frame rate under all conditions and continues operating if the server crashes. Errors generated are checked and sent back to the client periodically without interrupting the normal operation.
. Develop and implement the architecture of memory usage algorithms.
. Full implementation of multi-threaded server side code to be run with single-threaded OpenGL-EGL system.
. Implementation of the client-server test application running in GreenHills Integrity environment.

. Perform validation and testing of the entire architecture on GreenHills Integrity platform.
