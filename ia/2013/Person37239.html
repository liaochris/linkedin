Douglas Graves' Summary

Throughout my career, I've filled many different roles, including Web Developer / Designer, Game Developer / Designer / Producer, UX Designer, Engineering Manager, acting Creative Director, and Inventor. Now I can finally officially add "entrepreneur" to that list, which is extremely exciting!

Over the last few years, I've been flirting with pursuing my own endeavors full-time, and recently committed to making those flirtations a reality. As such, this year I've been focused on designing lo-fi game prototypes and building digital versions of them with Unity 3D—a powerful game engine with simple-to-use editing and WYSIWYG tools that makes porting to multiple platforms a breeze. In addition to developing our own IP, we also offer contract game design and development, game writing, and traditional media gamification services.

Historically, I also have a great deal of experience in the web design and development arenas, in which the most exciting projects to me involve building prototypes, single page web applications, and client-heavy apps that only interact with the rest of the stack via api calls. A keenness for invention and creative thinking is why I do what I do, and my aptitude for such is why I excel at them. I also find an odd pleasure in reinventing and restoring old, broken down sites to modern glory. Through necessity, over the years I've also become rather adept at fixing broken workflow and organizational processes, so I can lend a helping hand there too if you need it.

My skill specialties in order of interest include: Game development, Game design, Web Application Development, Mobile Web Application Development, HTML5 game development, Web development, Mobile web development, Leading multidiscipline agile / lean teams.

My technology specialties in order of interest include: Unity 3D, JavaScript, HTML5, CSS3, jQuery/Zepto, node.js (meteor, express, mongoose, & socket.io), MongoDB, Django, Python, WordPress.
