Matt Schulz's Summary

Veteran journalist. Award-winning multimedia storyteller and strategist. Experienced leader. Aspiring author.

I pride myself in my versatility. In two decades in journalism -- with newspapers, websites and TV stations -- I've had the good fortune to do a little bit of everything, and that's exactly the way I like it. I can turn marginal copy into a great story. I can write an irresistible newsletter headline. I can talk SEO. I can code in HTML and CSS and use most any blog software you've got. I can script, shoot and edit compelling video content for the web. I can produce a live satellite interview. I've even helped run an online town hall with the White House. And I can't wait to see what tomorrow holds.
