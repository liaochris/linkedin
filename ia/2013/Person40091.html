After the roadmap, I focused on the graphic design of the app, and coded the mobile server API backend (PHP5), before starting the actual coding of the actual iOS app (native Objective-C).

The native iOS app had advanced features, such as full offline capabilities, content data caching/preloading,device synchronization and state sharing, as well as full multi-language (English/Chinese/Hebrew/Greek) rendering via CoreText and a custom XML parser.

Additionally, the app has extensive social features (standard Facebook/Twitter sharing, as well as Facebook OpenGraph Activity posting. This was build upon our own implementation of "friends" and "badges", as well as location-based content "check-in" gamification.

This was all implemented on the backend using PHP5 and Kohana Framework, talking to multiple MySQL and MongoDB databases, (MongoDB used for the Activity feeds and API logs).

The app was successfully code and beta tested with an online team of 100+ selected users (using an Apple Enterprise Developer account), then pushed, accepted and released in the Apple AppStore.
