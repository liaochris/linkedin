Alec Muffett's Summary

A globally recognised security evangelist, architect and software engineer, focused on security technologies, systems design, architecture and performance. Well versed in the security challenges faced by major multinational Technology, Telecoms, and Financial Services organisations.

Adept in public speaking, mentoring, education, communication and social media technologies. Pragmatic, with a proven track record of security innovation in areas including security software development, cryptographic optimisation, systems integration, access control, host and network intrusion detection, vulnerability scanning, systems hardening and security incident management.

Author of several open-source security projects including the Crack unix password cracker, and CrackLib password-integrity testing library.
