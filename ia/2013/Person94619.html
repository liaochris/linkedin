1) Activator – powers the inferred social graph at Y! which is used for syndication of public feed updates and connection suggestions

2) Address Book – stores contact info for applications across the Y! network

3) Aggregator – harvests feed activity from public sites like YouTube, Last.fm, and Twitter for re-distribution into the Yahoo! feed

4) Broadcaster – pushes Yahoo! feed updates to Facebook and Twitter

5) Credential Store – a secure repository for 3rd party credentials that power integrations like Facebook Connect

6) Display Widgets – lightweight social UI components that standardize, reduce deployment costs, and facilitate feature enhancements across the network

7) Social Directory – offers an API to create, store, and manage user identity and relationship information

8) Social Messaging & Notifications – allows users to message/invite each other without an email address

9) Update Feeds – Yahoo!’s open newsfeed platform

Many of our capabilities are offered in widget and API form to both Y! network properties and 3rd party developers via the Yahoo! Developer Network (example: http://yhoo.it/gamZ7q).
