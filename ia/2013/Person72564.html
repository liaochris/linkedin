Kasper Baagø Jensen's Summary

Hey there,

My name is Kasper Baagø, and I’ve been developing web applications for more then 8 years.
In the last couple of years I’ve been running a web company named Edentic, together with my co-owner Frederik Stahlfest. As team we help companies find the best web solutions for their needs.

Over the time I’ve done a bunch of different projects, from writing advance back-end stuff in various PHP frameworks to front-end development of WordPress themes using HTML5, JavaScript, SASS and Compass.
I use technologies such as GIT and Trello, to help me keep a good workflow when developing projects both for clients and myself.

When I’m not developing I like to read up on new web technologies.
Listening to podcasts while taking a long walk is one of my favourite activities. Get some fresh air, while listening to Chris Coyer in Shoptalk is just fantastic!

If you have an idea for an awesome project or a job you think my company can solve, please feel free to contact me using http://kasperbaago.dk/mail/ or @kasperbaago on Twitter.
You can read more about my projects and me at my website http://kasperbaago.dk.

Remember to check out my projects below, and if you’re a coder my GitHub page at http://github.com/kasperbaago

Kasper Baagø Jensen's Projects

  • Send et smil - Movia bus driver campaign

    • August 2013 to Present
    Team Members: Kasper Baagø Jensen, Frederik Stahlfest Møller

    In cooperation with Moose Creative, Edentic have made a responsive website for Movias campaign: "Send et smil"(Send a smile), where bus passengers send in there stories about great drivers.
    My job on the project was to develop the main frontend architecture and extend WordPress functionalities to receive SMS data from a SMS gateway.
    On the project we kept a tuned eye on the performance part, because we knew this website where gonna to be visited in places where slow connections exists.

  • Intelligent Headset

    • May 2013 to Present
    Team Members: Kasper Baagø Jensen, Frederik Stahlfest Møller

    In cooporation with the bureau Moose Creative, Edentic have developed a responsive website for a upcoming product from GN Store Nord A/S.
    My job on the project was to develop the frontend and extend WordPress functionality for the project needs. The result is a performance optimized site that works on all kinds of devices.
    On the project I've been working with technologies such as SASS, GIT and image optimization.

  • MyMicroframework

    • November 2012 to Present
    Team Members: Kasper Baagø Jensen

    MyMicroFramework is a PHP5 framework I’ve written for solving my school assignments.
    With great inspiration from other PHP frameworks out there, i’ve made a micro framework that consists of a very small codebase.
    Models can then be used to extend the framework in every possible way.
    The framework is free for all to use.

  • Infotavle

    • January 2012 to Present
    Team Members: Kasper Baagø Jensen

    Displays train departure information from the nearest train station.
    It uses the Google Geocoding API for location, and DSB Labs API for train information.
    The app shows the user a list of trains, with countdown time to next train departure. The user can also follow a train from station to station.
    jQuery mobile was used as JavaScript framework.
