  • Due Diligence Viewing rooms

    Team Members: Mark Wilde

  • Printer/Scanner uplift

    • 2011 to Present
    Team Members: Mark Wilde

    Challenge: Poor support from current Printer/Scanner provider. Nodes down for long periods of time and re-repairs common. Complaints about difficulty logging in (due to inaccessibility of keyboard). Fielding many requests for either personal scanners or printers at individual workstations.
    Solution: Switch from 7 “brand X” machines (manual login) to 8 “brand C” (with badge-in technology). Also included built-in faxing as a simple “value-added” arrangement. Setup training and documentation for all users to make conversion easier. Include training information in IT’s New Employee document.
    Results: All end users surveyed expressed high level of comfort using the machines. Service calls from new provider repaired 90% of issues within 1 day and provided solutions where that was not possible. Tracked usage increased by 30-50%. Became necessary to add 2 further brand C machines to handle additional volume. NOTE: While we considered this a success, we also were frustrated by the ease-of-use causing higher paper-waste. We were fortunate that the company’s recycling program is strong.

  • User State Migration - Win XP to Vista

    • 2007 to Present
    Team Members: Mark Wilde

    Solution: With such a large user base, and a varied environment, advise users to make backups of all important data. Setup test environment and pilot user base. Create script with most open parameters to capture as much user data and settings as possible. Test script using all 7 known versions of laptop and 3 versions of desktops.
    Results: Automatic upgrade performed seamlessly on almost all machines, with only a small percentage of those remaining requiring special handling. All data captured and transferred with a minimum amount of individual handling and saved countless hours of manual labor.
