Ed Parsons' Projects

  • European Association of Urology Guidelines

    • September 2011 to Present
    Team Members: Ed Parsons, Tom Madden

    The EAU had partially designed a web application to dramatically speed up their production of guidelines procedure. The project was started in-house but fell behind due to the complexity of what is required.
    I was brought in to the project and inherited what had been done to date but quickly realised the architecture of the system as designed, would not be able to support the features required.
    I advised the EAU that the best course of action would be for ekpSoftware to design a brand new system.
    The new system is still in development but has the majority of the features required and has been demonstrated internationally to a good reception. The system will go in to beta testing at the start of the New Year, well in advance of the previously expected date.

  • Warwick Business School iTutor App

    • August 2011 to Present
    Team Members: Ed Parsons, Nick Wake

    Dr Wake of the Warwick Business School had an idea for revision app for students on specific modules at the University. The business school did not have the expertise in-house to develop the solution, so ekpSoftware tendered and won the contract.
    I was tasked with designing and developing the solution from start to finish over a period of two months before the next academic year started.
    I set about creating a multi-platform mobile app that combined with a web based back end to engage both students and faculty to have an easy to use system. I used Agile development, which enabled the project to move fast and hit its short deadline.
    The app started on a pilot scheme in October 2011 and the result of which has been that the app is now being approved to move into other parts of the university.

  • PHPCalDav

    • February 2011 to June 2011
    Team Members: Ed Parsons, Marco van Beek

    PHPicalendar is an open-source web calendar package used to render ical files to a browser. However, the software is read only and does not allow the user to make any changes to the events.
    I was tasked with extending PHPicalendar into PHPcaldav so the software could render ical files from a caldav and push back edits.
    I built the software while maintaining as much of the original PHPicalendar code as possible.
    The result has been that PHPcaldav is now used on a number of sites.

  • LogLogic Real-Time Dash

    • May 2010 to July 2010
    Team Members: Ed Parsons, Andrew Morris

    While attempting to win a contract with Deutsch Bank, LogLogic were asked by the Customer if their software could run a real-time dashboard application on top of their appliances. LogLogic had developed an in-house web services API but didn't have the expertise to utilise it for a real-time dashboard.
    Andrew Morris of LogLogic came to ekpSoftware and asked if a prototype real-time dashboard using their web services API was possible. I took on the project. As this would be the first application to use this API I had to work through bugs and deal with poor documentation.
    I made the decision to code the project in C# as this was the language that LogLogic engineers were used to. This meant I had to learn both the API and C# languages in a very short space of time.
    The real-time dashboard prototype was built within the time frame that enabled LogLogic to win the business.

  • Huntzz Android App

    • October 2011 to November 2011
    Team Members: Ed Parsons

    Huntzz Ltd have developed a successful iOS application that includes advanced features such as in-app payments and location based services.
    Huntzz Ltd wanted to expand into the Android market place but did not have the expertise in-house to undertake this development.
    ekpSoftware were tasked to develop a comparable Android application for Huntzz Ltd.
    I used my previous experience with Android Development to implement and develope the application. This involved extensive Java, Android API and Google API skills.
    The result is that at Huntzz Ltd's first major partner release, the company will be able to offer their application on the two leading mobile platforms.
