Joe Moon's Summary

I am a software developer with combat experience and a formal education in literature.

As a developer for AppFog, a cloud platform-as-a-service company, I contributed features to our command line tool written in Ruby, but spent most of my development time writing JavaScript for the browser and server. I contributed features, bug fixes, and unit tests to our web console and our API service. I particularly enjoy the functional aspects of JavaScript.

As a support engineer, I handled a broad range of technologies up and down the stack while troubleshooting both user applications and our cloud platform. I was rigorous about looking for the most effective solutions, e.g. I rewrote all of our user documentation content to better address users' needs, reorganized the content from a collection of articles to a set of narrative guides, and built a documentation collaboration system based around GitHub to allow our developers and our users to contribute.

I struggle with the implications of Dunning–Kruger.
