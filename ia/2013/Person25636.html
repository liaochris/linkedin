Colin Watson's Publications

  • OWASP Cornucopia - Ecommerce Web Application Edition

    • Open Web Application Security Project (OWASP)
    • 2012
    Authors: Colin Watson

    A card game for software architects, designers and developers to help derive application security requirements for e-commerce websites. The cards provide cross-referencing between threats and the OWASP "Secure Coding Practices - Quick Reference Guide", the OWASP "Application Security Verification Standard (ASVS) for Web Applications 2009", the OWASP "AppSensor Project", Mitre "Common Attack Pattern Enumeration and Classification (CAPEC)", and the SAFECode "Practical Security Stories and Security Tasks for Agile Development Environments". Initialy based on Microsoft's Escalation of Privilege (EoP) threat modelling card game.

  • (Application Security) Logging Cheat Sheet

    • Open Web Application Security Project (OWASP)
    • 2012
    Authors: Colin Watson

    Concentrated guidance for software developers on building application logging mechanisms, especially related to security logging.

  • Creating Attack-Aware Software Applications with Real-Time Defenses

    • CrossTalk The Journal of Defense Software Engineering
    • 2011
    Authors: Colin Watson, Michael Coates, John Melton, Dennis Groves

    Attack-aware software applications provide attack detection and real-time defensive response with a very low false-positive rate. This technique allows an application to detect and neutralize a threat before the attacker exploits a known or unknown vulnerability. The approach is especially suited to software applications with high information assurance requirements such as in the defense, critical national infrastructure, and financial service sectors to protect against cyber espionage, fraud, business logic abuse, tampering, and theft. The Open Web Application Security Project (OWASP) has developed a methodology, documentation, code and pilot demonstration which can be freely used to apply
    the concepts; this project is called AppSensor.

  • Attack Detection and Response with OWASP AppSensor - An Implementation Planning Workbook

    • Open Web Application Security Project (OWASP)
    • 2011
    Authors: Colin Watson

    Guidebook on planning the implementation of application-specific real-time attack detection and response. Based around the OWASP AppSensor Project.

  • CobiT Security Baseline Applied to Business Web Applications

    • ISACA Journal
    • 2009
    Authors: Colin Watson

    CobiT Security Baseline can be used by organisations to make a significant—methodical and verifiable—difference in the security of their web applications, by assisting the selection and implementatioon of appropriate security controls.
