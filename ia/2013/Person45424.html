Florent Pillet's Projects

  • NSLogger

    • October 2010 to Present
    Team Members: Florent Pillet

    A popular logging framework for iOS, OS X and Android applications & Mac desktop viewer that allows apps so send tagged logs, binary data & images to the desktop viewer. Live remote browsing, high throughput support, log filtering and tagging are among the most popular features of NSLogger.

    This solution is highly valued among developers of complex applications.

  • CommandFusion iViewer

    • July 2008 to Present
    Team Members: Florent Pillet

    iViewer is a powerful automation control product for iOS and Android, allowing automation integrators and DIY enthusiasts to remotely control any networked device. Based on the CommandFusion Core Automation and User Interface Engine I developed, iViewer allows the development of powerful and beautiful user interfaces with very few or no code. A rich API is available to expand and completely drive the product from onboard JavaScript code.
