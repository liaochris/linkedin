  • AR-PostOP

    • October 2013 to Present
    Team Members: Alexander French

    AR-PostOp is an unobtrusive "augmented reality surgical simulator that presents 3D reconstructed scenes from laproscopic video of surgical procedures"[Johnson 2013] The simulator was developed as a method to reliably test depth perception in 3D environments. AR-PostOp was developed inside the Unity game engine and uses the H3D-api to allow for user interaction via a haptic device (Phantom Omni).

    For this project I was tasked with following objectives:
    1) Develop code to bridge the Phantom Omni with the Unity game engine.
    2) Develop a system which would measure the distance from the user's in-game tool to the closest reconstructed organ or surface tissue.
    3) Integrate and test surgical environment with Oculus Rift.
    4) Develop game system to properly control user interaction with environment.
    5) Demonstrate and test system against participants in the engineering and Medical fields.
    6) Conduct post-participation surveys.
    7) Collect and organize data from AR environment and hand it off to graduate student for data analysis.
    8) Proof read final paper for publication in conference proceedings.
