Jose Galdamez's Summary

I'm a firm believer in building applications that solve real business needs while providing an optimal user experience. I currently do front-end Web development, server-side application development, database design, and database programming.

I have fourteen years of experience working with front-end technologies such as HTML, CSS, and JavaScript. I have ten years of experience working with ColdFusion, SQL Server, and Oracle on the back-end.

My goals for 2013 include learning mobile development (iOS, Android, and PhoneGap), building responsive Web sites, learning a new JavaScript framework (AngularJS, CoffeeScript, Ember.js, or Backbone.js) and writing applications that utilize up and coming HTML5 technologies.

I also teach an introductory JavaScript course at Montgomery College, a community college based out of Rockville, MD.

Specialties: ColdFusion 5-10, SQL Server 2005/2008/2012, Oracle 10g, HTML 4, XHTML 1.0, CSS2/CSS3, JavaScript, jQuery
