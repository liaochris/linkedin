Curtis Muir's Projects

  • SSO Integration of Ping Identity Federation with .Net Applications

    • December 2012 to Present
    Team Members: Curtis Muir, Prodip K. Saha, MBA

    SSO Integration of Ping Identity Federation with .Net Applications is a real-world proof of concept project. Ping thoroughly documented the product online. I was able to download Ping Federation, install and configure it for IdP and SP roles. Downloaded the .Net Integration Kit to integrated with ASP.Net application via plug and play IHttpModule. .Net Integration Kit with OpenToken provided greater control. It was easy to take SAML 2.0 token and build RolePrincipal with FormIdentity to connect the dots to existing custom Membership/Role/Profile providers.
