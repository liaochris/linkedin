    • Involved in check in, check out and preparing branches, tags using SVN.
    • Involved in writing configuration files [XML].
    • Involved in injecting dependencies into code using concepts like IOC of Spring Framework
    • The Database Persistence is managed with JPA, Entity mappings for each database table is contained
    as Pojo objects and managed with the Spring JPA modules through standard Spring Bean Mappings.
    (applicationContext.xml)
    • Designed and developed web pages and functionality in JAMS module
