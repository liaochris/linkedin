Curtis Farnham's Summary

I'm a freelance software engineer focusing on web-based applications, backends, and API's. My interest started as a hobby around the year 2000. In 2006 I woke up and realized this was my real passion. I've been doing it more or less full-time since then.

I have 12+ years of experience writing custom web applications and working with open-source CMS's, frameworks, and libraries. I'm also fluent with the Linux command line, having installed and configured numerous Linux distributions and servers. My key technology strengths include PHP, Zend Framework, Java EE, Objective-C (iOS), MySQL, JavaScript, jQuery, Sencha's ExtJS, HTML, and CSS.

I have a passion for User Experience design. Apple's UX efforts are a huge influence on me. I strive to make my user interfaces as simple as possible, but no simpler. In all of my development efforts, I strive to write code that is readable, testable, and maintainable. Messy code is broken code.
