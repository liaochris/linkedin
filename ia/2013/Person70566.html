Julian Buss' Summary

I am a consultant, software-engineer and project manager for mobile and IBM software based in Hamburg, Germany.

Since the year 2000 I helped many customers to solve their business problems with software. Either with individual or standard products. My focus is in the mobile, web application and IBM software space.

Together with YouAtNotes (www.youatnotes.com) I created various standard software products for the enterprise, among them complex and full flavored CRM and Workflow software. A recent software product development is a mobile development environment for creating native mobile Apps as front end for IBM Notes applications.

I have a broad skill set for all things software development for both project management and software engineering. In addition, I successfully worked in manager positions including staff and budget responsibility.

In the IBM community I'm well known for my expertise in IBM XPages and IBM Notes application development, including designing beautiful and easy-to-use user interfaces, performance optimization and application security.

In the mobile space I am an expert in mobile web applications and Appcelerator Titanium based native apps.

You can hire me on an hourly or daily basis. Just send me an email to jbuss@julianbuss.net.

  • Developing an XPages Custom Control to Make Filtering a Categorized View Easy

    • THE VIEW
    • April 30, 2013
    Authors: Julian Buss

    Imagine a view with thousands of categories and tens of thousands of documents. If you display this view with the standard view control on an XPage, users will complain, because unlike in Notes, they cannot simply type some characters to jump to a specific category. Instead, they have to use the pager to navigate to a category, which takes a lot of clicks if that category is buried deep down in the view. And having categories with subcategories makes this problem even worse.

    This article explains how to develop a custom control that displays all categories and subcategories in a tree. This way a user can easily select one category that will allow the view to show documents for only that category. Your users will love it.

    You also will learn how to use a Dijit (a Dojo-based UI control) Dialog, embed a Dijit Tree Control in such a dialog, and create a backend XPage to feed data to that control.
