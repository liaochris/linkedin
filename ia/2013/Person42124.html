Erich Lin's Summary

Erich (Hung-Che) is a new media designer with experience in web and broadcast mediums; specializing in creative concept development, UI design for Flash&Silverlight, illustration, and standard HTML/CSS website.

Fluency in best practices for web-based information architecture and design.

He works with developer team very closely, and created several high profile websites such as Microsoft Desktop Player, Microsoft UC, and SharePoint.com

Specialties

User interface design, creative concept development, pixel art, web standard CSS, user-centered design, wireframing and prototyping.

Extensive experience in Silverlight, WPF, Flash,
visual Design, and motion graphics.
