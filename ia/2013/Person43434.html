Evan Savage's Patents

  • Differentiating Between Good and Bad Content in a User-Provided Content System

    • United States Patent 12/946,731
    • Issued November 15, 2010
    Inventors: Evan Savage, Christopher Stein, Lev Popov

    This patent describes a system for real-time spam detection.

    The system first detects anomalous content by applying an online rolling exponential mean/variance algorithm to extracted feature counts (links, link domains, subnets, etc.)

    It then collects user feedback to distinguish between good and bad anomalous content. Once enough feedback is collected, the feature with anomalous count is either whitelisted or blacklisted depending on the aggregate results.

Evan Savage's Projects

  • XHPy: XHP for Python

    • January 2011 to Present
    Team Members: Evan Savage

    XHPy extends Python syntax such that XML document fragments become valid Python expressions. It is based off XHP, a similar framework for PHP.

    Under the covers, XHPy uses a PEP 302-style import hook together with a hand-rolled streaming Pratt parser to dynamically rewrite code as it is imported. This probably earns it a place among the most terribly awesome Python hacks of all time.
