Gilderlan Braz's Summary

Uma das minhas grandes paixões é a filosofia, passei minha adolescência lendo Schopenhauer, Nietzsche e outros. A forma como esses pensadores estruturam um problema para chegar na solução mais lógica me fascina. Acredite!   • Mailer

    • outubro de 2011 até novembro de 2011
    Membros da equipe: Gilderlan Braz, Luciano Bezerra

    Mailer é um sistema de envio de emails e mensagens sms. A partir de informações coletadas na base de dados do ERP do Grupo Tubform, o Sistema gera relatórios e os envia periodicamente para diretores, gerentes e representantes.

    Dados técnicos:
    O Sistema foi desenvolvido com JRuby. Usamos Ruby Oci8 para conexão com o banco de dados e ActiveRecord para mapeamento de tabelas. Para o envio de email usamos a lib ActionMailer, já as mensagens sms são enviados através de uma API HTTP.
