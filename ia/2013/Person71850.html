Software Researcher at RECONF 2 IST project

Universitat Politècnica de Catalunya

Educational Institution; 1001-5000 employees; Research industry

February 2003 – December 2004 (1 year 11 months) Barcelona Area, Spain

European Project (IST–2001–341016):
The project aimed to build a full-product solution for the dynamically reconfigurable FPGA devices. I developed the Front-End CAD tools supporting dynamic reconfiguration in FPGA devices. The tools, written in C++ with MFC libraries include:
1. Manual Partitioning Tool (drag and drop support for the user to easy split VHDL processes from the input files to the output “dynamic modules”),
2. Configuration Controller Generator (a VHDL code generator ensuring correctness of the design after the partitioning process)
3. VHDL-Post Processing Tool (functional simulation of dynamically partitioned design).

For a detailed description, please refer to the research section at my homepage ( http://personals.ac.upc.edu/kkedzier/research/reconf.html ).
