Don Gilmore's Summary

Software Architect, Designer, Developer, available for remote work on beneficial projects. I still love the excitement and tabula rasa creativity of startup opportunities.

I've designed and built many complex software systems where primary requirements included optimized human interaction, fail-safe quality and high performance.

Specialties: I could honestly list hundreds of tags from http://stackoverflow.com/tags and I know these control search, but at this stage in my career, search me for experience and conscientiousness.

OK, I'll just mention a bit about what I've been doing recently: web app and mobile device development using modern frameworks and tools, including Laravel 4, FuelPHP, HTML5, CSS3, jQuery Mobile, ORM-style database interfaces, lots of experience with UX design, graphics, videos, Objective-C, Java, PHP, etc. (I've used over 20 languages).

Don Gilmore's Experience

Independent Consultant

Self

In the context of software, describe requirements and I can design and build a system. Show me a system and I can find flaws and suggest improvements. However, I don't have a clue how to build artificial consciousness, and this keeps me humble.

Founder

Aubjex

2000 – 2007 (7 years) Maui, Hawaii

Co-founded Aubjex. We built the Aubjex adk to be a superior platform for Java development and distributed data visualization. Used the adk to build the most full-featured Java IDE in existence, prior to Eclipse. It analyzed, navigated, edited, refactored and managed Java code.

VP Software Engineering

Cardiopulmonary Corp

Privately Held; 11-50 employees; Hospital & Health Care industry

1991 – 1999 (8 years) Massachusetts and Connecticut

Re-invented ventilation. Startup company's first product was an ICU life support ventilator and patient monitor. Designed and wrote all the touchscreen user interface software in Objective-C using a NeXTStep platform, and most of the embedded software in C on an attached processor. See US Patent 5,931,160. This life support medical device was accepted by the FDA, the European Union, and the German TÜV. Many original ideas are now being copied by others.

Ease of Use Project Manager

Cognex

Public Company; 1001-5000 employees; CGNX; Industrial Automation industry

1990 – 1991 (1 year)

Led an advanced R&D project to make it easy for customers to program the Cognex vision processor. This processor was a world-leading tool for visual inspection of manufactured parts, but was difficult to program.

Systems Architect

ON Technology

Public Company; 201-500 employees; ONTC; Computer Software industry

1988 – 1989 (1 year)

"ON's original vision is known today as the web." I was one of the seven original architects for this high profile startup that was out to change the software world by creating a pure object oriented software stack from the hardware to the future world wide web, including a new language and virtual machine. My specialty was globally distributed persistent objects with peer-to-peer connectivity, permanent identifiers, digital rights.

Database Group Manager

Aries Technology

Privately Held; 51-200 employees; E-Learning industry

1984 – 1987 (3 years)

Designed Artemis, a high performance persistent memory database for the ConceptStation, a 3-D mechanical CAD product. Programmed in C and my own language, like Objective-C. Included schema evolution utilities, data transfer of design objects between workstations, revision control of designs.
