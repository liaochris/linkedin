Christopher Burke's Projects

  • Visual Information Extraction through the use of Data Mining

    • September 2011 to March 2012
    Team Members: Christopher Burke

    For my final year thesis at Bangor University I was approached to look at the problem of shirt colour associated with team performance as an extension of Nature 435: 293. To determine the answer to this hypothesis I created a bespoke solution that used data mining techniques to find patterns in large data sets to retrieve the team data, and information extraction and natural language processing techniques to retrieve the data and place it into a data structure allowing for the data set to be extended by both further data mining and information extraction, and by a custom image query API. I was awarded a grade of 77% for this work, and my solution (and part of my thesis) has been used in post-doctoral research, which has been submitted to Nature.

  • Poseidon (Video Game)

    • September 2011 to March 2012
    Team Members: Christopher Burke

    Poseidon is a fully 3D video game written in C++ using OpenGL. I have used no third-party game libraries to assist me; instead have done everything from the ground up. Some of the features in this video game that secured me top marks are: the procedural generation of a large realistic looking random terrain, linear fog, lighting controls, collision detection, alpha-blended water effects and Quake MD2 modelled enemies. For my initial demo hand in I received top marks in the school (96%) and my final hand in also achieved top marks (89%)

  • crossproduct (HTML5/WebSockets Game Server)

    • January 2013 to Present
    Team Members: Christopher Burke

    Crossproduct is a game server written in Java using Vertx.io to serve the HTML5 game, dotproduct, which is written in JavaScript using the Google Closure project. Crossproduct is completely open source, licensed under the GPL2 and is available on GitHub.
