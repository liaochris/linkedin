Daniel AG's Summary

I am working as a SQL Server 2012/2008/2005 Production DBA & Team Lead for a custom application development team of SSRS report developer, .NET developer and SharePoint 2010 administrator. Work with management, developers, end users and software vendors to implement best practice based production promotion processes.

Responsible for 24x7 technical support of 132 instances of 64/32 bit SQL Server 2012/2008/2005/2000 Enterprise/Standard infrastructure (Physical & Hyper-V) with Multi-Node Clusters (Active/Passive),SQL Server Replication, Database Mirroring, Log Shipping, IBM SAN solutions, DPM backup, SCOM Monitoring.

Business intelligence development using SSRS, SSIS, SSAS & MDX - More than 300 SSRS reports in SharePoint 2010 integration mode - SSAS solution for grade analysis - More than 40 SSIS data feeds between internal application databases and state education department.

Completed several large T-SQL programming projects for State education departments using stored procedures, views, triggers.

Successfully Upgraded (in place & side by side) SQL Server 2000/2005 to 2008 R2, SSRS 2005 to SSRS 2008 R2, DTS to SSIS.

Performance monitoring/tuning SQL Server, TSQL, ETL process, & SSRS. - Analysis Disk, CPU & Memory bottle neck using SCOM, SQL Profiler, Perfmon, Resmon, DMV, DMF & PAL. - Totally automated SQL Server index maintenance and statistics update process through SQL Agent.

Implementation of comprehensive disaster recovery solution. – Totally automated SQL Server native backup through SQL Agent & tape backup using DPM. – Full/differential/Transactional backup for critical systems.

I have a passion for teaching & I also do part time training for those who want to be a DBA OR Database Programmer OR Business Intelligence Developer. I trained more than 500 students so far.
