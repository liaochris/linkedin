Bruno Nery's Summary

Brazilian, nomad. Born in Jundiaí, lived in Campo Limpo Paulista, Mogi-Mirim, Socorro, Itapira, Mogi-Guaçu, Campinas, São Carlos, Paris, Lisbon, Zürich - now lives in San Francisco.

Studied Informatics at COTUCA (Unicamp) and studied Computer Engineering at EESC (USP) in São Carlos; studied Robotics and Embedded Systems at ENSTA and Artificial Intelligence and Decision at UPMC (Paris VI), in Paris. Recently was a PhD candidate at ISR/IST, in Lisbon.

Worked as a software developer (UNISOFT, Software Design and ICMC/USP), as a project manager (UNISOFT), as a system administrator (LABIC-ICMC/USP), as a researcher (LABIC-ICMC/USP and ISR/IST), as a development engineer (Skysoft), as an adjunct professor (ESML/IPL), as a software engineer intern (Google), and as a software engineer (Thousand Eyes). Now works as a software engineer/site reliability engineer (Google).

Interested in artificial intelligence, bio-inspired computing, robotics, embedded systems and distributed systems, open source software, software development, system administration, web development and entrepreneurship opportunities.

Brazilian Portuguese mother tongue, proficient in English and French, understand Spanish, basic notions of German. Fluent in C, C++, git, LaTeX, Linux administration, Network analysis, Python. Familiar with Ant Colony Optimization, ARP, ASN.1, Automake, Automatic code generation, Bash script, CMake, Code review process, CSV, Ethernet, Fuzzy systems, Gentoo Linux optimization, Haskell, libpcap, Object-Oriented Programming, openMosix, Probabilistic Robotics, PyUnit, Relational databases, ROS, SNMP, SQL, SVN, Testing techniques. Acquainted with Ant, BASIC, Compiler optimization, Delphi, Google Chrome plugins, HTML, Java, JSP, Ladder, Linux kernel hacking, Lucid Synchrone, Lustre, Noise reduction techniques, PASCAL, PHP, Rails, Ruby, Scheme, Tomcat, URBI, Visual Basic, VxWorks, Weka, Wordpress, x86 Assembly, XML.
