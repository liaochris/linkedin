Instant Access Technologies

Privately Held; 11-50 employees; Internet industry

August 2012 – Present (1 year) London, United Kingdom

IAT Ltd is a technology company that specialises in creating and monetising online user bases. In August 2012 IAT initiated a project to upgrade an existing in-house e-commerce product search platform. This platform consumed affiliate network feeds and exposed products to end-users via a series of on-line shops and portals. The upgrade rapidly transformed into a ‘Greenfield’ rebuild project, as initial analysis revealed fundamental limitations with the existing platform architecture.

• Recommended entire technology stack for the project, including application frameworks, code quality tools (Sonar), build management (Maven), CI (Jenkins) and deployment platforms (AWS EC2).
• Worked with a team of consultants (both UK-based and ‘near-shore’) to design an architecture that supported new requirements generated by the business. Implemented components within the project as SOA-based modules that communicated via REST-like services.
• Primarily responsible for creating new affiliate feed cache component, which extracted products from third-party affiliate feeds on a scheduled basis and created versioned deltas for consumption by the search component. Utilised Spring Core, MVC and Data combined with MySQL and MongoDB.
• Fully responsible for creating the product search component using Spring, MySQL and Solr 4. Implemented multifaceted pre-index pipeline in which third-party products were mapped to a consolidated ‘IAT Product’ (using Dozer API) and enriched by entity extraction (using Lingpipe and java.util.regex.Matcher).
• Utilised TDD methodology with JUnit, Mockito and PowerMock in combination with integration tests run via Jetty, H2, SDFongo (an in-memory Mongo server forked and modified from GitHub) and a bespoke InProcessSolrServer component (an in-memory Solr Server).
• Provisioned EC2 instances using Puppet, and deployed application components as RPMs.
