Debra Willrett's Summary

Debra designed and developed the award-winning project management application MacProject which was awarded the Readers Choice Software of the Year Award for many years running by MacWorld magazine, and was the first third-party application to be release for the Mac. MacProject was the follow on product to Debra's LisaProject, the first graphical project management software, which was distributed on Apple’s Lisa.

Debra designs and builds web applications for medical image processing at Stanford Medicine in Java and JavaScript. She is an experienced technical lead in web-based and desktop applications development with project management skills, such as assigning tasks, monitoring progress, resolving conflicts and team building.

As a skilled user of web tools for communicating specifications and for supporting collaboration among geographically dispersed team members, she interfaces with management, developers, and customers. Over 25 years software development experience in all phases of product development, from conception through multiple releases to the market on many different computing platforms. Specialities include: jQuery, HTML5, CSS3, SQL, C, JavaScript, Java, GWT, OOP, MVP and MVC, CMS platforms such as Drupal, as well as scripting languages such as PHP.

She is the technical lead for the ePAD web application development project, with key involvement in all phases of design, development, and implementation of the application. She created ePAD web front-end design using mockups with Balsamiq and collected user feedback. She defined ePAD architecture and features based on requests, customer input, and experience in GUI design, and directs local and remote software developers and other team members, consultants and collaborators.

Debra addressed critical performance issues with a custom image compression scheme for DICOM data. She resolved issues in server-side web services for XML database by providing web services for native XML database for our AIM documents. Debra created a customized JavaScript version of RaphaelJS to meet ePAD’s particular graphical requirements for drawing annotations on DICOM images, and re-architected the interface between ePAD and backend server processes to resolve reliability issues. She created comprehensive client and server side plug-in architecture.

Debra created the ePAD wiki to organize project communications, including ePAD specifications, task assignments, schedules, and dialog among the geographically dispersed team members. She designs new features – create mockups, interview users, get consensus, write specifications and supervises collaborators and students who contribute software for modules to ePAD and coordinate resolution of engineering design issues. Debra uses object oriented programming in Java/ GWT, JavaScript, jQuery, CSS3, JSNI using open source libraries and tools such as RaphaelJS in JavaScript, for SVG and WebGL
