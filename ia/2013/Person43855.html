Fabiano Franz's Summary

Software engineer, developer and team leader with 10+ years experience in Java and web related technologies. Focusing on being a full-stack web engineer, which means being able to develop all layers of a web project (from the frontend to the backend).

Senior Software Engineer at Red Hat OpenShift PaaS, working on spare time at Coolendar (http://coolendar.com), a personal project which passed 30,000 registered users, all developed by me.

Some previous personal and open source projects are: ThinFeeder Java RSS Aggregator (creator and developer), JImageTaglib (creator and developer), GreenBox - Code Generation Framework for NetBeans (developer), literar.org web 2.0 portal (creator and developer) and pribi.com.br portal (creator and contributor). Participant on the NetBeans translation team.

Sun Certified Associate for the Java Platform.
Scrum Master (trained by Boris Gloger).

Java (SE, EE, ME), JSP/Servlets, Struts (1 and 2), Hibernate, JPA, EJB 2 and 3, Spring, Grails, Play!Framework, JSF (Java Server Faces), Design Patterns, Swing, Thinlet, MIDP, tag libraries, Lucene, Solr, Velocity, XDoclet, Ant.
