Dave Quick's Patents

  • Trimmed And Merged Search Result Sets In A Versioned Data Environment

    • United States Patent 7765195
    • Issued July 27, 2010
    Inventors: Dave Quick, Bill Griffin, Pat Miller, Bruce Jones, Naresh Kannan

    In corporate content management systems, there are different versions of the same document authored by the users to be potentially viewed. Access to unpublished documents is restrictive, yet multiple users may want to retrieve and view them. The invention facilitates trimmed, merged and ordered search results based on an user and user access rights. The method automatically impersonates the user with varying access rights and executes successive queries using the varying user rights to retrieve both published and unpublished documents. In the multi-version document storage environment, corresponding user accounts are automatically generated and searches automatically conducted from each of these accounts against the data store for the query content. A predetermined sort order is employed to maintain results ordering when merging into the final results set. The claims are directed towards searching and merging search results based on a user having different access rights.

  • Output Cache Manager

    • United States Patent 7660950
    • Issued February 9, 2010
    Inventors: Dave Quick, Pat Miller

    Dynamically generated web pages are easier to maintain than static web pages, but require more processing to render. Output cache buffers allow previously generated dynamic web pages to be served thus saving the processing to regenerate the web page. However, maintaining current output cache buffers require laboriously modifying directives associated with individual web pages. The invention addresses this problem via cache profiles that store directives for sets of web pages rather than individual web pages. Additionally, profiles may inherit properties from other profiles. Thus when a single profile is modified, all associated web pages are modified.

  • Social Distance Based Search Result Order Adjustment

    • United States Patent Application US-2011-0087661-A1
    • Filed April 14, 2011
    Inventors: Dave Quick, Alex Mallet, Sinan Ingec, David Nichols, Nikolai Smolyanskiy, Mike Mroch, Mohamed Battisha

    Both search criteria and a friends list associated with a user are obtained. One or more data items that satisfy the search criteria are identified as search results. An order of the search results is adjusted based at least in part on multiple additional friends lists, each of which is associated with a data item of the one or more data items. Each of the multiple additional friends lists can also be updated to reflect changes in the friends lists made by users associated with the friends lists.
