Cesar Alvernaz's Projects

  • NINJA Framework

    Team Members: Cesar Alvernaz, Paulo Pires, João Miguel Pereira, Miguel Figueiredo

    New Infrastructure for Network management systems Java Applications (NINJA) is a proprietary model-driven (MDA) framework that aims to fasten module development for TNMS, the Nokia Siemens Networks NMS suite for transport networks.
    I was responsible for architecture design, implementation, testing and documenting the entire project which consisted mainly in code generation based on UML models and tooling that would enforce best-practices. We targeted at simplifying and speed-up the development process for TNMS modules.

    Relevant used technologies include Java EE (EJB, JMS/MDB, JMX), Spring, Hibernate, JBoss AS, Junit, Ant and AndroMDA.
