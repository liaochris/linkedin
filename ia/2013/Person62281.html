Jeremy Andrus' Publications

  • The Design, Implementation, and Evaluation of Cells: A Virtual Smartphone Architecture

    • ACM Transactions on Computer Systems (TOCS)
    • August 2012
    Authors: Jeremy Andrus, Christoffer Dall, Oren Laadan, Alexander Van't Hof, Jason Nieh

    Smartphones are increasingly ubiquitous, and many users carry multiple phones to accommodate work, personal, and geographic mobility needs. We present Cells, a virtualization architecture for enabling multiple virtual smartphones to run simultaneously on the same physical cellphone in an isolated, secure manner. Cells introduces a usage model of having one foreground virtual phone and multiple background virtual phones. This model enables a new device namespace mechanism and novel device proxies that integrate with lightweight operating system virtualization to multiplex phone hardware across multiple virtual phones while providing native hardware device performance. Cells virtual phone features include fully accelerated 3D graphics, complete power management features, and full telephony functionality with separately assignable telephone numbers and caller ID support. We have implemented a prototype of Cells that supports multiple Android virtual phones on the same phone. Our performance results demonstrate that Cells imposes only modest runtime and memory overhead, works seamlessly across multiple hardware devices including Google Nexus 1 and Nexus S phones, and transparently runs Android applications at native speed without any modiﬁcations.

  • Teaching Operating Systems Using Android

    • Proceedings of the 43rd ACM Technical Symposium on Computer Science Education (SIGCSE 2012)
    • February 29, 2012
    Authors: Jeremy Andrus, Jason Nieh

    We introduce a series of ve Android kernel programming projects suitable for a one semester introductory operating systems course. Each project teaches a core operating system concept infused with Android or mobile device specific context, such as Android specific process relationships, use of sensors, and design considerations for resource constrained mobile devices. We also introduce an Android virtual laboratory based on virtual appliances, distributed version control, and live demonstrations which gives students hands-on Android experience, with minimal computing infrastructure.

  • Cells: A Virtual Mobile Smartphone Architecture

    • Proceedings of the 23rd ACM Symposium on Operating Systems Principles (SOSP 2011), Cascais, Portugal, October 23-26, 2011
    • October 23, 2011
    Authors: Jeremy Andrus, Christoffer Dall, Oren Laadan, Alexander Van't Hof, Jason Nieh

    Cells is an operating-systems based virtualization solution for mobile systems. It fully runs several instances of Android simultaneously with fully accelerated graphics and with native performance, wifi, audio, sensors, and all Android applications.
