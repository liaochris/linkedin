BugSplat Software

Privately Held; 11-50 employees; Computer Software industry

December 2010 – Present (3 years)

BugSplat is a leading provider of online cross-platform crash analysis services for the software industry.

BugSplat provides a comprehensive set of tools that enable you to track end-user crashes in your own products. With BugSplat, you get an easy to integrate library that monitors user crashes. In the event of a crash, your user is given the option to upload crash information to our online service. Your engineering team can browse all crash information at the BugSplat web site and has access to valuable information including a symbolic call stack that can be used to isolate the actual problem.
Online crash reporting gives you a wide set of tools to analyze trends and isolate the most common user crashes.
