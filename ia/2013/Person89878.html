Massimiliano Bigatti's Publications

  • E-commerce con PayPal

    • Apogeo Editore
    • 2008
    Authors: Massimiliano Bigatti

    Questo libro, partendo dalla spiegazione del funzionamento e dei motivi del successo di PayPal, si rivolge a chi, gestore di negozi di e-commerce o sviluppatore, vuole implementare una soluzione di pagamento su base PayPal. Si parte illustrando lo sviluppo di soluzioni standard, per poi introdurre l’uso delle API e l’implementazione di soluzioni sempre più complesse. Note, suggerimenti e appendici permettono di approfondire e affrontare le problematiche più comuni, come quelle connesse ai pagamenti multipli o ricorrenti, ai rimborsi e alla reportistica.

  • UML e Design Pattern

    • Hoepli Informatica
    • 2006
    Authors: Massimiliano Bigatti

    UML è una famiglia di notazioni grafiche che vengono utilizzate per descrivere e progettare sistemi software di qualsiasi tipo, con particolare indicazione per quelli orientati agli oggetti. Un Design pattern è una soluzione comprovata per un problema ricorrente di progettazione di software orientati agli oggetti. Questo testo presenta un esempio concreto di applicazione, che viene sviluppato utilizzando Design pattern e UML senza imporre al lettore un’eccessiva preparazione teorica.

  • Java e Open Source

    • Tecniche Nuove
    • 2005
    Authors: Massimiliano Bigatti

    Java e Open Source mostra come utilizzare componenti open source per realizzare un’applicazione web completa in Java. Mediante la realizzazione di un semplice weblog di esempio, il testo insegna a utilizzare strumenti e componenti open source liberamente scaricabili da Internet per abbreviare i tempi di sviluppo e dotare le proprie applicazioni delle funzionalità più avanzate.
    Vengono inoltre indicati gli elementi utili per costruire il proprio ambiente di lavoro Java di base, illustrandone l’installazione e il funzionamento. Tra gli strumenti utilizzati si segnalano: Eclipse IDE, Ant, JBoss Application Server, MySQL Database Server, Log4j, Hibernate, Castor, ObjectRelational Bridge, Struts, Mokapackages, Spring Framework.

  • Java Hacking e Sicurezza

    • Edizioni Master
    • 2005
    Authors: Massimiliano Bigatti

    Nell'era della rete, in cui tutti i dati passano attraverso internet, nell'epoca in cui ogni informazione viene in qualche modo gestita da calcolatori elettronici, la sicurezza del software assume una rilevanza straordinaria. Ancora di più scrivere codice basato su regole di sicurezza di affidabilità comprovata diventa una necessità quando si parla di Java, che per le sue caratteristiche di multipiattaforma, per le sue estensioni verso la rete, per la sua capacità di essere usato per scrivere applicazioni web come standalone costituisce uno dei bersagli preferiti per pirati di vario genere. Massimiliano Bigatti ci guida attraverso le estensioni del linguaggio mirate a realizzare applicazioni a prova di Hacker. Gli esempi pratici come le spiegazioni teoriche contenute nel libro costituiscono una base essenziale per sviluppare applicazioni "sicure".

  • Java e Multimedia

    Lo scopo di questo libro è quello di presentare le API Java Sound, Java2D e JMF, fornendo esempi di riproduzione, elaborazione, acquisizione, sia di clip sonori che di contenuti video.   • Da Visual Basic a C#

    • Edizioni Master
    • 2004
    Authors: Massimiliano Bigatti

    Con l'avvento di .NET, lo scenario per gli sviluppatori Visual Basic è cambiato. Ora, per sviluppare su Windows (e non solo?) c'è una nuova piattaforma, per tanti versi nuova e per altri simile a quanto realizzato in passato. In questo ambito, si apre anche l'opportunità di cambiare linguaggio di programmazione: evolvere a VB.NET o passare al più potente C#.

    Lo scopo del libro è quello di insegnare, per differenze, a sviluppare con il linguaggio C# e con la piattaforma .NET, a sviluppatori Visual Basic 6.0. Non è necessario essere esperti di VB6, e non si diventerà esperti di C#. Si apprenderanno piuttosto le basi del linguaggio e come sviluppare applicazioni a finestre e servizi web.

    L'apprendimento per differenze è solitamente più rapido, e spesso le nuove tecnologie non sono poi così diverse da quelle del passato; alcune volte cambiano i nomi, oppure i concetti subiscono una certa evoluzione e, quindi, imparare per differenze aiuta a risparmiare tempo e fatica. Il passaggio da Visual Basic a C# non è semplice né indolore, richiede un cambio di mentalità, per passare da un linguaggio (principalmente) procedurale per Windows a uno orientato agli oggetti per una piattaforma diversa dal sistema operativo di Microsoft.

    Questo testo cerca di fornire un percorso guidato attraverso le novità di C# in modo da presentare queste diversità in modo comprensibile e con il linguaggio a cui il programmatore Visual Basic è abituato.

  • Java Web Application

    • Edizioni Master
    • 2003
    Authors: Massimiliano Bigatti

    L'avvento del World Wide Web non solo ha rivoluzionato il modo in cui le persone comunicano, si incontrano, fanno affari, ma ha anche cambiato, radicalmente, l'approccio allo sviluppo applicativo. Se prima veniva privilegiato lo sviluppo di applicazioni dalla ricca interfaccia utente, magari monolitiche e con una forte presenza sul client, l'affermazione di Internet è riuscita a promuovere un'architettura applicativa nella quale l'importanza si sposta sullo sfondo: i server applicativi centrali.

    Attraverso lo sviluppo di un'applicazione di e-commerce con interfaccia web, il libro affronta le tecnologie Java necessarie a realizzare questo tipo di applicazioni, come le Servlet, JSP, JSTL e anche JDBC.

    Nel testo non vengono solo trattate le API sopra elencate, ma anche altri importanti argomenti. Nella realizzazione dell'applicazione viene infatti utilizzato UML per descirvere il sistema e vengono applicati diversi Design Pattern e accorgimenti tecnici tipici per le architetture web. Il testo diventa, dunque, una preziosa risorsa per chi desidera non solo realizzare, ma anche strutturare in modo corretto una Web Application Java.

  • Da Visual Basic a Java

    • Hops Libri
    • 2002
    Authors: Massimiliano Bigatti

    Con Visual Basic oggi si possono produrre applicazioni allo stato dell’arte: event-driven, Internet-ready, con logica a componenti. Tuttavia Visual Basic ha una limitazione evidente: funziona solo in ambienti Microsoft. Nell’era di Internet, dove sono ormai frequentissime reti locali e geografiche popolate da numerose piattaforme differenti, la facilità di sviluppo di Visual Basic si scontra contro l’impossibilità del deployment su piattaforme non Windows.
