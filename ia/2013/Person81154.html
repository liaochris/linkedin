Johan Lindfors' Projects

  • SATS for Windows Phone

    Team Members: Johan Lindfors, Apegroup

    Helped design and implement the SATS application for Windows Phone together with Apegroup. With the app, SATS members can book, cancel and keep track of their upcoming scheduled exercises. All users of the app can search for upcoming exercises based on simple search or advanced search with custom calendar, categories, centers and instructors. Searches can also be pinned to start screen for immediate search and "two click" bookings!

  • TriPeaks Solitaire

    Team Members: Johan Lindfors

    Implemented the popular TriPeaks Solitaire game together with Fabrication Games for Windows Phone. Based on existing iOS source code and new customized graphics the game was migrated to XNA in record time.

  • Posten Riktiga Vykort

    • November 2012 to November 2012
    Team Members: Johan Lindfors, Johan Sjöberg

    Coderox helped Posten build the Riktiga Vykort (Real Postcards) experience on the Windows Phone platform. With the app you can either from a captured camera image or any image stored on the device, create a physical postcard which will be sent to up to five recepients.
