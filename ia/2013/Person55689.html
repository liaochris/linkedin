Ionuț Dănilă's Projects

  • ubiGuide

    • February 2013 to May 2013
    Team Members: Ionuț Dănilă, Mihaela Ghimiciu

    An essential tool for any tourist, ubiGuide's unique capabilities give users an amazing Windows Phone 8 experience.

    Technologies: .NET Framework 4.5, WCF 4.5, Windows Phone 8 (WP8), Entity Framework 5, SQL Server 2012.

    Web Services and Libraries: Microsoft Translator, Microsoft Routing, Microsoft Geolocation, Speech Recognition, Speech Synthesis, GART (Geo Augmented Reality Toolkit), Google Places, ABBYY Cloud OCR, Twitter, Flickr, YouTube, AlchemyAPI.

    Most exciting features:
    ☑ search and filter nearby places available on a map;
    ☑ navigate to a certain point of interest using driving and walking voice-guided directions;
    ☑ enhance the user-device interaction through augmented reality;
    ☑ provide useful details (contact info, reviews, pictures, etc.) about chosen locations of interest;
    ☑ makes possible a conversation between two foreign-language speakers through automatically understanding, translating, and vocally synthesizing any phrase into many known languages;
    ☑ show the back camera view to help users focus on the conversation;
    ☑ recognizes -- using optical character recognition (OCR) -- certain items (like street signs, posters, menus, etc.) written in foreign languages, and translates them into a language of choice;
    ☑ display suitable social media results (tweets, images, videos) from other users for any given subject;
    ☑ analyze sentiments and opinions regarding tweets;
    ☑ choose from a rich palette of settings: many speech recognition & synthesis languages, multiple OCR and translation languages, customizable voice, integration with Windows Phone 8 current theme, and many more;
    ☑ integrate with Windows Phone 8 platform features: Live Tiles and Music + Videos Hub integration;
    ☑ fast responsive application using our cloud services on the basis of Microsoft technologies.
