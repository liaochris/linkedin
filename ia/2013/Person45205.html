Fatih Mehmet Güler's Publications

  • Natural Intelligence – Commonsense Question Answering with Conceptual Graphs

    • ICCS'10: Proceedings of the 18th International Conference on Conceptual Structures: From Information to Intelligence
    • July 30, 2010
    Authors: Fatih Mehmet Güler, Aysenur Birturk

    Natural Intelligence (NI) is a question answering system based on Combinatory Categorial Grammar (CCG) as the theory of grammar and Conceptual Graphs (CG) for knowledge representation and reasoning. CCG is a lexicalized theory of grammar and very suitable for semantic analysis. Conceptual Graphs is a special kind of semantic network which can express full first-order logic. It aims to address the problem of commonsense reasoning in question answering, by using the state of the art tools such as C&C tools, Cogitant and Open Cyc. C&C tools are used for parsing natural language, Cogitant is used for Conceptual Graph operations, and Open Cyc is used for upper ontology and commonsense handling.

  • A Conceptual Graph Based Question Answering System

    • In the Workshop on Graph Structures For Knowledge Representation and Reasoning (GKR 2009) at the Twenty-first International Joint Conference on Artificial Intelligence (IJCAI-09), July 11-17, 2009, Pasadena, California, USA.
    • 2009
    Authors: Fatih Mehmet Güler, Aysenur Birturk

    In this paper, two kinds of semantic network based question answering systems are explained. In the first one, constructing semantic representations from the output of combinatory categorial grammar parser is explained. Semantic network is generated from the output of Discourse Representation Structures (DRS). The second one is based on conceptual graphs, which is a kind of semantic networks. Since the conceptual graphs are suitable for knowledge representation and reasoning, in the second one, semantic network is generated from directly Conceptual Graph Interchange Format (CGIF) input, with the ability of doing reasoning. Implementation details for both systems are examined, concluding with the idea of merging these systems.

  • Natural Intelligence – A Question Answering System

    • eChallenges e-2009 Conference Proceedings, Paul Cunningham and Miriam Cunningham (Eds), IIMC International Information Management Corporation, 2009, ISBN: 978-1-905824-13-7.
    • 2009
    Authors: Fatih Mehmet Güler

    In this paper, a question answering system based on the semantic representations created from the output of combinatory categorial grammar parser is explained. Combinatory categorial grammar (CCG) is a lexicalized theory of grammar, and very suitable for semantic analysis. Available tools are used for parsing and creating semantic representations. Our question answering system uses these semantic representations to build a semantic network from natural language sentences and answers questions based on this network.
