Fernando Cortés Hierro's Summary

My professional career was mainly focused in the development of Internet / intranet applications but in the last five years I am specialized in RIA applications with Silverlight and Windows client applications using WPF from the beginning of these technologies. I develop Windows Phone and Surface application as well.

I have held a variety of different roles and I have done different things like:
• Implementation of proof of concepts from functional specifications
• Admin and design databases
• Intranet / Intranet web applications
• Develop client / server applications
• Develop Silverlight and Windows Presentation Foundation applications
• Develop Windows Phone and Surface applications
• Automated Testing
• Create technical documentation
