Leonardo Lanchas' Projects

  • Distributed Pacman game for 2 players

    • mayo de 2009 a julio de 2009
    Miembros del equipo: Leonardo Lanchas

    This project was developed for the Concurrent programming course during the 3rd year of our degree.
    The main objective was to implement a parallel application that made use of the message passing technique, among others, as a synchronization tool.
    With this in mind, we decided to modify the old Pacman game by adding a second player so that both, in a cooperative way, overcame the various stages of the game, and this would be more fun.
    We also included a chat, rather basic (as it was not the main task and we only had two months), made in sockets (experimental), so that players could talk.
