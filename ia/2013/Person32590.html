David Skowronski's Projects

  • Floral Production & Ordering System - Access

    Team Members: David Skowronski

    Multi-user, menu-driven Access database allowing sales to enter orders for floral arrangements. The system then generated order slips to purchasing to acquire required flowers and production schedules based on due dates and flower life.

  • IRS Audit Adjustment Tracking Database - Access

    Team Members: David Skowronski

    Multi-company, menu-driven Access database allowing entry of proposed audit adjustments by Agreed/Unagreed, Basis/Timing, Entity, and Description. Multiple reports allowed internal staff to monitor overall tax effect as audit progressed.

  • Calendar Payment Management System I - Access, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    Multi-user, menu-driven Access database tracking corporate tax filing requirements, due dates, payments, and responsibilities. System also generated (a) general ledger account analysis, (b) open item lists, (c) EFT/check payment authorization signature forms, and (d) payment file for import into accounts payable payment system.

  • Calendar Payment Management System II - Access, SharePoint, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    Similar to Calendar Payment Management System I except that tax filing requirements, due dates, payments, and responsibilities are stored in a SharePoint list rather than an Access database. Front-end user interface exists in an Access database.

  • Global Tax Provision, Deferred Tax Entries, and Footnote Computations - Access, Excel, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    Access database to gather local currency tax data from over 200 Excel files. System computed the world-wide tax provision and annual report tax footnote amounts in US dollars. System also calculated the deferred tax journal entry along with the deferred tax item 'inventory' in both US dollars and local currency.

  • Electronic File System - Access, SQL Server, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    SQL Server database containing metadata for over 60,000 physical and over 28,000 electronic image files. Access front-end interface allowed user to search for files based on user-specified criteria. Electronic image files can be displayed via mouse-click.
    System also allowed for the update of physical file location based on bar code reader scan.

  • Office Rent Billing System - Access, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    Access database to generate monthly rent invoices for 40-story office building. Invoices included additional amounts, if any, for billable services performed by building management.

  • Departmental Outboard - Active Server Pages (ASP), Access

    Team Members: David Skowronski

    Application to display on a daily basis the location of departmental staff: In, Sick, Vacation, Out-of-office Meeting, etc. Data is stored in an Access database. Front-end was based on Active Server Pages (ASP), allowing users to interface with the data via a web browser.

  • Service Call Managment System - Access

    Team Members: David Skowronski

    Multi-user Access database to store equipment service requests by equipment type with initial diagnosis. Actual problem and repair time entered after service call. Reports allowed management to look for repeating problem types or equipment failures.

  • Consolidated Budget Analysis and Preparation - Excel, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    Excel application allowing users to enter budget data for lowest level reporting unit and ultimately consolidate data into district/region/consolidated amounts.

  • Tax Data Collection Kits - Excel, CorpTax Office Add-In

    Team Members: David Skowronski

    Created a series of domestic and international Excel workbooks for tax data collection utilizing the Corp Tax Office Add-in for Excel. This add-in allows Excel files to either (a) 'pull' data from the Corp Tax program or (b) 'push' data back to Corp Tax.

  • Spreadsheet Application Conversion - Excel, Visual Basic for Applications (VBA)

    Team Members: David Skowronski

    Converted spreadsheet applications programmed in other programs such as Lotus 1-2-3, Quattro Pro or Smart spreadsheet into Excel using Excel VBA.

  • Expense Report Streamling and Template - Excel

    Team Members: David Skowronski

    Developed template for streamlining expense reports utilizing standardized descriptions and Excel drop-down boxes. Expenses were totaled into classifications (ex. meals, hotel, transportation, etc.) and general ledger accounts based on items chosen in drop-down boxes.

  • Global Hospitality System - Access

  • Hyperion to CorpTax Data Import Transposition - Excel,Visual Basic for Applications (VBA)
