Andrew Comeau's Summary

My primary experience is in the programming and development of Windows applications, online tools and database solutions. I am productive with a variety of technologies including Visual Studio, SQL Server and Microsoft Access. I am a Microsoft Certified Professional in Windows Application Development with Visual Studio.NET and am familiar with the C# and VB.NET languages as well as all phases of the software development life cycle. I have several years of experience developing applications for areas including manufacturing reporting, laboratory testing, employee training and pharmacy operations.

In addition to independent consulting, I operate the websites AndrewComeau.com and Drewslair.com in order to share my software and programming knowledge. This has included corresponding with people who have found the sites helpful and answering their questions. I've published the book "Microsoft Access for Beginners" which is an expanded and updated version of a tutorial series originally featured on Drewslair.com.

I am currently open to new consulting opportunities. My goal is to continue my work as a programmer and technology writer and to expand my current skills to include additional technologies. I also plan to continue developing online services to offer resources for those interested in learning about programming.

Andrew Comeau's Publications

  • Microsoft Access for Beginners

    • Andrew Comeau
    • January 3, 2013
    Authors: Andrew Comeau

    This book is an introduction to Microsoft Access intended for people who have never used the program before and might be unfamiliar with database software in general. It introduces the reader to the basic concepts of a relational database system as well as some of the things that are possible with Microsoft Access. This book explains the concepts and provides a good foundation on which to continue learning and creating applications with Microsoft's premier desktop database software. Rather than being a simple step by step tutorial, "Microsoft Access for Beginners" demonstrates how to use Access well. The second edition, published in January 2013, includes an introduction to Visual Basic for Applications, the programming language which enables Access users to create fully automated and sophisticated applications.
