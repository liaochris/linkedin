Ellis Hettinga's Projects

  • EURBANLAB Climate-KIC portal

    • November 2012 to Present
    Team Members: Ellis Hettinga, Philippe Lanceleur

    The EURBANLAB website of the European Climate Knowledge and Innovation Community (Climate-KIC) is a portal that displays information coming from connected websites.
    It displays for instance listings of People connected to the Climate-KIC project internationally.
    If you want to see more information on a person it shows this information inside the portal, so you don't have to visit other sites.
    Besides this it is working in partnership with a system created in another project: SAM - Services based relation manager.
    Next to the detailed information on a person actually residing in another website it shows the relationships each person has with information saved in other attached websites.
    All data displayed in the EURBANLAB portal is retrieved from connected websites through webservices.


  • Willem Wever Widget

    • November 2012 to December 2012
    Team Members: Ellis Hettinga, Rudy de Kok, Carole Grootenboer, Kevin van Daatselaar, Sandra Verdel

    Willem Wever is a NCRV television programme in which children get answers to questions they have asked.
    The supporting website contains all questions and answers of the past.
    The Willem Wever Widget is a new website that offers a search and find on the questions and answers. It is presented in a widget that partners can place in a corner on their own website.
    The widget offers partners an interface to select the questions they want to show initially. The layout of the widget can be adapted for every partner.

  • TNO (DINO, Data and information of the dutch subsurface)

    • June 2012 to October 2012
    Team Members: Ellis Hettinga

    Adaptation and extension of several websites (dinoloket, broinfo, broinname).
    DTAP deployment with Features, Hudson, Maven

  • SAM - Service based relation manager

    • October 2012 to Present
    Team Members: Ellis Hettinga, Philippe Lanceleur

    SAM is a service based infrastructure that handles relationships between nodes in multiple drupal sites.
    It consists of SAM-Base to store the relations, and a client module to be installed on remote systems that want to be connected to SAM-Base.
    The client module contains two SAM-satellites: one to administer the relations in SAM-Base from the remote system, and one to show related nodes of the node you are visiting on the page. Currently there is a D6 and a D7 version of the client module.
    This project is in development for REEDS (International Centre for Research in Ecological Economics, Eco-innovation and Tool Development for Sustainability, part of french University UVSQ)

  • NEXIT architects

    • July 2012 to September 2012
    Team Members: Ellis Hettinga, Linda Swaap, Wilbert de Haan, Addy de Boer

    Online presentation of projects, news, concepts and BIM (Building Information Model) information.
    Graphic design: Accu Graphic design
    There is a design for screen, tablet and mobile making it possible to adapt to the device that is used.
    Technologies used: Drupal 7, HTML 5, Media queries, Responsive design, Webfonts.

  • Yggdrasil

    • October 2011 to March 2012
    Team Members: Ellis Hettinga, Jean-Marc Douguet

    Yggdrasil (Tree of life) is an online presentation of the Catalogue of Teaching programmes of OVSQ (Observatoire des sciences de l' univers de l'UVSQ).
    In a way similar to Broceliande (former project for UVSQ REEDS) visitors will be stimulated to explore related content.
    While exploring related content, visitors keep track of the Teaching Programme they have chosen before, to prevent them from getting lost.
    Therefore site administrators must be able to create all kinds of relationships. This combined with the fact that the site is multilingual made it a very interesting project for me to work on.
