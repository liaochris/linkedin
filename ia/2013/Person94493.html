Michael Nash's Summary

Michael Nash is the President and founder of JGlobal Limited, a software development, support, training and research company based in Saskatoon, Saskatchewan, Canada. Michael has been in the entrepreneurial software development business for the past 20 years.

He was the creator of the Expresso web-application framework and was team lead for Expresso when it won the 2001 Java Community Award for Outstanding Group Technical Contribution to the Java Community. Michael is the author of the book "Java Frameworks & Components: Accelerate Your Web Application Development" (Cambridge University Press) and the upcoming "Explorer's Guide to Java Open Source Tools" (Manning Publications), as well as various articles and presentations.

Michael has worked with Java since its introduction and his development experience includes EIS systems for multi-national company intranet systems, development of web application systems for extremely large high-volume interactive internet sites, and international financial management applications, among others. He is a member of the Sun Microsystems Expert team for JSR-127, the upcoming JavaServer Faces UI API for web applications.

Michael is the Lead Developer, Chief Architect, and author of the initial design as well as much of the initial code base for the Keel meta-framework project. Through JGlobal Limited, Michael offers training and support services for numerous open source Java projects.

Since 2004 Michael has worked mostly in the Scala language, and is currently the Senior Software Developer at the Point2 division of Yardi Systems

Specialties

Computer Software Developer. Expert Java Programmer, specializing in server-side development for web applications. Open-source Java project expert. Technical author - two books, numerous articles.
