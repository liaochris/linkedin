Steve Krueger's Summary

Steve Krueger is a founding partner of a Vancouver based Drupal start-up, The Jibe Multimedia, Inc.

He has personally worked with several large organizations for web-based projects such as SPIN Magazine, Sony BMG, eHarlequin, Placeblogger, Macs Convenience Stores and has contributed code to the Drupal community such as the Basic theme.

Specialties

Drupal, front-end development technologies (jQuery, CSS, HTML), css preprocessors, semantic markup, usability and accessibility
