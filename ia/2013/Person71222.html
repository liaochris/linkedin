  • Bachelor's Capstone - Signature & Anomaly Based IDS/IPS Prototype

    • July 2012 to October 2012
    Team Members: Justin Hutchens, CISSP, CNDA, CEH, ECSA, CHFI

    - Developed a local area network signature & anomaly based IDS/IPS (Intrusion Detection/Prevention System) prototype
    - Configured a spanning port to monitor and collect traffic on a local VLAN
    - Used Python scripting with scapy to analyze network traffic and send alerts to a backend MySQL database when events were triggered.
    - Additional scripts were developed to perform metadata analysis on these trends to implement rules regarding upper and lower bounds of traffic frequency and proper packet sequence, thereby implementing anomaly based analysis.
