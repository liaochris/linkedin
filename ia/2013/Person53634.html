Horacio Nunez's Publications

  • Functional Representations of Numeric Matrices

    • dotNetMania
    • September 2010
    Authors: Horacio Nunez

    This article introduces a data type to represent a numeric matrix programmed in F # that uses functions instead of numeric arrays. It describes how this new abstraction can help reduce the consumption of memory, while giving the evaluation of matrix operators a lazy behavior as well a coding style similar to mathematical notation. More over is presented how this features and support of the iterator pattern can be used to express computations that can be processed by Parallel LINQ. Finally, is presented a customized version of the tool F# Interactive that supports the data type introduced in the text
