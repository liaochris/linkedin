Aaron Botnick's Summary

While a degreed Meteorologist, I have a minor in Computer Science and a passion for computer programming, software development, and software engineering.

Currently, I'm a software engineer for a company that has created and deploys utility sub-meters. I write code that simplifies in-house processes and manage data collection from remote meters into a Ruby on Rails web application. Primarily, we're using Python on an embedded system with ZigBee network communication as well as our Ruby on Rails app.

Prior to that, I worked for a company that creates software for law enforcement agencies and am specializing in mapping technologies. For the 3 years prior to that, I combined the best of both worlds while working for a weather software company as a software engineer, constantly improving an online/interactive weather map through both server-side and client-side development.

I strive to continue learning new techniques, technologies, and languages while seeking solutions to fun and unique problems in software development / engineering.

Specialties: Python, Flex, Objective-C, C++, C#, .NET, Java, Perl, Fortran, Matlab, scipy, matplotlib, numpy, shell scripts, Regular Expressions (RegEx), MySQL, PostgreSQL, MS SQL Server, PostGIS, OS X, Linux, Windows, ArcGIS SDKs

Meteorologist / Software Engineer

Privately Held; 51-200 employees; Information Technology and Services industry

February 2009 – September 2011 (2 years 8 months) Norman, OK

While working with these data formats, I frequently wrote automated scripts parsed text files into databases and/or handled client/server communication. Data conversion and pre-processing was accomplished using Python with scipy, numpy, and matplotlib, as well as Perl, PHP, MySQL and PostgreSQL databases, and PostGIS.

I also worked on a low-bandwidth marine weather display application called ClearPoint Weather, using C# .NET through the Visual Studio IDE. Throughout this time, we developed in teams using Agile development practices using SCRUM methodology and Subversion source code management tools.
