Maxime Petazzoni's Summary

I graduated in France in 2009 with a Masters degree in computer science and embedded systems from the UTBM (University of Technology of Belfort-Montbéliard), and in 2010 with a second Masters degree in the same field from the UHA (Université de Haute-Alsace).

Most of my background revolves around free and open source software and embedded systems programming and my main interests are from low to high-level programming for embedded, communicative or mobile devices and wireless sensor networks. I did a lot of kernel and embedded systems development at MontaVista Software. More recently, at Turn, I have been working on high-availability, high performance, large scale and distributed systems that handle 1M+ requests/sec.

I have more than 15 years of experience working with GNU/Linux systems as a daily user, developer and system administrator. I am a free and open source software enthusiast, developing or contributing to various open source projects.

My programming languages of choice are C, Java and Python (in no particular order) and I also have some front-end development skills for the occasional UI/dashboard need.

Maxime Petazzoni's Projects

  • MapOSMatic

    • September 2009 to Present
    Team Members: Maxime Petazzoni, Thomas Petazzoni, David Decotigny, David Mentré, Jérôme Petazzoni

    MapOSMatic is a free software web service that allows you to generate maps of cities using OpenStreetMap data. MapOSMatic can generate two kinds of maps:

    Single page posters, where the map and the street index are rendered side by side. The map itself is split in squares so that streets are easy to find. This format is typically useful for printing on large paper sheets.
    Multiple page booklets, where the map and the street index are split into several pages. This format is typically useful for printing on standard paper sheets.
    The generated maps are available in PNG, PDF and SVG formats and are ready to be printed.

    As the data used to generate maps is coming from OpenStreetMap, you can freely (under the terms of OpenStreetMap license) reuse, sell, modify, ... the generated maps.

  • ttorrent

    • June 2011 to Present
    Team Members: Maxime Petazzoni

    Turn's BitTorrent library, ttorrent, is an open-source, pure-Java implementation of the BitTorrent protocol, providing an easy-to-embed implementation with a BitTorrent tracker and a downloading/seeding client and tools to manipulate and create torrent files. Both the tracker and the client provide hooks for easy integration in a larger application, allowing for tracker control, download progression updates, etc.

    ttorrent benefits from frequent contributions and improvements on GitHub and is being used outside of Turn in other enterprise environments or open-source projects.
