Anthony Nicholls' Projects

  • Multi-channel Reverberation (Dissertation Project)

    • September 2012 to Present
    Team Members: Anthony Nicholls

    Using a source image model for calculating reflections inside a simple three-dimensional room, a complex network of low-pass, all-pass and comb filters is created with similar effect to traditional algorithmic reverberation approaches. However each reflection (as a reflection of the source) generates a vector between itself and a listener located within the room. With this data each reflection can be accurately panned in a three-dimensional sound field, this allows for the modelling of virtual microphone techniques such as double-mid side for a first order B-Format output (higher orders are also possible).

    The project was initially implemented using Processing (with OpenGL for 3D graphics) and SuperCollider (for signal processing). However it is expected in the near future that the project will become an open source cross-platform project utilising C++ and the JUCE library.

  • Granular Reverberator / Particle Synthesis (Processing and SuperCollider)

    • October 2012 to December 2012
    Team Members: Anthony Nicholls

    The task was to build a creative project using SuperCollider (an open source programming language and environment built specifically for sound synthesis).

    Demonstrated in this video (http://vimeo.com/59725806) is an example in which OSC (Open Sound Control) messages are sent to SuperCollider from Processing (an open source programming language and environment often used in visual arts). With every mouse click the Processing sketch creates a cluster of particles inside a virtual 2D room; each particle is sent in a random direction and is allocated a set of properties (size, speed, and colour). Particles bounce around the room reflecting off walls, as these particles travel they gradually dissipate until they are no longer visible. When particles pass close to the cursor position, an OSC message containing details of the particles' individual properties is sent to SuperCollider. The OSC message is used to control synthesis parameters within SuperCollider. Further control over the sound is achieved using a simple GUI created in SuperCollider.

  • Psychology Precision Test Application (C# .NET)

    • 2012 to Present
    Team Members: Anthony Nicholls

    A final year undergraduate student required a specific application that displayed faces portraying different emotions to a screen for a set period, followed by an interval, before another face appeared and the user would respond with key stroke to indicate if the face they saw was the same as the previous face or not. The timing accuracy required between measuring a face appearing on the screen and the key stoke needed to be greater than 10 ms. To get as close to these results as possible, multi-threading techniques were combined with the present() method found in Direct-X to detect when a vsync occurs. The student went on to find significance in their results

  • VST Tremolo Plugin (C++)

    • 2011 to Present
    Team Members: Anthony Nicholls

    Using the Steinberg SDK and Microsoft Visual Studio we were set the task to create a simple Tremolo plugin. My version incorporated a invert phase switch, adjustable oscillator waveform, frequency, and depth control.
