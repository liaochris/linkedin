Dan Johnson's Summary

Recently left the semiconductor industry to focus more on the embedded design space with a particular interest in ARM Cortex A15/A9 processors and MALI GPUs. Also exploring the Nvidia Tegra 3/4 rendering/processing capabilities. Interested in the development of ARM based mobile platforms for 3D virtual worlds and applications which require 3D rendering/modeling resources.

Embedded Java multimedia applications are also of interest especially those which use QuickTime, MPEG-4, OpenCL, OpenAL, OpenGL-ES. and advanced 3D modeling/rendering technologies. Java technologies of interest include J2SE, J2ME, Java3D, JINI, JMF, Android, & JavaFX. Using Xcode, Eclipse, and Netbeans for Java development projects.

I also develop virtual world content and software for Second Life and OpenSim. I have experience building and configuring OpenSim servers on Ubuntu Linux, Mac OS-X, & Windows platforms. My test servers are currently running on Via Technologies Mini-ITX cards. I am also interested in porting OpenSim to ARM based platforms.

I have extensive experience porting open source EDA tools to Mac OS-X and various Linux platforms. Broad exposure to operating systems (Mac OS-X, Fedora 10, Ubuntu 8.4/9.05/10.04/12.04, CentOS, Android, Chrome OS, Solaris 10 ) and software development using Java, C, C++, Objective-C, and some TCL/TK & Qt.
