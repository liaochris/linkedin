Albert Nadal Garriga's Summary

I was working from year 2005 to the begining of 2008 at ArenaMobile (Reus, Spain), a global multimedia mobile content provider that worked with partners like Vodafone, Movistar, Orange, Claro and many other telcos worldwide. At 2008 I joined the Internet Web Serveis company (Lleida, Spain), just coinciding with the arrival of the first iPhone SDK from Apple to the mobile market. IWS is a local ISP that provides all kind of custom made Internet web services with customers like Terra, Grupo Prisa or Fon.

During the year 2011 I collaborated with some start-ups and companies analyzing, designing and developing complex iOS applications like vertical social networks and other business-oriented mobile applications. Just entered in 2012 and with the aim to improve and learn new realtime technologies I decided to join Telefonica Digital (Barcelona), participating in an initiative developing innovative videoconference and instant messaging system software on iOS and Android platforms.

Now I'm currently working at Golden Gekko (Barcelona), a leading mobile and web solution provider in Europe and North America. I'm performing the analysis and development of software for the iOS platform.

Apart from all this, I also started some successfull projects in my personal career, all based in mobile and web technologies. You can see a detailed portfolio of my recent works at http://lafruitera.com Feel free to contact me for any suggestion or innovative proposal at anadalg@gmail.com

I have more than 10 years experience in mobile and web software development. Both are my specialities. I like programming computers since my early days. I had the opportunity to develop with old and modern programming languages like Basic, Logo, Lisp, Pascal, Visual Basic, Delphi, Java, JavaScript and PHP.

Now I'm focused developing iOS mobile software in Objective-C, C and C++ using the Xcode IDE. Although I use to work with gnu/Linux development environments if possible.
