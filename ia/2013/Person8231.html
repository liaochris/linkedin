Web Developer, Business Analyst, Project Manager

First Data Commercial Services

Public Company; 10,001+ employees; FDC; Financial Services industry

January 2001 – November 2007 (6 years 11 months)

Created processes for Web site production: designed and used standard templates for Web sites and landing pages; implemented content management system, and created Web site and landing page request forms. With these processes, Internet Services group successfully managed over 90 different Web sites with only 2 employees and found large cost savings for company: Web site design costs were lowered from $35K each to only $4K each when a standard template was used.

Solved technical issues to implement return on investment (ROI) tracking for Web sites. Prior to this, ROI tracking was not used for any Web sites and many did not have traffic reporting or conversion tracking. Once implemented, ROI tracking allowed First Data to understand effectiveness of all Web sites as a sales-lead generation tool. ROI tracking showed that 93 Web sites generated over $167K per year in revenue. Removed 58 sites which were not producing sufficient ROI.

Designed and created fourteen new Web sites and 47 new landing pages in 2007 alone. New sites used best practices and learnings and included ROI tracking. Through site traffic analysis, found the most effective sites, the best sources of traffic for these sites (those most likely to convert to a sale), the least effective sources of traffic, and the performance of marketing campaigns.

Since these were often illegible, they were difficult to process. The new system replicated the paper application as a Web page and stored all application information in an Oracle Database, allowing TeleCheck to print legible service applications.
