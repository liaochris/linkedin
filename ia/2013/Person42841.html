Ernest de León's Summary

Cloud Computing and Virtualization Solution Architect with over 15 years experience in Unix, Linux and Windows architecture, engineering, cross-platform integration, network security and information security.

Current Focus: Fortune 500 Private Cloud Infrastructure (IaaS) based on Eucalyptus Systems IaaS, Amazon Web Services (AWS), VMware vCloud Director, Apache CloudStack and OpenStack.

Integration of Private Cloud with leading enterprise storage (SAN) systems from NetApp, Dell EqualLogic, Compellent and EMC. Integration of scalable distributed back-end storage platforms such as Ceph and Gluster with Cloud object storage such as Eucalyptus Walrus and OpenStack Swift.

Integration of Public and Private Cloud infrastructures with change management and automation platforms such as Puppet and Chef to include recipe design and configuration. Design of automated systems to rapidly deploy more nodes in Private Cloud IaaS, and eliminate the issues related to manual installation and configuration which hamper horizontal scalability speed.

Architecture and design of highly available applications that span multiple clusters and availability zones (locally and globally dispersed) in both Private and Public Cloud infrastructures. Integration of these applications with work queue, automation and provisioning tools which allow rapid elastic scalability (up and down) as demand fluctuates.

Expertise focusing on enterprise private cloud computing deployments that offer Amazon Web Services (public cloud) functionality and API compatibility, at scale, within the datacenter. Integration of compute nodes, storage nodes, SAN/NAS, 10gE networking, load balancing, multi-tenant configuration (security), high-availability, infinite scalability, application refactoring, net new application development, and much more.
