    In the course, we built a compiler for a language loosely based on Go. It included not just syntax checking but also type checking, support for functions/closure, and pluggable code generation backends based on an AST transformations.

  • Schemato

    • July 2012 to Present
    Team Members: Andrew Montalenti, Emmett Butler

    Mr. Schemato is a validator for semantic web metadata implementations in RDFa and HTML5 Microdata. Written in Python using Ivan Herman’s pyRdfa and pyMicrodata libraries, it validates the implementation of the rNews, Schema.org, Facebook Open Graph and parsely-page standards for embedded metadata by reading the latest versions of these standards’ ontologies and comparing against a graph built from a page’s metadata. Mr. Schemato is meant to aid publishers of web content in the implementation of semantic web standards, in order to ease integration with tools like Parse.ly. Open-sourced at the @timesopen Bigger Data and Smarter Scaling event, schema.to is designed to be modular and easily extensible for support of additional standards.
