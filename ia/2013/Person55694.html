Ken Minx's Projects

  • FTP Download Server

    • October 2012 to Present
    Team Members: Ken Minx

    This project was created to answer a problem for an engineering company. Customers reported that they could not get to the corporate FTP site but still needed to send very large files in for analysis. Some customers stated that even if they didn't have connection issues, they didn't want to tie up their servers sending in these large files. This set of scripts allows for customer downloads to be scheduled. Customer's FTP server login, password and path to files is required to schedule download. Notification emails are sent to engineers when download has started, completed or errored for some reason. There is an FAQ on the site home page that explains the features of this project. Project is available and can be easily modified to fit most corporate needs.
