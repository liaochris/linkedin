Dan Frederiksen's Experience

President / CEO

Domino Developer Network, Inc.

Privately Held; 1-10 employees; Information Technology and Services industry

December 1998 – Present (14 years 10 months) San Francisco Bay Area

President and founder of Domino Developer Network, Inc. (DDN). DDN provides Lotus Domino application development and system administration consulting services in addition to hosting services for over 500 clients worldwide. As an Application Service Provider (ASP) DDN allows Lotus Notes/Domino professionals around the world to easily publish Lotus Notes databases on the Internet using advanced, self-service tools.

Consultant

Providian Financial

Public Company; 10,001+ employees; PVN; Financial Services industry

April 2000 – March 2001 (1 year) San Francisco Bay Area

Developed bidirectional data exchange procedures with Sybase and Oracle back-end data sources. These mission critical data feeds moved data both upstream and downstream between multiple enterprise systems such as Lotus Notes, Remedy and PeopleSoft.

Enhanced and supported corporate-wide Phone Directory application accessible via Lotus Notes and browser clients. Integrated Mobile Services for Domino (MSD) and Replix FAX services within the Phone Directory application for single-click paging and faxing.

Developed Mass Mail Agent application for sending important, large-scale e-mail distributions. This application allowed distribution lists to be dynamically built based on current department/location demographics and changing employee reporting structures. Approval cycle functions were also provided for central management of all e-mail distributions.

Consultant

Bank of America

Public Company; 10,001+ employees; BAC; Banking industry

August 1996 – April 2000 (3 years 9 months) San Francisco Bay Area

Member of the Domino and Notes Architecture & Implementation group whose primary function was to design and implement enterprise-wide administrative solutions and other infrastructure improvements to a Lotus Notes domain of 70,000+ users and 400+ servers worldwide (including Notes servers on both Windows NT and IBM S/390 (AIX) platforms).

Performed key role in the collaps e (merging) of two of the world's largest Lotus Notes implementations.

Architected and designed a complete system of Lotus Notes applications to manage and control all Lotus Notes administration functions. The system provided automated user procedures with back-end LotusScript processing to perform everyday functions such as creation of user IDs, control of server load (user capacity), modification of ACL groups, automated creation and roll-out of Notes databases and synchronization of Notes addressbooks with mainframe HR applications and foreign e-mail directories (Exchange, cc:Mail, OfficeVision and TAO).

Developed a simplified process using Lotus Domino Connectors for LotusScript to import and update data from several SQL databases containing e-mail directory information for all non-Notes e-mail environments (this process replaced a much more complex process that involved Casahl's Replic-Action).

Developed a Lotus Notes change control application to collect and track changes required to all administration applications.

Developed a Lotus Notes application to track Domino server statistics and on-site contact information. The application included an on-call paging function with automated escalation paths for support issues and server outages.

Developed a Lotus Notes time management application to track employee projects/tasks, travel, training, time-off and meeting notes.

Provided high-level application development support to 30+ Notes developers across the enterprise. If necessary escalated issues to Lotus technical support for resolution.

Consultant

Level 3 Communications

Public Company; 10,001+ employees; LVLT; Telecommunications industry

January 1996 – August 1996 (8 months) San Francisco Bay Area

Assisted in project management role to design a Windows NT network infrastructure and Lotus Notes R4 migration strategy for 4,000 users.

Developed a Lotus Notes application to provide a process by which users could create, approve, publish and revise "online manuals" which included corporate-wide standards and also location-specific procedures. The application included many customized workflow procedures written in LotusScript. Among the key features of this application were ease of administration and an intuitive, efficient user interface.

Developed a Lotus Notes application to track issues reported by WAN users. Multiple forms were used for each the end-user, dispatcher (telephone operator), manager and assignee roles of the application. Several agents were developed to notify managers of neglected issues and assignees of issues left in prolonged "hold" status.

Developed a Lotus Notes application to function as an electronic "dot board". The application enabled other Lotus Notes applications to determine the availability of a particular resource and return a substitute resource if applicable.

Developed curriculum and instructed several Lotus Notes R4 Administration classes for over 100 network administrators.

Consultant

IBM Global Services

Public Company; 10,001+ employees; IBM; Information Technology and Services industry

Tasks involved coordinating R4 client installations, supervising a temporary help desk and overseeing the pilot through completion. Also developed a Lotus Notes application containing numerous tasks for pilot users to complete and then enter their comments. The application contained views showing individual progress and dynamically generated charts displaying overall completion status of the pilot.
