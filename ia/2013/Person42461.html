Eric Weddington's Projects

  • WinAVR

    • 2002 to Present
    Team Members: Eric Weddington

    WinAVR (tm) is a suite of executable, open source software development tools for the Atmel AVR series of RISC microprocessors hosted on the Windows platform. Includes the GNU GCC compiler for C and C++.
    Project creator and maintainer. Popular open source development toolchain package for the AVR on Windows. Used as toolchain in Arduino software on Windows.

  • AVR-LibC

    • 2002 to Present
    Team Members: Eric Weddington

    The C runtime library for the AVR family of microcontrollers. For use with GNU toolset (Binutils, GCC, GDB, etc.)

  • avrdude

    • 2003 to Present
    Team Members: Eric Weddington

    avrdude is an open source project for downloading and uploading software to AVR microcontrollers

  • AVaRICE

    • 2004 to Present
    Team Members: Eric Weddington

    AVaRICE is an open source project that connects AVR GDB to the Atmel JTAG In-Circuit Emulator (ICE).

  • AVR-LLVM

    • November 2009 to Present
    Team Members: Eric Weddington

    AVR port of the open source LLVM compiler.
