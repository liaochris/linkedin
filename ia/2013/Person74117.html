Ken Boggs' Patents

  • System and Method for Providing Emergency Call Services

    • United States Patent Application 61/490,109
    Inventors: Ken Boggs

    [Embodiments herein are directed to systems and methods for reducing the number of non-legitimate NSI calls that reach PSAP without affecting public safety.
    An embodiment provides a method that uses E911 Selective Routers or NG-911 Emergency Service Routing Proxies (ESRP) to detect and send emergency calls to an interactive voice response (IVR) system. The IVR may present options that are dependent on the controls determined by an administrator of a PSAP. By way of illustration and not by way of limitation, controls may include global options that apply to the system and options that apply to individual callers. For example, the global control options may include features such as system bypass, Block all NSI callers, allow Non-Emergency Operator intervention, controls on how the system interacts with the PSAP equipment, how the system handles invalid responses from caller, default settings on IVR menu options provided to the caller, Registration and Billing Options. The individual caller controls are similar to the global controls except that they only apply to individual callers.
