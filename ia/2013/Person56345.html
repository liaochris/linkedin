Marty Hall's Summary

Develop and present customized J2EE onsite training for customers all over the world. Android, Ajax, jQuery, GWT, JSF 2.0, PrimeFaces, Servlets and JSP, Java 7 programming, Spring, Hibernate, Hadoop, RESTful Web Services, etc. Author of 3 video series from Pearson and 6 popular J2EE texts from Prentice Hall, including "Core Servlets and JavaServer Pages" which was worldwide bestseller and is published in 15 languages. Have done Java EE, Ajax, and Android training courses in the Philippines, Australia, Canada, Mexico, Puerto Rico, Japan, Norway, India, Cambodia, and many US venues. Email me at hall@coreservlets.com for a quote on customized training for your company!

Consult on Web application design and Java EE software development.

Organize onsite developers for clients (USA only). Very experienced Java EE developers, most with high-level security clearances.

Organize offshore developers for clients (any country). Inexpensive but experienced developers in the Philippines. Design and code *personally* supervised.

Details at http://www.coreservlets.com/

Specialties

Teach training courses on Android development, Servlets, JSP, JSF 2.0, Ajax, Spring, Hibernate, Struts, SOAP-Based and RESTful Web Services, and Java 6 Programming. Specialties within Ajax include GWT, jQuery, Prototype, Scriptaculous, Ext-JS, and JSON-RPC.
