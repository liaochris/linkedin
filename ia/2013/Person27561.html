Craig Miller, MS's Projects

  • LocoMaps

    • February 2012 to Present
    Team Members: Craig Miller, MS, Lloyd Moore

    LocoMaps is a simple Qt geo application that was created to demonstrate Qt's cross-platform capabilities at LinuxFest NW 2012. The application is built using a single code base and the compiled to run natively on Mac OS X, Linux, MeeGo, Windows and Symbian. The application contains NO platform specific code, and development was also carried out in a cross-platform development environment using Mac OS X, Linux and Windows.

  • Mobile: Intel

    • 2011 to 2011
    Team Members: Craig Miller, MS, Ivan Cockrum, Mark Lacas, Lloyd Moore
