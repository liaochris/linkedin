I started out at HP working on SCSI driver for HP-UX adding support for SCSI controllers. I wrote the first IDE CDROM driver for HP-UX. I then worked on enhancing and adding new hardware support in SCSI driver for Linux kernel. I added support for HCDP to enable serial ports on legacy free computers. HCDP support code was merged in mainline kernel.

I designed and lead the implementation of a Linux operating system targeted at Communications, Media and Entertainment applications - Debian GNU/Linux with HP Telco Extensions (Debian HPTE). I was the architect and technical lead for this product through 3 major releases that are currently deployed in field. Debian HPTE has been the key factor in HP being awarded 4 large contracts with tier 1 NEPs totalling ~$500 million in hardware, software and services revenue. One of these deployments currently handles 800 million calls a day around the world and there have been no bugs reported on this deployment in the last 6 years. I added support for kexec on ia64 architecture to Linux kernel as part of this project .

I have worked on projects related to network packet processing, like deep packet inspection, packet steering, load balancing etc at line speeds. I have redesigned I/O driver architecture for HP-UX, and OS image customizations for application specific deployments, as the I/O Architect for BCS Servers.

I am currently leading the technical design of Mission Critical Linux OS for HP Business Critical Server division.
