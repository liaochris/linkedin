Jiangcheng Oliver Chu's Projects

  • Mint Programming Language (Latest Version: 2.97)

    • April 2012 to Present
    Team Members: Jiangcheng Oliver Chu

    Official Website: http://mint.oliverchu.com OR http://www.mint.oliverchu.com
    [Contains tutorials, lines of code for Mint, downloads, source code, and much more.]
    Mint is a new programming language that I invented during the summer of 2012. Mint runs on top of Java and is able to use most of the Java library methods and classes, including methods and classes from Java Swing's GUI system. It is dynamically typed and supports heterogeneous lists. You can take screenshots and save them as image objects in the language, and write them to image files. You can connect to Internet Relay Chat to gather statistics and offer file transfer services. You can open a web browser or scrape an online HTML page in 2 lines of code. "Batteries included" the Python way - that is the Mint Standard Library way.

  • Intel x86 Assembler / Assembly Compiler (Latest Version: 1.31)

    • February 2011 to August 2011
    Team Members: Jiangcheng Oliver Chu

    The x86 Assembler (Compiler for Assembly Language) that I made comes with a built in GUI or IDE interface that allows you to edit assembly source code. This Java program converts x86 assembly code directly into executable bytecode, the kind found in C and C++ compiled .exe files.

  • SkySeeker Forums

  • JLime Code Editor
