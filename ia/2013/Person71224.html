Justin Jackson's Summary

I'm passionate about building products (and creating experiences) that delight customers.

Right now, I'm helping to improve Industry Mailout by using Agile, Kanban and Lean principles. We've working on delivering quality, well-designed software continuously. We're using Kanban to visually define our process: what to build, what we're building now, and what we can test. We're using Lean to improve our learning: listening to our customers, detecting trends, and making changes to our app based on this feedback. Eliminating unneeded features and code has also been a focus.
