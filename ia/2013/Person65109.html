Various Software Engineering and Project Management positions

Display Data Corporation

1979 – November 1988 (9 years)

Software Engineer initially in Applications, moving to Operating System Software Development.

Display Data was a maker of vertically integrated computer systems for several markets, primarily automobile dealerships. All aspects of the system were proprietary. Original hardware was based on the MicroData 1630 CPU, an 8-bit processor with 64K RAM.

Since all software was written in 1630 assembly language, migrating to something more capable was problematic. There were several efforts to grow past the 1630.

I was involved in four separate efforts to emulate the Display Data System using more modern hardware. All four projects were successful, the last of which lead to my joining Convergent Technologies.

Display Data Corporation was acquired by Convergent Technologies in August, 1986.
