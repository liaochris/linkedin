Hossein Ebrahimi's Summary

Innovation and production has always been the great interest. Therefore, there has always been exciting and challenging to work in IT. It's been exciting to work with programming and create new programs.
I started working when I was sixteen. After my studies in college, I started to work as a programmer. It started with programming in DOS 3.2 and later developed into various newer operating systems. During my years as a programmer, I have developed several different software packages. After having taken a course in "User Interface Design", I learned also how I would use psychology to create interactive applications more efficiently.
I mastered several computer programs used in the printing of various types (books, magazines, advertising, etc.). In this connection, I have translated several books from English into Persian, and have published a book in Farsi which centered on DPT. In addition, I taught several short courses in the DPT (Desktop publishing) and pre-printing.
I have worked with web-based applications and add-ons for cell phones. Another area that I had to work with is cartography (the science of principles and definitions in geography). I developed a program in digitized maps.

In programming languages, I am an expert in category C (C++, C#, Objective-C) but can also write other languages ​​that run on Windows, OSX and Linux, and OS/VS, DOS, OS2, Symbian and iOS.I have written a number of projects that run cross platform for both Windows and OSX.
