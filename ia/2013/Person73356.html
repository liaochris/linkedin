Kaladhar Bapu's Summary

A seasoned and effective User Experience Strategist & Information Architect with solid strategic design thinking and creative problem solving skills in complicated Web and Mobile application design.

Skills encompass communicating complex information through creating highly efficient user experiences, interfaces and visual design solutions for dynamic web-enabled systems and mobile platforms based on gathering quantitative and qualitative business and user needs using user-centric design methods.

Work experience spans 14 years in User Experience Design, Strategy and Information Architecture for digital mediums such as the Web, mobile, touch screens, and software applications using user centered methodologies, user research, emerging technologies.
