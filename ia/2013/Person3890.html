Alex Bleasdale's Summary

With over twelve years experience within web applications development (client and server-side) and multimedia production, I specialise in building and supporting web applications using XML-based technologies: XML Databases (including MarkLogic Server), XQuery 1.0, XSLT 2.0, XPath 2.0, XForms 1.0, XML Schema 1.0 and XProc.

I have several years experience building commercial web applications with Ruby, Scala and Java/J2EE and have worked with many well-known java libraries throughout my career.

I also have many years experience with JavaScript, ActionScript, PHP and Perl on many major commercial projects and have significant experience developing sites using Wordpress, osCommerce, Magento, Drupal/ubercart, Zope/Plone and DaisyCMS.

I have significant experience working both with relational databases (Oracle, mySQL, DB2 and SQL Server) and native XML databases (MarkLogic, eXist). I also have a sound knowledge of Apache 1.3 / 2 (and xssi), various unix-based servers (Linux/BSD/Solaris) and Windows Server.

I have several years experience working with Agile project management methodologies, source control systems (Subversion/Git), thorough code testing using TDD, BDD approaches (FlexUnit/JUnit/TestNG/ScalaTest), configuring build servers (Jenkins/Bamboo) and dependency management tools (Ivy/SBT/Maven).

I'm not currently looking for any new work opportunities.
