Morten Holdflod Møller's Projects

  • Football lab

    The project was done in .NET in a semi embedded setting, was completed on a very tight schedule and required a week long stay at the construction site.

  • Økofonen - QR-code based mobile storytelling application

    • November 2009 to Present
    Team Members: Morten Holdflod Møller

  • Village Table - H2 INTERACTION

    • May 2009 to Present
    Team Members: Morten Holdflod Møller

    Installation for an interactive exhibition simulating production, storage and consumption of renewable energy.

    I designed and wrote the controller software interpreting input from four RFID sensors, perform simulation calculations and expressing output to both a custom made LED-hardware setup and a Adobe Flash-based view. It was written to be operated as en appliance and runs on the .net framework.

  • Bluetooth firmware for LEGO MINDSTORMS NXT

    • January 2007 to Present
    Team Members: Morten Holdflod Møller

    I have designed, written and documented the firmware for the bluetooth subsystem of the LEGO Mindstorms NXT.

    The code was C written for the small BlueCore 4 chip. The documentation is published as appendix 3 of the NXT Bluetooth Developer Kit on http://mindstorms.lego.com/en-us/support/files/default.aspx

  • fona.dk ; e-commerce website
