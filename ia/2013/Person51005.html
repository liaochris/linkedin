Gustavo Roberto Rodrigues Gonçalves' Summary

Gustavo Roberto Rodrigues Gonçalves is certified LPI II , NCLA ( Novell Certified Linux Administrator ), DCTS ( Novell Data Center Technical Specialist ), with 15 years experience deploying Open Source Security Softwares ( Firewalls, IDP, IPS, HIDS ) .

Gustavo works in TJES ( Tribunal de Justiça do Espírito Santo - Espírito Santo Court of Justice), managing servers : SLES ( SuSE Linux Enterprise Servers - Novell ) and OES ( Open Enterprise Server - Novell ).

He has spoken at a number of open source and security conferences ( Vale Security Conference, Security Day - BB ), and some Technical Schools. He is OWASP chapter leader at Vitória, Security Researcher Enthusiast ( Malwares and Reversing ) and have overwhelming desire to take things apart.

Special Interests : PCI, Pentest, Physical Security
