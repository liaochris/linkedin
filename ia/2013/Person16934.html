Brent Edwards' Summary

Brent is an accomplished software developer, architect, mentor and speaker with over 8 years of experience in developing innovative solutions with Microsoft technologies. He has presented on such topics as jQuery, WPF with MVVM, Android development, Windows 8, and Windows Phone 7/8 development at several technical conferences and user groups. Brent is also a co-founder and leader of the Twin Cities Windows 8 User Group (http://windows8ug.com/). Additionally, Brent is the author of MVVM Fabric (http://mvvmfabric.com), an open source library for WPF which helps address communication, navigation and commanding in an application leveraging MVVM, and Charmed (https://github.com/brentedwards/Charmed), an open source library for cross-platform MVVM development with Windows 8 and Windows Phone 8.

Brent has substantial understanding of object oriented methodologies and design techniques and understands the need to adhere to best practices throughout the development life-cycle. In addition to middle tier development, Brent is well versed in front end technologies, including HTML, CSS, Javascript, WPF, Silverlight, and Windows 8. He understands the need for keeping clean separation of markup, styling, and functionality.

In addition to Microsoft technologies, Brent has a proven record of quality development with the Android OS. His apps have a download total of over two million. He was also selected as a Top Android Developer during Google’s Device Seeding Program.

Specialties: .NET, C#, ASP.NET, ASP.NET MVC, Javascript, Ajax, HTML, CSS, WPF, Silverlight, Windows 8, WinRT, Windows Phone 7, Windows Phone 8, XAML, MVVM, Dependency Injection/Inversion of Control, XML, Android, CSLA

Brent Edwards' Projects

  • MVVM Fabric

    • December 2010 to January 2011
    Team Members: Brent Edwards

    MVVM Fabric is a WPF-centric library which helps address common issues such as communication, navigation and commanding in an application leveraging MVVM.

  • MyVote

    • February 2013 to March 2013
    Team Members: Brent Edwards

    MyVote is a Windows 8 and Windows Phone 8 application developed for the Modern Apps Live! conference. MyVote was intended to demonstrate modern app development techniques, issues, tricks, etc. MyVote has a cross-platform presentation layer, written with C#/XAML, which leverages MVVM, Dependency Injection, Inversion of Control, and abstraction techniques to achieve it's goal of staying testable.

  • Charmed

    • April 2013 to Present
    Team Members: Brent Edwards

    Charmed is a cross-platform MVVM support library for Windows 8 and Windows Phone 8. Charmed isn't intended to replace your favorite MVVM library, but rather to enhance it. Windows 8 and Windows Phone 8 have some cool features, but they usually come with the cost of making your view models difficult to test. Charmed is intended to abstract those features out so that you can still use them, while keeping your presentation layer testable.

  • Meet Me Halfway

    • October 2012 to February 2013
    Team Members: Brent Edwards

    Meet Me Halfway is a Windows 8 app that helps you find a place to meet halfway between you and a friend. It features a clean design and integration with the Bing Maps SDK and Bing Maps API to show points of interest on a map and provide directions. Meet Me Halfway leverages key features of Windows 8, including Sharing and Searching, to provide an excellent user experience.
