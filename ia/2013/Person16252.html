Brad Garrison's Summary

ACCOMPLISHMENTS:

Automated on-boarding process between HR, Information Security and Information Technology by eliminating paperwork and shortening the process from weeks to days. This was accomplished by getting a new user setup with access to the network and email; and Automating email creation in the Cloud for new users using .NET, PowerShell and other scripting tools.

Automated Active Directory updates from company’s payroll system. Each night user objects in Active Directory received updates automatically to more than 30 attributes.

Replaced company’s manual e-mail distribution groups with dynamic distribution that auto-update. Greatly reduced manual effort for email communications process while improving data accuracy.

Created several Active Directory reports for Information Security showing user attribute exceptions. Reports used daily as a checklist to ensure all new users are created/deleted quickly with proper access.

Created company-wide file share using access-based enumeration. All users company wide map to the same drive letter and path, but only see the department folders they work in. Reduced company login script from 20+ pages to two pages.

Created automated process to purge user home drives and personal files from SAN when the users were removed from Active Directory.
