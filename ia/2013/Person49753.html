Graham Pentheny's Summary

I'm a computer scientist and game developer with industry experience in AI and graphics programming. I have extensive experience designing and maintaining large, custom game engines, with a focus on cross-platform portability and multi-threading. I have a deep interest in programming language design, which drives me to understand and apply many different programming styles and languages. I wish to continue to progress the field of AI through new perspectives and approaches to solving the problems faced in games and software today.

Specialties

Artificial intelligence, game engine architecture and development, realtime 2d and 3d computer graphics, game engine architecture, adaptive videogame music systems, iOS/mobile game development, object oriented analysis and design, server hardware software and maintenance.

Graham Pentheny's Publications

  • Efficient Crowd Simulation for Mobile Games

    • Collected Wisdom of Game AI Professionals
    Authors: Graham Pentheny

    This article provides an in-depth examination of the crowd dynamics system developed for the tower-defense game Fieldrunners 2. The system is capable of simulating organic, fluid, interactive crowds of thousands of autonomous agents efficiently on mobile devices. Motivations, design choices, implementation specifics, and integration strategy for the proposed simulation method are each explored.
