Martin Reddy's Publications

  • API Design for C++

    • Morgan Kaufmann
    • February 14, 2011
    Authors: Martin Reddy

    This book focuses on various Application Programming Interface (API) styles and patterns that will allow you to produce elegant and durable libraries in C++. It offers in-depth discussions of interface design, documentation, and testing, as well as advanced topics of scripting and plug-in extensibility.

  • Level of Detail for 3D Graphics

    • Morgan Kaufmann
    • August 5, 2002
    Authors: Martin Reddy, David Luebke, Jonathan D. Cohen, Amitabh Varshney, Ben Watson, Robert Huebner

    Level of detail (LOD) techniques are used by professional real-time developers to strike a balance between breathtaking virtual worlds and smooth, flowing animation. This book is a compilation of the mechanisms, principles, practices, and theory needed by every graphics developer seeking to apply LOD methods.
