Tarek Majdalani's Summary

Tarek Majdalani manages OmniBus Automation & Media Management System along with Harris Video Servers, AP ENPS News Room system, Miranda ImageStores, Beehive WASP3D Graphics, Video Logger solutions and Avid Thunder XL / FXDekoII / DekoCast systems.

He also supervises and maintains Audio/Video Editing suites, such as Harris NewsFlash/VelocityQ, Apple Final Cut Pro, Autodesk Smoke, Avid Pro Tools HD, Avid Media Composer with Avid Unity MediaNetwork and Avid DS Nitris .

He has an extensive background in IT as well. Graduated as a Computer Engineer from the Lebanese American University in Lebanon and a Microsoft Certified Systems Administrator on both Windows Server 2000 & 2003 and for the 6th consecutive year, he has been recognized by Microsoft and awarded the Microsoft Most Valuable Professional.

On the Internet Web Boards, he is known as ElMajdal. He is the founder of http://www.elmajdal.net, which is a dedicated website covering in it technical articles for TMG/ISA Server 2004/2006, Windows Server 2008, Windows 7, Windows 8 and MAC OS X. He is also a Moderator on both ISAServer.org Web Boards & Microsoft TechNet Edge Security Forums, with more than 6000 posts.
