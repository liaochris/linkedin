Joseph Hill's Projects

  • Jirex - Jira Extender

    • May 2012 to Present
    Team Members: Joseph Hill

    This is a productivity tool to enable faster input of bugs into Jira. The tools main use is for bug standardisation across all Project Teams which each require their own unique standards. The tool is dynamic and can easily be altered for new project demands as well as totally new projects. The tool is coded in VB.NET and uses URL manipulation to enter issues into Jira without a need for API / Back-end access.

  • BUMP - Bigpoint Unified Messaging Platform

    • October 2012 to Present
    Team Members: Joseph Hill

    This is a tool to enable a standardised, easy to see communication tool for such things as per Test Case completion tracker, bug entry tracking, task specific questions and query tracking as well as other forms of communication. The tool was mainly built to enable QM PM's in Hamburg, Germany see a real-time view of what is going on without any productivity being lost in IM communication. The tools basic method of entry is a textbox where "twitter" style updates will be posted by every tester on the team once they have tested assigned test cases.

    The tool is currently in a Alpha state and is being trailed by the Malta team
