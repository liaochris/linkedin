Larry Azevedo's Summary

I specialize in user experience/interface (esp. e-Learning and Financial), Web Design, HTML, CSS, Ecommerce, Web Standards, Web Accessibility, and Web Usability. I have a proven track record of designing elegant, easy to use web applications that empower regular people to perform complex tasks.

Along with a focus on usability, I also believe in clean and valid HTML/CSS. I realize that this lends itself to effortless maintenance, a broader audience, small file sizes, cross-browser consistency and higher search rankings.
