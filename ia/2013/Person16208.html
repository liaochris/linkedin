Brad Lilley's Summary

I am a 25-year-old we designer/developer living in Nashville, Tennessee. I specialize in standards-based XHTML and CSS web development with an emphasis on information architecture and user interface design. In addition to my technical knowledge, I have a strong background in studio art and extensive training in visual fine arts.

As an artist, I approach each and every site that I build as a work of art. This means that every site that I develop is built to be visually appealing as well as standards compliant and search engine friendly. I am also an advocate of web accessibility standards.

Specialties

HTML 5, XHTML, CSS, Site Mapping, Usability Testing, User Interface Design, Search Engine Optimization (SEO) , Project Management, Corperate Branding (i.e Letterheads, Brochures, etc.)
