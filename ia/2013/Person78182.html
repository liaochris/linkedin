Lance Osborne's Summary

Lance Osborne is a web design professional with more than 16 years experience in web strategy, interaction design, and open source web development.

Lance has exceptional knowledge of hand-coded HTML and CSS, web standards, accessibility, and usability testing. He is known for a user-centered approach to creating clean, simple, easy-to-use interfaces that meet business objectives and user needs.

Lance has a rare combination of creative and technical expertise with an excellent understanding of how design impacts development (and vice versa).

Specialties:

accessibility, e-commerce, hand-coded HTML and CSS, information architecture, interaction design, open source web development, ProcessWire CMS, prototyping, simplicity, training and education, usability testing, user interface design, web standards, wireframing, WordPress
