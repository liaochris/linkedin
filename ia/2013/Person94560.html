Mike Schinkel's Summary

If you only need to contact me, just use mike-(at)-newclarity.net.

I run NewClarity LLC and we architect and build web solutions based on WordPress as a platform for agencies, enterprises and software companies. We also offer team training for agencies who want to hire developers and bring them up to speed on using WordPress as an application platform.

I am also launching a new startup named TeamBldr with my co-founder Sherra Bell. TeamBldr aims to empower team leaders to build and manage better project teams via a software-as-a-service (SaaS.)

Highlights of my experience includes:

- WordPress Platform Architect of Coca-Cola's Global Marketing Intranet for THINK, Inc.

- CTO for Great Jakes, the company that offer the WordPress-based Law Firm CMS.

- Open source offerings for WordPress and RESTful APIs on GitHub at http://github.com/newclarity

- Blogging at HardcoreWP: http://hardcore.wp.com

- Launching "The Business of WordPress Conference"

- Running the Atlanta Web Entrepreneurs for web startups from 2007 through 2011.

- Founding and growing Xtras, ranked #123 on the 1999 Inc 500.

Current Specialties: WordPress application platform architecture and development, RESTful API architecture and development.
