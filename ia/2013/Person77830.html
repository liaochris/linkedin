Kyle Schweighauser's Summary

Kyle is an experienced Product Manager, Project Manager, and Business Unit Manager. With a technical grounding in Linux system administration and XML data integration, Kyle established and led teams that built online learning systems like Virtual Biology Labs, Sociology Now, and History Unbound for Thompson Learning (now Cengage Learning).

Kyle’s technical skills run the gamut from Java programming to PHP, to Perl, to advanced server setup and administration. As a Project Manager he’s worked in Scrum framework, Critical Chain, Extreme Programming, and Agile Programming methodologies. As a team leader and Project Manager Kyle specialized in helping disparate organizations work together, in motivating people toward a common goal. Clear communication and realistic technical planning are his strong suit. Kyle has worked for Lunar Logic, Google, Right Media and Yahoo!
