Andrew Woods' Summary

OBJECTIVE:
To craft intelligent and accessible websites that solve business problems and provide a positive user experience.

CAREER PROFILE:
Web Developer creating websites since 1999, utilizing the best of open source technology. Early on, my focus was on developing a strong server side skill set, developing database driven web sites and automating administration tasks with Perl. in 2003 I transitioned to developing web applications with PHP. Even today, PHP is my primary server side language. In 2006, I realized the importance of design, and began studying an array of design topics and their role on web design. Some of these are color theory and it's effect on accessibility, the importance of context on the user experience, and the use of grids for organizing information. This resulted in making me a more well-rounded web developer.
