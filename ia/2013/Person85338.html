Marc Shifflett's Summary

Mobile Responsive Design or bust!

Lately, I have been involved in a number of projects to create open source design tools and frameworks to help designers produce style tiles, iterative code, and ultimately functional web pages using the Sass CSS preprocessor combined with MVC & CMS platforms

Mobile devices are clearly the new modality to track and deploy day-to-day, minute-to-minute tasks. My goal is to communicate innovative designs into sane applications, ideally peered so that users can make choices with the confidence that their data is secure.

The skills I can bring to any team are in the areas of content analysis, template design, user interface, user experience, structured data and coding.

Specialties: Experience with interface and template logic for a variety of Blogging and CMS platforms: including Wordpress, Drupal, MoveableType and Krang.

Mobile frameworks include JQTouch and Sencha. PhoneGap mobile simulator and native compiler. Xcode. RubyonRails with Heroku.com

MAMP, SVN and Git for site development, management, and versioning. Adobe applications suite and a wide range of productivity apps on Mac OS X.
