Erik Woods' Projects

  • Swanson (internal email tool)

    • April 2013 to Present
    Team Members: Erik Woods

    I created a new process and application to support the process. This was instrumental in winning a large chunk of new business for our biggest client and will allow us to take on more work without hiring more staff. More money in, less money out, and a more consistent and high-quality product. What more could you ask for?

    Swanson creates and manages email campaigns. Emails are built up-front with placeholder assets and then development time after that is cut down to practically nothing. Creative teams utilize the tool to help them change the default assets, which also saves time across the board by getting rid of certain documentation and other files in the standard process.

    I built this tool using the CodeIgniter PHP framework along with heavy usage of jQuery. I pretty much built my own editor, as well, because I was not satisfied with the options available and wanted more control over the code that goes into the final product.

    I also worked on the design of the tool, along with the name of the tool (Swanson). I chose this name because I really enjoy the show Parks and Recreation - and I particularly like Ron Swanson's quote: "Never half ass two things. Whole-ass one thing."

  • DOW Powerhouse | Savings Estimator

    • June 2012 to November 2012
    Team Members: Erik Woods, Kate Peach, Jessica Bush, Holly Ensman-Attebury, Eliza Nascimento, Lauren Victory, Catherine Wagner

    I was told that there was applause during the client review for this project. THIS is the reaction I strive for!

    Using DOW's pre-existing site structure and creative assets made by team members, I was responsible for taking those assets and creating an easy-to-integrate section of code which would not have conflicting styles or scripts with the client's existing site.

    The application uses jQuery for the animations, XML to store the various data for results, CSS transitions for rollover effects, and CSS keyframe animations for some of the less important effects. (The cloud animations were added by the DOW dev team.)

    Direct link to jQuery written: http://www.dowpowerhouse.com/assets/financial-tool/scripts/calculator.js (Note: this is subject to change at any moment by the DOW dev team and may not accurately reflect my work done.)

  • Consumers Energy | Powering Michigan

    • July 2012 to September 2012
    Team Members: Erik Woods, Iain Lanivich, Elizabeth Linabury, Holly Ensman-Attebury, Mark Montri, Lynn Stark, Michael Pietersen,

    I wrote the HTML, CSS, and jQuery by hand. Throughout the project, I made the suggestion to keep the site responsive and worked directly with the designers and other team members to make that happen, even though it was not planned for. I really care about going the extra mile, when possible.

    Note: This project is no longer in production and therefore may have a few minor issues with iOS devices.

  • DEA | Survivor's Benefit Fund

    I assisted a team member with the basic front-end structure of this site by writing the HTML and CSS for the header, navigation, masthead, and footer areas as well as one secondary page - along with several reusable styles seen throughout the site.

  • CDC | Making Health Easier

    • August 2011 to June 2013
    Team Members: Erik Woods, Lynn Stark, Lauren Menna, Ben Wright, Mark Montri

    I came up with several creative solutions to allow us to stretch the Ning platform to its limits in order to provide our client with a social community which they use to promote healthy living. Using RSS feeds and jQuery, I've re-worked how the Ning platform is supposed to function - to force the experience the client needed. There were several instances in which the project was in jeopardy, and I came up with a solution to each of those problems.

    Being involved with this project gave me the motivation to share my knowledge further with the Ning Creator community.
