Brian Van Watson's Summary

I have over 15 years of solid experience as a Programmer, Technical Writer and Copy Editor, working mainly in the Banking and Finance industries. I have designed and written user guides, system admin guides, API/Developer documentation and training materials for major, international firms and organizations. I have also copy edited and rewritten instructional design materials and scholarly journal articles involving IT, Accounting and Finance. I have a basic knowledge of the following programming and scripting languages: Excel VBA and MS Word VBA, Perl, HTML, XML, XSLT, SQL, PHP, JAVA, C, and C#.

Specialties

In addition to technical writing, I specialize in designing and developing efficient, and cost-effective IT project inventory tracking and reporting applications using advanced MS Office/Excel VBA/.Net and SQL Server automation. These utilities are generally used by Support, Project Management, QA and Development to track project artifacts (such as stored procedures, database IDs, program modules and project resources). I develop scripts in Perl or VBA to automate repetitive documentation tasks.
