Keith Gibbs' Summary

I'm a web interface and mobile application developer with a broad range of skills (and an eclectic background), bringing the following to the table:

• Extensive practical experience in web application and interface architecture, design, and development, encompassing HTML5, CSS3, AJAX, XML, REST services, and JavaScript development (in standard ECMAScript and modern JavaScript toolkits). Recent, intense work with Backbone, Grunt, and Node.js.

• Proven knowledge of a variety of backend scripting and programming languages including PHP, Java, JSP, C#, Objective-C and ASP.net. Modest experience with MySQL, SQLite, MS SQL, and Oracle for data-driven web applications. Familiarity with Python, and Ruby on Rails.

• Skilled in mobile web technologies, standards and best practices. Developed native and hybrid iOS apps using C# (using Mono), Objective-C, and web technologies, in turn training and mentoring developers locally and throughout the System.

• Deep knowledge of IBM WebSphere Portal/WAS portlet and theme development, vastly customizing and extending out-of-the-box capabilities of platform for intranets. Experience with developing and customizing interface and backend function for variety of enterprise and open-source content management and collaboration systems.

• Skilled in writing text and multimedia documentation for broad user-base/public use, as well as broad group and one-on-one training and consultation experience. Presented at numerous national conferences within Federal Reserve System on intranets, content management, JavaScript interface development, HTML5, and mobile application development and design.

• Project leadership and management skills, including serving as deputy-chair for a national web communicator’s group for two years, responsible for planning and running an annual conference and orchestrating multiple technical workgroups. Skilled in agile and scrum development methodologies, serving as Scrum master on a number of critical projects.
