Doug Domeny's Summary

Experienced team leader and principal software engineer. Detail-oriented, quality-minded and customer-motivated. Contribute significantly as an architect, designer, implementer and mentor, to the success of the software department and company. In my role as software engineer, developed significant skills in software engineering and architecture, UI design, and technical writing.

Developed a browser-based, multilingual, business user-friendly XML editor written using XSLT, W3C XML Schema, DHTML, JavaScript, jQuery, regular expressions, and CSS. Served many years on OASIS technical committees such as XML Localization Interchange File Format (XLIFF) and Open Architecture for XML Authoring and Localization (OAXAL).

Bachelor's degree in computer science and mathematics from Gordon College in Wenham, MA.

Specialties: ASP.NET, HTML, XHTML, DOM, CSS, Web, Web Services, C#, VB.NET, XML, XSLT, XPath, JavaScript, jQuery, XLIFF (localization, L10N, I18N), W3C XML Schema (XSD), SQL
