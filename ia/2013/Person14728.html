Bart Bakker's Summary

My career started being a Delphi software engineer at Intersoft Software Engineering developing administrative desktop applications targeted specifically to the facility services industry. At this job I learned object oriented programming and learned a few of the key agile practices like using user stories, whiteboard designs, TDD, and pair programming.

About 4 years later I moved on to a Java software developer position at Compare Group where I was part of porting existing web applications to the Java platform. Compare Group owns the product and price comparison websites Vergelijk.nl, Elcheapo.nl, the Belgian Vergelijk.be, and the Finnish Vertaa.fi.

In June 2008 I moved to IPROFS where I became a Java consultant. IPROFS is specialized in Java consultancy and has about 60 employees.

My first consultancy job for IPROFS was at the Dutch central bank (De Nederlandsche Bank, DNB) where I worked on an update of an existing business application. The update consisted of the introduction of some new features and solving bugs. Due to my agile approach and close customer interaction the requirements got clear and the update got accepted and went into production successfully.

For my next and present consultancy job I remained at DNB and joined CCBM2, a large European Central Bank (ECB) project done by a cooperation between the National Belgian Bank (NBB) and DNB.

Specialties

Well-known with Java, JPA (1, 2), Spring Framework (2, 3), Hibernate, JUnit, JMock, JSF (1, 2), Ant, Maven 2, XML, and more.
Familiar with C++, Python, Bash scripting, Perl, XSLT, Apache web server, Tomcat, Glassfish, and more.
Operating systems: Windows, Linux, and Mac OS X.
