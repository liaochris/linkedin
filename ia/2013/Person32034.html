David Ames' Summary

As of September 2012 I have started teaching Computing to various classes at Holy Family in addition to teaching Maths. As a result I have been selected by Computing at School as one of their Master Teachers so that I can deliver CPD and support to colleagues who are teaching or looking to teach Computing without them necessarily having a Computer Science background.

I have been a classroom teacher for the past 20 years with all associated skills which this requires (eg communication, multi-tasking, planning, etc).

I have also recently completed a part time degree in Computing at the University of Bolton, which has involved lots of courses on programming (mainly C++, with some .net and a bit of VB6), and Databases (using SQL with MS.Access and MySQL and both SQL and PL/SQL with Oracle and Oracle Forms). I achieved a First Class Honours Degree in the process and also picked up a Governors' Prize from the University for Outstanding Academic Achievement.

In my spare time I have also learnt a fair bit of Python and some PHP, which has enabled me to develop the school website using Django and to set up the Schools VLE (Moodle).
In the last few years I've taught Animation using Flash and other tools to various groups of Pupils. I have also used various programming languages to create a variety of small applications which I use with classes (eg an application to generate questions on solving equations written in C++ and Qt). I would like to learn more Common Lisp (especially for web based stuff) but I have to fit this in amongst all of my other responsibilities.

In September I will also be developing and teaching our new GCSE Computing course, probably using Python as the main programming language.

A Masters in Computing is a possibility at some time in the future but not just yet.
