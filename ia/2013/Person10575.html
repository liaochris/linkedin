Andrew Stromme's Projects

  • Research: Creating the Chestnut Programming Language

    • May 2011 to Present
    Team Members: Andrew Stromme

    Chestnut is an experimental language that I created while researching ways to simplify general purpose GPU programming. This summer a prototype of the Chestnut compiler was implemented in Python; this compiler transforms Chestnut code into CUDA C. I worked with my advisor Tia Newhall from the Swarthmore CS Department and will continue my research this fall for my thesis.

  • OpenHouse iOS Music Player

    • March 2011 to Present
    Team Members: Andrew Stromme, Ryan Carlson

    OpenHouse is an iPhone music application with a running playlist. Guests can connect wirelessly and recommend songs and the host can see what is popular. I started this project with Ryan Carlson during our software engineering course and released a first release over the summer. OpenHouse is under active development.

  • RFID Card Entry System for SCCS

    • January 2011 to May 2011
    Team Members: Andrew Stromme

    I am part of a student-run group at Swarthmore that provides technology resources for Swarthmore students. We have a lounge and a game room on campus and needed a better way to allow access to these spaces. With the help of other SCCS members I created a system that uses the RFID chips already present in the student ID cards to allow access to this space. It involves an arduino micro-controller, door strikes and integrates with our LDAP account system.
