Bill James’s Summary

After college, Bill started work at SAIC on a ballistic missile defense simulator, working in C on Unix to develop the model and X11/Motif user interface for a quarter-million lines-of-code anti-ICBM system simulator.

Leaving SAIC for the games industry, Bill joined BlueSky Software and worked on game simulators and interface screens for the World Series Baseball games on the Sega Genesis platform (1994, 1995, 1996). As the company transitioned to the Sony PlayStation, Bill worked on several un-published baseball games as well as a PC-based NFL Managerial simulator.

After BlueSky, Bill worked at Sony Online Entertainment. First, in the Casual Games division as an engine architect and game developer in Java (for Applet-based games). Then, as a developer on the EverQuest MMORPG, developing features and user interfaces as well as fixing game bugs in all systems while publishing 7 expansions over 4 years.

Looking for something new, Bill joined Qualcomm, developing and maintaining web applications for use by internal and external customers. Bill is now a tech lead in his IT division, doing research on Web Services, JSF, AJAX, and other web technologies in addition to supporting his previous applications and training new staff.
