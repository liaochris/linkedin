Jim Gleason’s Experience

  • Vice President of Sales

    Qumranet

    (Privately Held; 11-50 employees; Computer Software industry)

    December 2007 — Present (2 months)

    Qumranet’s desktop virtualization solution Solid ICE, goes beyond conventional server virtualization and application streaming by completely virtualizing end user desktop machines. User desktops run in a KVM (Kernel-based virtual machine) on a server in the datacenter. Users remotely connect with a thin client or laptop via an optimized connection protocol called SPICE or with RDP. The end result is a seamless interactive work experience, just like a physical PC on a LAN.

    - Ramp up offshore developers securely & quickly on virtual desktops

    - Decrease downtime by moving a running virtual desktop from one physical server to another in under 1/100 of a sec

    - Reduce deskside visits and support because user PCs are virtualized and managed from a central server

    - Simplify software upgrades and provisioning of pools of desktop PCs

    - Support obsolete apps by running them on Solid ICE desktops

    - Add security & prevent the unauthorized access of data on a lost or stolen laptop
