Matthew Roche’s Summary

Matthew Roche is an experienced software architect, developer and trainer with a unique perspective into the Microsoft developer zeitgeist. His current role as Chief Software Architect of Integral Thought & Memory allows him to stay on the cutting edge of Microsoft’s tools and technologies, both as a hands-on software architect and business intelligence consultant and as a trainer, mentor, writer and presenter.

Matthew is a Microsoft Most Valuable Professional (MVP) for SQL Server and holds a wide range of professional certifications including Microsoft Certified Trainer, Microsoft Certified Database Administrator, Microsoft Certified Solution Developer, Microsoft Certified Technology Specialist, Microsoft Certified Professional Developer, Microsoft Certified IT Professional and Oracle Certified Professional.

Matthew Roche’s Specialties:

Business Intelligence, SQL Server, .NET, SQL Server Integration Services
