Maurizio Denna’s Summary

Senior SW engineer specialized in embedded, real-time and safety critical software design. Significative experiences on multitasking/multiprocessor and redundant systems.

Maurizio Denna’s Specialties:

Object oriented programming, event-based programming, embedded systems, device drivers, BSP, safety-critical applications, DO-178B standard, neural networks.

Languages: C++, C, Java, ADA, assembler (PowerPC, 680x0, 68HC12, M16C), base knowledge of C#, Python, SQL, HTML, XML.

Tools: Multi, Tornado, I-Logix StateMate, LabView, Vector Cover, Source Safe, PVCS.

OS: Green Hills Integrity, VxWorks, pSOS, MacOS, Windows NT, UNIX, OS/2

Maurizio Denna’s Experience

  • Senior SW Engineer / RTOS Expert

    Vibro-Meter

    (Privately Held; 501-1000 employees; Aviation & Aerospace industry)

    January 2007 — Present (1 year 2 months)

    Design and development of the Board Support Package for GHS Integrity, support device drivers and application framework for the Engine Monitoring Unit (PPC5200).

  • Project Leader/Real time designer

    InfoSolution

    (Privately Held; 51-200 employees; Aviation & Aerospace industry)

    April 2004 — December 2006 (2 years 9 months)

    InfoSolution SpA is a software consulting company working mainly with customers in the avionic and aerospace sectors. In particular I’ve been involved in design projects for Galileo Avionica SpA and SIA SpA:

    Design and development of the Board Support Package and support device drivers for the Galileo Avionica G4MCS board with PPC755 processor and Integrity Operating System using MISRA-C (standard DO-178B)

    Module Support Layer for the Attack Computer / Navigation Computer of the EuroFighter Tranche 2 project.

    Design and development of the control software for the Flutter Generation Unit of the Aermacchi M346 aircraft. This unit is part of the Flight Control System and is used for the dynamic structural test of the airplane. It is based on a PowerPC board with VxWorks operating system. The entire code has been generated automatically from the design using I-Logix StateMate.

  • Coordinator/embedded software designer

    Vesiel

    (Privately Held; 51-200 employees; Consumer Electronics industry)

    September 1999 — April 2004 (4 years 8 months)

    Vesiel SpA is an italian company operating mainly in the vending market, where it designs and produces embedded system for automatic vending machines and cashless payment systems. In Vesiel I had a double role of analyst/developer and coordinator of the software development activities.

    Definition of the software architecture for all new payment systems and vending machine controllers.

    Reorganization of the software production activities to develop reusable modules and introduction of defect tracking and version control methdologies using source code management tools.

  • Analyst/real-time software developer

    TekneData

    (Privately Held; 11-50 employees; Industrial Automation industry)

    October 1996 — September 1999 (3 years)

    TekneData s.r.l. is an italian company operating in the industrial supervisory and control market with a S.C.A.D.A. (supervisory control and data acquisition) software for Windows 95, Windows NT and OS/2.

    Specification and realization of the redundancy module for the acquisition, control and alarm management packages. This module allows to run these packages in parallel on two different computers, while keeping them synchronized to ensure that in case of a failure in one PC, the other takes over, keeping the plant under control and avoiding any loss of data

    Acceptance test of the control software for the Baggage Handling System of the Malpensa 2000 airport

    Realization of a distributed, redundant supervisory system for the security system of the headquarters of a major italian bank

  • Designer and real-time software developer

    Eurotherm

    (Privately Held; 51-200 employees; Industrial Automation industry)

    September 1986 — October 1996 (10 years 2 months)

    Eurotherm S.p.A. is a member of an international group operating in the process control market. My experience has been mainly related to the design and development of real-time control and supervisory systems.

    Development of the software for a data acquisition and alarm management unit, based on dedicated hardware with MC68000 CPU.

    Design and the development of a software for the supervision of industrial processes, based on the MacOS operating system. In particular I was responsible for the integration of the real-time kernel, for the realization of the trend and the graphical representation packages.

    Design and development of the software for a redundant distributed control system, based on hardware VME Motorola with the addition of proprietary hardware for the I/O and the control of the redundancy

    Design of a batch processes controller, based on Intel 80x86 hardware with VxWorks OS, in cooperation with the Eurotherm R&D centre in England.
