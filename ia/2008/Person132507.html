Mike Pihlman’s Summary

Actively looking for opportunities to continue to advance videoconferencing / collaboration and / or teach.

It is time to embrace telecommuting, not only to save gas but to save the environment. See my web site for more information and links: YeOldeTechy.com

In 1992, I beta tested the Cameo Desktop Videoconferencing system and since then have been a staunch supporter / evangelist of desktop videoconferencing.

I currently write Technical and Book Review Blogs. I feel the Blog is the perfect media to show the value of this (videoconferencing and collaboration) technology. Where paper is restricted to words and diagrams a Blog combines words, pictures, movies, hyperlinks, diagrams in a multimedia learning experience for the reader / viewer.

I'm interested (even after all these years) in helping to make remote collaboration as easy and as useful as being face-to-face. Videoconferencing is now becoming mainstream but there is more work needed to make it easy to use.....for everyone.

If anyone has a product or application I can review and write about on my Technical Blog, please contact me at: mike@telbitconsulting.com.
