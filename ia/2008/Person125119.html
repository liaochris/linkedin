Matthew Hogg’s Summary

Web developer with over 9 years of experience. Knowledgeable in all aspects of developing for the Web, but particularly enthusiastic about client-side technologies and user experience. Codes everything by hand. Loves Javascript, clean design, and learning something new. Inspired by the effect the Web has on people’s daily lives. Produces consistent, efficient code and simple, friendly end user experiences by thinking critically and solving problems in simple, but clever, ways.
