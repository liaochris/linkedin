  • Senior Web Producer, Blogs Platform

    (Public Company; 1001-5000 employees; CNET; Online Media industry)

    November 2007 — April 2008 (6 months)

The platform involved a rich user interface for adding structured modules - including inline editing and auto-fill. Most of the work involved was done using Javascript, HTML, CSS and Velocity (VTL)

    Improve the quality of the CNET web experience using modules such as - dynamic toolbar that can be plugged into any CNET property, dynamic carousels that pull in data as required, voting modules and dynamic info modules that allow interaction within the page without page-refresh etc., thus reducing page load time and page-weight and deliver a richer web experience. Used DHTML, CSS, Javascript, PHP to achieve this.

    Delivered location based services (LBS) like address lookup, recall and location based search for the mobile for WAP/XHTML devices as well as a new CPC action called "Click-2-Store".   • Software Engineer

    (Privately Held; 51-200 employees; Information Technology and Services industry)

I worked on discussion forums subsystem and an analytics engine for Appian Enterprise. Both of these subsystems had high scalability, availability and performance requirements.

    The analytics engine analyzes the work-flows in an organization to optimize costs, identify problem areas quickly and increase the turnaround time of processes within an organization.
