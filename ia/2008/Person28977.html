Barrett Snyder’s Summary

I am an avid computer geek, programming my first BASIC program that made a single pixel light up and change colors when I was 13 and I've been hooked ever since.

I've been working professionally in the computer software development industry since 1995 starting at CompuServe Online Services (competitor to AOL at the time) doing technical support and transitioned to the Network Services side as a developer. CompuServe Network Services was then bought by/merged with UUNet, and through a couple more mergers ended up as MCI. There I did a lot of web based Lotus Notes development as well as some C and Java.

In 2000 I moved on to a small startup company called L3Com whose main focus was Java development. There I helped to create web based software for electronic markets. This software was used to auction off billions of dollars in goods, in front of eBay for money transacted until 2004, for anything from cellular spectrum licenses for Italy to electricity for the state of New Jersey.

In 2003 I then branched out to do some out of office consulting at Cardinal Health where I completed a total of 7 Java based projects ranging from a web based money procurement system for requesting project money to order management systems that handled up to 150 transactions a second.

In 2004 I then left L3Com to do some solo consulting for AEP (American Electric Power) where I was a systems architect and tech lead on the first phase of a 5 phase project to create systems and a standalone C# .net client to manage coal inventory and procurement. The leap from Java to .net was much less painful then I thought it would be;)

At the end of 2005 we moved from Columbus to Las Vegas and I transitioned to stock trading full time while still doing some software architecture and development on the side.

With a new child on the way at the end of 2006 I landed at Vegas.com to do more Java based development and architecture.
