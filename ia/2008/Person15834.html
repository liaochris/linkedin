Ariya Hidayat’s Summary

Currently in the final stage of a doctorate research.

Experienced in cross-platform application development using C/C++. Familiar with practical aspects and good practices in software development. Experienced in GUI desktop application development using Trolltech's Qt.

Experienced in digital microelectronics design and implementation, from microcontrollers to FPGA. Extensive laboratory experiences, including (but not limited to) automation of system experiments and measurements. Familiar with core components in optical transmission system. Authored and coauthored over a dozen scientific publications.

6 years involvement in the KDE community, specifically for its open-source office suite. Experienced in reverse engineering of proprietary document formats (e.g. Microsoft Excel, WordPerfect, Lotus Ami Pro).

Involved also in some other open-source projects, such as libwpd (library for WordPerfect document), libwpg (library for WordPerfect graphics), and SpeedCrunch (multiplatform usable desktop calculator.)

Ariya Hidayat’s Specialties:

C/C++, Qt, software development, open-source contribution, embedded system, VHDL, FPGA, measurement automation, optical transmission

Ariya Hidayat’s Experience

  • Scientific Researcher

    Universität Paderborn

    (Educational Institution; Research industry)

    2003 — Present (5 years)

    Conducted research and experiments on the polarization aspects in high-capacity multichannel optical transmission system.

    Realized the world fastest, record-breaking ultra-fast FPGA-based endless polarization controller (for optical polarization multiplexing and PMD compensation). Involved in a team that has achieved world-record total C-band capacity of 6 Tb/s. Carried out a lot of development using optical components, digital microelectronics, FPGA, instrument interfacing, etc.

    Experiment results have been presented in conferences and journals, e.g. Optical Fiber Communication (OFC), European Conference on Optical Communication (ECOC), Asia-Pacific Optical Communications Conference (APOC), Journal of Optical Communication (JOC), Optical Fiber Technology, Photonics Technology Letters (PTL).

  • Software Developer

    KDE.org

    (Non-Profit; Computer Software industry)

    2001 — Present (7 years)

    Add features, fix bugs, optimize and polish KOffice applications.
    Implement over a dozen document conversion filters for KOffice.

  • Author & Core Developer

    SpeedCrunch (http://www.speedcrunch.org)

    (Computer Software industry)

    2004 — 2007 (3 years)

    Initiated and developed SpeedCrunch, a free and open source cross-platform scientific desktop calculator implemented in C++ and Trolltech's Qt 4. Available by default on Kubuntu Linux. Also ready to run on other Linux distributions, Windows and Mac OS X.

  • Developer

    libwpg project

    (Non-Profit; Computer Software industry)

    Sponsored by Google (Summer of Code 2006).
