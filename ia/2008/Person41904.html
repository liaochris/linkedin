    Celera

    (Public Company; 201-500 employees; CRA; Biotechnology industry)

    October 2001 — April 2007 (5 years 7 months)

    While at Celera, I designed and implemented a data warehouse, including a custom ETL system, using PL/SQL and Java scripts embedded within an Oracle 10 database. I was senior designer and developer of a LIMS system for controlling 50 ABI Polymerase Chain Reaction (PCR) machines running in parallel.

    I integrated Plate Tracking module into the LIMS using customized IDBS ActivityBase software. I designed and implemented all relational schemas for the LIMS. I wrote Java applications to access data within the LIMS. I also wrote C and C++ programs to implement Bioinformatics algorithms. I wrote Visual Basic programs to control Plate Cranes for feeding plates to liquid dispensing and PCR machines. I wrote requirements documents and HTML User Guides using RoboHelp.

  • Senior Software Engineer

    ID Business Solutions (IDBS)

    (Privately Held; 51-200 employees; Computer Software industry)

    March 1995 — September 2001 (6 years 7 months)

    At IDBS, I was lead designer / developer of a data warehouse utilizing the star schema model of Oracle 8i Enterprise Edition. I designed and implemented a customized ETL solution for the warehouse. I had sole responsibility for the design and implementation of the Oracle database for the company’s High-Throughput Assay Screening application – ActivityBase. I developed database triggers, PL/SQL packages, SQL scripts and application links to the database via the Oracle OCI library.

    I trained new developers and consultants to show them how to create, customize and maintain product databases. I wrote many programs, utilities and sub-modules using Microsoft Visual C++. I modularized ActivityBase code using static and Dynamic Link Libraries (DLLs), reducing the size of the application and allowing shared access to standard functionality.
