Stephen Adams’s Summary

Over 20 years of experience in Information Technology with in development positions in various vertical markets offering a wide range of valuable experience. An effective communicator, professional, with the ability to work individually, using own initiative or as a team, under minimum supervision with innovation and creativity.

Over the years I have participated in all areas of the Systems Development Life Cycle (SDLC).

I have architected, designed and developed systems that are currently being used by 6,000 plus users globally. This included gathering business requirements, data conversion, systems implementation, user training and writing supporting documentation.

I have an in depth knowledge of Microsoft Technologies and always participate in local forums and user groups.

During my last 5 years, I have focused on developing applications in Microsoft.Net 1.1, 2.0 and 3.0. I have specialised in developing business layer components using Web and Remoting Services providing data from Microsoft SQL Server databases and Oracle databases.

I have extensively used Microsoft Enterprise Library for data access, caching and security.

More recently I have focused on using Microsoft.Net 3.0 components like Workflow (WF) and Communication Foundation (WCF).

I have also built User Interface applications using both web and windows forms technologies as well as building a User Interface framework to provide a consistent look and feel to applications.
