Raj Borooah Web Design, Development, CMS, eCommerce’s Summary

I am active in the field of Web Design, Development and Search Engine Marketing since last 10 years. I co-founded X-istech in 1999. At X-istech we offer Web Design & Development and related consultancy services with core focus in Ecommerce Stores and Content management Systems (CMS).

I also founded X-istech LLC in 2006, which is the US extension of X-istech, and formed in order to be closer to our client base in the US. X-istech LLC will offer Web Design, Web Development, Graphics Design, Interface Design, XHTML table less interface development, WEB 2.0 sites using AJAX, Search Engine Marketing and IT Staffing services from New York.

We use ASP.NET with C#, PHP and JSP platforms for back end programming and MS SQL 2000/2005, MySQL and Oracle for database development. X-istech today consists of 45 people team and we offer bleeding edge web solutions, using very current web technology packaged within a value based delivery framework.

In last 7 years of organic growth we have been constantly updating our skills to keep abreast with the very current web technologies. We have been offering web applications using XHTML table less web design along with AJAX since last year and half. We also have had the privilege to work with some of the top web design houses in the world and continue to offer our web design services, which forms the cornerstone of my business.

I can help in:
1) Table less CSS based web designs.
2) Ajax / Web 2.0 web design and development.
3) Ecommerce web design and development.
4) Content Management System based web design and development.
5) Custom web development and programming.
6) Social Networking Sites.
7) Dating and community websites.

And at the enterprise level:
1) Management and Consultancy in offshore outsourcing.
2) Off site IT teams & Staffing.
3) On site consultancy and IT Staffing.

We have developed and have extensive knowledge and track record on all fields above and system demo's can be arranged on request.
