Pete Geiger’s Summary

23 years of extensive system hardware and software architecture, development and executive/management experience.

Developed over 19 products focused on multi-processor server, acceleration, scaling, networking, and graphics controller technology for the high-end technical, enterprise server and PC markets. Applied for more than 40 patents, at least 12 have issued.

Held various executive, director, program manager, product development manager, functional manager & project leader positions.

Staffed and managed various multi-disciplined hardware, software and test organizations for startups, transitional, medium-sized and large companies.

Cross-functionally managed multi-disciplined hardware, software, test, product marketing, and manufacturing programs from small/medium teams, up to 50 engineers, to large matrix-managed teams, up to 150 engineers spanning multiple organizations and sites.

Coordinated development efforts with strategic OEM, ODM, ISV and VAR partners and customers.

Created product development processes regarding program scope & priorities, roles & responsibilities, schedules, budgets, business cases, issue tracking, risk assessments, weekly status, and monthly program reviews. Ensured that product reliability, availability, serviceability, quality, performance, development and product costs, and time-to-market goals were met.

Pete Geiger’s Specialties:

HW - Server ASIC chipsets, scalability ASICs, graphics cntlr ASICs, compression & memory mgmt ASICs; motherboards, backplanes, CPU, memory & I/O PCBs; chassis, power, cooling.

SW - Kernel drivers, APIs, BIOS, embedded, utilities, performance simulators, benchmarks.

Arch - Distributed cache coherency, memory cntlrs, adaptive compressed caching & memory mgmt, 3D, 2D & digital video graphics, crossbar switches, ccNUMA & I/O bridges, MP interrupt cntlrs, system mgmt & network cntlrs.

Pete Geiger’s Experience

  • Director, Product Development & Architecture

    Violin Memory

    (Privately Held; 11-50 employees; Computer Hardware industry)

    August 2007 — April 2008 (9 months)

    Violin Memory Inc. is a high-performance terabyte DRAM & Flash storage appliance startup in Iselin, NJ. As a product development manager, led the development of Violin’s next-generation scalable, coherent extended memory technologies and products which coherently attach Violin’s terabyte memory appliances to AMD and Intel enterprise servers over Infiniband to massively extend host main memory for memory and I/O intensive applications in enterprise, high-performance technical, and storage computing. The first product included the development of novel FPGA, PCB, BIOS, OS, system management and embedded SW technology. As director of architecture, responsibilities included key technology investigations, tactical and strategic product concepts, roadmaps, architecture documents and participation in standard’s committees. As a principal manager in this startup, created and established Violin’s product development processes.

  • Product Development Manager, Architecture Manager

    Newisys

    (Public Company; 51-200 employees; SANM; Computer Hardware industry)

    2003 — 2007 (4 years)

    Newisys Inc. was a fabless semiconductor and enterprise server development startup in Austin, TX. As a product development manager and manager of architecture, contributed to the development of Newisys’ ExtendiScale technology, complex ASICs capable of coherently scaling AMD and Intel x86 enterprise servers up to 32-sockets for use in advanced enterprise, high-performance technical, storage and telecom solutions. In addition, led the development of the industry-recognized Newisys 4300, a 3U 4-socket AMD Opteron server, the Newisys ExtendiScale reference platform, and the Newisys ExtendiScale HTX adapter.

  • Cofounder, CTO, VP Engineering

    Interactive Silicon

    (Privately Held; 11-50 employees; Computer Hardware industry)

    1998 — 2003 (5 years)

    Interactive Silicon developed advanced server acceleration & graphic controller technology. Helped raise $20M over 3 rounds of funding & grow company to 50 people. Led development of TurboDIMM product, a novel adaptive compressed caching technology based on coupling a high-performance compression ASIC integrated on a SDRAM DIMM to sophisticated kernel drivers that transparently interfaced to the file system, VMM & disk drivers of Win2K & Linux. When plugged in a server, TurboDIMM dramatically improved performance of memory intensive & I/O bound enterprise & HPC applications by dynamically increasing effective size of main memory only when needed. Developed MediaFX technology, advanced 3D, 2D & digital video graphics ASIC IP cores, for use in PCs & set-top boxes. Using novel processing, pipelining, caching & compression techniques, MediaFX consumed much less bandwidth, memory & power than conventional graphics controllers.

  • Technical Manager, System Architect, ASIC Designer

    IBM

    (Public Company; 10,001 or more employees; IBM; Information Technology and Services industry)

    1992 — 1998 (6 years)

    Contributed to the development of 3 high-end technical servers at IBM as a principal architect, designer and technical manager developing systems, ASIC chipsets, PCB boards, firmware, system management, OS, performance modeling, chassis, power and cooling solutions for several PowerPC high-end technical servers, one of which was the 16-processor Power3 High Node based on the 3rd generation PowerPC 630 for the RS/6000 SP product line which was used to build the world’s first multi-teraflop supercomputer. The other servers included RamJet, a high-end 18-processor SMP technical server based on the 2nd generation PowerPC 630, and RioBravo, a high-end 18-processor SMP technical server based on a 1st generation PowerPC 630. Key architect of various scalable distributed cache coherency protocols and system management solutions.

  • Project Leader, System Architect, System Designer

    Dell

    (Public Company; 10,001 or more employees; DELL; Computer Hardware industry)

    1991 — 1992 (1 year)

    Project leader, architect and system designer developing enterprise server PCB, ASIC chipsets, chassis, power, cooling and firmware for the Dell Nautilus, a 4-processor Pentium SMP enterprise server featuring a proprietary ASIC server chipset, and the Dell 466D desktop PC.

  • Project Leader, System Architect, ASIC/PCB Designer

    NCR

    (Public Company; 10,001 or more employees; NCR; Information Technology and Services industry)

    1985 — 1991 (6 years)

    As a project leader, system architect/designer, ASIC designer, PCB designer with NCR’s Advanced Systems Development Group, contributed to the development of server ASIC chipsets, PCBs, firmware, adapters, kernel drivers, and utilities for several SMP enterprise servers based on various Intel and Motorola processors, one of which was the industry-recognized NCR 3550, a scalable 8-processor 80486 SMP enterprise server. Other programs included contributing to the development of the NCR Class 10 and Class 20, SMP enterprise servers based on the Motorola 88000 and Multibus II, the NCR 88000 Application Processor module for NCR Tower32/900, a scalable loosely-coupled enterprise server, as well as, the NCR ET105 PC Ethernet adapter.
