Dave Marro’s Summary

I have two separate areas of experience, Medical Device Process Development and Database Application Designer. In each of these I have managed and completed many projects.
As a process development engineer for more than twenty years, I have refined the following skills:
A trained and intuitive understanding of the physical disciplines of engineering.
Design of tests and experiments that are useful, efficient, and fair.
Use of DOE technology, or simple, iterative tests that reflect an understanding of how the device is used by the customer.
Skill at building test fixtures, using machine tools for “one of” construction.
Collection of data efficiently and accurately, using automated data collection, data sheets or lab books.
Writing of procedures, test protocols and reports for use in the FDA regulatory environment, using graphical techniques and statistical analyses.
Database design to make testing more efficient. Important data is safely stored and easily retrieved.
Communication and support for colleagues in engineering, regulatory, quality, documentation & management.

As a database applications designer:
I have build many databases both in an FDA regulated environment and in other industries. I have primarily worked in SQL server and Microsoft Access. I have also build applications with ASP.net front ends. I have done much data conversion from legacy systems as well as working to develop new business processes.
