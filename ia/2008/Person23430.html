    (Computer Software industry)

    December 1999 — June 2004 (4 years 7 months)

  • Technical Consultant

    Computer Communications Specialists, Inc.

    (Privately Held; 51-200 employees; Computer Software industry)

    March 1985 — 1996 (11 years)

    Computer Communications Specialists (CCS) would create virtually any custom system it could sell for the IBM Series/1 EDX platform. As time went on we shifted to the IBM PC. I started work there doing Series/1 programming for Consolidated Edison of New York (ConEd). I later shifted from the Series/1 to the PC and I learned C programming.

    The systems I wrote for ConEd underwent several changes over the years. Originally there had been five programmers working on their PET system and as many working on their FPET system. Eventually I assumed responsibility for all parts of both systems, including code written in assembler, Nantucket Clipper and C, all of which I had modified as ConEd's needs changed.

    ConEd is only one of many customers I have worked with. I worked on many ACCA receiving systems for wholesale companies.

    C is one just one of the languages I have used. I have done many systems with CCS' ScriptWrite. As such, I am primarily an IVR programmer.
