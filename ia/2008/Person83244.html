Jim Raymond’s Summary

Application Developer, QA Engineer, and Project Manager at home in a wide variety of industrial scenarios. Interested in teaming up with other consultants or prime contractors on a project by project basis to help customers develop n-tier web-based IT solutions for their intranets and extranets, using J2EE and state of the art development tools and database management systems. Particularly interested in working on projects where meaningful value can be added, for example by taking part in strategic planning and evaluation of different courses of action before implementation begins. Always interested in implementation projects, where I can build on my technical skills.

Jim Raymond’s Specialties:

Design and implementation of technical information systems; multimedia authoring and production; test automation; l10n; project management; web development; front end requirements analysis and design. Certifications: SCJP (J2SE 1.4). Experience in the automobile, paper, publishing, securities, and computer industries. Bilingual English/German, almost useful French.
