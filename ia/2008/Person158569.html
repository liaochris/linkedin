Reinout Wolfert’s Summary

Mijn passie ligt bij webanalytics en al wat onder die paraplu te scharen valt. Daarnaast heb ik voor het redesign van snsbank.nl twee projecten geleid. Voor één project was ik verantwoordelijk voor het herontwerp van al onze formulieren op het web en voor het andere project was ik verantwoordelijk voor de implementatie van ons webanalyticspakket; Moniforce webSensor.

Samen met Ton Wesseling heb ik Webanalisten.nl opgericht. Een blog over, jawel, web analytics / conversie optimalisatie. Op Webanalisten.nl behandelen we veel praktijk cases en geven we tips om de conversie te optimaliseren.
