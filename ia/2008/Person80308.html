Hugo Shebbeare’s Summary

My role as a consultant is to handle your data, using Microsoft's SQL Server platform or even Access (smaller scale), whether it be for operations, reporting, mining or e-mail marketing. I love to lead teams on technical projects that involve the web and, of course, databases.
My goal is to be an Exceptional DBA as defined by Brad McGehee.
Often, significantly reduce the workload before the end of the mandate - part of a under promise, over deliver methodology ingrained since my internship with Deloitte in '98 (consultants are not permanent, thus I don’t build up a fiefdom to protect). My own network, setup in 2003, is pratically auto-administering itself since my last major network change in 2006. Here are the things I like to do for my clients:
-Project Management with, of course, a heavy database nature or web-application oriented.
-Performance tuning specialization thanks to a myriad of SQL Server management tools.
-Proactively manages his environment by continuously tweaking, until months of stability are established. This is usually done thanks to running automated SQL Server Agent Jobs, with incorporated alert system by SQL DBMail.
-Enjoys writing documentation to allow for a quick, safe rebuild of network infrastructure or for ease when expanding infrastructure.

Hugo Shebbeare’s Specialties:

Consultant, Microsoft SQL Server specialist (MCDBA certified since 2001), managing web-related projects since 1998, certified for project management since 2005. Most recent Database mandates have involved some intense performance optimisation.
