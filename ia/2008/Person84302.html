    (Public Company; 11-50 employees; Information Technology and Services industry)

  • Senior Java Developper

    Azertia (currently Indra)

    (Privately Held; 10,001 or more employees; Information Technology and Services industry)

    October 2003 — October 2005 (2 years 1 month)

    In this period, I was working as a Java Developer designing and implementing a layered business application. It was an evolution of the previous project in which I was working previously (MASTIN), but this one was completely designed and implemented with Java in mind and to work over JBoss and IAS with Oracle as DB. This new version was called AzTramite and included a lot of new functionalities. Amongst it's strong points were:
