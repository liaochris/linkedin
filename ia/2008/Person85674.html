Jason Carter’s Summary

I have been an IT professional since 1993, with 13 years of experience. My employers have found me a highly adaptable, dynamic and multi-purpose resource. My main duties have been to provide technical design, development and testing resource. I have experience with project management and team leading, including liaising with internal and external development teams and providing consultancy and relationship management with clients world-wide.
My main technical abilities are in developing Client/Server applications using Microsoft C++ on all Windows operating systems, incorporating both the Win32 API and MFC. All server applications have been high performance multi-threaded, multi-tiered, scaleable services. Most client projects have been Windows based MFC applications incorporating a number of third party extensions. I have experience in producing API’s for both Server services, allowing custom defined extensions and Client services, allowing the ability to embed into other applications. These API’s have included wrappers for VB, Com, .NET and bespoke O/Ss. I have some experience in Visual Basic, Java and C# development. I also have some experience in developing applications for multiple O/Ss including Unix and bespoke O/Ss. Other experiences have included developing for the Web. I have extensive experience in Relational/Network Database and Query design strategies in many of the Main Stream DBMS Servers including, but not limited to Velocis, Oracle and SqlServer.
