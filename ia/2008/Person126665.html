Michael Boman’s Summary

Michael Boman has throughout the last 7 years specialized in IT security. He has performed numerous security assessments for Singapore local Banks and financial institutions as well as Singapore government linked companies and agencies.

Michael has also authored and lectured several IT security courses on topics such as Linux and UNIX Security, Windows Security, Network Security and Web Application Security. Apart from writing courses he has also been involved in writing and implementing several IT-related policies in line of ISO17799 requirements.

Incident Response, IDS Analysis, Secure Network Design, IT Security Audit, Penetration Testing, Web Application Security Assessment, Source Code Review, Exploit Development, IT Policy Development
