Ronny Crop’s Summary

With over 9 years SAP experience on 11 SAP implementations, Ronny Crop fulfilled both functional, technical and project management roles in multi language SAP projects in Europe and the Middle East.
Within these projects he fulfilled several roles from Senior HR consultant and ABAP/4 Developer to Module Leader and Team leader.
As a certified Functional Human Resources consultant extensive experience in PA - RC - PT - OM - T&E - ESS - PD - CATS.
As a certified ABAP/4 workbench consultant extensive experience in Database Updates, Performance Tuning, Data Conversions, Migration and Interfaces (LSMW, Batch Input BDC, Direct Input SXDA, BAPI, RFC), Dialog Programming, Report development with interactive Lists and Object Oriented ABAP programming, Sap Query and Logical Databases, Enhancements and Modifications.
ABAP developments cover SAP modules HR/PS/MM/SD/PP/CS and CATS.

Ronny Crop’s Specialties:

SAP Human Resources functional Analyst, Technical Analyst, ABAP Developer and solution architect for SAP modules HR, CATS, CS.
