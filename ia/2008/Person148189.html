Paul Salamone’s Summary

Holds a Masters of Science, Computer Science Degree from Boston University and a Bachelor of Science, Electrical Engineering, Magna Cum Laude from the University of Massachusetts Lowell. Have served as Director of Engineering, Chief Technology Scientist, Project Manager, Project Leader, Systems Analyst, and other senior principal engineering roles over the course of a 20-year career in the software business.

Started my own consulting company ten years ago providing software consulting services to several companies including PC Connections, Bemis Associates, Fidelity Investments, Millipore Corporation, Simplex Time Recorder, and Andover Controls.

Specializes in the design and development of Windows or Web Applications for PC-based products or embedded processor-based appliances. Have developed software products for several vertical markets including the Intelligent Building (HVAC), Internet, Intranet, Windows, Multimedia, Video Conferencing, Security, Manufacturing, and Financial industries using local and wide area networks.

Extensive experience with Microsoft operating systems and software technologies including: HTML, ASP, ASP.NET, XML, XSL, SOAP, OCX, COM, SQL, MSMQ, MFC, C++, C#, Visual C++, JavaScript, Visual Basic Script, Java Applets, ActiveX controls, Window Services, and Web Services. Developed software applications for several operating systems including: O/S 2, Windows 98, 2000, NT, ME, XP, Windows Server 2000, 2003. Provided database design and development services using Oracle 8, Oracle 9, SQL Server 2000, 2003, and 2005.

Companies have found my depth of experience in all tiers of Client/Server application development as well as software product management from conception to deployment a compelling value proposition over other outsourcing alternatives. Providing on-site development services using the software industry’s best practices and tools ensures timely delivery and support of critically needed business solutions.
