Bart Bakker’s Summary

I started being a Delphi software engineer at Intersoft Software Engineering, where the core competence of the company was to deliver administrative desktop applications, specially targeted to the facility services industry. At this job I learned developing maintainable software using Object Oriented practices.

About 4 years after this start I moved on to my next job. This is a Java software developer position at Compare Group. The core business of this company is to do product and price comparisons of any kind of product in different websites across Europe, among them are Vergelijk.nl, Elcheapo.nl, and the Finnish comparison website Vertaa.fi.

After working 2 years for Compare Group I have taken the position of Java developer at IPROFS. IPROFS is a software company specialized at Java design and development, and maintaining Java application.

In my spare time I have learned about Design Patterns and agile development practices like XP, Scrum, TDD. I am particularly interested in such aspects of software development and I tend to practice them in professional projects.

While using Firebird RDBMS professionally I was one of the developers of Flamerobin in my spare time. Flamerobin is an open source database management system for Firebird. Flamerobin is written in C++.
