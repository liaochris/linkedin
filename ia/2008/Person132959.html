Miles Bintz’s Summary

Miles is an ambitious and creative embedded software engineer with over 20 years of programming experience. Of those 20 years, 8 have been spent in the consumer electonics sector and cover the gamut of embedded systems development. Miles's most recent responsibilities include leading a team of developers responsible for developing and supporting an embedded Linux offering for Conexant's set-top-box products.

In addition, Miles has ported various RTOSes (developing Board Support Packages); ported existing drivers into new environments; designed, documented and implemented new drivers; evaluated and selected JTAG debug tools; designed and deployed build systems; designed boot loaders with stringent size requirements; focused heavily on embedded Linux kernel development, including BSP, cross-compiler toolchain generation, C library selection, root filesystem generation; ported software onto customer hardware designs; and supported customers through the complete development, integration, test, and production phases.

Miles Bintz’s Specialties:

ARM Processors, C (compilers, linkers, and C runtime libraries), driver design and debug, embedded Linux development, process development
