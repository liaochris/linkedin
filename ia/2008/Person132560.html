Mike Rocha’s Summary

I enjoy software development in C#.NET and Java. I performed C#.NET development most recently, but I did Java GUI application development for about 2 years at Motorola.

Web design gives me a lot of satisfaction. At Motorola, I developed intranet web applications in PHP and Java that interfaced with MySQL and Oracle databases. I have experience with Java, Javascript, XML, XSLT, Schemas and DTD’s, CSS, JSP, servlets, applets, Avalon Framework, Cocoon, Struts Framework.

I find configuration management and custom software tool development gratifying. I wrote many Perl applications (on Windows, Unix, and Linux) and Unix scripts, that integrated software build processes with various tools such as ClearCase, ClearQuest, make, etc.

Embedded Linux development holds an appeal for me as I know how to build working kernels, write basic modules, manage rootfs, packaged RPMs.

Finally, I hold a BSEE in RF and Digital hardware engineering. I performed work in these areas for 8 years at Motorola.
