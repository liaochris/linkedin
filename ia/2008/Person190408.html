Todd Stiles’s Summary

Recent work:
A recent project I was involved with was the creation of a new, centrally administered, enterprise-wide messaging system based on Exchange 2007, designed to replace the multiple e-mail systems used within the customers departments. The new system was targeted to be operational by the end of the calendar year 2007, and involved the migration of approximately 5,600 GroupWise 6.5 users and 1,800 Exchange 5.5 users to the new platform. Our team was also tasked to design and implement a monitoring and reporting infrastructure based on System Center Operations Manager 2007 (SCOM). System Center was implemented to support the operations of Exchange 2007, Windows 2003 Active Directory, ISA Server, and Windows File and Print.

Another engagement that I was involved with this past year was with a customer that was large municipality in Southern California.
Project highlights and accomplishments include:
- Performing a detailed Discovery Assessment of the existing Windows NT and Active Directory infrastructures
- Testing and implementing coexistence for the new AD infrastructure and the legacy NT environment.
- Migrating and supporting 1,700 users
- Migrating and supporting over 350 end-user workstations and laptops
- Migration of user data
- Migrating users off of roaming profiles and creating restricted Group Policies
- Testing and migrating applications into the new Active Directory
- Migrating all Printers to Enterprise Print Servers
- Providing communications, hands-on support, and training to all customer personnel
