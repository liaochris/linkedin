J. C. Wren’s Summary

Embedded systems developer with a wide range of experience on various CPU platforms. Non-GUI based application design for product support, testing, configuration and field service. Very experienced C & assembly programmer, with significant experience in Forth, Perl, PHP, and a number of lesser used programming languages. Strong background in protocol design and implementation. Other strengths include system bring-up (initial debugging), low-level hardware/software interfacing, and several real-time operating systems.

Projects have ranged from small PIC-based protocol converters to x86-based embedded Linux systems for telematics applications to large scale networked access control systems.

Interests include implementing web-based applications, antique computer systems, exploration of new processors, PCB layout, and anything that appears interesting at the moment.

J. C. Wren’s Specialties:

Embedded systems programming (ARM7, MSP430, AVR, PIC, x86, 8052, Z80, etc), Linux applications (C, Perl), software architecture, protocol implementations, hardware design.
