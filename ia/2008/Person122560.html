Mark Rustad’s Summary

• C software developer, programming of all sorts since about 1970.

I enjoy finding simple solutions to programming problems. I like making code simpler and easier-to-understand. I am very often working at the interface between hardware and software, whether it is boot code, kernels or device drivers. I am quite comfortable working in multiprocessor environments, and find avoiding synchronization issues an interesting challenge.

Included on seven issued patents. So far.

I have no interest in going into management, perhaps because I appreciate good management. That is, management requires one's full attention, but my attention is always drawn to technology.

Mark Rustad’s Specialties:

Linux kernel, BSD kernel, device drivers, C programming
