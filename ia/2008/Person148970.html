    (Privately Held; 11-50 employees; Computer Software industry)

Responsible for developing a comprehensive patent portfolio. Participan in Business Rules standards such as OMG PRR & SBVR and W3C RIF.

  • Enterprise Java/Business Rules Consultant

    Western Asset

    (Public Company; 501-1000 employees; Computer Software industry)

    July 1999 — February 2000 (8 months)

    Researched and evaluated all of the industry leading business rules technologies and Java application servers. Designed and implemented a solution by performing the industry’s first integration of Weblogic (currently BEA) application server with the NeuronData (currently HNC Blaze) rule server. This system prevented trades that violate any portfolio's management constraints from being processed. This project uncovered many of the fundamental shortcomings and architectural flaws of business rules technologies available at the time. This was the motivation for founding Corticon.

  • Software Consultant
