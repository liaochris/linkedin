David Belson’s Summary

I am a web designer and front-end developer with over 8 years of industry experience.

My mission is to create simple, easy to use web sites that are accessible to as wide an audience as possible.

All of my work is done by hand following web standards and best practices. That means I am using the W3C’s guidelines for creating valid HTML and CSS to build your web site. By adhering to these methodologies I create web sites that are easily maintained, will load faster and function properly regardless of the browser or operating system.
