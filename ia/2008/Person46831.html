David Hunt’s Summary

Microsoft Office 2001-Present

Senior development engineer lead leading a team of five others building a platform for an Office product to be released in the Office 2007 suite. Employed technologies including: .Net, C#, XQuery/XPath. Duties include: project management and project design.

Microsoft Research -- Component Applications Group 1999-2001

Senior development engineer researching techniques related to developing highly specified and testable systems. Included authoring detailed specifications that were turned into code for tests. Achieved 100% block and arc test coverage over the systems. Over subsequent years of use we encountered only 1 bug.

Developed an internal build system using .Net technologies. The system was completely componentized for extensibility. It used plug-ins to build a database of information about the source files. The source files could be of any type or housed by any system (e.g. standard source files all the way to sources embedded in SQL, for example). This system minimized the amount of information that a developer had to specify/maintain to describe the build and reduced a cross-platform build to a single command. This minimal specification was possible because the system knew all of the dependencies between the various portions of the system and thus the user only specified what needed to be included using pattern matching specifictions. This system is still in use today.
