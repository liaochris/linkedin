Gary Young’s Summary

I've been using Flash making games in 1999, After Flash mx 2004 was present, I felt Flash Player will be the only 1 platform can confront Microsoft in browsers, so I decided to put all my time into studying Flash Platform; I predicted Flex product, by the end of 2004, I quit my job, started to study every aspect of Flash Platform, and I picked Java as the server side language .

Real time collaboration is the future of internet, and my aims were simple and clear, learn everything about Flash Platform, and build an engine for real time collaboration, at that time Flash was treated as a drawing and animation making tool, I have to do everything by myself, I choose Flash Communication Server as synchronizing server, and I coded it 3 times from scratch, from the beginning of 2005 to the Mar. of 2007, I used Flash mx 2004, Flash 8, Flex 1.0, Flex 1.5, Flex 2, Flex 3, Flash communication server, Flash Media Server, Tomcat 4, Tomcat 5, Tomcat6, Flash Remoting, Java Open AMF, Java Spring, Java Hibernate, Mysql 4, Mysql 5 and a lot of other assistant tools.

I'm using Red5 building realtime interaction application instead of FMS now.

After the Mar. 2007, I was totally broken, I found a job where i use Flash building small applications, and I then met De'an Liu, who wants to start a career focus on user making product online, and sharing it. I thought that it's a good idea, so I spent 7 months building http://joloftdev.flashflexpro.com, which using Flash9, Flex2, Flex3, Java Open AMF, Java Spring, Java Hibernate and Mysql5.( it's not finished yet, we had to halt at the end of Oct. 2007, because of some non-tech problem ).

So that's my little simple history which believes in Flash and I devote myself into it.
