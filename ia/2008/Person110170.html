Laia Subirats’s Experience

  • Intership

    Telefónica R&D

    (Privately Held; 501-1000 employees; TID; Telecommunications industry)

    October 2007 — Present (6 months)

    In the "Navigation Innovation" part I make a software in C++ for S60 Symbian OS (Nokia) using concepts of image signal processing which in order to recognise the movement of the mobile using cameras. This functionality is useful to be able for instance to send a message with Bluetooth tecnology to somebody which is pointed by the mobile phone, pass the pages of a calendar of the mobile phone, etc. I also work in the PULGAR project studying the extraction of traffic and mobility information from the mobile network. In this line, we will explore different methods of data extraction and develop different algorithms to process the extracted data in order to infer traffic density and mobility matrices. In this project my work consists in writing the documentation of the used technology, operation, implementation, achieved improvements and future work. Moreover I work in the testing part of the platform and the software application.
