Keith Hamilton’s Experience

  • Software Architect

    Microsoft Corp.

    (Public Company; 10,001 or more employees; MSFT; Computer Software industry)

    2007 — Present (1 year)

    Designed and built a real-time application monitoring solution for Microsoft.com. The solution is handling 10,000+ requests per second. The architecture for the solution is SOA built on the distributed publish/subscribe event system using asynchronous eventing. The solution provides real-time aggregated views of all application running across the entire server farms at both data centers. The aggregation service provides a web service (REST) interface to provide data to the monitoring client console. The monitoring solution provides monitoring across all servers within 3 seconds of what is happening on the front-end servers and provides alerting if the front-end applications exceed their service level agreement (SLA). The solution is integrated with Microsoft System Center to where actions can be taken.

    Published a whitepaper internally on the overall transition of technology from the current Windows paradigm to a software appliance paradigm driven by virtualization and multi-core.
