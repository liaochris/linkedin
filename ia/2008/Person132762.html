Mike Wilson’s Summary

Software engineering professional specializing in high performance search, instrumentation, compilers, data communications, and Lawful Intercept. Solid experience in project management. Well versed in fault-tolerant computing. Enjoys working with diverse, global teams and solving complex problems. Major accomplishments include:

* Redesign and implementation of Tandem Failure Data System on Intel Itanium platform. This new design has a significantly smaller process footprint and is much more stable than the previous implementation on the original platform.

* Lead developer in project to bring the ISO standard C++ run-time library to the HP NonStop platform.

* NonStop Systems Development liaison to WMQIB project responsible for coordinating the efforts of several development groups, and representing HP Cupertino in all project meetings.

* Design and implementation in C++ of library to convert proprietary telephone switch call data messages to J-Std-025-A, an ANSI Lawful Intercept standard.

Mike Wilson’s Specialties:

System level software design, Compilers, LAN design.
Program using C, C++, and pTAL.
Project Management expertise.

Produce technical documentation including design specs, functional specs, white papers, and manuals.

Formal presentation to the annual International Tandem User's Group meeting and led the customer chat room for discussion of compilers and run-time libraries for the NonStop Enterprise Division.
