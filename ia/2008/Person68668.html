Oscar Gensmann’s Summary

Interface designer, programmer and all round developer and system architect, now co-founder and CEO of the web applications development company GMTA ApS (www.gmta.info), creators of the newsletter administration platform "eBrev" (www.ebrev.info - danish).

Selected references and work experiences through time: Jubii.dk, Lycos Europe, De Gule Sider (European directories), Bold.dk, Urban blog, Link Logistics, Soco System.

Looking for business and networking opportunities with people and business', who has an interesting and refreshing point of view on this whole internet thingy. If you need help designing and/or developing a - more complex than average - web based application, leave a message after the beep.

Oscar Gensmann’s Specialties:

User interfaces, application development, electronic newsletters, online marketing, user experience, community building, search engine optimization, w3c standards, CSS layouts using XHTML and semantic markup.
