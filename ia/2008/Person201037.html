William Rodgers’s Summary

I began my career as a Web Dev/Programmer/etc. by developing static HTML web sites, but soon began to develop sites using ASP (classic 2.0), CSS and Access Database. Along the way I have been involved in developing mostly customer/client facing sites (Internet and Intranet) and web-based applications using JSP, PERL, DHTML, XHTML, XML, JavaScript, PHP, MySQL and Oracle.
