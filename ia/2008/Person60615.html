Erik Renaud’s Summary

Software architect, perfectly bilingual with eight years of experience with emerging technologies. Autonomous, organized and self-teaching, has acquired experience for every step of the software development cycle. Exceptional skills for problem description, analysis and resolution. Special interest for management tasks and system’s architecture. Intuitive and a natural leader, also has abilities in tutoring and teaching. Entrepreneur with a lot of initiative, aims to stand out of the crowd.

Global experiences:
– Software architecture using UML and the MDA paradigm, using common design patterns and the ICONIX/Agile Modeling methodologies.
– Agile project management as a certified SCRUM Master.
– Object oriented development in C#, C++, Java and VB.NET using AOP and TDD techniques.
– Experience in modular development using Microsoft.NET, SOAP, XML, COM/DCOM and ActiveX.
– Experience for linking disparate systems together, object data models, data transformation and system architectures.
– Platform specialist for Microsoft technologies (.NET, COM and Win32).
