    The Toy'd Project is a student project developed by a small team of students at the European Institute of Technology (EPITECH).

    Toy'd is an acronym standing for Take Over Your Desktop.
    The goal of the Toy'd Window Manager is to implement a fully portable, full-featured next-generation Window Manager, that would allow the standard user to take advantage of a unique User Interface under Microsoft Windows or UNIX/GNU Linux systems.
    it relies on the Aoof-Wm abstraction framework (Advanced Object-Oriented Framework for Window Managers), which takes care of all platform independent / operating systems' internals and ensures Toy'd can have an absolutely identical behavior on any Aoo-Wm supported host.
