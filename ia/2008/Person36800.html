Chris Dart’s Summary

I have been in the Information Technology field as a professional since 1996 when I started moonlighting as a FileMaker Pro database developer and Macintosh computer support consultant.

Since then I have added general network support including both Mac OS X Server 10.3 and up and integrating Macs into Windows server environments and vice versa.

My main focus has always been providing technical assistance to organizations with limited budgets. Although sometimes very frustrating, I enjoy the challenge of making shoe-string computer setups work. This makes me particularly adept at solving problems others would claim are intractable.

Chris Dart’s Specialties:

Wide-ranging skills with computer support, specifically for Macintosh, also Windows 2000 and XP and Ubuntu/Debian Linux.

Expert at FileMaker Pro development, data manipulation, design, migration to or away from FMP and general troubleshooting.

Very skilled at AppleScript and integration with shell scripting. Solid skills at Unix/Linux shell scripting. Growing familiarity with Java and Perl programming.

Adept at web programming with php, MySQL, and javascript including AJAX.
