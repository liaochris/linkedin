I've spent all of my career developing web applications. I started with creating ecommerce websites as well as online web communities. More recently, I've helped develop a vertical search engine (http://www.simplyhired.com). My most recent experience is with integrating and building an Ads System to support CPC campaigns similar to Google AdWords using PHP and Perl.

I've always used the latest internet technologies. I have experience from the back end to the middle tier to the front-end. On the back end, I have extensive experience with SQL, database design, writing complex queries, data mining and data enrichment using MySQL, Perl, and SQL Server. In the middle tier, I've developed web application platforms for software that's served very large online communities and a search engine using PHP, VB.Net, ASP.Net, C# and Java. I also have some basic experience with the front-end using HTML, CSS and Javascript. I've done some minor work utilizing AJAX, Memcache, and database replication.

I have experience mining very large data sets and data enrichment. I have some minor exposure to crawling and lucene technology. I've spent over a year developing and integrating an Ads system that performs auctions and serves CPC campaigns. I've also developed optimization algorithms.

I consider myself a Jack of all Trades and I thrive in a fast-paced start-up environment. I'm very motivated and feed off an energetic atmosphere. I like to surround myself with similar individuals whose goals are to achieve excellence and have a true passion for their work. I'm happiest when I'm being challenged and even happier when I learn new things.

My goal is to work at an early-stage start-up that's doing an internet-based product. I'm seeking a Lead Software Engineering or Engineering Manager position.
