Alan Kidd’s Summary

I have been designing, developing and troubleshooting web and database applications, system integrations and business intelligence reporting systems for more than 10 years.

I am currently employed as a Software Team Leader with the Qld Department of Local Government, Planning, Sport and Recreation delivering the Smart eDA Web Application. This application allows the online preparation and lodgement of Development Applications for facilities such as new estates and shopping centres.

The application is is composed of java code, a business rules engine and a BPMN tool called Intalio. I am responsible for designing the business process, and integrating to the databases, internal and external web services to integrate the system to councils and other agencies such as the EPA and Queensland Transport. I am leading a team of programmers using an agile methodology.

Prior to this I was employed on a contract with the Queensland Police Service designing interfaces for a SQL Server-based application with more than 12,000 users. This project used Biztalk to integrate external agencies with the new QPRIME Police computer system.

I was employed as a SQL Server trouble-shooter for Dell Computers where I resolved performance issues with an 80 site SQL Server implementation of a medical application and set up maintenance, monitoring and backup procedures.

I was Principal Consultant with Volante Solutions for 3 years providing technical leadership over a team of 14 web and Windows Forms developers. Our specialty was the integration of web and Windows Forms front ends with ERP and database systems. I have been designing, deploying and troubleshooting SQL Server since 1998.

Alan Kidd’s Specialties:

Excellent problem solving skills, Application Support and Performance Tuning, Business Intelligence Analysis and Design, BPMN, UML and Data Modelling, Business and Technical Analysis, ERP interface design and implementation, Infrastructure design, Project Management, Data Analysis, Supply chain management and logistics, Excellent written and verbal Communications Skills, Public speaking and training, Technical writing for documentation and specifications
