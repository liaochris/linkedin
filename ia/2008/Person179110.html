Paul Dant’s Summary

Well versed and experienced information security consultant specializing in software and application security. Primary focus is on the secure design and implementation of applications leveraging the Microsoft .NET framework, including ASP.NET web applications, written with C#.

Interests and experience also include network and application penetration testing, security tool development (including exploits and shellcode), risk analysis and policy development.

Paul Dant’s Specialties:

Secure coding and development practices, security-focused code review (C, C++, Java, ASP, ASP.NET), vulnerability assessment, application penetration testing.

Web application development with ASP.NET and C#; database design and development with Microsoft SQL Server, TSQL and ADO.NET.
