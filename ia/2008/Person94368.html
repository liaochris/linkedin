John Price’s Summary

I currently have over 12 years of IT industry experience. In my position as a Senior Solutions Consultant, I design, architect, and implement Microsoft and Citrix technologies. In addition, I research new technologies and develop solutions associated with those technologies. My responsibilities also include the implementation of solutions based on technologies such as Microsoft Active Directory, Microsoft Exchange Server, Microsoft Operations Manager, System Center Operation Manager and Configuration Manager, Microsoft Systems Management Server, Microsoft ISA Server, Symantec Enterprise Vault, Softricity, SoftGrid, Citrix Presentation Server, Citrix Password Manager, Citrix Access Gateway.

I have recently launched www.learnITstuff.com, an online e-learning site devoted to task-based training on a variety of subjects. With the purchase of LearnSystemCenter.com, we now have over 100 training videos online, specifically Operations Manager and Configuration Manager. I will soon be launching a series of channels that will include Active Directory (Server 2008), Exchange 2007, SQL, Powershell, and more.

Prior to joining MSI, I also worked for six years for a Citrix Platinum and Microsoft Gold partner, where I was responsible for the management of the local Microsoft and Citrix consulting teams. This included the planning, architecting, implementing, and migrating various Citrix products as well as Microsoft Active Directory and Exchange.

I have collaborated on the following published books, both as an author and as a technical editor:

• Mastering Active Directory for Windows 2008 (April, 2008)
• Mastering Active Directory for Windows 2003 “R2”
• Windows Server 2003 Active Directory and Network Infrastructure Design Guide
• Active Directory Best Practices 24seven: Migrating, Designing and Troubleshooting
• Mastering System Center Operations Manager 2007
• Mastering System Center Configuration Manager 2007
• Citrix Metaframe XP 1.0 Administration

John Price’s Specialties:
