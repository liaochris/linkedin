Tom Handal’s Summary

Tom Handal’s Specialties:

Systems and Kernel Programming on Windows 2k/XP/2k3/Vista, Linux, Solaris, HP-UX, AIX and others, Network programming, Cryptography, Security software including Firewalls, VPNs, and Access Control suites. Navigation software, GPS, INS, mapping. Advanced experience with MS DirectShow, RTSP, media player software development.

Tom Handal’s Experience

  • Lead Engineer

    Verimatrix Inc.

    (Privately Held; 51-200 employees; Computer Software industry)

    August 2006 — Present (1 year 11 months)

    Lead Developer on Verimatrix ViewRight(TM) PC Player.

    Lead a team of three developers working on ViewRight PC Player. Responsible for the entire product life-cycle.

    The player employs Microsoft DirectShow technologies, including self authored custom DirectShow filters.

    Capable of processing and viewing content from the Verimatrix Content Authority System (VCAS).

    Integrates VideoMark(TM) technology and embeds this water mark technology into the video stream.

    Designed and developed RTSP Protocol Client DirectShow filter written from scratch, including buffering and trick play techniques.

    Designed and developed Digital Video Recorder (DVR) software for media player. This includes live rewind, future channel recording and playback.

  • Senior Software Engineer

    Websense Inc.

    (Public Company; 501-1000 employees; wbsn; Computer Software industry)

    September 2004 — August 2006 (2 years)

    Development of desktop security product (Client Policy Manager)
    The product included:
    Application blocking, Network Access Control, Removable Media Control and Lock down, and many other features that include kernel and user space work.

    Worked on File System Filter (FSF) drivers, Transport Driver Interface (TDI) drivers and system call hooking technologies.

    Lead designer and developer of Remote Filtering Software, which filtered web traffic on remote laptops using a secure connection to a Remote Filtering Server.

    Sole designed and developer of Secure Transport Protocol, which is similar to SSL, but completely written from scratch to be friendlier and encapsulated in a C++ class. This protocol was used for the secure communication between the Remote Filtering Client and Remote Filtering Server. It included Diffie-Hellman key generation, user-choice of encryption and authentication algorithms and protection against replay-attacks.

  • Senior Software Engineer

    Computer Associates

    (Public Company; 10,001 or more employees; CA; Computer Software industry)

    May 2001 — September 2004 (3 years 5 months)

    Senior developer on design and development of eTrust Firewall and eTrust Access Control.

    Completed kernel work for both products on both Windows and several Unix flavors, including Linux, AIX and HP-UX.

    Lead designer and developer on Windows NDIS IPSEC Driver for eTrust VPN.

    Awarded Patent NO. 7,076,770 – “Apparatus and method for modifying a kernel module to run on multiple kernel versions” – Filed Jan 2004 and Awarded July 2006.

  • Software Engineer

    (Government Agency; 1001-5000 employees; Computer Software industry)
