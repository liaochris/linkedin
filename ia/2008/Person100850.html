Justin Hambleton’s Summary

Currently a technical account manager, focused on large direct advertisers within the retail and shopping comparison verticles. I manage the technical relationship with some of Yahoo's most valued advertisers, providing consultation and technical support for Yahoo's Search Marketing API, RightMedia Exchange API, Search Submit Pro and Product Submit.

Formerly an interaction/Interface designer, working primarily with complex web based applications. Extensive background in information architecture and interface design for web based content management systems, CRM tools, data visualization and insurance rating/quoting and policy management systems. Experienced client-side programmer (Java Script, Ajax, CSS, XHTML). Experience with system integration and data exchange using web services.

Experience with various API's including Yahoo Map API, Google AdWords API, del.icio.us, Flickr.

Search engine marketing knowledge - particualrly PPC management strategies with Google, Yahoo and Miva. Extensive knowledge of web analytics principles using WebSideStory HBX and Google Analytics. Campaign tracking and optimization. A/B testing and landing page optimization.

Justin Hambleton’s Specialties:

Use case development, work flow design, information architecture, usability studies, competitive analysis, interface analysis and design, client-side programming (CSS, Ajax, Java Script, XHTML), server-side programming (Cold Fusion, Ruby, PHP, .Net), search marketing, ppc management, api integration, project management, business requirements and scope, functional requirements and form/function documentation.
