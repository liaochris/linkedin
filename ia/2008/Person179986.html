Stefan Iancu’s Summary

Since Sep 2008 I’m Transversal Project Manager inside Business Engagement department.

Jan 2007 – Sep 2008, I was appointed Project Manager for important / urgent IT&C projects with high implication on Orange Romania image and customers base. Main projects list: Single Sign On – Identity, USSD Services, USSD Premium Services, Messaging Gateway.

The main projects I’ve managed between 2001 and 2006 are: Mobile Banking, Over-The-Air, Wireless Information Query, Roaming Call Back, OWP Content Providers Integration, over 250 Mobile Equipments assessments on more then 20 providers, SIM/USIM projects, Security Tools, MOC/BEW, Machine2Machine, PC2SMS, PopUp Info, BlackBerry solution and many others.

Until end of 2006, for almost 1.5 years, I’ve took over SIM / UICC activities / projects. In this period Orange Romania implemented and commercially launched 3G Services. For cards I’ve managed Milenage algorithm implementation, UICC batches validation for all subscribers categories and profiles (for 2 different providers) and last but not least (a) new version of STK application for 3G and 2G cards.

Since Mar 2004, for almost 3 years, I was appointed Lead Engineer for all ORANGE Romania Technical projects / activities related to Mobile Equipments assessment. From this position I’ve coordinate all Mobile Devices technical assessments and I manage all technical interfaces inside validation flow (Mass-Market / Corporate Marketing; all devices suppliers (around 20), Orange group, Break-Down Services, Technical After Sales, Logistics, and Customer Care.

Starting with Apr 2001 my position changed to Value Added Services Expert inside MobilRom / Orange Romania Development Department. For the first 3 years I was appointed Project Manager on different VAS projects, with budget between 10K to 2000K EUR. My main responsibilities were to manage all activities for specific VAS projects, starting with marketing idea and delivering an up and running service to the market.
