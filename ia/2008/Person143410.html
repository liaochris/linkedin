Josh Butner’s Summary

The internet (and web design, for that matter) is still a new frontier. I'd like to think while I'm not the first cowboy to try and make a difference here, this certainly isn't my first rodeo.
I've spent quite a few hours, days, weeks, months constantly reevaluating what it is to be a web designer and trying to push myself to make designs that are fresh and effective, while still maintaining an artistic edge. It's a balance that has to be found with every pixel or tag.
I strive every day to be more effective and expressive with the skills at my disposal, whether that be xhtml, css and javascript or PHP and MySQL or even Flash.
