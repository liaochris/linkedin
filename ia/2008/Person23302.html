Bill Gottschall’s Summary

Bill Gottschall’s Specialties:

Project management, project planning and control, business process analysis, workflow planning, requirements analysis, business systems analysis, software architecture, configuration management, release management, software development using traditional and agile techniques, and hardware and software vendor selection.
