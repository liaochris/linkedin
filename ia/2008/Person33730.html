Charles Harless’s Summary

With a varied background in computer science, education technology, web-based software design, bio-medical technology, and user interface design, my career focus is on providing unique, easy to use & manage web-based applications for clients in areas including education, entertainment, travel & tourism, biomedical engineering, and healthcare.

I have extensive experience developing web-based systems using Java on both the client and server side specializing in the java servlet layer and front-end technologies. I have extensive experience using various data models and data storage environments.

Through all my various development, I have strived to develop my communication, management, and leadership skills including project management, technical writing & presentations, and team building.
