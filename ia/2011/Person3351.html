Antoine Meyer’s Summary

I am a Java developer most experienced in Front-end web development and my current primary interest is to develop web applications using Google Web Toolkit. I have been using this technology for nearly 2 years on various projects for companies like CCBill and more recently for DNG Consulting where I had the chance to work with Sami Jaber, author of 'Programmation GWT2'.

I am able to build a whole GWT 2.1 application using a classic MVC pattern or the MVP architecture (preferred!). I am well aware of the best practices and always keep an eye on the performance, testability and re-usability of my code and am experienced using the latest features of GWT including UiBinder or libraries like Gin, gwt-dispatch, etc...
I am really interested in open source projects and try to contribute to the community when I can. For instance, I recently created a small LGPL project based on Atmosphere for gwt (http://code.google.com/p/atmosphere-gwt-domain/).

As a java developer, I am of course versed in back-end technologies: I have been using ORM with JDO on Google App Engine (in the Cloud) with Dependency Injection via Guice and JSPs/MySql with various web servers like jetty, jboss, tomcat.
I am comfortable using Eclipse IDE and tools like Maven or ant.
I am a good team player and am familiar with agile methodologies (Scrum).

Always looking to improve myself and to keep up with technology innovations, I attend conventions like Google I/O 2010 and I am part of the user groups related to Java, Web or Google technologies.
