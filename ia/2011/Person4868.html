Ben Ilegbodu's Summary

I am a User Interface Engineer with a strong CS background. In sum, I'm interested in web programming, either on the client-side via JavaScript or in the middle-tier via web technologies like .Net, PHP, etc. I'm well versed in web standards, XHTML/HTML5 and CSS2/3, but I prefer the programming aspects of web development. Recently, I have found myself leaning towards widget architecture type projects that make it easier to create highly-interactive web pages.

I also contribute heavily to the open-source JavaScript framework UIZE (http://www.uize.com/).

Specialties

* Object-oriented & unobstrusive JavaScript (frameworks & widgets)
* UIZE open-source JavaScript framework
* ASP.Net (C#)
* DHTML/AJAX
* XHTML/HTML5
* CSS2/3
* Optimization techniques (file weight and perceived/actual rendering speed)
* SEO techniques
* Localization/I18N principles
* PHP
* Usability testing
