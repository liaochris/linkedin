Keith Terry, PMP's Summary

Award-winning experienced IT professional with demonstrated success in planning, developing, and implementing cutting edge solutions to address business opportunities. PMP certified (Project Management Professional) and certified in ASAP (Accelerated SAP Implementation). Knowledgeable and experienced in regulatory and compliance issues.

HIGHLIGHTS

• Project Manager for $4 million implementation of SAP integrated with GE Proficy MES (Manufacturing Execution System) to manufacture automotive lithium-ion batteries. Directed a team of 7 consultants for: SAP PP, QM, MM, GE Proficy Suite, and PLC (Programmable Logic Controllers) integration.

• Project Manager to implement SAP Best Practices for a Japanese Automotive Manufacturer. Directed a team of 12 SAP consultants (3 offshore) for SAP FI/CO, BI/BOBJ, and BPC (Business Planning and Consolidation).

• Project Manager for Proof of Concept project regarding the use of SAP PP, MM, SD, FI/CO, and EDI for a Louisville manufacturer to re-engineer a failed $15 million implementation of SAP ECC6.0.

• Project Manager for $5 million project to replace legacy system with SAP (SD, PP, QM, EH&S) using Variant Configuration, Mill Products Industry Solution, and interface with MES (Manufacturing Execution System) resulting in faster integration time of new acquisitions and improved self-service capabilities for customers.

• Project Manager for $1.3 million SAP ASAP software implementation (MM, PM, FICO) that reduced need for month end reconciliations and improved visibility and trust in organizational data.

• Project Manager for SAP BW, Business Warehouse, for Finance (AR, AP, GL) and provided centralized reporting repository for business data.

• Finalist for 2007 America’s SAP User Group Impact Award. Judged by ASUG for criteria including return on investment, strategic alignment to goals, SAP product scope utilized, and service improvements.
