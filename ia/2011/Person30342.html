Michael Wu PhD’s Experience

  • Principal Scientist

    Lithium Technologies

    (Privately Held; Computer Software industry)

    July 2008 — Present (2 years 7 months)

    * Collaborating with academics using Lithium's customer network data, we were able to quantify the value of WOM by simulating the customer lifetime value of each community user.
    * Taking into account of the rate at which users, posts, page views, and kudos accumulate, and whether a thread is solved, I derived and algorithm that can assigned a hotness scored to a thread base on the thread's popularity, recency, and decay rate.
    * Using metric space embedding, I was able to cluster communities into similar bins based on their size, age, focus, audience, and industry. This allowed us to perform benchmark based on both date and age of the community.
    * Using behavioral profiling, I was able to identify at risk users and classify them into different classes base on whether they are product experts or enthusiasts. I also develop a module that was able to visualize the 5 dimensions of each user's behavioral profile data.
    * Borrowing from economics the principles of Lorenz Curve and Gini coefficient, I was able to characterize the participatory inequality and derive an algorithm for identifying superusers (hyper contributors) in communities and in general outlier detection based on participatory data.
    * Using social network analysis and algorithms and the social graphs created from the communication network within Lithium communities, I was able to identify various types of influencers based on different types of centrality measures and other network statistics.
    * Using boosting for regression and variable selection, I was able to identify factors that are predictive of community health. And using nonlinear analysis, I was able to create the community health index (CHI).

  • Research Scholar in Gallant Lab

    UC Berkeley

    (Educational Institution; Higher Education industry)

    2001 — 2008 (7 years )

    * Combining principles of wavelet analysis and early visual processing, I was able to design and build a biologically inspired wavelet pyramid, call the Berkeley Wavelet Transform (BWT).
    * Applying kernel regression using polynomial kernels in an combinatorial stimuli design to study the nonlinear summation of V4 neuron, I was able to understand how they process multipart objects.
    * Using Bayesian probability and the principle of maximum a posteriori (MAP) estimate, I was able to unified all spatio- or spectro-temporal receptive field (STRF) estimation algorithms. This facilitate the building of new algorithms and understanding of similarity and difference of old algorithms under the MAP framework.
    * By modeling the neuronal response distribution as a negative binomial (NB) distribution rather than a Poisson distribution, I was able to model the over dispersed nature of spike data. And using delta method on the NB model, I was able to derive a correlation bound on the predictive validation correlation coefficient.
    * Applying kernel algorithms: support vector regression, kernel ridge regression and kernal partical least square, I was able to predict the neuronal response as well as neural netowrks.
    * Using theory of conformal mapping and Riemannian and differential geometry, I began to understand the nonlinear higher-order statistics of natural scenes and how neurons might deal with these nonlinearities.
    * Using neural network I was able to fit a predictive model of how neurons process naturalistic informations and interpret what the neurons are doing.
    * Applying principle component analysis to neuronal spike train data, I was able to understand how downstream neurons interpret an incoming spike train.
    * I modeled the development of receptive fields (RF) via feed forward network between LGN and V1 with realistic local Hebbian and STDP learning rules and synaptic connectivity of these neurons and showed that RF development is robust.

  • DOE CSGF Researcher

    Los Alamos National Laboratory

    (Government Agency; Research industry)

    June 2003 — September 2003 (4 months)

    * Using date dependent hypothesis class and theory of uniform convergence, I was able to derive prediction error bounds on classes of models.
    * Using manifold theory and differential geometry to study the geometric structure of images distributions, I was able to understand some principles of invariance in visual processing.
    * Using principal component analysis and analysis on the possible variation of facial expression, I was able to come up with a more accurate face recognition algorithm by taking account of the invarince due to facial expression.
