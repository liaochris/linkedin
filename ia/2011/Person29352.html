Matthew Baxter-Reynolds’s Summary

I work as an independent software development consultant helping businesses achieve more from their software teams.

My particular interests lie in open standards, the Microsoft server stack and the "big four" mobile computing platforms.

I'm currently writing a book - Multimobile Development: Building Applciations for any Smartphone - that looks at the individual technical challenges of HTML5, Android, iOS, Windows Phone, Windows Mobile and BlackBerry when approached from the perspective of building the same application for each.
