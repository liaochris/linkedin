Christophe Debruyne’s Summary

My research interests include: Ontology/Knowledge Management, Ontology Engineering (methodologies), Ontology Reuse, Ontology Lanuages (RDF(S), OWL, etc.) and Web 2.0 applications.

Other interests include programming, AI, ML, databases and database theory. My MSc thesis was titled "A Method and Tool for Lexon Reuse in the DOGMA Ontology framework" and presented a means for gathering possible relevant facts on different semiotic levels to reuse while constructing ontologies. My current research focusses on giving the community a more prominent role in ontological commitments.

I have experience in working in a software development team on small to medium-scale projects. I’m familiar with agile programming methodologies such as extreme programming and incremental waterfall based programming methodologies such as RUP. Extensive knowledge of software design, information system design and database design using UML, ORM, ER and EER.

I am proficient with following programming languages: Java (J2EE and J2SE), SQL, Visual Basic, C#, PHP. Skills in other languages such as JavaScript, Ruby (on Rails), ActionScript, Lisp, Prolog and Scheme and an advanced understanding of Web and Semantic Web technologies such as HTML, XML, XSLT, RDF(S) and OWL.

I was responsible for developing several modules for research prototypes that consisted of an Eclipse product during my time at VUB STARLab as a MSc student, for which specific plugins needed to be developed. On the client side and a JBoss application server backend connected to a PostgreSQL relational database server using Hibernate on the server side.
