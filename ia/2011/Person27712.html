Started the project to make Linux shell accounts cheap and affordable without strict limits imposed by commercial hosting companies. Today Rootnode owns 20 professional production servers mainly placed in Amsterdam, the Netherlands and for security reasons in France and Poland (backup policy, secodary MX, secondary DNS). Another objective of this project was to create a community of IT professionals. Because there is no official support all users communicate with each other and solve their problems through mailing lists. You can find there administratos, all kind of developers, web designers and sometimes people not directly related to IT sector for example marketers. Most of them work in international companies and are the true experts in their field. During this project I also had occasion to be a project manager.

Significant achievements:
• Satan – a sophisticated command line based service manager written entirely in Perl. Responsible for adding domains, databases, mail accounts, managing DNS etc.
• Fastweb – a web system based on Nginx, Varnish, MySQL, Memcache, PHP, Python, Ruby run on 12–core servers with 15k SAS drives with ability to inject your own Nginx configuration via XML files. Extremely fast and flexible.
• E–mail system that supports over 2200 user accounts and requires no administrator attention. Solved the problem of handling single domain with multple Postfix servers without
the common storage and providing automagic failover.
• Backup servers based on rsync software and my own Perl wrapper to create snapshot–style backups. All integrated with Satan service manager to define backup types, schedules and include/exclude lists.
