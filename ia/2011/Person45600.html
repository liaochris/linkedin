Neville Lee's Summary

Neville is a highly experienced web developer specializing in Adobe's ColdFusion®, a state-of-the-art software tool for developing advanced web applications. He has developed a number of dynamic, interactive, database-driven web applications which include inventory, shopping cart, and eCommerce systems. His repertoire includes the creation of web interfaces to Microsoft SQL Server and Microsoft Access databases.

Neville has a B.S. in Chemistry from Mount St. Mary's University, Maryland, and numerous graduate-level courses in Management Information Systems from the University of Missouri, St. Louis.

His career in Information Systems includes software development positions at organizations such as McDonnell Douglas, St. Louis (health systems), metaTRACE, Inc., St. Louis (environmental analysis), American Express, St. Louis (health systems), First Data Corp., Charlotte (health systems), and Transamerica Reinsurance, Charlotte (insurance).

Prior to switching to Information Systems, Neville spent a number of years in the scientific field, encompassing work in the fields of chemistry, biochemistry, microbiology, immunology, and organ transplantation compatibility.

Specialties

Webfuzion.com internet applications specializes in dynamic, interactive, database-driven web applications developed in Adobe's ColdFusion®, a state-of-the-art, cross-platform, web applications tool.
