Marcelo Giorgi's Summary

I'm a is a passionate web developer who has been working with JEE for a while, until he met Ruby On Rails. Since that moment, I decided to throw away all the Sun certificates that I already had and advocate myself to the more dynamic scripting languages and edge web technologies. In particular, I've been working as a freelancer using RoR since 2007.

About development process, I prefer agile methodologies instead of the more heavy weighted processes, such as RUP.

My development platform is based on Linux, Ubuntu is my preferred distribution and I'm using it on a daily basis since 2007. Also, my IDE is gVIM, as I consider a very lightweight and powerful tool.
