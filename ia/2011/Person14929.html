Florent Messa’s Summary

I am a PHP developer and also a Python enthusiast and evangelist with JavaScript skills.

Using new web development techniques such as AJAX or the architectural pattern Model-View-Controller, I produce light and powerful interfaces for both users and developers. Well-aware of the fast evolution of my profession, I pay attention to maintainability and cutting-edge ways of coding.

For instance, I keep working on software development techniques like Test-driven development and REST software architecture through a web application framework.

To build robust web applications, improve performance and avoid waste of time on important developments, I mainly use PHP with Zend and Symfony frameworks but also Python with Django framework.

I really appreciate development teams who aim at providing original experiences for the customers.
