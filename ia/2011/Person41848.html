    MSc. Thesis: Network traffic anomaly detection based on NetFlow protocol and chosen data mining algorithms.

    The work focused on analysis of issues regarding detection of anomalies, and especially threats, that are inherently related to packet traffic in IP networks. The scope of the thesis contains investigation of network security concepts and concentrates on using NetFlow protocol potential and chosen data mining algorithms to automate network threat detection. Thorough analysis of two open-source packages SiLK and Flow-tools, used for managing collected NetFlow records, allowed for development of several attack detection procedures. Java based Flowmine application, developed in the thesis, implements chosen data mining techniques and constitutes an extension to one of the described packages. Production and laboratory environment tests have confirmed usefulness of NetFlow protocol and developed application in detecting network traffic anomalies.
