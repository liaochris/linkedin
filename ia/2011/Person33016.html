Noah Sussman's Summary

Noah Sussman is a senior software engineer, with a background in literature and graphic design. For the last decade, he has focused upon building Web applications that support Tim Berners-Lee's vision of a platform-independent, "read-write Web."

Noah is deeply invested in learning to build a great user experience for software applications, especially on the Web. He is dedicated to the vision of a truly World-Wide Web, that is accessible to people of all cultures, economic strata and levels of physical or mental ability.

Specialties

Automation and tools for developers; specializing in functional GUI test automation tools and continuous build environments.

Tools for Agile software teams, including expertise in the configuration and administration of SVN, ClearCase, MediaWiki, Confluence, Jira, QuickBase, Cruise Control and Hudson.

Highly proficient in building UI with platform-independent Web standards: table-free layouts, unobtrusive JavaScript, semantic CSS and Flash-free interaction.
