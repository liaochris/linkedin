Brian Buckley's Summary

Hands-on Technical Architect with wide experience in Java SE, Java J2EE and Java ME as well as CORBA and C++. I spent 9 years in the telecommunications industry before moving into the web application space. I have also coded a number of Symbian OS applications for Nokia smartphones but don't ever ask me to do that again :-)
