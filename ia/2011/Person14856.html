FJ van Wingerde's Summary

I make User Interfaces, and have been working over a decade in every aspect of it. From concepting, specifying, user-centered design, to implementing and delivering, in any technology that made sense, for mobile, web, TV, or desktop applications. From the first interactive web medical records to advanced mobile service concepts, from start to finish, with paper & pen, Excel & Word, usability labs with one-way mirrors or just observing over a shoulder, JAVA / Swing, HTML / CSS / JavaScript / AJAX, Python / Plone, J2ME, FlashLite, I make it work and usable.

Specialties

(mobile) interaction design, (mobile) ui R&D, prototype creation, concept exploration, software engineering, visualization, electronic social networking, reality check, making systems work

FJ van Wingerde's Publications

  • Using HL7 and the World Wide Web for unifying patient data from remote databases.

    • Proc AMIA Annu Fall Symp. 1996:643-7
    • October 1996
    Authors: FJ van Wingerde, J. Schindler, P. Kilbridge, P. Szolovits, C. Safran, D. Rind, S. Murphy, G. O. Barnett, I. S. Kohane

    W3-EMRS is an architecture designed to access clinical data from remote heterogeneous electronic medical record system (EMRS) databases. We describe the technologies used in an experimental implementation of W3-EMRS that concurrently collects data from several sources and presents them in an integrated set of views. After describing some of the organizational constraints, the architectural decision, implementation methodology, and operation of the completed project are discussed.

  • Linking multiple heterogeneous data sources to practice guidelines.

    • Proc AMIA Symp. 1998: 391–395
    • October 1998
    Authors: FJ van Wingerde, Y. Sun, O. Harary, K. D. Mandl, S. Salem-Schatz, C. J. Homer, I. S. Kohane

    The BiliLIGHT system is a World Wide Web (Web) based system that integrates an interactive clinical practice guideline with real-time patient-data retrieval from remote heterogeneous data sources to help clinicians manage newborn jaundice at the point of care in three clinical settings. We briefly describe the system, how actual information exchange of medical data across institutional boundaries was achieved, and how the data were connected to a CPG. In particular, we examine the requirements for patient identification, exchange protocols, authentication, and a standard vocabulary.
