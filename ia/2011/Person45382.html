  • Software Engineer

    INRIA

    (Government Agency; Research industry)

    February 2007 — December 2009 (2 years 11 months)

    Software engineer working on SOA, SCA and component-based architectures.

    I'm the lead developer of one of the core modules in the FraSCAti OW2 project: http://frascati.ow2.org, a next-generation SCA runtime.

    The module I develop is in charge of establishing distant bindings between components through different transport protocols. Its open, plugin-oriented nature makes easy to add new binding protocols.

    Several technologies are used:
    - Fractal, as the core component model, http://fractal.objectweb.org
    - Maven2, as project lifecycle management tool
    - Java as main implementation language

    The code is released as open source and currently available at:
    http://svn.forge.objectweb.org/cgi-bin/viewcvs.cgi/fractal/trunk/fractal-bf/
