Loren Scott's Summary

I began my formal photography studies when I was a sophomore in High School (class of '81), and have been involved in photography ever since. I shot my first wedding when I was just 19 (in 1982), and I currently only accept between 15-18 weddings per year. My studio/gallery is located in Murrieta, CA, and I photograph events worldwide, recently shooting events as far away as San Francisco, Arizona, Hawaii, and the U.S. Virgin Islands.

As Director of Product Development for Pictage I was chief architect and developer of two major web-based services for our photographer clients -- P3 (Pictage Payment Processing) and Bridal Recommendations. P3 was developed for a Linux web server using JSP pages and a PostgreSQL database backend, and the Bridal Recs system was developed for a Windows web server using ASP pages with VBscript and a SQL Server database backend. As of November 2006, I left my full-time position at Pictage to grow my own photography business, yet still remain on staff with Pictage as a part-time consultant to offer development/maintenance support for the P3 and Bridal Recommendations systems.

Specialties

photography, public speaking, community development, web development, database management, SQL, graphical user interface (GUI) design
