Adam Ullman's Education

University of Sydney

MSc (Research) Candidate, Information Technology

2006 – 2007

Thesis Topic: WikiNavMap - A Visualisation to Improve Navigation and Encourage Collaboration using AJAX

This Masters project involved the development of WikiNavMap, a plugin for the Trac Project (http://trac.edgewall.org). WikiNavMap visualises the wiki and issue tracking system as a directed graph using Graphviz (http://graphviz.org) and has a “GoogleMaps like” interface for navigating around the map and viewing information about nodes. It was written in Python with heavy use of SQL, as well as JavaScript using the Prototype (http://prototypejs.org) and Scriptaculous (http://script.aculo.us) Frameworks. The design of WikiNavMap is based on research in the areas of visualisation and user interface design. Several studies were also conducted to evaluate the usability and usefulness of the visualisation. A demo of this project can be found at: http://trac.edfac.usyd.edu.au/demo/navmap?helpmenu=active Source code: http://trac-hacks.org/wiki/WikiNavMapPlugin
