Will Gilbert’s Summary

I have been active in the Biotechnology/Bioinformatics arena or over 30 years, working in both academia and industry as both consultant and perm. I have built a career on successfully evaluating and implementing emerging open source technologies.

Primary strengths in:

Object-Oriented design and programming, emerging technologies, adapting Open Source projects to solve client problems

Extensive Java experience, client-side Swing (J2SE), server-side Tomcat (J2EE), JDBC

Extensive experience with XML, XSLT, XPATH, FOP

Open Source development technologies: Maven, ANT, Subversion, JIRA, and JUnit.

LAMP: Linux, Apache, MySQL, PERL, PHP4 and PHP5

Web 2.0 technologies including HTML, XHTML, HTML5, CSS, WebKit, RIA, AJAX, and Javascript

Have implemented JavaScript components for both the Echo3 and GWT webapp UI frameworks

Excellent knowledge of both the GWT and SmartGWT frameworks using the MVP triad architecture

SQL and DBA experience with MySQL, Sybase, SQLite; Hibernate ORM

JBoss Rules Engine (aka DROOLS)

Workflow Management: OSWorkflow (former committer) and JBoss RuleFlow

Very exited about using Groovy to bring greater agility to the JVM (Java) platform; Learned the Gradle build system directly from Hans Dockter

Extensive experience with MacOSX as user, developer and administrator

No experience with Microsoft Windows, SQLServer, IIS, .net, ASP or MS-Office and would like to keep it that way.
