John "unscriptable" Hann's Summary

John Hann is a master-level, freelance software engineer whose early interest in the web became a career and an obsession. Since 1996, he has successfully delivered over 70 enterprise-class, web-based applications. Notable projects include a Web 2.0-class product in 2000 (US Patent 7,016,751) and several Rich Internet Applications ("Ajax apps") as early as 1996 (using browser plug-ins before 1999 and hidden IFRAMEs until 2004). In addition to grokking dojo, jQuery, and prototype, he has also written three comprehensive JavaScript frameworks from scratch.

Mr. Hann also harbors a passion for pioneering into technological frontiers. In his spare time (or when projects demand it), he likes to prove-out his mantra, "Nothing is impossible. Even on the web." Examples:

2004 Draggable, modal HTML dialog widgets (using multiple inheritance to add common dialog behaviors)
2006 JSON as a client-server data format (before JSON-RPC Working Draft)
2007 "Background threads" in Javascript (actually non-preemptive multitasking before Google Gears)
2009 CSS3-compliant CSS Transitions emulation in all browsers, including IE6

Specialties

Web 2.0 / HTML5 / Rich Internet Applications using industry standards such as (X)HTML, CSS, XHR, and Javascript. dojo 1.x expert. also proficient with jQuery and prototype
