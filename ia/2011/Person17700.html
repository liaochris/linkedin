Hugh Griffith's Summary

What's so special about me? I'm skilled at both interaction and visual design, so not only can I make a web page, web application, mobile site, or iPhone app look appealing, I can make it easy to use as well. (Ask someone who knows, and they'll tell you that's a rare combination.)
