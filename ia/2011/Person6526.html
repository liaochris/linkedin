Bryan Comstock's Summary

What drives my passion for test engineering, has been the problem solving and the team process. My test engineering positions have provided me with significant experience in both software and electrical test engineering, in the computer products and industrial protection industries. I have test engineering experience in both automated and manual test disciplines. My programming experience includes automated test scripts in, TCL/ Expect and C programming languages.
Developed test cases and plans, executed test cases, and analyzed test issues, Executed tests using Linux, Solaris, and Windows OS servers. Performed the test project lead role on two Fibre Channel switch test projects, and for a GenRad in-circuit test automation project. Seeking a Software Test Engineering position where my test case development, test plan execution, test programming, team orientation, and problem solving experience can bring success to a technical software development project.

Specialties

Performed tests on a large Storage Area Network (SAN) configured, and automated, via a physical layer switch. Developed TCL/Expect programs for Fibre Channel computer networking switches, and C program enhancements for disk drive test. Tested communication protocols including Fibre Channel, FCoE, SNMP, and CIM. Strong problem solver, skilled at product software, and hardware failure resolution.
