Doug Alcorn's Summary

I am a senior level programmer steeped in agile development. I've done development using traditional "waterfall" methodoligies at Big Companies; but I much prefer to use lighter processes like SCRUM and eXtreme Programming at smaller companies that are still hungry. My past is littered with Unix systems programming, interprocess communications, and kernel hacking. Now I'm focusing exclusively on web applicaiton development using Ruby on Rails.

Ruby on Rails, Objective-C, Cocoa Touch, iPhone, HTML, XHTML, CSS, SQL, Javascript, Web Development, Agile and Extreme Programming (XP), SCRUM, Test Driven Development (TDD), Automated Testing
