Don Watters’s Summary

An accomplished database professional, working in both administrative and development roles for OLTP and OLAP systems, with experience in large scale deployments related to SQL Server, Aster nCluster, Netezza, MySQL and Oracle platforms. Technical reviewer for magazines and database related books for APRESS. Objective Domain writer for SQL Server Database Admin and Database Development tests for Microsoft. Member of the Microsoft SQL Server Customer Advisory Network (SCAN) since inception. Member of the Aster Customer Advisory Board since inception. Conference Speaker. Currently working as the Chief Data Architect for the Data Platform group at MySpace.

Don Watters’s Specialties:

SQL Server Development and Administration for highly available, scalable and secure database solutions.
