Craig Dowell’s Experience

  • Senior Software Engineer & System Programmer

    University of Washington

    (Educational Institution; Higher Education industry)

    December 2006 — Present (3 years 2 months)

    I am a key member of an international team of scientists and engineers designing and implementing an open-source discrete event simulation tool for large-scale internet research and analysis: the ns-3 Network Simulator Project.

    I have a major role in system requirements definition, architecture, design, implementation, documentation, testing, release, and maintenance.

    I work in environments ranging from lowest level drivers (such as Ethernet, WiFi, and Tun/Tap) to operating system services (real-time schedulers, sockets, I/O, logging, and tracing) through various network protocol stacks (TCP, UDP, and routing) to various client-server applications which drive simulation process. Use primarily C++ and Python in object-oriented designs using common design patterns and idioms.

    In a separate life at the UW I have been writing code for the alignment system for the muon spectrometer of the Large Hadron Collider at CERN. I participated in assembling the muon spectrometer "big wheels" (big fun) and did some commissioning work on the detector. I have also done embedded system programming for PET scanners at the UW Image Research Library using Rabbit development systems, and separate ARM embedded system programming using GNU tools.

Designed and implemented componentized media system used in Microsoft ActiveMedia products; enabled streaming of various kinds of video and related information across internet. Collaborated with researchers on implementing MPEG-4 codes for low bandwidth video applications
