Vishal Rao’s Summary

I am a software engineer with over nine professional years of multi-domain experience developing several different types of software that span the range of the software technology stack, including but not limited to:

• high performance, multithreaded, server-scale, cross-platform system software
• database plug-ins, indextypes, and adapters that customize and/or extend ORDBMS services
• design and extension of evolving, customer facing APIs
• middle tier servlets and web applications that run in the context of an application server
• thick client GUI tools using Java Swing, Java AWT, and Eclipse RCP.

I have experience with all parts of the software development lifecycle – gathering and proposing feature requirements, writing functional specifications and design documents, estimating human and hardware resource costs, designing inter-component interfaces, implementing component behavior, writing test cases (unit, end-to-end, stability, performance), and fixing bugs.

I am a Java expert and have significant experience with C/C++.

Additionally, I have experimented with the following in non-production settings:
• RIAs (rich internet applications) using GWT (Google Web Toolkit) and Adobe Flex.
• Mobile application development with Google Android.
