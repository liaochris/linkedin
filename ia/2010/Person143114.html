Michael M David’s Summary

I have researched and designed commercial database and query products for major database software vendors for over twenty five years; first involving hierarchical databases using IBM's IMS, then relational databases, and now hierarchical databases again with the advent of XML. I am now introducing ANSI SQL navigationless hierarchical access and processing to the database industry. This enables non technical users to finally have transparent access to XML data which allows them Enterprise 2.0 interactive processing of the entire hierarchical structure in a single query. This advanced structure-aware automatic full hierarchical processing dynamically increases the value of the queried data and insures correct hierarchical results. For more information on my database research and findings, see my papers and articles on my site below and on the web under "Michael M David".

Michael M David’s Specialties:

I designed the first ANSI SQL transparent XML interactive hierarchical processor, SQLfX®, now available online for testing. Its inherent SQL hierarchical database processing capability naturally solves the long outstanding relational/XML data integration problem. This hierarchical processing supports powerful multipath (LCA) queries made possible by its navigationless processing. The discovery of Lowest Common Ancestor (LCA) processing occurring naturally in SQL was made during this project.
