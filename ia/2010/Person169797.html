Ron Barrett’s Summary

Ron Barrett is an independent technology author, speaker and consultant. Ron has been a technology professional for over 13 years, working as a hand-ons IT Manager for several major financial services firms and dotcoms. Ron is a specialist in network infrastructure, security, and IT management. He is the author of Office Communications Server 2007 R2: How-To, as well as co-author of Windows Server 2008: How-To and The Administrator’s Guide to Microsoft Office 2007 Servers. Ron has been a Technical Author and/or editor for several other books on Windows administration. Along with book writing, Ron has contributed to several industry magazines such as Redmond, Datamation and Windows IT Pro. Beyond writing, Ron has spoken at several technology conferences for CPAmerica, AICPA and MCP’s TECHMENTOR.

Ron Barrett’s Specialties:

Windows 2008, Exchange Server 2007, Office Communications Server 2007 R2, Sql Server 2008, Windows 7, Open Source and Cross-Platform Technologies.
