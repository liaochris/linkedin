  • Variable Data Programming Internship/Web Design Rework

    Japs Olson 2009

    (Graphic Design industry)

    June 2009 — September 2009 (4 months)

    Worked in Data Processing department of Japs Olson learning customized processes and software for variable data processing, including scripting, sorting and compiling data. Work also included letter set up with Photoshop, InDesign and PrintNetT. Worked on implementing CSS design changes on the Japs Direct website. Data types that were used in the process were Excel files, CSV, disk, or tape. The data was formatted in tab-delimited format to be run through programs such as First Prep or TSS. Scripting had to be applied to the data for specific purposes and then the data would have to be parsed through one or more of many custom designed programs before it's final destination in letter setup. Letter setup utilized a program called PrintNetT in which the data is parsed via XML on the background of the application and then brought into design layout format for manipulation and output to inkjet and laser printers. All data has to be verified every step of the way to make sure there were no changes or discrepancies in the data which would result in spoilage on the press.
