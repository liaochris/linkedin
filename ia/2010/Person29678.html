James Broome’s Summary

I am a senior technology consultant specialising in developing web applications using agile delivery methods. For the last three years I've been working at EMC Consulting (formerly Conchango) delivering innovative retail and media projects using Microsoft technologies.

Most recently, I've led the delivery of EMC Consulting's first full cloud-based solution - seethedifference.org - a charity based web application built on ASP.Net MVC, running fully on the Windows Azure platform.

Over the last year I've championed the use of .Net OSS in EMC Consulting's Microsoft Development practice, through leading the development of fancydressoutfitters.co.uk - an e-commerce solution built with ASP.Net MVC, Sharp Architecture, N2CMS, Spark View Engine, SOLR, Fluent NHibernate and xVal. The success of this project has been pivotal in EMC Consulting's e-commerce strategy and has directly led to further similar engagements with new clients.

I have a strong focus on engineering practices and constantly seek to reduce project delivery friction. Since attending J.P. Boodhoo's Nothin But .Net training 'bootcamp' in 2008, I've been championing the use of BDD within EMC Consulting and have strived to push the Microsoft Development practice forward through internal Community Day sessions and blogging.

Community and Open Source:

I'm co-creator of the Who Can Help Me? project – an open source web application, providing a searchable skills profile matrix. Its main goal is to demonstrate an enterprise level architectural approach for an ASP.Net MVC web application. Who Can Help Me? is the official demo application for the Sharp Architecture project.

I've also created two contributory projects for the MSpec BDD framework:
Machine.Specifications.Automocking attempts to reduce noise in specifications when creating mock dependencies using Rhino.Mocks.
Machine.Specifications.Mvc is a set of extension methods for testing ASP.Net MVC specific types with a fluent assertion syntax.

James Broome’s Specialties:

.Net, C#, ASP.Net, ASP.Net MVC, N2CMS, Agile, Engineering Practices, Client Handling, Behaviour Driven Development, Test Driven Development, Scrum, e-Commerce, Retail, Project Delivery, Web Applications
