Jonathan Wood’s Summary

I have been a software developer since 1987 using tools such as C#, ASP.NET, C/C++ with MFC, Visual Basic (classic), C, and assembly language on the Microsoft Windows and DOS platforms.

My current goals include consulting positions developing websites using C#/ASP.NET, and desktop applications using C++/MFC or C#. I'm also interested in establishing new website businesses from scratch.

I've enjoyed a close association with Microsoft, earning their MVP (Most Valuable Professional) award for helping other developers on the Microsoft newsgroups. I'm also developing many "HOW DO I" videos for the Microsoft MSDN website on topics related to C++ with MFC.

I was a paid section leader for Visual Basic Programmer's Journal CompuServe forum and Internet newsgroups September 1994 through January 2003. This position involved answering technical questions online.

Authored Ask the C++ Pro for DevX February 1998 through September 1999.

Wrote the first chapter in the book The Official Visual Basic Programmer's Journal Guide to Visual Basic 4, Que Corporation, 1996. This chapter was entitled Object-Oriented Programming Visual Basic 4.

Have worked with people in Microsoft to write several knowledgebase articles for distribution on the Microsoft Developer's Network CD.

Have written a number of articles in the past for various languages including Detecting the Display Hardware, Windows/DOS Developer's Journal, January 1992.

Have also contributed many free utilities and source code examples via the Internet and CompuServe. One example was my widely-used VB-ASM DLL which allowed 16-bit Visual Basic programs to access interrupts and I/O memory. This DLL appears in several issues of the Microsoft Developer's Network CD (starting April 1995). Another example is my extensive PICTOR video library for C (windowing text mode library for DOS), which was reviewed in C/C++ Users Journal, June 1995, and entered into their CUG collection as volume 428.

It is with great appreciation that I laud this sort of content in your journal.
