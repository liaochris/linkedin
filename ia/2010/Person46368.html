    I developed software for the F/A-18 Maintenance Data Processing System. This system was developed in Ada, utilized a MOTIF user interface and an Oracle database for the back end.

  • Roboticist

    NASA Goddard Space Flight Center

    (Government Agency; 10,001+ employees; Defense & Space industry)

    September 1989 — February 1992 (2 years 6 months)

    Software designer and developer for the Space Station Flight Telelrobotic Servicer Test Bed. The test bed was a 7 DOF teleoperated system with Force Reflection. All software was developed in Ada for the 80386 processor. I developed forward and inverse kinematics for both master controller and remote manipulator systems.

    Although this program was canceled by NASA in 1991, the unit test oriented software development approach yielded a library of code that continues to be used in systems from satellites to later robotic programs.

    This was a tremendous learning experience for me. I worked closely with Stephe Leake, one of the greatest minds I have yet to encounter in the field of Software or Robotics. As the software development world continues to improve its practices and procedures, I continuously encounter some new element of Agile or Extreme programming and say, "We used to do that at GSFC, in 1990".
