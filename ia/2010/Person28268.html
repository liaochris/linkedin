Brian Dillard’s Summary

Versatile software engineer with nearly 15 years' experience developing rich user interfaces for consumer Web applications; experienced technical lead with solid management and code-stewardship skills; jack-of-all-trades writer with lots to say.

I know my way around SQL, Java, ASP and other server-side technologies, but my passion is the front end: rich, object-oriented JavaScript and AJAX, scalable CSS, semantic markup and all the cool things you can build with them. From consumer-facing Web sites and B2B tools to widgets and mash-ups, the UI is where it's at.

I'm also a staunch defender of the end user: the person who's actually going to use the code I spent so much time developing. Usability, Web standards, progressive enhancement and user-focused design are not only achievable, but essential.

With gigs everywhere from non-profits and start-ups to consulting firms and household brand names, I know how to navigate any type of organization. But I'm also happy sitting alone in front of TextMate cranking out reams of code and copy.
