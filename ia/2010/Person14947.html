Anthony Merrill’s Summary

Started as a Gameplay Programmer at Shiny Entertainment ( now Double Helix Games ), where I worked on "The Golden Compass", by Philip Pullman. Was in charge of Scripting specific levels, and other game-play tasks.

After it's release, I started on a new project 'G.I. JOE: Rise of the Cobra'. Where I was still helping script specific scenarios, was given tools programming tasks. Wrote a Synchronization tool to help automate the disk building process and also made an interface for one of Shiny's advanced excel sheet for character implementations.

In April, 2008 I moved back to the East Coast ( Home for me ) and started working at Venan Entertainment where I was a programmer working on mobile games. Working on NBA Street, and NBA Live 2010 as mainly the UI programmer I also helped create a tool called "Pixie", which was a 2D Image Compositing tool for artists.

After a year and half of mobile development my views changed and I felt the need to work on software that did more for the world then just provide entertainment, now I work in Cambridge, MA working for MAK Technologies doing military simulations.

Objective:
To pursue a career in a cohesive team working environment that strives for innovative and profitable software.
