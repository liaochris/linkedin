Lew Cirne’s Summary

I'm a software geek / entrepreneur with a passion for awesome products and building amazing teams. I had a blast founding and building Wily Technology, and (perhaps due to mild memory loss) am excited to be back into the game with my next company, New Relic.

New Relic is the premier application performance management solution for Java and Ruby on Rails web applications. Provided as Software as a Service, our customers can literally be up and running in full production within minutes, and immediately get excellent visibility into what their web applications are doing in production, in realtime. For more information, go to www.newrelic.com.

Lew Cirne’s Specialties:

Web infrastructure software, application performance management, operating systems, enterprise systems software.
