Don Seiler’s Summary

I have been working in Oracle database administration since 2001, starting with Oracle 7.3.4. In 2008 I earned OCP certification in Oracle 10gR2 database administration. I've worked with various version of Oracle up to 11g on Solaris, Linux and HPUX.

I also have much experience with many open-source technologies, languages and applications. These include various distributions of the Linux operating system, languages like perl, php and python, and applications like the MySQL database and Apache web server.
