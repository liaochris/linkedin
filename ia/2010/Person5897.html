Alessio Pace’s Summary

My research activities deal with the design, implementation and evaluation of distributed systems and algorithms, in particular large scale Peer-to-Peer communication protocols.

Passionate, versatile and curious in creating and sharing technology and innovation, facing challenging and varied problems in a stimulating environment.

Alessio Pace’s Specialties:

Pragmatic problem solving, Distributed systems and algorithms, P2P systems, Componed Based Software Engineering, Object Oriented Programming, Testing, Refactoring, Java.

Alessio Pace’s Experience

  • PhD candidate

    INRIA

    (Government Agency; Research industry)

    2008 — Present (2 years )

    * Research focus on distributed systems and algorithms for large scale P2P communications:
    - Spam filtering in the context of P2P information dissemination
    - Data replication algorithms in Distributed Hash Tables
    - Design and evaluation of a novel NAT-resilient gossip peer sampling protocol to maintain unstructured networks (In Proceedings of ICDCS 2009)

    * Software development to support research actitives:
    - P2P simulator design patterns leading to the development of various P2P simulator personalizations suited for different research investigations (gossip peer sampling, gossip broadcasting, NAT traversal, data replication in DHTs, ...) and written in different programming languages (Java, Python, C++)
    - Design and development of a lightweight event-based Python framework for unstructured overlay maintenance and NAT-traversal. Using the Twisted Framework for asynchronous programming model. Two modes of execution: distributed through sockets (deployed on the PlanetLab network), or in a single machine cycle-based simulated environment
    - Design and development of a Java event-based framework for scalable topic-based publish/subscribe P2P protocols. Component based (based on the Fractal component model), using the Java NIO libraries for event dispatching, extensible through a lightweight plugin system.
