Please note our new automated regression testing software toolkit & services offering to expedite application releases, which is from our retail delivery bank systems consulting practice, Gallagher Systems Group, Inc.

Here are the details:

Software testing has transformed from a cumbersome, inefficient manual activity to an automated process using standard software designed for testing across generic horizontal application environments. For this standard software testing tool to be useful, test parameters and test scripts must be written to tailor the generic testing software to a specific software application. By leveraging our industry knowledge and software technical expertise, Gallagher Systems Group has developed a retail delivery (banking) software testing toolkit with object models, testing methodology software, and best practice testing analysis that bridges the gap between the industry’s standard generic testing software and a specific retail delivery testing environment. This retail delivery software toolkit, which dramatically reduces time in testing and, thus, time between application releases, forms the basis of our new services offering.
