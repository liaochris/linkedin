Adam Buenz’s Summary

My primary expertise is in the development and security of SharePoint. My security specialty branches to associated Microsoft platforms focused around the hardening and identity space such as Identity Lifecycle Manager (ILM), the ForeFront stack (Forefront Security for SharePoint, Forefront Threat Management Gateway, and Forefront Unified Access Gateway)and Active Directory Federation Services (ADFS).

I also heavily focus on Team Foundation Server and building adaptable source control environments for dynamic organizations. My specialty in the TFS space is in build automation and geographically disparate environment inclusion, bringing orphaned development units into cohesive workspaces.

Clearances - Defense Security Service (DSS) – SECRET Clearance
