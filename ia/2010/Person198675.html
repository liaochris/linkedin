Stephen Wynkoop’s Summary

Microsoft SQL Server MVP 2003, 4, 5, 6, 7, 8, 9
Editor, SSWUG.ORG

Stephen Wynkoop is the founder and editor for SSWUG.ORG – the SQL Server Worldwide User’s Group where he writes a daily column and maintains the site overall. SSWUG features a weekly video program about the database and IT world, webcasts, articles, quarterly online virtual community events and vConferences several times a year. Stephen is a Microsoft SQL Server MVP and the author of more than 10 books, translated into at least 7 languages. Stephen has been working with SQL Server since the very first version, with a prior experience in database platforms that included dBase and Btrieve.

Stephen has started several companies and standalone corporate divisions, including:

NetServices - a nearly stand-alone division/business within IKON Office Solutions providing for cutting edge internet development and tools.

SWYNK.COM - an online community site featuring Microsoft IT technologies

Bits on the Wire, Inc. - a company focused on the application of technologies online in new and innovative ways.

vConferenceOnline - a virtual event solution that will let people hold virtual events, trade shows, workshops, classes - for paid and free attendees.

SSWUG.ORG - an online community site featuring database technologies for more than 500,000 registered members (growing daily) and providing continuing education, peer-to-peer articles and sharing, best practices and more. Stephen also writes the daily column for the site's newsletter, delivered daily to 100's of 1000's of people.

OfficeUsers.ORG - an online community site for the Microsoft Office user base, providing help on the use of Office products - from how-to information to tips, tricks and tools.

Stephen Wynkoop’s Specialties:

Building community, A/V on the 'net, high-capacity web-site development, "web psychology", experience-based online design, development and deployment.

Database implementations to enable real learning from information, along with the constant application of innovative technologies are a driving force in my life. I love to combine technologies in new ways that drive new benefits and features for the market.
