    Kripa works as a robotics software developer for Mind project. The project is meant to familiarize school children with robots. The developers are currently working on the 5th version of Iris robot. Iris robot is in elementary stage and does certain things such as talking, playing tic-tac-toe, drawing etc. The fifth version of Iris Robot is intended to give an eye and navigation engine for the Robot. Kripa works as the major contact on this part of the project. As of now the software can detect different colored blocks. The aim is to make the Robot recognize different letters in the block and to spell them correctly. A sensor for Iris Robot is also under development. The navigation engine for the robot will help to make the robot move on its own.
