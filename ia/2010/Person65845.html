Eric Friese’s Summary

Software Security Consultant with experience in:

• Delivering onsite consulting services including the installation, configuration, integration, and customization of products that when effectively deployed protect organizations from the threats posed by security flaws in software applications.

• Performing static code analysis of software applications written in a wide variety of programming languages including Java, C#, C/C++, ASP, and PHP to fix potentially exploitable vulnerabilities at the root cause early in the software development cycle.

• Performing dynamic analysis of software applications to find vulnerabilities that become apparent only while an application is running and to actively defend vulnerable applications until appropriate remediation steps are developed.

• Teaching training courses that provide students with the knowledge needed to recognize and avoid critical software vulnerabilities, perform thorough source code audits, write secure code, and continuously generate secure software applications.

• Interfacing with Project Management and Product Engineering to enhance deliverables based on knowledge gained deploying products in the field as well as customer feedback.

Certifications: CEH, Security+
