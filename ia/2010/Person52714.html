Dmitry Balin’s Summary

Entrepreneur and database technologist.

Founded a social shopping Web 2.0 startup - http://www.Pikaba.com - we just won Open Web Award 2008 at mashable - wow!!!

Founded a software startup (DB Best Technologies) in Silicon Valley at 2002, developed database migration technology, and sold it to Microsoft at 2005. He built a software team in Ukraine from zero to 75+ employees – the only company in Eastern Europe developing a full software product for Microsoft. After the acquisition Dmitry was Director in the SQL Server group at Microsoft, where he led their “Migrate to SQL Server” program generating over $100M annual revenue for Microsoft database business.

He’s a frequent speaker on database topics at TechEd, PASS, TechReady, and other conferences.

DB Best provides now software development services, database consulting (upgrades/migrations, performance tuning, remote DBA services, BI/ETL/SSRS/SSAS projects), and Web services (site design, development, SEO, promotions).
