Ali Khan’s Summary

As a Senior Consultant for SharePoint Products and Technologies, Ali has real-world experience evaluating, planning, developing, and deploying complex IT solutions. Ali excels at communicating technical issues and best practices to customers, analyzing requirements, and delivering solutions with the customer’s best interest in mind.

Ali's current emphasis is on architecting collaborative solutions using Microsoft Office SharePoint Server 2007 (MOSS) and SharePoint 2010 server products and providing engineering support for disaster recovery strategies, designing and implementing highly-available and redundant SharePoint Farms.

Providing valuable technical information to support proposal teams responding to large Federal Government RFPs related to SharePoint technology implementations.

In addition, developing systems using the Microsoft .NET framework using the latest object-oriented design patterns and SOA architecture, establishing integration points with other Microsoft enterprise products including: SharePoint, BizTalk, InfoPath, SQL Server, Reporting Services, PerformancePoint Server and Project Server.
