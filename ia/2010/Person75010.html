Gail Rahn Frederick’s Summary

Architect, mentor, author, instructor and evangelist for standards-based Mobile Web and mobile application development.

Author of "Beginning Smartphone Web Development: Building Javascript, CSS, HTML and Ajax-Based Applications for iPhone, Android, Palm Pre, Blackberry, Windows Mobile and Nokia S60", published by Apress in December 2009.

Establishes product requirements and technical specs for Mobile Web and native applications on smartphone and featurephone platforms. Directs mobile software engineers developing mobile search, discovery and analytics applications.

Located in Portland, OR.

Gail Rahn Frederick’s Specialties:

Mobile application development, Mobile Web development, enterprise software for the mobile industry, technical management, technical communication, customer interaction.
