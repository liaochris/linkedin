Jason Slater’s Summary

Independent blogger, writer and software developer interested in Internet and new media technologies.

Involved in the field of Information Communications and Technology since the early 1980s during which time I developed a number of computer software applications, both business and entertainment. Particular areas of focus is Internet technologies including PHP, ASP, CMS, Java.

Experienced IT and IS Manager having managed IT systems since 1989, including MRP, ERP, T+A, HR, SQL.

Awarded City+Guilds 746 in Computer Programming in 1984 and City+Guilds 417 Applications Programming in 1987.

Strong programming skills including Cobol, VB.NET, C#, PHP, Sun Java and C++. Strong database skills MS SQL and MySQL. I also qualified as a Novell CNE (Certified Network Engineer) and actively support and develop a Windows Infrastructure.

Blogger on a number of sites including JasonSlater.co.uk, SL8R Code Zone, Eggnchips Search Engine Project, IT Pro, Networld World Community and Gizmos for Geeks.

Author of a number of online cartoons including Bev and Catey, Monk and his I.T. Junk and Abbey Days.

2009 - Qualified as Certified Sonicwall Security Administrator (CSSA)

2008 - Awarded EDS Prize for Best Masters Project

2007 - Qualified as Portable Appliance Tester (PAT).
