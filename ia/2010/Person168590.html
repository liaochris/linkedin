  • Internship (Master) at Department of Anatomy, Embryology and Physiology (AEP), Heartfaillure-centre

    AMC

    (Privately Held; Hospital & Health Care industry)

    November 2009 — Present (8 months)

    Automatic categorization of gene expression type in developing hearts

    Gene expression in a stained heart is currently annotated as being either negative or positive. Annotation of an area or a group of areas is currently defined as: No expression, Scattered, Regional, or Ubiquitous.

    A possible way of analyzing each image is by computerised, automated recognition of expression type. Based on my two recent intern-ships I have developed some hypotheses and created a Matlab application which enables the annotation of scanned images of sections of the developing heart. This application uses decision rules to assign an expression type to the regions in the image.

    Description
    Overall there is a need to automatically annotate gene expression types in stained sections of the mouse heart. Furthermore the results have to be inserted in a 3D database in order to perform queries later on.
    The current application is not based on real images but on simulated images. In addition, there is a problem with the definitions of expression types that exist at the borders between the above mentioned expression types. Similarly, the annotation of gene expression type on the edges of an expression domain is a problem. Finally, the application to biological images requires an algorithm to make valid decisions on re-categorization of gene expression types based on neighbours in a certain area.

    Research questions
    · How to perform automatic categorization of gene expression types in developing mouse hearts?
    o What methods for image analysis and annotation decision can be
    used?
    o How to determine definitions of gene expression types?
    o How to validate the sensitivity and specificity of automated gene
    expression annotations?
    o How to use the information from different anatomical regions in the
    heart?
    o What is the biological relation between gene expression areas?
    o To what extend does aggregating information from in the 3rd dimension improve the decision mechanism?
