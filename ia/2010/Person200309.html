Tony Cannizzo’s Experience

  • Managing Director

    DynamicDB - Dynamic Database Solutions

    (Computer Software industry)

    December 2009 — Present (5 months)

    DynamicDB provides break-through technologies that solve Performance and Security problems that before now were impossible.

    Implemented at the SQL*Net Protocol Layer, ActiveBase is a patented SQL*Net Proxy that acts as an Oracle listener. Through a simple rules architecture, it is able to inspect inbound SQL, identify undesirable code and dynamically replace it with preferred code before it hits the database.

    From a security perspective, Dynamic Data Masking is the new frontier in Data Privacy as it is the first solution that can mask Sensitive Private Information (SPI) in the Production environment. By masking data at the presentation layer, it can even protect SPI from Outsourced Production DBA's from browsing while allowing them to do their job.

    From a performance perspective, often the answer is "Rewrite the SQL". But this requires access to source code. Customers of ERP/CRM, BI/DW and Development Tools are not permitted to change the source. ActiveBase can identify offensive SQL before it hits the database and replace it with optimized code on the fly. The database recieves the better code and executes it instead.

    (Computer Software industry)

    December 2008 — Present (1 year 5 months)

    Silos-Connect Technolgies provides software and services solutions for optimizing application performance and business performance.

    Data Privacy and Security - Gartner refers to this technology as Dynamic Data Obfuscation, and also considers it to be unique in the market. It is expected to appear on the next Gartner "Hot List". Sensitive data can be masked, encyrpted, hidden or disguised in a dynamic production environment. This solution is even able to improve the execuition of 3rd party applications where customers do not have access to the source code. The performance imrprovements occur without changing the source or the database.
