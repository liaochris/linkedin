Stephen Millea’s Summary

Currently I am a Creative Director for User Interface/Web designer for Facebook applications, iphone applications, and web application in my company Betapond: from the initial requirements, to the project specs and documents, to the design, development and implementation of these applications.

I started my professional career as a User Interface Designer, with a special interest media rich websites.

Afterwards, I keep following the advancements of Web 2.0 User Interface Design, concentrating on topics like web standards, usability, accessibility, mobile technology and social networks.

Holding senior positions at many important companies, I have been expanding my skills in Front-end User Interface Design and gaining wide knowledge of technologies like Ruby on Rails Information Architecture and User Experience Design, Joomla, Word Press and Rails Refinery.

I also have vast experience in Search Engine Optimization, Web Analytics, Online Marketing Strategies, and Facebook Marketing Strategies allowing my companies to gain in popularity.

In the last years I have particularly been focusing on all the features of Web 2.0 User Interface Design, Facebook Application Design and iphone Application Design.
