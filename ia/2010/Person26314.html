Bruno Pedro’s Summary

Bruno Pedro is a systems engineer with over 10 years’ experience in database related applications who’s been developing Web Applications since 1995.

He’s the author of, among other Open Source software, Auth_Container_SOAP, PEAR’s SOAP based plug-and-play authentication package.

Currently he’s interested in Web services and Large Scale Web applications using Erlang, PHP and most recently node.JS.

On his spare time he enjoys exploring the Internet of Things with the Arduino platform.

Bruno Pedro’s Specialties:

Computer languages and protocols: C, C++, Erlang, PHP, Python, XML, (X)HTML, Java, JavaScript, CSS, LaTeX, DocBook, TCP/IP, HTTP, DNS, FTP, SSH, SMTP, POP, IMAP, SOAP, XML-RPC, OpenID, OAuth, AJAX.


Operating systems and software: Linux (Debian, Fedora), Mac OSX, SUN Solaris, Microsoft Windows, MySQL, PostgreSQL, Apache, Squid, QMail.
