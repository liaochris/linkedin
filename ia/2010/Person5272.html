Alan Tai’s Summary

Software architect with over 13 years C++ and Java J2EE experience in Web and vertical search, mobile search, real-time search, social networking, SEO analysis, and data mining of search logs on the 5th most popular search engine. At AOL, led development and launched over 16 major products, with a focus on high-performance, high-scalability, fault tolerance, and security. Comfortable coding at any tier: client-side JavaScript, Java servlets on the front-end, and Web services and MySQL integration in the back-end.

Alan Tai’s Specialties:

Mobile: Java ME on BlackBerry, iPhone and Windows Mobile DHTML
Web: Java, J2EE, Javascript, AJAX, Dojo, JSP and Servlets, Struts, Spring, JDBC, Swing, SOAP, XSLT, XHTML/CSS, Python, Perl, Ruby, TCL, Tomcat
Back-End: C and C++, Lucene, SOLR, PLS, MySQL, Berkeley DB, Sybase, SOA, Hadoop, ETL
