Anthony Bryan’s Summary

My current research and focus is on Internet standards architecture, development and deployment, specifically:

Metalink/HTTP: Mirrors and Checksums in HTTP Headers (requesting review)
http://tools.ietf.org/html/draft-bryan-metalinkhttp

FTP Extensions for Cryptographic Hashes (early stages, requesting review)
http://tools.ietf.org/html/draft-bryan-ftp-hash

RFC 5854 Metalink Download Description Format

RFC 5843 Additional Hash Algorithms for HTTP Instance Digests

Metalink is an open specification that harnesses the speed and power of traditional downloads and peer to peer networking in a single click.

For users and content providers, Metalink makes downloads simpler, faster, and more reliable.

I work on the Metalink framework which offers high availability self healing (automatic error correction) downloads and is used for improved file distribution by commercial and open source organizations such as OpenOffice.org, openSUSE, Sabayon, Ubuntu, Fedora, Mandriva, Eiffel Software, UniProt Consortium, Swiss Institute of Bioinformatics, and other Linux/BSD/OpenSolaris distributions. It's used to legitimately distribute software, games, music, and movies.
