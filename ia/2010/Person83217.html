  • Intern

    AT&T Research Labs, Inc

    (Computer Software industry)

    May 2008 — August 2008 (4 months)

    Prototyped a thread synchronization model for SIP Servlet Containers. SIP is a preferred signaling protocol used by IP multimedia applications (such as VoIP applications). These applications have proved to be difficult to program. One source of such complexity is the need to synchronize concurrent threads. Synchronization is difficult in this domain because a thread tends to have complex synchronization constraints. The model I proposed and developed is to separate such complexity from the routine programming task of the functional code.
