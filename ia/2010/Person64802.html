Emmanuel Gasquez’s Summary

I am a Software Engineer who has had a wide variety of work over the course of my career. I have experience in a range of programming languages including C, C++, Java, Python, PHP, JavaScript, SQL and a few Linux Scripting languages including bash and Perl.

My C experience has been in Linux device driver development in the area of communications hardware. (Serial PCI Card and the MAC layer for a communications system).

I used C++ to write application level code to control the same serial device I helped write the driver for. This was heavily threaded and worked flawlessly.

I made modifications to a GUI written in Java. The GUI displayed radio node locations on a geo-positioned map. My work was to modify the GUI to allow for more than 3 nodes connecting via 2 different and switching networks.

My experience in Web development languages PHP, Python, JavaScript and SQL began as a hobby to create a website for some friends of mine. I expanded this experience to my job at BAE. I first modified a Web Interface which handled communications between various communications systems in order to support a further com system. I then began working on an administrative tool which helps manage users, groups, and various system configuration settings for a mobile system which uses three different servers to handle the tasks listed above. I continue to work on Web Development things at home on my free time to learn and expand my knowledge. Lastly, my skills led me to MIT Lincoln Laboratory where I helped write a Web UI for controling and monitoring a complex system.
