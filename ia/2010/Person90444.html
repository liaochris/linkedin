Jason Fugate’s Summary

I have worked in the field of software engineering for more than a decade, and games for more than half of that time. My first job was in a tiny company doing 1 man/1 application, then a Fortune 50 company where a classic waterfall methodology was used, at Midway Games, on a 10-15 man tools and tech team, then at EA Chicago, as a lead engineer shipping 360 and PS3 titles, and now at Hypnos, working on top secret but cutting-edge game tech. Through this variety of experience, I have developed a core set of personal best practices while maintaining the flexibility necessary in diverse situations.

  • Senior Software Engineer

    Hypnos Entertainment

    (Privately Held; Computer Games industry)

    January 2008 — Present (2 years 10 months)

    At Hypnos, we get hired by other game companies needing expertise at either starting or finishing a project.

    My first project at Hypnos (which I joined in the middle of) was to ship Viking: Battle of Asgard for Creative Assembly on the PS3. I optimized the particle system using VMX SIMD instructions to get it within an acceptable percentage of frame time as well as all the usual crunch time optimizations and bug fixes that go into finaling a project.

    Our next project was to lay the foundations of a next generation console MMO engine for Turbine. I refactored a PC only build system to support any platform, ported a 4 million line code base that used many Microsoft-specific structures to 100% clean code on GCC, and refactored the memory system so that all memory allocations were tracked and reported, among many other things.

    Our most recent project was another MMO, this time for NCsoft. Details are under NDA.

    I am now working on interesting tech for Sony.
