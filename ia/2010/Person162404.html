Perttu Tolvanen’s Summary

Designing Web services and managing complicated Web projects is what I do best.

My daily work is doing user research, leading concept design workshops, developing information architectures and user interfaces for large Websites or intranets and using different kinds of measurement tools to support decision making.

Perttu Tolvanen’s Specialties:
