Anders Buvarp’s Summary

Systems engineer working in the field of multicore real-time wireless communications; WiMAX, LTE, Software Radios, SDR, DSP, Architecture.

MATLAB, Turbo Coding, BCH, Viterbi decoding of QAM/QPSK signals. Burst detection of RACH, FACCH, DKAB and TCH traffic.

Unscented and Square-Root Kalman Filter and Linear Prediction Coding, Code Composer, TI 672x.

Down-conversion from IF to baseband. I/Q sorting.
Correlation algorithms. Direct P(Y) acquisition using FFT.
Indoor Time-Of-Arrival navigation using pseudolites.

Extensive background in various DSP algorithms; FHT, FFT, IIR, FIR, Windowing, Decimation, Interpolation, LPF, Hamming Windows, DMA, PCM, DPCM, ADPCM, AAC, MPEG, speech recognition and synthesis.

Anders Buvarp’s Specialties:

DSP algorithms for multicore wireless systems.
