David Gerlach’s Experience

  • Senior Software Developer

    AccessVia Inc.

    (Privately Held; Computer Software industry)

    August 2005 — Present (5 years 4 months)

The template layout tool is used by marketing/Photoshop people and as such has features that rival the best authoring environments and modern UI design. The engine and print controls run on many platforms and output native to every major output technology. Currently used in over 50,000 locations in the US, Japan, China and parts of Western Europe. They also rank in the top 50 software products that are used in a retail/POS environment. www.AccessVia.com

    Also worked on SaaS model that allows Retailers to host their services in a system that was multi-tenant using SQL Server and web services. Front end was legacy ASP/DHTML.

  • Software Engineer

    Cardiac Science

    (Public Company; CSCX; Medical Devices industry)

    September 1999 — August 2005 (6 years )

    Lead engineer on several cardiac monitoring products to include stress testing EKG and invasive cardiac catheterization as well as EKG management product called Pyramis, which is a web based application for use in storage and retrieval of all cardiac radiology and image data. It has become the market leader in the past few years since it was release. I worked the .NET interop to a legacy backend. The front end was ASP.NET that talked thru WSDL to the back end services.

    Key player in the development of the Q-Stress Exercise ECG Testing System which was released April 3, 2001. Developed loosely coupled publish/subscribe eventing component (Event Manager) for use by 200+ COM components. Developed XML template approach for all real-time reporting/charting and DHTML/JPEG output. Researched use of Adobe Acrobat 5.0 PDF output for diagnostic quality output of traces. Developed process to image OS/service packs/MSDE install and supervised all setup issues with the inclusion of an InstallShield script. Q-Stress is a soft real-time system with 16 channels of ECG data coming from a data collector card containing a PLX PCI bus controller.
