  • Database Developer

    Merkle Inc

    (Privately Held; Marketing and Advertising industry)

    January 2007 — March 2010 (3 years 3 months)

    After cutting my teeth as a production specialist, I moved onto more developer focused tasks. Not only did this involve purely SQL development, but it also involved a lot of COTS integration projects. One of the biggest challenges was to move from a SQL Server based environment to a Netezza focused one. This involved a platform migration, as well as the integration of several other disparate systems (Tidal, DMExpress, SQL Server and Netezza to name a few!). The database update runs more than twice as fast as it did before and that's really saying something for a multi-terabyte (tables with over a billion records!) system! Overall, a very good learning experience.

    Today I continue to build on, and expand that platform, getting some .Net code into the mix among other things. This is the great thing about working at Merkle; there's always some new boundary to push up against and something new to learn along the way.
