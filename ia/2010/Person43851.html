Wil Everts’s Summary

I'm a web developer obsessed with building beautiful, pragmatic, and surprising web experiences. I've been writing HTML by hand for 16 years. I started my professional career as a web geek out of High School in 1996.

Most recently I was the UI Architect/Community Advocate at Zivity, where I worked for 2 years.

Today, I work as a contract-based developer helping companies optimize their front-end code, hire qualified developers, and create fun/profitable web applications.

Wil Everts’s Specialties:

UI Design, Front-End Web Development, Templating (Smarty/Template Toolkit, PHP, Rails, Wordpress), Cross-platform hacking, Valid HMTL, CSS, jQuery, W3C, Unobtrusive Javascript, and PHP.
