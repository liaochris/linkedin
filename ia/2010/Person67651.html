  • Project Lead, Atlas Core platform

    Diagnostic X-Ray Engineering, GE Medical Systems

    (Public Company; 10,001 or more employees; GE; Medical Devices industry)

    September 1999 — April 2001 (1 year 8 months)

    As a software project lead, I was responsible for closely working with a team of 15 bright engineers to deliver infrastructure components platform that was to be used across Vascular, RAD, R&F and Mammo X-Ray systems. Atlas core platform needed to be highly scalable distributed platform addressing the needs of very diverse X-Ray systems which ranged from 150K USD to 1.5 Million USD. This challenge was further complicated by the fact that the development team is distributed between USA, Hungary and India and integration teams were in Milwaukee, China and France.

    Atlas core platform was built on the same code base to work on Windows, Linux and Vxworks OS's and was deployed on embedded SBC and multi-CPU PC based systems. To enable this we had to design and develop highly efficient distributed communication mechanisms and used CORBA for location transparency of software objects. Service applications were delivered on a web platform and required Java and C++ interfaces.
