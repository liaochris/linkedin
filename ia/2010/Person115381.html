  • Senior Software Engineer

    Bridgeborn

    (Privately Held; 11-50 employees; Computer Software industry)

    February 2002 — February 2006 (4 years 1 month)

    Inventor, product manager, and technical lead for Bridgeborn's core technology, Bridgeworks. Bridgeworks is a software platform for advanced visualization and human computer interaction that runs on a standard PC. As technical lead, Kevin articulated requirements and made design decisions, implemented software libraries in C++, and created Bridgeworks applications for Bridgeborn customers. Bridgeworks is a lightweight (< 5MB) client runtime engine that provides a high level abstraction for 3D models, maps, terrain, charts, graphs, and text. It is built on the OO principles of reflection and serialization, making it convenient to manipulate any aspect of a visualization at runtime by sending and receiving XML or other well-formed text (ex., JSON, CSV). Bridgeworks can be embedded in browsers, Office documents, and integrated with C++, Java, and .Net applications.
