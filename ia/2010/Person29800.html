Bruce Foley’s Summary

* Database Programming - Design and development of data migration and data integration architectures for large data warehouses and major software applications (e.g. JD Edwards). Over 12 years experience with Teradata, Oracle, SQL Server and DB2 using SQL, PL/SQL, T-SQL and numerous database administration/access/loading tools (ERwin, Toad, Quest, SQL*Loader, Mload, DTS, Perl DBI, etc.).

* ETL Processes - Analysis of source systems and data quality. Development and performance tuning of complex Extract, Transformation, and Load processes. Automation of ETL processes with scheduling tools and exception-handling procedures. Development and execution of ETL test plans. Considerable experience with data cleansing processes to ensure accuracy (parsing structure & content, correction & validation, standardization, matching to business rules, supplementing with existing data)

* Informatica - over 8 years experience building repositories, mappings (including re-useable objects such as mapplets), implementing sessions/batches with Informatica Power Mart and Power Center. Development of robust Korn shell scripts for integration with third party schedulers. Considerable experience with loading both Oracle and Teradata databases using both file and relational writers.

* UNIX - over 12 years experience on AIX, Solaris, HP-UX platforms. Development of complex shell scripts for running ETL and integration processes. Development of standards and script templates for team projects.

* Windows - over 17 years experience on Windows 3.1 through Windows XP. Developed rock-solid automated load and extract processes using CMD batch scripting and Windows Task Manager.
