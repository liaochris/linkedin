Jillian "JK" Hudson’s Summary

From print to web to multimedia and finally software interfaces, when it comes to design, I've done it all! I understand that design's function is to support the message and branding for each client and usability and user experience for each user. It has been my experience that consistency in image and message is the key to great branding.

I've been in design so long that "good design" has come back in fashion. It's now re-branded as Usability, User Experience and PET. I've been doing it all along, they just invented a new vocabulary for it!
