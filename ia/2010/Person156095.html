  • Software Developer and Project Manager

    Lawrence Berkeley National Laboratory

    (Government Agency; Research industry)

    November 2007 — Present (3 years )

    From 2007-2009, I was the project manager for OBO-Edit, an open source ontology editing tool written in Java. In addition to directing the activities of another software developer, I ran weekly user group meetings, collected user requirements, fixed bugs, improved the user interface, issued software releases, and wrote user documentation.

  • Software Consultant

    Vinq, LLC

    (Privately Held; Information Technology and Services industry)

    2008 — 2009 (1 year )

    I helped to improve the performance and user interface of RADAR Mail. Written in perl, Mason, Javascript, and Ajax on top of a MySQL database, RADAR Mail is an integrated email and task management system that incorporates machine learning algorithms to help users manage their workload as it relates to their email communications.

  • Director of Communications and Senior Software Engineer

    Omicia, Inc.

    (Privately Held; 1-10 employees; Biotechnology industry)

    July 2006 — November 2007 (1 year 5 months)

    Omicia is an early-stage company focusing on genetic risk prediction. As a Senior Software Engineer, I greatly improved the speed, reliability and user interface of the company's web-based genetic variant visualization tools. In my role as Omicia's Director of Communications, I wrote press releases, handled media inquiries, contributed to grant proposals, wrote business documents, and prepared slide presentations for meetings with potential collaborators or investors. I helped to shape the company's business and marketing strategies. Through my efforts, Omicia was chosen to participate in the NIH's Commercialization Assistance Program, aimed at providing business guidance to biotech companies that have received NIH funding.

  • Software project manager and developer

    Lawrence Berkeley National Laboratory

    (Government Agency; Research industry)

    June 1995 — July 2006 (11 years 2 months)

    As project manager for the Apollo Genome Annotation and Curation Tool (http://apollo.berkeleybop.org/current/), I led the transformation of a barely working prototype to one of the most popular annotation tools available today (with close to 10,000 downloads so far). Apollo is an open source Java application that allows researchers to explore genomic annotations at many levels of detail, and to perform expert annotation curation, all in a graphical environment. The improvements I made to the Apollo package included developing a framework that allows Apollo to be installed quickly and easily on any platform, and writing extensive user documentation.

    Before the Apollo project, I designed and implemented Genotator, an interactive graphical workbench for sequence annotation. Genotator has been downloaded by thousands of groups worldwide, and several groups have used it as a basis for developing their own annotation systems.

  • User interface consultant

    DoubleTwist

    (Privately Held; 51-200 employees; Biotechnology industry)

    1996 — 1998 (2 years )

    As a part-time consultant, I helped to improve the company’s Web site and the user interfaces of several bioinformatics products. I also added a module to GeneWorld to enable remote processing of certain sequence similarity searches.

  • Software developer

    Arris Pharmaceutical Corporation

    (Public Company; 51-200 employees; Pharmaceuticals industry)

    1993 — 1995 (2 years )

    I participated in the design and implementation of Navigator, a molecular database visualization tool written in Lisp. Navigator offered intuitive ways for medicinal chemists to view and analyze structure-activity relationships among candidate drug molecules.

  • Software Developer

    University of California, San Francisco

    (Educational Institution; 10,001 or more employees; Research industry)

    1992 — 1993 (1 year )

    My work at UCSF involved four-helix bundles, an important protein structure motif. I derived several criteria for recognizing traditional and less-traditional bundles from atomic coordinates, and wrote code that offered insights into helix-helix packing based on geometrical analysis.

  • Software Developer

    National Center for Biotechnology Information, NIH

    (Government Agency; 1001-5000 employees; Research industry)

    1991 — 1992 (1 year )

    David States, Lawrence Hunter and I developed and implemented an efficient unsupervised classification algorithm for use on very large sequence databases. Our algorithm (HHS) was used to recognize and group together sequence motifs that form the building blocks of proteins.
