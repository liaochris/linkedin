With the application it is possible to integrate all businesses and all forms of eCommerce, with print and screen media. The code (2D data matrix, QR-code, 1D barcode or color code) or other type of tag (picture, OCR) gives mobile phone users multiple access and interaction, for an example but not only to internet content simply by pointing their phone at a code.
