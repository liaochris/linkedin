  • Undergraduate Research Assistant

    UOIT

    (Educational Institution; Higher Education industry)

    April 2009 — September 2009 (6 months)

    Current touchscreen and tablet technology limits the user to attempting to interact with the computer using the same interface as a mouse would use. This makes navigating webpages difficult and cumbersome. To help with this issue cross browser jQuery and GreaseMonkey plugins have been created. The jQuery plugin allows a web administrator to enable touch controls on their website in just a few lines. The GreaseMonkey script allows any user to make any page they view use the touch screen friendly interface. These controls include guessing at the most likely meaning of a touch and responding accordingly either by executing an action, scrolling the screen, displaying an onscreen keyboard, or when uncertain giving a selection of the most likely event. To obtain this a state machine was created with the ability to enable and disable the features or swap them as necessary. The resulting touch system effectively compensates for the lack of a mouse and its precision.

  • Undergraduate Research Assistant

    UOIT

    (Educational Institution; Higher Education industry)

    April 2009 — September 2009 (6 months)

    Despite the extensive mobile resources available at the University of Ontario Institute of Technology, there are few examinations or evaluations performed on these devices due to lack of security, software conflicts, and possible cheating. This research presents a new protected and controlled environment in the form of a customized, non-persistent Linux image on a USB flash drive. When the user loads the operating system, a configuration script is run that will set up the user environment for the specific test or exam being written. The professor or lecturer issuing the exam will select which programs, Internet addresses, and resources the student will be allowed to use, prior to the exam. The configuration script is implemented and the secure environment begins. The Linux image has been modified to not allow access to other storage devices on the computer internally or connected externally. The rest of the system is controlled by monitoring software that include an internal firewall, software watchdog, and restriction of permissions to executable binaries. This technology will be used during 2009-2010 academic year by a number of professors to demonstrate feasibility and help find areas for improvement.
