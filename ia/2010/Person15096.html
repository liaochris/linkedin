Antoine Meyer’s Summary

I am a software and web developer specialized in Java development.
I have been working for several companies throughout the world (France, Malta, California) and have thus seen and experimented with various technologies and ways of working.

I am most experienced in Front-end web development and have been using Google Web Toolkit during my latest work experiences.
I am able to build a whole GWT 2.0 application using a classic MVC pattern or the MVP architecture. I am well aware of the best practices and always keep an eye on the performance, testability and re-usability of my code. I am experienced using UIBinder, defining RPC services or using the gwt-dispatch library, using an EventBus ...

I am confident with the related web technologies like XML & JSON for data communication, HTML & CSS for style and layout.
I have also worked on the back-end side using simple PHP scripts (Apache / MySQL), Object Persistence via JDO on Google App Engine (in the Cloud) with Dependency Injection via Guice and JSPs (Tomcat / MySQL).
I am familiar with Eclipse IDE and some tools like Maven, Ant ...

Always looking to improve myself and to keep up with technology innovations, I attend conventions like Google I/O 2010 and I am part of the user groups related to Java or Google technologies.

Other development technologies I studied are EJB and Struts in Java, C# on .net 3.5, PHP and PL/SQL. I am familiar with networks, system administration (Windows Server, Linux) and database administration (Oracle).
I have also learned some law and marketing basics linked to Computer Science, as well as management (like ITIL or Agile methodologies (i.e. Scrum)).
