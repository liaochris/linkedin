Brandon Wirtz’s Summary

I am currently working as an SEO/SEM/Online Marketing consultant.

I was formerly compressionist and technical lead for Microsoft TV's Merlin Lab. A group dedicated to best practices and research in content processing for IPTV. I worked with HD content for both Live and On-Demand in VC-1, H.264, and Mpeg2.

I was Chief Operations Officer for Griffin Digital Solutions. My primary function was as an outsourced consultant to companies working with Streaming Media, Distance Learning, Deployment, and Security.

I have built a network of clients and developed custom solutions based around the Windows Media Platform through London based Digital Media Communications LTD . Working with DMC, I created solutions for tracking the viewership of Viral Marketing clips that were used by companies including Virgin, Mazda, Microsoft Xbox, Electronic Arts, MTV and numerous others In the US and UK.

When Windows 2003 Server was released to beta I saw opportunity to use its improved streaming server as a platform for paid content. Prior to the ship date of Windows 2003 Server I developed QANTA a complete authentication and metering platform for monetizing Windows Media content. QANTA used the robust WMS API to create a set of server plug-ins that allow users and passwords to be stored in a SQL Database and Quota to be applied to those users. Combined with the fast start, and stream seek features built in to Windows Media Server 9 it allows for a better than “on demand” pay per view system for online content.
