Rachel Slaybaugh’s Experience

  • Rickover Fellow

    The University of Wisconsin - Madison

    (Educational Institution; Higher Education industry)

    September 2006 — Present (3 years 8 months)

    I am doing research toward my PhD. I am working on eigenvalue acceleration techniques for a deterministic code. The focus is on scaling issues and using preconditioners in a massively parallel environment.

    Past Reasearch:
    - Developed Monte Carlo fusion sources: takes plasma physics data, create a source distribution function, and sample it in Monte Carlo. For both toroidally symmetric and arbitrarily shaped plasma sources.
    - Developed secondary source sampling method for DAGMC (a software library which can be used with MCNP(X) with a CAD package so that transport can be done directly on any 3-D geometry). An arbitrarily shaped photon source created through activation can be sampled.
    - Tested DAGMC's capabilities, for example comparison with criticality benchmarks.

  • Fellow

    Knolls Atomic Power Laboratory

    - Helped couple the Monte Carlo code MC21 to the deterministic code Jaguar via the interface code UNIK for the purpose of variance reduction.
    - Added weight windows to MC21 while maintaining previous importance sampling capability.
    - Investigated and recommended methods for creating weight window values for MC21.
