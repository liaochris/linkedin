Abdul Hafiz Ibrahim’s Experience

  • Founder

    The Hyper-Cloud Venture

    (Computer Software industry)

    2008 — Present (2 years )

    Hyper-Cloud(tm) is a virtualising real-time kernel I developed after nearly two decades of personal research to solve the problem of how general purpose concurrent software could be developed by any general skill level programmer without having to resort to multi-threading or critical code segments. Additionally, how to dynamically emulate non-native instruction sets with high performance to handle legacy code as well as to decouple software source code from specific hardware environments.

    I designed the Hyper-Cloud(tm) kernel to be deployed in huge data centres for hosting games, mobile applications and IT services to be accessed by a range of wireless and wired access devices.

    A key differentiator of a Hyper-Cloud(tm) based data centre is that software applications/services can be rapidly developed, deployed and maintained as almost pure UML models with automatic generation of parallel code and load balancing for scaling, as well as mirroring for both process and data level redundancy.

    The software design methodology expressed through the UML models follows biological computing principles yielding a high degree of parallelism and redundancy (like in biological systems). Additionally, the architecture supports efficient implementation of neural networks. This opens the door to a new class of biological algorithms with pattern recognition and learning capabilities way beyond that possible using current computing approaches.

    Currently a Hyper-Cloud(tm) implementation exists for the Cell processor (as used in the Sony PlayStation 3 and IBM's Cell Blades).
    Versions of Hyper-Cloud(tm) are being developed for the ARM processor as well as for the Loogson (Dragon Chip) processor.
