  • Software Design Engineer

    Hewlett Packard Company

    (Public Company; HPQ; Computer Hardware industry)

    June 1990 — February 2004 (13 years 9 months)

    I led HP’s Unified Event Database project. This Oracle Enterprise Server data warehouse serves as the centralized repository for message codes generated by all hardware products from HP to enable HP product divisions to use only unique message codes and HP support groups to more quickly identify problems using those codes. In addition to leading the globally dispersed team, I also functioned as a data architect, data modeler, database developer, and software developer. The data warehouse was internationalized using the Unicode character set, setup for country-specific language localization, and populated from individual division’s data marts using XML for extraction, translation, and load (ETL). Tools included PL/SQL, Oracle Forms, XML, and VBA.
    We used the Waterfall Model of the software development life cycle (SDLC) from inception, and as a result, we wrote the Requirements Definition, Project Plan, Project Schedule, Work Breakdown Structure (WBS), Design Specification, Data modeling (Entity Relationship E/R, UML, application flow, data flow, storyboard, use cases), External Reference Specification, Character Set Migration Guide, and Server Setup Guide. This resulted in far lower support costs for HP.

    I developed a data warehouse and website for HP’s partnering consultants to advertise services, using Oracle Enterprise Server, Apache, JRun application server, Java Servlets, and JDBC. Testing was automated using WinRunner and LoadRunner to track the performance of our website.

    At first I was responsible for improvements to the ALLBASE/SQL relational database engine, including query parser SQLcore and physical storage manager DBcore, and their subsystems: buffer, disk, file, lock, latch, log, transaction, replication, and ODBC. Then, I led a team of engineers in the design and development of new ODBC drivers using Visual C++ and Assembler.

    I performed software maintenance on relational database products (ALLBASE/SQL) and ODBC.
