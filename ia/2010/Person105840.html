Josh Lockhart’s Summary

I build simple, usable, and well-designed websites using web standards. Get in touch to learn how I can help your organization create a website that is simple, fast, and optimized with modern web technologies.

I am a front-end web developer with over 7 years of industry experience, proficient with client-side and server-side technologies including HTML, XHTML (Transitional or Strict), CSS 2.1 + 3, Javascript (jQuery), PHP 5, Ruby, and SQL.

While I hand-code all of my work, I also use web frameworks and content management systems to expedite the development process. My favorite tools are ExpressionEngine, Ruby on Rails, and the Recess PHP Framework.

My clients hail from the United States, Western Europe, Africa, and Asia. My past projects include content management, e-commerce, social networking, blogging, business, and personal websites.

I also develop professional Rapidweaver themes and plugins at http://www.joshlockhart.com/. Rapidweaver is an easy-to-use website creation tool for Mac OS X, created by Realmac Software.
