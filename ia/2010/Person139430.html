  • CNC Operator

    Usher Tool and Die

    (Machinery industry)

    August 2005 — August 2008 (3 years 1 month)

    I started off working here as a general machinest, preparing blocks of metal to be precision machined in one of our CNC mills. This included me squaring the blocks properly so they would properly rest in the machines used. I would then have to size them very precisely, .01 of an inch, sometimes less.

    I eventually ended up running a CNC mill of my own, which I did take some classes at a local community college in order to do so. This job entailed me reading a blueprint for a block and deciphering what needed to be done to it in my mill. I had to then make sure I had all the correct tools in my mill in order to complete the blocks requirements.

    I would then have to program every aspect of what needed to be done to that said block IE: starting points and end points for tools, speed of the tools, paths of certain tools, ect... One slight mistake ended up in a machine crash, so this was very important to make sure your program was set accordingly.

    I learned a lot from this job and it was a fantastic work place experience where I learned a great trade.
