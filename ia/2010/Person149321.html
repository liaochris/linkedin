Anthony McClay’s Summary

Tony McClay

Solutions / Application Architect
Java Enterprise Developer / Team Lead
Tibco Architect / Administrator / Developer

The solutions I have designed and developed include solutions based on Service Oriented Architecture Techniques (SOA) such as Orchestration, Business Process Management, Complex Event Processing (CEP) and the techniques that Design Patterns from the Gang of Four (GoF), J2EE, and Integration Patterns has to offer to solve complex problems.

As a Java Developer/Team Lead I have architected and built solutions using a number of Java Vendors which include BEA/Oralce Weblogic, Redhat’s JBoss Application Server, IBM’s Webshere, Sun’s Java Application Server (Glassfish). These applications include the ability to scale, secure, manage, and monitor these solutions. Using Traditional and Agile Project Management Techniques, I have worked with onshore and offshore resources to produce these solutions.

I have been called on specifically for performance tuning, optimization and profiling of Java Core, Java Enterprise Applications, and the entire application infrastructure.

As a Tibco Architect, Administrator, Developer I have been a productive as a Tibco Business Works Developer (Orchestration). I am also proficient in Tibco Business Events (CEP). I am also a Tibco Administrator (Admin), Tibco Hawk (Management) and Tibco EMS Administrator (JMS Admin) of the Production, QA, and Development Tibco Domains. This includes over 50 servers and 13 Tibco Based Applications. This includes planning for security, scalability, manageability, training, and Project Schedule. I am the current leader of Tibco Center of Excellence (COE).

As an Architect, it has been my responsibility to teach and train developers how to use technology that is in sync with the company’s technology goals. I have taught classes for Mainframe developer to Java, Java Certification Courses and Tibco Business Works.
