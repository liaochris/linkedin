Experience includes:

development of SalesForce / Force.com solutions with VisualForce, Apex, Portal and Sites.

mobile applications for iPhone and Symbian OS.

server-side development using Python, Perl, Erlang, and/or Java

advanced VoIP solutions using Asterisk, FreeSWITCH including bespoke module development

In-depth security experience at both the application and network level.
