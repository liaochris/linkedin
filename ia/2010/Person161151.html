    Since 2004 I've been working with MapWindow GIS.
    First as a user and later as a moderator at the forums, helping members with their questions, bug finding and developing plug-ins.
    Since 2009 I'm leading an international team of developers working on MapWindow and try to manage the releases and act as a configuration manager.

    MapWindow GIS desktop application is a free, open source, standards-based standalone software package that you can use to view and edit GIS data in many file formats.

    The software includes plug-ins for various geoprocessing tasks (e.g. buffer, merge, etc.), watershed delineation, accessing on-line data sources, and an experimental geodatabase plug-in. The attribute table editor can be used to write simple queries, and there is a complete scripting editor that allows you to write and run scripts in VB.NET and C# directly in the MapWindow program.

    What makes MapWindow GIS special is that it is the ONLY mature .NET based open source GIS. We've had over 253,835 downloads (December 2009) so far! Averaging about 200 per day!
