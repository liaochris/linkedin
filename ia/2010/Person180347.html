Sarah Dutkiewicz’s Summary

As a professional developer, my main focus has always been on database-driven applications. Whether it's migrating old database-driven systems to newer technology or creating new database-driven systems to automate processes, I enjoy working with the automation of tasks involving a lot of data through programming and databases. Having had database and server administration roles, I have a lot of options in my toolbox for getting problems solved.

I am active in the developers community in the Cleveland-area, attending various .NET and SQL user groups. When I'm not attending a meeting or a mini-conference, I can be found on the committees that plan these events or networking with a lot of the influential people in our event community. I also maintain a blog where I talk about various technologies that I'm working with and area events that other developers might be interested in.

In what free time is left, I've been growing a website to show how expansive Cleveland's tech community is - launched with 5 links in October 2008, Cleveland Tech Events is at 30+ user groups strong. I've also recently launched a sister site called Columbus Tech Events, that launched in June 2009 with 28 user groups and 6 events.
