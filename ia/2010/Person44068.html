Craig Dedo’s Summary

Application developer and software development project manager in a wide variety of industries and application areas. Application areas include: Engineering, financial analysis, manufacturing, telecommunications, shop floor control, product tracking and scheduling. I have a strong preference for robust, disciplined software development methods using classic SDLC and CMM methods.

Languages & Tools: Fortran, C, SQL, SQR, DCL, C++, Ingres 4GL, BASIC, 80x86 Assembler, CMS, FDL, Install, Message, MMS, System Services Library
Databases: Oracle, Rdb, MS Access, Q&A
Business System Analysis: SDLC, project management, requirements definition, systems analysis, functional design, modular decomposition, data flow diagrams, entity relationship diagrams, construction, testing, configuration, documentation, software size and cost estimation
