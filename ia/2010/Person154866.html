Nic Wise’s Summary

I am highly skilled in software development, design and architecture, with a focus on the Microsoft .NET platform (C#), and most of the sub-components (ASP.NET, MVC, Compact Framework, BizTalk (2004), SQL Server).

Whats a GHD? Try here:

http://www.fastchicken.co.nz/index.php/2009/08/17/architects-or-why-business-cards-need-to-be-bigger-or-have-smaller-text/

and here:

http://www.fastchicken.co.nz/index.php/2009/08/19/more-on-architects-and-titles/

My preferred method of working is agile, but I'm not a zelot about it - I prefer to be highly pragmatic, using any and all tools and techniques to get the job done.

I also have development and operational experience with Windows Server systems (2000-2008), Active Directory and Exchange Server (2000-2010) as well as Linux and MacOS.

I dabble "on the side" with Ruby on Rails, Python (Google App Engine), Java and Mac OS X development. I'm no designer, but I can find my way around HTML, CSS and Javascript. I've done a major project in PHP (Zend), using a host of internet-scale sub-systems - CouchDB, Memcached and other tools.

In general, I am a jack of most trades, and the master of a few.

I have given a number of presentations, usually at Microsoft events or User Group meetings, including TechEd New Zealand, which I presented at three years running (2004-2006).

I have a MCP in SQL Server management and ASP.NET development.

Nic Wise’s Specialties:

Software development, design and architecture, specializing in backend systems - services, integration, biztalk, and some infrastructure. Jack of most trades would be a good description.
