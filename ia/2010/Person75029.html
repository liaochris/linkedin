Glenn Bernard’s Summary

Working as a DSP/SW engineer since 1988. Extensive experience in algorithm development, signal processing, ELINT, SIGINT, acoustics, RF, and communications and RADAR.

Founded DSP Consulting, LLC in 1999, a consultancy specializing in algorithm development, implementation, and signal processing, signal analysis systems.

see: http://www.dspconsulting.com

DSP Consulting has developed a niche line of signal processing algorithms tailored to the SIGINT community available for OEM integration to select clients. For Signal Recognizer, Automatic RF Communications Network Analysis, and RF Communications Network Simulator product inquiries, contact bernard@dspconsulting.com

http://www.signalanalysistech.com

Glenn Bernard’s Specialties:

Software design, debug, maintenance, including porting, digital signal processing (DSP), system engineering, DSP HW/SW/FW design/development. RTOS and Embedded Linux as well as Windows supported.
