Vijay Dewangan’s Summary

I have worked as a Project Assistant, at Indian Institute of Science (IISc), Bangalore, India. At IISc, I have worked in CommNets Algorithm group, where I was responsible for designing, developing and simulating the prototype algorithms specific to Wireless Sensor Networks (WSNs) Applications. I have worked on developing prototype application for Intrusion Detection and Localization. I got an opportunity to develop a dynamic routing protocol Ex-GRAB, which was robust to jamming, For WSNs, we developed and simulated self-organization algorithm which was an addition of a feature specific to WSNs in MAC 802.11 (b). All the above developments were extensively simulated using a commercial network simulator QUALNET and also implemented on resource constrained (memory and processing power) Wireless Sensor Boards (Atmel processor Atmega128L), running on TinyOS, using the light weight programming language NesC.

At OSU, as my Master's graduation project I have worked on "Resource Allocation in Cognitive Radio Networks for delay constrainted applications". We formulated the above problem as utility maximization problem and proposed resource allocation and congestion control policies for different applications. We proved our policies to be optimal and convergent in polynomial time. This work has given me a good understanding about the current state of art in wireless network protocols, and also gave me good insight about the future developments for the Next Generation Data Networks.

Goal: To work in challenging environment of designing, developing and testing Advanced Data Networks Protocols and next generation Wireless Communications Systems.
