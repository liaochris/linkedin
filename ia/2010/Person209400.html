Umut Gökbayrak’s Summary

Software engineer with expertise on a variety of systems but especially web and mobile platforms. His experience is mainly on J2SE, J2EE, J2ME, C, Objective-C, iPhone SDK, Cocoa and also all sorts of web based technologies.

Worked at internet service providers, independent software vendors, as well as financial institutions, jewelry industry and currently working for the telecommunications sector.

He has been an open source evangelist and an active open source developer for more than 10 years. Some open source projects involved:

Postaci Webmail: One of the world's most popular webmail clients during the years 1999-2001.

Claros inTouch: Full featured Java communication suite with IM, Mail, Calendar, Contacts, Notes and much more. One of the most popular apps in the same category.

Claros Chat: A web based Ajax XMPP client.

Claros Mini: A mobile webmail application for small screen devices such as mobile phones and PDAs.

Gelecek Linux: First commercial Turkish Linux distribution.

Isola Framework: A Javascript abstraction layer built upon (but not limited to) legendary Extjs framework.

Please check http://www.claros.org and http://labs.turkcellteknoloji.com.tr for more details.

Besides beeing a developer, he has experience on Unix/Linux and Windows systems administration. He is a Microsoft Certified Systems Engineer and also owns a Linux Certified Professional certificate.
