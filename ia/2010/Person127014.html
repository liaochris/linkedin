Ian Lloyd’s Summary

Long-standing evangelist in the world of web standards and accessibility. Authored, co-authored and been technical author on a number of books on the topic of building and designing web sites.

Owner of respected and long-running accessibility resource Accessify.com. Previously member of the Web Standards Project (WaSP).

Ian Lloyd’s Specialties:

accessibility, web accessibility, standards, web standards, technical authoring, technical editing, search engine optimization (SEO), search engine optimisation, usability, HTML, CSS, javascript, unobtrusive javascript, jQuery, JSON, PHP, MySQL, evangelism, public speaking, markup
