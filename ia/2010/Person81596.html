Henrique Araujo’s Summary

Formado em Sistemas de Informação pela Faculdade Natalense Para o Desenvolvimento do RN em 2007. Atualmente Analista de Sistemas - Webdesigner da IVIA alem de freelancear com projetos de web sites comerciais há mais de 7 anos. Foi Web designer na MAXMEIO Comunicação (Natal ,2007) e na Agência Sales RT (Natal,2006) como programador no departamento de Web. Além de projetista e desenhista de interfaces para Web é desenvolvedor front-end com alguma experiência back-end e teve breve atuação como programador Java em 2005.

Possui um blog onde compartilha conhecimentos provenientes da constante pesquisa por novidades tecnológicas na área, bem como insights, tutoriais, dicas e pequenas soluções para websites tais como Simple Captcha (CC) e Easy Calendar (CC): www.henriquearaujo.com.br. Também ja foi colunista do antigo site Help Masters (www.helpmasters.com.br).

Henrique Araujo’s Specialties:

Projetista de interfaces para a Web. Especialista em concepção, criação e desenvolvimento de front-ends usando as tecnologias XHTML, CSS e Javascript estritamente dentro dos padrões. Também trabalha com programação PHP e banco de dados MySQL.
