Septimiu Fabian Mare’s Summary

I'm mostly interested in software development, especially security related software. Since 2000 I have programmed a wide range of software applications, from the simplest end-user tools to sophisticated industry infrastructure-based programs (distributed server management, voip, criptography, web based services, anti-viruses, firewalls).

I'm mostly interested in researching new security techniques, inventing or improving existing ones. With several patents pending, I am currently working in developing a new security scheme made to secure internet connections, while using some unconventional, never before used techniques.

With a programming experience of 10 years, while working for several international software companies I have developed very good skills in software management and planning which makes me good project manager. Having an overview and anticipation of how the final product should look like, with my programming skills, I'm able to distribute, manage, divide and coordinate the work of multiple programmers (and mine of course) to obtain a well-functional final product that would satisfy the customer. I'm a very severe software tester in both user interface and functionality, but also in coding issues (source code).

I like the code clean, simple and optimised. I always try to find the most effective, less time/resource/hardware consumming method to obtain maximum efficiency.

My motto is: "Quality is just a word, perfection is the right direction"
