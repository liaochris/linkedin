  • Senior Software Engineer

    Circuit City Stores, Corporate Headquarters

    (Public Company; CC; Retail industry)

    September 1995 — May 1998 (2 years 9 months)

    Conceived and led the effort to convert the custom Circuit City Point-of-Sale system, DPS, from Pascal software running under the Motorola VersaDos OS to C software running under UNIX as the first step toward a GUI system. The ultimate goal of the project was replacement of the custom POS hardware with a UNIX workstation, replacing the terminal-based interface with a graphical interface, and redesigning the software in C++ or JAVA. Performed benchmark tests using LINUX, Lynx, and RTMX (a port of OpenBSD 4.4) resulting in the selection of RTMX. A vendor specializing in code translation performed the task of translating 1,000,000 lines of Pascal code into C language. While interfacing daily with the translation vendor, led an in-house team of 8 responsible for porting the low-level VersaDos software to UNIX.

    Designed, coded, tested, and integrated a Silicon Graphics UNIX process in C to allow communication between the Circuit City stores and Corporate Headquarters for the processing of credit applications and payments. The process receives X.25 communications from the stores, determines the type of communication, reformats the information, and uses a UNIX socket to communicate to the Credit Process running on a DEC Alpha. Because it is imperative that communications be handled in order and not lost, the process maintains an internal timestamped Circuit Table and an internal FIFO queue to allow resend attempts should the socket become full or the Credit Process terminate.

    Chairperson of C Coding Standards Group to establish coding standards for the 600 person MIS group.
