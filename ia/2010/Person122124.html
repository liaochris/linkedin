Laura Galindo’s Summary

Originally from Bogota, Laura earned a BFA in Graphic Design from Columbia College. She's worked as an Interactive Art Director and Flash Developer for some of the best-known brands in the US including Sears, Craftsman, Kenmore and Eureka. Laura’s work ranges across several disciplines from identity and branding, to print layouts to web design, programming and architecture. She approaches each project with passion, enthusiasm and the belief that strong conceptual thinking is key to developing effective design.

A strategic thinker, she is passionate about creating and executing effective interactive media strategies. Handles all aspects of web and interactive project management from concept to production. An expert Flash developer with a keen eye for integrating interface and user experience.

Laura Galindo’s Specialties:

_Expert in designing within the latest user interface programming languages, class libraries and tweening engines.

_Experience in designing, developing and executing site wire frames, Flash experiences, microsites, viral campaigns and interactive applications.

_Complete mastery of Adobe Creative Suite products, Flash C33 and and OOP using Actionscript. Additional programming experience with HTML, Javascript and CSS.
