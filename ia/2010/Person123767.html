Michael LeMay’s Summary

I am a computer security and reliability researcher with experience in embedded system security and reliability, particularly in the field of embedded systems with high assurance requirements, such as advanced electric meters.

  • Graduate Student

    Southwest Research Institute

    (Non-Profit; Research industry)

    May 2009 — August 2009 (4 months)

    Hardware Hypervisor Research: Developed processor extensions for SPARC soft core on Xilinx FPGA to permit fine-grained monitoring and control of system operation. Developed software for ARM coprocessor to achieve configurable security objectives by coordinating processor extensions. Constructed several demonstration scenarios: 1) Blocked access to critical system state through test ports such as JTAG, 2) Monitored host process CPU usage without host cooperation, 3) Detected and optionally prevented kernel rootkit infection. Developed methodologies for using Maude model checker and other tools to formally verify that system provides isolation, avoids harmful interference with host operation, and correctly defends against attacks.
    Advanced Metering Infrastructure Vulnerability Assessment: Analyzed security of optical port interface on electric meter using custom software tools, software reverse engineering, and Sulley fuzzing toolkit. Investigated processor buses internal to meter using logic analyzer, UART signal injection circuitry, and custom software tools. Performed attacks on ARM processor debug interface pins. Reverse engineered ARM processor firmware.
