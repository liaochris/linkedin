Tim Berglund’s Summary

I run a software consulting firm called the August Technology Group, which provides training and development services to customers building web applications with open-source tools on the Java Virtual Machine. My technology interests span web applications, business integration, data architecture, and software architecture, but my greatest passion is to help developers improve in their craft. I'm also a conference speaker internationally at various conferences and domestically with the No Fluff Just Stuff tour. I work locally with IASA Denver (www.iasadenver.org) and the Denver Open Source User Group (www.denveropensource.org)

Tim Berglund’s Specialties:

JVM, Groovy, Grails, open source, technology change, training, mentoring, agile, Internet-based applications
