Boye Oomens’s Summary

Seasoned front-end developer with over eight years of in-depth web development experience and passionate about web standards. Always aiming for semantic, cross-browser & cross-platform consistent code with a high maintainability level.

Comfortable and proficient with writing accessible semantic HTML, efficient CSS and strong understanding of how to enhance progressively by adding elegant interaction via unobtrusive JavaScript (both traditional as well as libraries like jQuery).

Demonstrated aptitude with web fundamentals (such as HTTP), version control (SVN, Git), the LAMP environment, Object Oriented Programming and third-party API's.

Committed to the professionalization of the front-end developer profession by volunteering for Fronteers (the Dutch Front-end Association).

Eye for details, well organized, and the ability to take direction as well as provide guidance to production and peers.

Boye Oomens’s Specialties:

(X)HTML, CSS, JavaScript/ECMAScript (Core, DOM API, JSON, jQuery), XML, SEO, semantics and other (W3C, WHATWG) web standards, combined with usability and accessiblity aspects. (e.g. WCAG, WAI-ARIA)
