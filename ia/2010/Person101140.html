Joeri Pansaerts’s Summary

I build web applications from scratch in .NET, using ASP.NET and C#. Furthermore, I'm always looking to learn about new developments and technologies.

My latest project is an Audiovisual recording application built in WPF. Additionally I have developed a web application to stream the videos of the presentor and presentations on conferences and seminars built in asp.net.
A few of the features :
- Encoding to smooth streaming format (.ism) via Expression Encoder SDK.
- Webvideo player built in Microsoft Silverlight
- Recording the video devices via an encoder h.264 to .avi files
- Capturing audio devices
- Capturing video devices
- Live Streaming application
Now I am creating a commercial website to promote these products and I am finetuning the design of the videoplatform itself.

I am also working on a multimedia and interactive platform for Vlerick Leuven Gent Management School:
- Trendy design
- built in ASP.NET MVC & C#
- Video Platform with Silverlight Player and videos in Smooth Streaming format
- Podcast Platform with MP3 and AAC-format
- Mobile Application for Iphone & Mobile version for mobile browsers
- Integration with Social Media
- ...
