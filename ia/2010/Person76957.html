Giampiero Bonifazi’s Summary

I'm a senior Web Developer involved in developing a wide variety of web applications. I’m also an Adobe Certified Instructor (ACI) and Adobe Certified Expert (ACE). As Adobe Specialist I've been selected as Master Instructor for Italy and involved as speaker in several roadshow and public meetings.

As ColdFusion and PHP Developer I can develop all aspects of web development: business logic, back-end architecture, unit testing, monitoring, optimization, implementation with RDBMS and others application server and/or gateway (i.e. AJAX, REST WebService, Flash Remoting, Flash Media Server etc.).

As Web Designer I can manage all aspects of web design: layout and interface design, CSS implementation, front-end features and JS client-side development (Prototype, Scriptaculous, JQuery, JQuery UI library).

As Web Project Manager I manage the roadmap of development process from the start-up to the launch, from the layout to the architectural design, from the database design to the system integration.

When I do not work I like to spend my time with my wife and my little girl or do sports (biking, spinning...), take photos, videos, watch movies and read books
