Richard Gerber’s Summary

More than 10 years helping scientists build, run, debug, and optimize codes on supercomputers. Experience evaluating, characterizing, configuring, and monitoring large high performance computing systems from an application user's perspective. Designed and implemented web technologies as an interface for monitoring High Performance Computing (HPC) system performance and giving users access to run information and statistics. Leads training efforts and current President of ScicomP, the IBM HPC Systems Users Group. Developed online user surveys and trouble ticket systems using LAMPs technologies.

Richard Gerber’s Specialties:

Consulting, running jobs on supsercomputers, debugging, optimization, workflows, building code, training, outreach, data analysis, system characterization and monitoring.
