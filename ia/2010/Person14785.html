Anthony DeBonis’s Summary

Anthony DeBonis is a Senior Technologist and Partner with Troy Web Consulting in Troy, New York. Anthony is responsible for designing, developing and deploying RIA/web applications that are required to be secure and highly available in Flex, ColdFusion and Java Platforms for both government and corporate entities. Anthony is currently working on Flex / ColdFusion Applications for several New York State Agencies including highly secure, mission critical dashboard system. An avid developer community member, Anthony manages the Java Developer's Network in Albany NY and presents at many user groups and Nation Conference.
