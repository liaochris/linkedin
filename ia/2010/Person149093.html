Moutaz Shams’s Summary

Moutaz Shams is an experienced software developer specializing in planning, analysing, architecting, designing and developing enterprise-class rich web applications using Microsoft .NET related technologies.

For several years (since 2003) he has been focusing on C, C#, OOP, WindowsForms, GDI+, ASP .NET, Service Oriented Architecture, Microsoft SQL Server and recently Microsoft Silverlight. Before that.. he was using Microsoft Visual Basic 3 through 6 in developing windows-based applications.

Moutaz is currently the Head of Development at DigitalLane Egypt leading the IT team developing dozens of sophisticated web applications for clients in the UK and Egypt using Microsoft Silverlight, ASP .NET, Microsoft SQL Server and SharePoint.

Other than code, project management and system administration, he got the design talent and experience (HTML, CSS, Flash, Expression Blend and Silverlight) that makes him produce attractive usable stuff.

Moutaz is an MCP 2.0 and a Microsoft Certified Professional Developer.
