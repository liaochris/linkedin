Chris Yanc’s Summary

I've become deeply rooted in Interactive Design for 5+ years of as an Interactive Designer at Optiem. My included projects cover Web and Email Design, User Interface Design and Data Visualization. Along with designing, I'm always discovering how to create the code behind developing a usable front end through Actionscript 2 and 3, ASP, DotNET, PHP, Javascript, XML and XSLT, MySQL, CSS and XHTML.

A new technology I've been working on is multi-touch computer use interaction via Flash. I have built my own touchscreen with help from NUIGroup users across the world and have been posting tutorials on my progress learning how to use Actionscript 3 to create multi-touch applications on my personal website www.cyancdesign.com/blog. My new goal is to bring the ideas of this new style of application to clients willing to break the mold of user interaction.
