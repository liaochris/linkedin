Dan Orlando’s Summary

Dan Orlando is currently a Sr. Flash Platform Developer for Universal Mind, specializing in enterprise class Flex and AIR applications. Dan has also written a number of articles and white papers on the Flash Platform for PHP Architect Magazine, IBM developerWorks, Amazon Developer Connection, Flash & Flex Developer Magazine, Adobe Developer Connection, and Zend Technologies. Dan was recently the Technical Editor for Professional Papervision3D (Wiley & Sons) and is currently authoring Flex 4 in Action (Manning Press), along with fellow authors Tariq Ahmed and John C. Bland II.

Dan is also a Principal Founder and Director of Video Technology for View Control Inc, and has played a role in the development of several important innovations for the Adobe Flash Platform, including the AWS S3 integration library, an AMF server that runs on C++, and an open source REST library for Flex.

Dan Orlando’s Specialties:

Enterprise application development, project management, business management, Actionscript 3.0 development, 3D programming, bleeding-edge user interface development, Unix/Linux networking, PHP and Java programming, and Flash, authoring software development and programming literature
