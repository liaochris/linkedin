  • Applications Developer (C#/ASP.NET/SQL/Natural/Adabas)

    Texas A&M University System

    (Educational Institution; Education Management industry)

    April 2008 — April 2010 (2 years 1 month)

    SQL Server, SSIS and C# development. Part of one of the largest projects I have been involved in: integration with a procurement solution from SciQuest. Started on this project back in 2008 and by the beginning of 2009 my main role was developing a solution for ongoing data flow between SciQuest (via SQL Server) and mainframe (FAMIS). I took the initiative since I had background in database development (education and previous experience with other RDBMS) and it has been a great project to work on. Lots of SSIS/SQL/T-SQL development (development of views and stored procedures; development of SSIS packages for one-time data loads during testing; development of SSIS packages for ongoing data transfers), lots of changes and optimization in between. Being a standard solution, I based my initial development around SSIS, but due to maintainability issues I replaced most usage of SSIS with a much more maintainable and more flexible .NET/C# solution that I designed to dynamically map fields from SQL Server tables/views/SPs to corresponding fields in mainframe programs (under the SSIS solution when a developer made changes to mainframe programs lots of other things needed to happen before everything was re-mapped and working again - under the .NET/C# solution only xml files need to be modified to provide the new mappings, and unlike SSIS, changes to the mainframe programs would not break things) My personal priority was making things easy for everyone on the FAMIS team. We went live with SciQuest early 2010.

    C#/ASP.NET Web Development, work on CANOPY, web interface to mainframe based FAMIS (Financial Accounting Management Information System), development and maintenance of web and mainframe applications (including Natural/Adabas Mainframe Programming). I contributed to a few projects and provided help to other developers along the way.

    Quality Assurance. Implemented and/or provided recommendations for improvements, including external applications such as SciQuest.
