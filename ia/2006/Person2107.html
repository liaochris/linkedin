Jonathan Poh’s Summary

Web Designer who is able to bridge the creative and technical sides, producing functional and usable sites with beautiful minimalist aesthetics. All code is w3c-validated, XHTML 1.0 Strict, and CSS 2.0 compliant. Most of them are even constructed using table-less CSS layouts and are semantically coded. What this means is that future updates and changes by the client are easily done due to clear and concise code, and also the web sites are scalable to different web browsers, versions and platforms.
