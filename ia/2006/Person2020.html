Justin Mason’s Summary:

Software developer with a strong software development and systems admin skill-set, especially in the internet and security fields on UNIX platforms.

I released the first version of SpamAssassin as open source software under the Perl Artistic license, in 2001. Since then, the project has gone from strength to strength; it has joined the Apache Software Foundation, and is now the market-leading open-source spam filter product, in use at thousands of sites, filtering email for somewhere between 100 and 200 million users worldwide.

See http://spamassassin.apache.org/ for more details.
