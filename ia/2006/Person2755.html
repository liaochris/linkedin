A resourceful, imaginative IT Consultant with a broad ranging experience of programming and design using Oracle tools. Flexible, committed individual with excellent communication skills, capable of rapidly adapting to change. A provider of innovative solutions to complex problems, aiming for a greater challenge based around Oracle RDBMS technology.

Key Skills
- Well versed in Oracle technologies including Forms, Reports, Oracle 8i, Oracle 9i, Oracle 10g, Oracle 9iAS Interconnect, Oracle Application Server and Advanced Queuing.
- Extensive experience of developing workflow solutions using Oracle Workflow including Business Event System.
