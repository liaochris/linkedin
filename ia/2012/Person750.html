Jason Perlow's Summary

Systems Documentation and Authoring / Published Expertise: Jason is a frequent contributor to various computer industry magazines on various technology subjects. His most recent articles can be found on ZDNet.com and Linux-Mag.com

Mission Critical Linux Configuration: Clustering (Penguin Scyld, Beowulf Clusters), High Availability web servers (Apache, MySQL, Postgres), LAMP/PHP applications on Red Hat and SUSE servers. Linux SAN configuration and SAMBA servers. Kernel tuning/recompiling, RAID and journaling file systems. 10+ years with UNIX/Linux on the x86 platform. 5+ years with Linux on Itanium and IBM POWER architecture.

Open Source Linux Infrastructure Strategy: Microsoft Windows integration and Linux Migration, High Availability and Thin Client server computing, Disaster Recovery, Enterprise Systems Management, Internet/Intranet infrastructure implementation, Wide Area Networking and Cyber Security.

Specialties

VMWare and virtualization technologies. Comprehensive Microsoft and Active Directory integration experience with Linux. Mobile operating systems (Android, iOS, QNX) and Wireless, Datacenter consolidation and server optimization
