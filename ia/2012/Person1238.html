Dean Michael Berris' Summary

Dean is a Software Architect by trade and Software Developer by heart with significant Systems Development experience using the C++ programming language. He has been developing solutions for the mobile content provider industry in the Philippines since 2004. He is also Interested in advocating the use of the C++ programming language, developing high performance computing systems, and solving challenging problems through software.

Dean has done consulting work to build highly scalable architectures for web applications. He is also currently actively working with other developers on the Boost C++ Library as well as the C++ Network Library as one of the maintainers.

Currently, Dean is a Technical Solutions Engineer at Google Australia part of a global engineering team.

Specialties

high performance computing, parallel computing, network programming, agile software development, system design and architecture, systems programming, c++
