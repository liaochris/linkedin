  • Senior Software Developer

    ProCredit Bank

    (Privately Held; Banking industry)

    July 2005 — August 2008 (3 years 2 months)

    The “Pro-Credit Bank” had an old loan application system based on client-server architecture which was developed during the past years.
    There was a wish and ambitions to develop new app server based on lightweight technology which would develop in two years. Part of this app system would be loan application logic on server side and wizard like client.
    Methodology:
    We used agile-scrum development methodology.
    Short description:
    The system is based on 3-tier architecture where client is written (C#) programming language.
    Server is written in pure Java technology.
    Server is based on Spring like container and Hibernate like ORM mapper.
    On client side I developed parts as follows:
    1) Channel system – serves for communication with app server
    2) Data binding system (involved into all frames)
    3) Dictionary API on client side (App for dictionaries and complete logic for storing refreshing and restoring dictionaries from disc local. Dictionaries are stored in XML format)
    4) Unified Localized message subsystem
    5) Meta-data setup environment
    6) Lot of wizards with implemented logic.
    7) Nant-build script.
    8) Lot of customizable components (for example MoneyTextBox, NumberTextBox etc.)
    All my development was covered by tests.
    On server side I developed parts as follows:
    1) Channel system – serves for communication with client and rest of environment.
    2) Domain driven design based on loan logic .
    3) Object to document fragment processor which serves for transformation object to fragment.
    4) Ant build script – (build script was rewritten by Ruby-Rake script)
    5) Hibernate mapping for lot of businesses.
    All developed classes were covered by tests.
    After developing the system the first performance test was in August 2007. I adopted Grinder performance tool to accomplish tests. Scripts were written in java programming language.
    Complete performance test was monitored by OptimiseIT profiler.
