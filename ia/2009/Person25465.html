A statistical data mart was designed and deployed as well as a Juvenile data mart. The juvenile data mart allowed for specific analysis of juvenile dockets to better serve these cases.

    (Public Company; 1001-5000 employees; BAT; Tobacco industry)

Several data marts were also built once the warehouse redesign was completed. These marts used BusinessObjects as a front-end reporting tool. The mart structure supported query analysis across several distinct subject areas and was presented transparently to the user through advanced BusinessObjects universe design. The backend DB was Oracle.
