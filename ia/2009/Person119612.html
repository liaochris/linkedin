Jon Rose(jonathon-rose@comcast.net)’s Summary

Over 11 years of experience in digital ASIC/FPGA design, including all front-end design tasks(specification development, detailed module spec development, Verilog/VHDL synchronous/asynchronous design coding, verification matrix development, testbench development, synthesis, static timing analysis, RTL level simulation, DFT) and back-end design tasks(DRC/LVS netlist for layout, clock-tree, clock-grid, ECO netlist, static timing analysis, vector generation, gate level simulation with back-annotation from SDF, SPEF to SDF). Specialized experience in RHBD techniques, ASIC library design, simulation models for memory including Verilog and VITAL. Skilled at using Synchronous design practices, and in debugging gate level designs using Static and Dynamic analysis methods.
