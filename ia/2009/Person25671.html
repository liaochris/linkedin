Brian Eldridge’s Summary

I've been a consultant since 1999 when I began work for KPMG Consulting. Since then I've been largely independent helping larger companies build and deploy Blue Martini commerce and content management sites. I've also been helping smaller companies get a web presence in my spare time.

I am well versed in JAVA through versions 1.4 and have had some experience with 1.5. I am a capable programmer using C++, PHP, and PERL. I've recently started working with OpenLaszlo and Ruby on Rails.

I have recently started developing useful tools to help with deploying and maintaining Blue Martini websites, including, but not limited to, custom Blue Martini tags, a PERL interface to read DNA files, and an Eclipse plug-in to interface with the Enterprise Desktop content management section.
