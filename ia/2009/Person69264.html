Brendan McCarthy’s Experience

  • Sr. Architect/Tech Lead

    Apple, Inc.

    (Public Company; 10,001 or more employees; AAPL; Computer Hardware industry)

    November 2006 — Present (2 years 11 months)

    Led a tiny senior Agile team which implemented fundamental technologies for Apple's MobileMe service.

    Designed, developed and delivered MobileMe's carrier-grade "cloud computing" data platform. This Java-based platform provides the a sophisticated and highly scalable datastore for MobileMe's wep application front-ends (Contacts, Calendars and Bookmarks.)

    The platform also transparently synchronizes MobileMe data with iPhone, iPod Touch, Mac OS X and Windows systems using SyncDav and OTA protocols.

    Designed & developed MobileMe's OTA (over-the-air) wireless synchronization protocol, client, and server software which powers MobileMe's Push feature to iPhone and iPod Touch.

  • Senior Software Engineer

    Apple, Inc.

    (Public Company; 10,001 or more employees; AAPL; Computer Software industry)

    March 2002 — November 2006 (4 years 9 months)

    Designed and implemented client and server technologies in Mac OS X (Tiger/Leopard) for distributing lightweight notifications between computers on the Internet.

    Designed and implemented backend indexing and search to integrate .Mac homepages and iWeb.

    Designed and implemented SyncDAV protocol and client: structured data synchronization semantics layered over WebDAV. Enables Mac-to-Mac sync in Mac OS X (Tiger/Leopard).

    Lead the small team which implemented the high performance Java-based synchronization management server which powered Apple's Mac-to-Mac-to-Web iSync.

    Represented Apple within the SyncML standards committee.

  • Principal Engineer/Architect

    fusionOne, Inc.

    (Privately Held; 51-200 employees; Internet industry)

    March 2000 — March 2002 (2 years 1 month)

    Delegate to the SyncML Core Expert Group. Participated in the development of the SyncML protocol. Significant contributions to the protocol include: filtering, soft delete, secure authentication, message chunking.
    Other enhancements intended to improve usability of solutions built around the open standard.

    Developer of fusionOne's [then] cutting-edge and first-generation SyncML Client and Server solutions. Developed highly scalable Java and C++ based server solution. Developed embedded C client software stack.

  • Senior Software Engineer

    Filemaker, Inc.

    (Privately Held; 51-200 employees; Computer Software industry)

    October 1996 — March 2000 (3 years 6 months)

    Completed design and initial implementation of FileMaker Mobile Companion for Palm.

    Sole MacOS engineer for FileMaker Server 5 with many performance, threading, networking, and remote management UI improvements.

    Early adopter of MacOS X technologies.

  • Lead Software Engineer

    Claris Corporation

    (Privately Held; 201-500 employees; Computer Software industry)

    July 1988 — October 1996 (8 years 4 months)

    Was the principal designer and engineering lead for FileMaker Companion for Newton (the most flexible and best unreleased relational database for Newton.)
    Handled application UE, architecture, database implementation, data synchronization and networking implementations.

    Developed the MVC portions of a proprietary OO Macintosh/Windows application framework.

    Contributed significant features to MacProject II 2.0 and FileMaker Pro 1.0 - 3.0.
