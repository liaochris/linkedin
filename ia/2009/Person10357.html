Altug Gurkaynak’s Summary

1980 yılında İstanbul’da doğdu. İlkokuldan itibaren öğrenimini Ankara’da tamamladı. Başkent Üniversitesi – Bilgisayar Donanımı Bölümünü tamamladı ve bu bölümdeki öğrenimi süresince internet ve internet teknolojileri, web tasarımı üzerine çalışmalar yaptı.

Mezun olduğu yıl okul içerisinde Web Sitesi Tasarımı üzerine verdiği seminerle ertesi yıl müfredata bu konunun eklenmesini sağladı. Ardından girdiği Anadolu Üniversitesi Açık Öğretim Fakültesi, Turizm Otelcilik bölümüyle beraber mesleki çalışmalar, kurs ve seminerlere devam etti. Microsoft Sistem Mühendisliği (MCSE) Eğitimini bu dönemde tamamladı. Bunun dışında International Brainbench Windows 2000 ve Brainbench HTML Web Tasarım + Web Development sertifikalarını aldı.

Birçok işlere girdi, çıktı. Askerliğini tamamladı, emlakçılık bile yaptı ama yine doyamadı o işe de web bulaştırmayı başardı.

Nisan 2007′de çalışmaya başladığı GittiGidiyor Bilgi Teknolojileri A.Ş.’de birçok web sitesinin yapım ve geliştirilme safhalarında web tasarımcı olarak görev aldı. Firmada çalıştığı süre boyunca, sırasıyla; web tasarım elemanı, web tasarım uzmanı ve web tasarım yöneticisi konumlarına yükseldi. Bu süre içerisinde xhtml ve css teknolojileri alanlarında uzmanlaştı. Ek olarak Photoshop ve kodlama bazlı metin editörlerinde (Editplus – Dreamweaver v.b.) pratik ve uygulamalarda bulundu.

Hâlihazırda yine hobi olarak Japonca öğrenmekte ve çizgi roman senaryoları yazmaktadır. Askerlik hizmeti sırasında da devam etmiş olduğu Anadolu Üniversitesi Turizm – Otelcilik Bölümü’nü bitirmesinin ardından dikey geçişle İşletme Bölümü’ne geçiş yaptı. Diğer yandan da web tasarımı çalışmalarına bağımsız olarak devam etmekte olup, evli ve Çanakkale’de ikamet etmektedir.

Altug Gurkaynak’s Specialties:

Web Design, CSS & XHTML applicating, PSD to HTML, Photoshop, SEO Search Engine Optimization, e-publishing, Editor
