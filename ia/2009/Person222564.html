Stella Hwang’s Summary

Customer-focused senior UI designer experienced with information architecture, interaction design and usability for consumer websites and software applications. Interested in learning and applying new technologies to simplify and improve user experience.

Solid understanding of user interface design principles, product build cycle and working within a cross-functional team under tight deadlines. Excellent written, verbal and personal communication skills.

Experienced with: Photoshop, Illustrator, Fireworks, Word, Excel, PowerPoint, Visio, Acrobat, Demoshield, Dreamweaver, HTML, DHTML, JavaScript, CSS, XML, Java, Flash, AJAX.
