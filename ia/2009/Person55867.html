Larry Briscoe’s Summary

I began my IT Career in Oklahoma City, Oklahoma in 1988 as a Field Service Technician. That lead to plenty of Basic and C programming with Basic being my preferred programming language.

I have been a SQL Server DBA since 1992. I started with Microsoft SQL Server 4.2 for 16-Bit OS/2 v1.3 (that’s Microsoft OS/2 v1.3) . I have worked in many industries during my very satisfying career. At current the industry I work in doesn’t really matter. Whether I’m working in the Credit Card Industry, Accounting, Banking or Mortgage Industry, “It’s just another database”.

I have been a Senior Level DBA in my current position since 1999. I’ve actually been on the same campus since 1997, with two years as a Consultant. The company has changed names a few times, and I’ve been on many different projects and departments.

My expertise is in Performance Tuning Production SQL Server installations. I have been influential in many high end applications with performance issues. I also have had extensive experience with Database Architecture, Development and Design.

Currently my group supports over 200 SQL Servers from SQL 7.0 (don’t ask), to SQL Server 2008 x64. I have built and support many SQL Server Failover Clusters from SQL Server 2000 to SQL Server 2008. This includes SSAS and SSRS in a Web Farm.
