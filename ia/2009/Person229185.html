Tao Jiang’s Summary

Worked in many different areas for over 10 years, including: embedded software, server software, application framework, web applications, driver software in semiconductor testing. Excellent skills in analysis, design, problem-solving, communication, multitasking. Proven ability to lead design, implementation and delivery.
* Lead the effort of developing new generation of jet engine test system in Cenco from architecture concept and project goal, prototyping, sales demo, development and testing all the way to the first on site system installation and grand open ceremony for the 30+ millions USD branded new test cell in Germany. The major technical highlights include:
* A SOA solution to turn traditional data acquisition and control system into an open platform to embrace large variety of client side technologies, including: Web, Java and .NET as well as any future technologies.
* Declarative programming paradigm by creating an application framework based on SVG to allow flexible design and easy configuration on client side display for quick project turn around. Further, all user interfaces can be proportionally resized.

* Lead the effort of developing PLC and Linux PC based low cost data acquisition solution for testing small engine, which became big sales growth area for Cenco.
