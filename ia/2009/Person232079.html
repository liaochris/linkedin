Thiago Almeida’s Experience

  • Senior Integrations Consultant

    Datacom

    (Privately Held; 1001-5000 employees; Information Technology and Services industry)

    March 2007 — Present (1 year 11 months)

    Involved in several integrations consulting engagements from business requirements to project delivery for some of Datacom’s largest clients and projects.
    Technical lead, advisor, team lead, and solutions architect in a variety of projects involving business-to-business and EAI, e-commerce, legacy data integration and systems troubleshooting and configuration.
    Extensive use of integrations technologies such as Microsoft BizTalk Server 2006 R2 and older, Microsoft SQL Server 2005 and older, WCF, and supporting technologies (XML, XSLT, SQL).


    Started the Connected Thoughts blog (http://connectedthoughts.wordpress.com). Leader of the Auckland Connected Systems User Group (http://www.acsug.co.nz), and active on the Microsoft BizTalk forum (http://forums.microsoft.com/Forums/ShowForum.aspx?ForumID=1470&SiteID=1).

  • Senior Integration Specialist

    Conexa

    (Privately Held; 11-50 employees; Computer Software industry)

    October 2005 — April 2007 (1 year 7 months)

    Technical lead of the integrations team that develops advanced integration solutions for several Conexa® clients.
    Create and control business to business data integrations using BizTalk 2004/2006 and SQL Server. The solutions range in complexity from moving flat files between folders to orchestrating complex business to business interactions using BizTalk, XML files, Web Services, SQL Server databases and custom C# .NET applications/DLLs.
    Introduced BizTalk 2004 to the integrations team. Upgraded BizTalk 2002 and DTS packages solutions to BizTalk 2004. Main integrations analyst/developer in the business-to-business e-commerce integration between one of the world’s largest energy companies operating in Australia and its clients.

  • Developer/Analyst

    Simpl

    (Privately Held; 51-200 employees; Information Technology and Services industry)

    April 2003 — October 2005 (2 years 7 months)

    Part of a team that supports and develops applications for Simpl’s client Auckland District Health Board, New Zealand’s biggest district health board with more than 7,500 staff.
    Development of Patient Management Systems with C#.NET, XML Web Services, Visual Basic 6, COM+, and large SQL Server 2000 databases.
    Implement advanced integration solutions using BizTalk and e*Gate.
    Work with Quality Assurance and Acceptance teams to provide reliable software to the Hospital’s system. Duties range from general support and defect fixes to large projects impacting the entire organization.
    Project requirements and other documents design together with business analysts and clients.
    Manage small projects and perform team leader activities.
    Implemented an interface in BizTalk and e*Gate that translates a web generated XML file containing patient discharge information into HL7 documents and PDF files.

  • Senior Developer/Analyst

    ADP

    (Public Company; 10,001 or more employees; ADP; Information Technology and Services industry)

    October 2000 — February 2003 (2 years 5 months)

    Support and develop all applications for the company’s 300+ Costumer Service Representatives Call Centre running Clarify and SQL Server.
    Design and develop auxiliary applications (CTI, Reports, Systems Integration) using Delphi 7, Visual Basic 6, and SQL Server 2000.
    Recommended and implemented the Call Center import system. ADP Salt Lake works with Payroll and Benefits for client companies, and the Call Center services their employees. The import system was created to automatically update the client companies’ employee data on the Call Center’s database. It was developed using Object Orientation, Delphi COM Objects and ADO for reporting, a control application using Visual Basic to set up the schedules and manage clients, and SQL Server DTS, Jobs, and stored procedures for the updates.
