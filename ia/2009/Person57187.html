David Dang’s Interests:

**What is Cloud Computing?** - When software runs as a service. When it runs, not on PC, not in office network, but some where on the Net. There is no installation, no upgrade, no hardware, no download, no virus, no boxes, no copy. Consumers of Cloud Computing purchase computing capacity on-demand and are not generally concerned with the underlying technologies used to achieve the services capability. All they are concerned is the value of services in the cloud. The winners are Users, Consumers. It gained attention in 2008. - In the next 5 years, new businesses with little infrastructure will create cloud market first before big businesses get there. "- Cloud Computing opens a whole new philosophy in software design is that now we should shift focus more on what Consumer's needs, benefits, and value of services rather than on what technology's requirements, needs that create more technology's complexity, cost." - LIVE Podcast, V.P Gartner, Inc.-Cloud Computing at NetthruOffice.com
