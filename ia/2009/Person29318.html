Bjorn Roche’s Summary

Skills include:
* DSP and Scientific Computing: Extensive experience with numerical, statistical and scientific computing methods, as well as DSP, especially audio DSP for both high-end and low-cost applications.
* Java: Experience building Java GUIs with JFC/Swing and AWT. Proficient in major Java classes including java.lang, java.io, java.net, java.util, java.nio, java.awt, javax.swing as well as JDOM, JNI, JMF (Java Media Framework), JAI (Java Advanced Imaging), and QTJ (QuickTime for Java).
* Application Development: Experience designing complex multi-threaded, portable, client-server applications using various languages including C/C++, Java 1.4, Java 1.5, Perl and Unix Shell.
* Scientific Computing, Statistics and Research Methods: Experience with statistical analysis in SPSS, as well as in low-level programming languages, such as C/C++ and Java. Familiar bio-medical imaging research applications including MRI, CT, and PET, fluorescence and confocal microscopy.
* Other Software Experience: Broad knowledge of GNU/Linux software packages as well as Apache and BSD project software. Basic management skills include Solaris, Mac OS X, FreeBSD, Irix, Windows, NFS, Samba, PC NetLink, Apache, Tomcat, WebDAV, and MySQL. Basic knowledge of MySQL, JSP, and LATEX, in addition to standard office applications.
