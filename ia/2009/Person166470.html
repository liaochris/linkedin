Marcus Johnson’s Summary

Marcus has seven years of professional experience and almost eight years of experience in software development. His development experience involves creating, maintaining, integrating and testing software in Windows, Solaris, and Linux environments. Marcus developed plug-ins and other components using Eclipse Rich Client Platform (with SWT) that were a part of an application that performed network analysis. Previously, that application was in the form of a web-based portal that utilized WebLogic. He developed several portlets for the portal and wrote software design documents for those portlets respectively. Marcus also developed data service layers for the system back end.

Marcus also developed web portlets for a performance assessment web portal within a J2EE environment. He was also responsible for building the releases which were subsequently distributed to the client. In addition to this, Marcus wrote documentation for this project which included user guides, design documents, and online help pages for all of the applications in the portal. Marcus also performed software testing duties which included documenting test plans and procedures, writing various test applications using C++, HTML, and various scripting languages (including Bourne shell and Perl).
