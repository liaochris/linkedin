Tom Wilson’s Summary

President of Jack Russell Software(http://www.jackhq.com), and Chief Technologist, 15 years of software development experience. Currently focused on building web based business applications. Working in both Windows and Linux application stacks.

I really enjoy working with Ruby on Rails. This software platform is the most efficient platform for web application development I have seen in the last 15 years. Its use of convention over configuration makes problem solving and software change management a breeze. The Model View Controller architecture is solid and is starting to take shape on all Platforms. Microsoft is in the process of releasing ASP MVC, which takes advantage of some of the MVC principles.

Before Jack Russell Software was created, I was Vice President of Software Development at ExcelleRx(http://www.excellerx.com), and led a team that built a custom web based enterprise software application called Xeris. Xeris was built using ASP.Net and SQL Server technologies. We started development of Xeris as soon as Asp.Net Version 1.0 was released. The web based technology enabled the company to grow and scale incredibly fast.

Before ExcelleRx, I was Director of Software Development at CarePoint(http://www.carepoint.com). I was the architect of GuardianRx, one of the first Windows, client/server based pharmacy systems. It is an effective local pharmacy solution.

I am a software technologist, who enjoys working with all types of software technology and finding and creating solutions to solve business problems and meet real business needs.
