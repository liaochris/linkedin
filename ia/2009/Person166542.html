    (Partnership; Computer Software industry)

    November 2002 — Present (7 years)

    Blue Brain Project (http://bluebrain.epfl.ch):
    • Designed and developed an advanced cross-platform, 3D graphics based scientific software for neuron reconstruction to be used as a processing system for the microscopic data of brain cells and neurons.
    • Designed framework to support loading and navigation of extremely large data sets.
    • Developed algorithms for automatic estimation of dendritic radii in image stack.
    • Researched and implemented algorithm for semi-automated tracing of neurite branches. This algorithm employs streerable gaussian filters to perform image processing, eigen analysis of the Hessian matrix to determine branch flow direction and Djikstra's algorithm to calculate the optimal route via the defined cost function.
    • Performed extensive optimizations and innovated techniques to allow this processor and memory intensive algorithm to be able to run in real-time on huge data sets even on desktop machines without the need of high-end workstations.

    OrthoClear Inc

    (Privately Held; Medical Devices industry)

    • Worked on a research project to automate the fabrication of medical device.

This approach proved successful in bypassing an already patented technique for this process.
    • Customized RapidForm 3d scanning software by writing plug-ins to automate the post scanning steps.
    • Co-inventor of a few patents assigned to OrthoClear.
