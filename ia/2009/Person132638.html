Kinjal Dixit’s Summary

Telephony Domain. Worked on telephony since 1997. Done a lot of work on IVR, SMS and Dialers. Worked on Dialogic and Asterisk. Developed additional applications like CRMs, voice loggers, workflows for end to end processes and so on. Architected a complete contact center suite, including CRM, outbound/inbound dialer and report server. Role was of a specialist and senior member of the team.

Procurement Domain. Worked on web based tendering, auctions, reverse auctions. Specially developed PKI signer for the web based application. Designed a layer of web services which allow upstream and downstream integration with other applications.

Other than that done a lot of work on diverse technologies like speech recognition, digital image processing, social networking algorithms, accounting systems, predictive dialing algorithms, PKI cryptography.

Have exposure to a variety of languages starting from C, C++, Clipper, Java, C#, Javascript, VBScript, Pascal, Scheme, Perl, PHP, Python, Ruby, Assembler. Partial exposure to other languages such as Fortran, Smalltalk, Forth, Visual Basic and Visual Basic.net, F#, Modula, Boo, Groovy, LISP, Prolog, COBOL and many more.

Worked on many frameworks, including Struts, MVC, Rails, ASP.Net, and on javascript libraries like yui, moo-tools, prototype, scriptaculous, among others.

Have studied application architecture and design patterns extensively. Have studied application security, code security, project management, agile methods like scrum and xp, and so on, also pretty extensively.

Am an effective communicator and trainer on the subjects.
