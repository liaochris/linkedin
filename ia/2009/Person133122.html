Kurt Pires’s Summary

Expert knowledge of the UNIX operating system at both user and kernel level. I have developed applications, system and kernel software for systems running BSD UNIX (4.1a through 4.3), DEC Ultrix, Sequent UNIX, SunOS, DEC OSF/1.0, BSDI (1.1 through 4.0), RedHat Linux, FreeBSD (4 through 6), Solaris (8 through 10) and AIX (5.2 and 5.3); I have also acted as a system administrator for all of those systems. Core-level knowledge of: Sendmail, Apache, GIF & JPEG image formats. Working knowledge of ClearCase, CVS, RCS & SCCS source code control systems as well as Bugzilla and ClearQuest bug tracking tools. Programming languages include: C, C++, Lex, Yacc, Perl (including OO modules), awk, sed, sh, csh, HTML, PHP, PostScript, Java, DB2, Oracle, MySQL and sendmail.cf.

Kurt Pires’s Specialties:

Network Monitoring, Perl, Protocol Design, Automated System Administration
