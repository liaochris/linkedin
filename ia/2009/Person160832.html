Michael Benford’s Summary

Desenvolvedor de software com 7 anos de experiência, trabalhando no projeto e criação de aplicativos desktop, Web e embarcados, concebidos através do ambiente Delphi e da plataforma .NET com a linguagem C#. Experiência em sistemas de controle de acesso, arrecadação de pedágio, segurança de informação, desenvolvidos através de soluções multi-camadas, multi-threaded, escaláveis e de alta disponibilidade. Interessado em soluções de software baseadas no uso de orientação a objetos (OOP), orientação a serviços (SOA) e padrões de projeto.

Michael Benford’s Specialties:

Projeto de sistemas, Delphi, C#, .NET Framework, .NET Remoting, Windows Communication Foundation (WCF), ASP.NET, AJAX, Web Services, API Hooking, orientação a objetos (OOP), orientação a serviços (SOA), aplicações multi-threaded, aplicações multi-camadas, criação de componentes, controles e componentes da DevExpress, design de interfaces gráficas para produtos digitais.
