Ricky Greer’s Summary

A very well seasoned software developer with over 20 years of programming experience. In the 80's I starting writing software for BBS's (Bulletin Board Systems you connected with thru a dial up modem). Most of these were multi player games like Power Struggle, World Dominance, and War Zone 5.

Due to the worldwide popularity of those games, I also created a universal interface system (door.sys) which opened up the way for other developers to start supporting multiple systems. IE... GAP, PCBoard, Wildcat, RBBS, and several others.

Back then we had to create our own connection routines, database structures, and encryption code. In 1996 I started creating "websites"! The internet was obviously going to be the new thing, and it was here to stay. I had to learn a lot of new things to make this work, and am still learning even more each day.

I am currently seeking a LAMP Developer/Administrator position working with the latest technologies. I will also train and mentor others while advancing into any “hands on” type of management positions. I have very little tolerance for sloppy coding practices and crappy database structures.

If you have any type of site problems... I can always fix them!
