Paul Bain’s Summary

Lawyer and information technology (IT) professional.

PaulBain@pobox.com.

These days, I am primarily interested in legal technology, especially electronic data discovery (EDD, or "eDiscovery"), computer forensic examinations, and software that supports litigation (e.g., Concordance, Summation, iPro, iConect, LiveNote, etc.)

I tend to begin using good IT tools and methods long before they become popular. For example, I installed Linux on my home computer in 1997, years before Linux became popular. Furthermore, I have used Slashdot since 1998, server-side Java (J2EE) on Linux since 1999, the Apache web server and the MySQL database server since 2000, the Tomcat web server since 2001, and Debian Linux since 2002.

I have long believed that the most important trend in IT today is the open source software (OSS) revolution. Furthermore, this belief underpins nearly all of my activities and goals that are related, however tenuously, to IT.
