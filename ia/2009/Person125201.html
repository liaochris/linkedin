James West-Sadler’s Summary

Some examples of my work include:

Won a six month contract with a major bank in the City of London. Working on a number of leading edge C# development projects.

Architected and developed a website for the Drinking Water Inspectorate (DWI) using AJAX and ASP.Net version 2.0. The application is used by the DWI to monitor all the water companies in the UK. It is used to log complaints, incidents and the required sampling data from each of the companies. It also includes complex mathematics to determine frequency and/or compliance breaches.

Designed, developed and implemented (end-to-end) a bespoke network data gathering tool. The tool uses remote agents to gather information about the network and the environment. - LogPoint® a server management system for Web, Windows and Window Home Server environments.

Developed a CCTV video review and export application for a major metropolitan transportation network. Using .NET Winforms technologies and COM interop the project was coded in VB.NET and required extensive auditing and solid architecture as it will be used in mission critical scenarios.

Won a contract with a major US insurance company to convert their existing reporting system to .NET. This system was completely rewritten from the ground-up using Webforms, ASP.NET, VB.NET, C#.NET, XML, Agile development and remoting. It is currently serving out reports across this companies entire world-wide infrastructure.
