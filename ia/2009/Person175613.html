  • Software Engineer

    Faculty of Technical Sciences

    (Computer Software industry)

    December 2004 — December 2006 (2 years 1 month)

    I developed business application framework for desktop applications while I was at this position. The framework was based on generated XML code for GUI forms (generic CRUD functionality) and generic reports (based on Jasper framework). The presentation (front-end) layer was implemented by using Swing GUI library. The back-end layer was based on the SQL queries dynamically generated in runtime.

    In order to speed up development process we developed "JForm" - Eclipse plugin for XML code generation (forms). The meta-data for code generation was acquired from PDM (PowerDesigner's Physical Model - XML file). The parser that we used then is Open Source project currently known as "Mosquito PDM". I still keep it up to date here http://mosquito-pdm.sourceforge.net/.

    A few big systems, within the University of Novi Sad, are currently using this framework.
