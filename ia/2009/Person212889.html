  • Developer

    RhymBox

    (Computer Software industry)

    1999 — 2000 (1 year)

    I built a search engine for file sharing networks. The project was not intended for public usage as I was a secondary school student and could not afford to do so financially or time wise.

    The application was a web service built in Perl with an HTML based front end. Protocol support included Napster, OpenNap, CuteMX, Scour eXchange and Gnutella. Each protocol had its own search agent which reported back to a central master collecting and caching the results and displaying them to the user. Extensive reverse engineering, especially for CuteMX, was required and I became familiar with network packet analysis, protocol design, distributed network topology and graph traversal scaling problems.
