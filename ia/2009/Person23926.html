Barbara Ballard’s Summary

Mobile UI design. Mobile user experience architect. Mobile product strategy.

Work on all mobile content excluding advertising and games, but including device, operating system, services, text, use cases, NFC, location, web, applications, and more.

Evangelist for improved wireless product design, incorporating device proliferation and user experience. Designer of mobile user interface patterns.


Author, Designing the Mobile User Experience (Wiley 2007).

Professional services include mobile and wireless:
- user interface design consulting,
- usability consulting,
- graphic design,
- business strategy,
- user experience business process improvement,
- web development consulting, and
- application development consulting.

New service, quick UI review: http://www.littlespringsdesign.com/blog/testing/expert-review/

Barbara Ballard’s Specialties:
