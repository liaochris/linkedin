Moutaz Shams’s Summary

Moutaz Shams is an experienced software developer specializing in analysing, architecting, designing and developing enterprise-class rich web applications using Microsoft .NET related technologies and relational database management systems.

For several years (since 2003) he has been focusing on C, C#, OOP, WindowsForms, GDI+, ASP .NET, Service Oriented Architecture, Microsoft SQL Server and recently Microsoft Silverlight. Before that.. he was using Microsoft Visual Basic 3 through 6 in developing customizable windows-based ERP applications.
