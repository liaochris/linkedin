Gregory B Pepus’s Summary

Greg Pepus works in two distinct areas:

Supporting the US intelligence community by helping them identify and adopt technology from venture backed companies with a particular focus on knowledge management, multilanguage analytics and service oriented clients/rich internet applications.

Integrating analytical tools including foreign language support, data mining, video/audio mining, search, geospatial, visualization support, link analysis - all using rich internet application development approaches for loosely coupled, composite applications. Current focus is on building event driven applications using Adobe Flex, AIR, AS with a focus on tagged based programming approaches using model view controller frameworks.

Clearances: TS/SCI 2008 full scope polygraph 2008.

Gregory B Pepus’s Specialties:

Information Technology: Knowledge Mgt Technologies, machine translation, text extraction, multimedia exploitation, search, visualization, analytic and predictive systems, middleware/SOA, rich internet applications

Rich Internet Applications: Integration of a range of analytical tools using RIA development with Adobe Flex, AIR, Action Script. Focus on loosely coupled components, event driven architectures with dependency injection.
