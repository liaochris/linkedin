Federico Feroldi’s Summary

I have more than twelve years of experience in designing, developing, integrating and managing complex solutions for Internet-based companies. I have a history of successfully using Open Source software to achieve company goals with an excellent ROI.

I'm a certified Scrum Master, member of the Scrum Alliance.

Recent work interests include: highly scalable web architectures (to tens of millions users), distributed computing architectures (Map-Reduce, Hadoop clusters), dynamic web technologies (AJAX, Comet) and JS frameworks (YUI, JQuery, Prototype); social network technologies (Google's OpenSocial, Facebook Connect); web 2.0 information management like Wikis and document tagging.

I'm currently writing my BS thesis on "Scalable recommendation systems" based on the Map/Reduce paradigm, implemented on Hadoop based technologies.

I'm a Ruby language enthusiast and Ruby On Rails evangelist.
Recent interests includes Erlang, a concurrent/distributed functional language.

Freelance technical writer for Infomedia (Login, Computer Programming) about innovation, languages, web technologies.

Visit my blog: http://www.pixzone.com/blog/

Federico Feroldi’s Specialties:

Design and implement large scale Internet application and systems. Build robust and scalable solutions for hard to solve problems.

Development skills (C/C++, Java, Perl, Ruby, PHP, Javascript). Security (IDS, firewalls) and networking (expecially on load balancing and optimization issues). Database programming (mysql, oracle/plsql, sqlserver). XHTML, XML, DOM, JS, DHTML. Familiar with a wide range of Open Source projects and operating systems (freebsd, linux, solaris), Apache.
