Michel Dufresne’s Summary

I was the CTO of Medco Health Solutions, a Fortune 50 Company, from 2003 to 2006, reporting to the COO. At Medco I was responsible for establishing the Application Development Standards and developing and maintaining the Enterprise Application Framework based on Java J2EE platform.

I have extensive experience in Business Application Development and the elaboration and execution of IT Strategic plans. My strengths are in the area of overseeing the development of enterprise web-based business applications using various open-source technologies (C++, Java/J2EE, Python, MySQL, Postgres, Oracle, etc.) and Artificial Intelligence. In particular in Knowledge Representation and Rule-based Business Applications.

I was the Lead Architect at Empire Blue Cross Blue Shield of New York for their eBusiness initiative. This initative consisted in the development of Member, Employer, and Provider Portals in J2EE with real time transactions with their legacy CICS mainframe.

Recently I developed Top Engine - The Semantic Web Engine for the Enterprise. Top Engine is a Business Rules Engine written in C++ and Python programing languages that uses Semantic Web Technologies in an innovative way. In particular, Top Engine uses W3C's Web Ontology language (OWL DL) as the modeling language of the Domain Model for rule-based applications.

    (Computer Software industry)

    2006 — 2008 (2 years)

    Developed Top Engine – The Semantic Web engine for the Enterprise.

    Top Engine is a business rule engine that uses OWL Description Logic (DL) ontological definitions for vocabulary primitives. Top Engine provides OWL DL Reasoning services to compute satisfiability, subsumption, equivalence, and disjointness of the ontology classes. Ability to write rules on top of ontology using a fusion between DL and Logic Programs formalism. Inference engine in C++ using a rete-based algorithm with forward and backward chaining, and querying capability. An RDF graph is used as working memory.
