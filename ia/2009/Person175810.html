Nimrod Abing’s Summary

I started programming at age 15 on an Atari 800XL home computer, moving on to bigger and better things in college when I wrote my first C program on a PC running MS-DOS. I dabbled in games programming for a while using the Allegro Game Programming Library. Eventually, Linux found its way onto my hard drive courtesy of Debian and I've been hooked ever since. My first stab at web programming using PHP was for PhilWeavers.net. At that time they needed a quick and dirty message board and membership database. I was able to finish the job in two days time.

In my time as a professional, I have worked as a computer technician/go-to-guy, as a programming instructor for college students, system administrator, network administrator, database administrator, and freelance computer programmer.

In what little spare time I have left, I am working towards getting certifications for Java Programming (SCJP and eventually SCJD), MySQL (CMDEV or CMDBA), RedHat (RHCT and eventually RHCE). I still have a long way to go though.

Note to recruiters: I am currently not accepting any employment or recruitment offers.
