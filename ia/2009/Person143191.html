António Cruz’s Summary

In the last 20 years, I worked as a consultant, lead developer, trainer, architect and manager for several entities in Defense, Banking, Industry, Services and Telecommunications sectors, so I have a lot of real world projects experience, particularly using all versions of the .NET Framework and related technologies. I also got sparse knowledge of Java and several other languages and technologies.

As a software architect, I have recognized experience in distributed software architectures, software development best practices, SOA, EDA, web services design and implementation, design patterns, messaging patterns, application integration patterns, software analysis and design.

I also have relevant knowledge of Software Factories, Domain Specific Languages, SAML, WS-Security, WS-Trust, WS-Federation and BPEL-WS.

I'm a big fan of best practices and methodologies for software development, including Source Control, Testing, Continuous Integration, Contract-First Development and Scrum.

I'm a Certified Scrum Master and Certified Scrum Product Owner and I'm currently using Scrum with 14 people in 3 projects: Sapo Services Bus, GIS and Web Analytics.

My technical skills, professional deliveries, excelent presentation and communication skills are highly considered so I've been invited and doing a lot of presentations, deliver training courses and act as juri for technical sessions, courses and several well-known national conferences and events around the country.

My current interests include: Context-Aware Systems, Complex Event Processing, Semantic Web, Cloud Computing, Geo-Marketing and Behavioral Targeting.

My favorite hobbies are to read and travel.

You can access a glimpse of what I think through my blog: http://www.arquitecturadesoftware.org/blogs/antoniocruz
