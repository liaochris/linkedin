Christopher Murtagh’s Summary

I've spent most of my professional life either writing software and building custom hardware (mostly musical interfaces to software). For 6 years I was the development lead, system administrator and web systems architect for McGill University's Web Services Group's home-grown content management system. I have given talks on designing web systems using open source software, mostly targeted at higher education institutions.

Topics that interest me:

Linux, both server and desktop
Networking, system administration and security
Web application architecting, coding and design
Audio recording, production and streaming
Computer music and computer music composition
Avant-garde and new music
