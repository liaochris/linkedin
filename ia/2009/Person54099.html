  • Domain migration and OS deployment

    Home Group

    (Non-Profit; 501-1000 employees; Non-Profit Organization Management industry)

    April 2007 — September 2008 (1 year 6 months)

    Originally I was contracted to the HOME Group to assist with testing for the migration of 2700 users PC’s from two older Windows NT 4 and 2000 AD based domains to one common Windows 2003 Active Directory domain. My main responsibility was the UAT testing of all backend server aspects of the project scope.
    Part of the original project scope was that as user accounts where migrated, the users’ PC’s would be refreshed with a standardised copy of Windows XP using Microsoft’s OSD framework. Unfortunately the OSD process that was in place fell short of expectations and failed UAT for multiple reasons and so was dropped from the project goals.
    As I had spent some considerable time understanding the organizations requirements and constrains for OS deployment I was asked to design, develop and test a system that would allow a single master windows XP image to be deployment at any PC in the environment. The end result was a customized OSD based deployment from a USB pen.
