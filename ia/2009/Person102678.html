James Lai’s Summary

My focus as a web designer and programmer is twofold: to create memorable user experiences through pleasing visuals and highly usable sites, and for my clients to be absolutely thrilled with the work.

I have been both designing and programming sites for nearly a decade, providing services for a myriad of industries both nationally and internationally. I have designed and developed countless web products using an equally vast array of technologies.

Today I am more involved and surrounded by the industry than ever. Through reading, blogging and actively participating in the industry, learning and exploring new frameworks, libraries and languages, keeping up with design trends and pushing the boundaries of what we feel web sites are able to do. Web is my passion.

James Lai’s Specialties:

Web design, web development, CSS/XHTML, web standards, usability, PHP, MySQL, jQuery, Wordpress, Expression Engine, Code Igniter, XML, JSON, AJAX, eCommerce integration, Wiki integration, web application development, marketing and rollout support.
