Performed and implemented the storage design of an 11TB Oracle/Solaris 10 performance/mission critical solution on DMX2, while at a global investment bank.

Worked in various performance committees, from a storage perspective, for Oracle and Sybase solutions running on Solaris. Also worked closing with Microsoft for the largest financial implementation of Microsoft BizTalk (SQL Server 2005)

Perl Scripting:

1. Automated an entire SRDF implementation at a global investment bank, whereby device and composite groups were dynamically populated and modified depending on device changes on the host.
2. Completely automated the storage provisioning process in two global investment banks by creating a toolset that both upheld the storage provisioning standards, as well as performing complete checks to ensure that there were no storage provisioning mistakes. In some cases, the reduction time spent on provisioning ranged from 5-10 man days per week, down to 1 man day with the provisioning tool in place.
3. Created a storage resource management tool for a global investment bank, providing end-to-end visibility of the environment, as well as providing a tool which was capable of storage forecasting / reporting.
4. Created a CGI/Perl "terminal broker" which would take a user's ID, compare against a database and present the VMware hosts, Terminal Services or Citrix servers against which the user was registered.
5. Created a CGI/Perl VMware auditing tool which alerts/displays VM migrations/configuration changes that have taken place.
