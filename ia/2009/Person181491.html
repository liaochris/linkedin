Patrick Glynn’s Experience

  • Game Programmer

    Novint Technologies

    (Public Company; NVNT; Computer Games industry)

    January 2007 — February 2009 (2 years 2 months)

    Implementing Novint Falcon into games including Left 4 Dead, The Orange Box ( Half Life 2, Half Life 2 Episode One, Half Life 2 Episode Two, Portal, Team Fortress 2 ), The Ship, and Advent Rising (Unreal Engine 2).

    Designed and created Novint HFX which is a high level haptic effect handling API utilizing load library to avoid driver dependency and a pure abstract effect class for game specific forces which update on a millisecond interval.

    Through Novint HFX I have helped implementing it into many AAA titles:

    Electronic Arts: Need For Speed ProStreet, Battlefield 2, and Battlefield 2142.
    Valve: Left 4 Dead and The Orange Box.
    Eidos: Project SnowBlind.
    Outerlight: The Ship

    Various game mods from Novint also use Novint HFX for falcon implementation such as Id's Doom 3 (including expansion), HumanHead's Prey, and a mod i personally did within a month for Monolith's F.E.A.R. Novint HFX continues to be use in Falcon implementation.
