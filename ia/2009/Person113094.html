Jim Jagielski’s Summary

I've been lucky enough to work in many layers of IT, from design to programming through architecture, implementation and evaluation. I like opportunities that are unbounded and allow each person to collaborate in a healthy and open environment.

I founded a successful ISP and web hosting company, have developed code and solutions for numerous F100 companies, served as CTO for two very well known and respected companies (Zend Technologies and Covalent Technologies) as well as having been a driving force and developer within the open source community.

Jim Jagielski’s Specialties:

In-depth knowledge of numerous programming and scripting languages; Real-world and practical expertise in all aspects and phases of large-scale, high-availability and high-performance application development; Network design and implementation; Security and performance audits; Extensive depth and breadth of web/net technologies

    May 1995 — Present (14 years)

    I was one of the 1st people to join the (then called) Apache Group, which developed the Apache Web Server, using the NCSA Web Server as the foundation. Apache has grown to be the most widely used web server on the Internet and is arguably the most successful Open Source project ever. In June 1999, we created the Apache Software Foundation, a 501(c)3 non-profit corporation to help foster the development of the Apache Web Server and other Open Source projects under the ASF umbrella. Since its inception, I have served on the Board of Directors of the ASF and as Exec VP/Secretary of the foundation, I am currently Chairmain of the ASF, in addition to being an active developer on several of the ASF PMCs (projects).
