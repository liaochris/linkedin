Bart van den Eijnden’s Summary

I am dedicated to using open source GIS software since this has a lot of benefits. For instance, when using proprietary GIS software and you find a bug, you might need to wait several months or even a year before this bug will be fixed. This can hold up your project significantly, or if you're lucky you might be able to find work-arounds. With open source, you can look at the code yourself and suggest a fix, or you can hire somebody to fix it quickly. This is a huge benefit. Also, the support can be a lot better and more responsive than for proprietary GIS vendors. I've worked with many proprietary GIS software packages and open source is way better. Also, the things you create will be available to everyone, which is a good thing and a higher cause.
