Martin Kleppmann’s Summary

Martin is a highly motivated, efficient and personable technologist and entrepreneur. With 10 years of experience in developing web applications, proven project management and team skills, and a strong academic background, he is constantly striving for shared success in his work with clients and partners. Martin approaches assignments passionately, realistically, with creative and analytic thinking and highest integrity.

Recently Martin has been concentrating on Go Test It http://go-test.it/li -- the best way to test that your website works correctly in a broad range of web browsers. Go Test It enables you to easily record stories of how a user interacts with your web app, to edit these collaboratively online, and to play them back with many different real web browsers. When the tests are done we notify you of the results. We take care of maintaining the testing platforms -- no longer do you need to waste time maintaining several different Windows, Mac and Linux installations yourself.

Martin founded Ept Computing in 2007, with a view to helping companies make best use of state-of-the-art web technology to generate a competitive advantage and delight users. Ept works on web applications which are not just clones of existing concepts, but which take what is possible today to the next level. With a broad range of experience, including high-end Ajax sites, web on mobile devices, automated regression testing and semantic web technologies, this start-up has all the tools to realise those web projects which others find too hard; with solid project management, great people, high integrity, a strong focus on the user experience and business value, and delighted clients, Ept Computing is good to work with and can be trusted to get things done well.

Martin Kleppmann’s Specialties:

Web application development (Ajax, Python, Ruby on Rails, Java etc.), web application testing, semantic web, mobile applications, user interface design, project management, information architecture, interaction design, scalable application architecture, innovation, creativity, communication.
