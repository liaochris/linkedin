Zigurd Mednieks’s Summary

I am the co author, with Rick Rogers, John Lombardo, and Blake Meike, of Android Application Development with the Google SDK, published by O'Reilly Media, and currently under revision for a 2nd edition. I have written books and book chapters before, including one of the first books on self-hosted Macintosh application development: C. Programming Techniques for the Macintosh, which was published by Howard W. Sams & Co. in 1986. Throughout my career, I have been inventing, explaining, and using computer UI systems.

At D2 Technologies, I am the architect of two multimedia, multi-protocol, IMS-capable user interface systems: One that runs in it's own user interface framework in a standard Java runtime (e.g. JamVM and GNU Classpath, or Java SE), and one that replaces core elements of the Android communications UI with components that are capable of both IP communication and circuit switched telephony.

At JumpTap I created a downloadable user interface for mobile search location services that broke new ground in performant high-concurrency J2ME user interfaces in 2.5G networks and is one of the most sophisticated non-game UIs in mobile software.

i have expertiseis in areas as diverse as microcode diagnostics and debugging, speech processing, IVR, visual user interface design, CPE and infrastructure telephony, unified messaging, wireless mobile data network infrastructure nodes - GGSN and PDSN, mobile infrastructure integration with applications, prepaid and postpaid charging and billing in mobile networks, and context-sensitive networking. At Chasma, I applied my knowledge of mobile infrastructure to enable people to play connected games on their mobile handsets.

Read what I think about mobile software at 4thscreen.blogspot.com

My other blogs are telirati.blogspot.com, which is an archive of earlier writings on the telecom and software industries, and jamdats1blog.blogspot.com, which covered the IPO of Jamdat, a successful mobile games publisher.
