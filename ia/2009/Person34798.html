Bruce Wenholz’s Summary

I have worked mostly with embedded systems. My first years in the industry were doing verification and validation half time on missile system command and control which entailed communication using both RF and Cable based communications and the other half time designing and implementing tools to assist in the analysis to the entire real-time command and control system.
At my last job I was also responsible for finding and fixing all the fatal errors in the software for this medical device. The device would never been sold without my correcting the fatal errors that occurred during in-house and clinical-trials. I was responsible for the complete rewrite of the telemetry which missed many messages into a process that missed only messages that had RF interference.

I did a similar complete rewrite for a Baker Hughes division as part of adding a complete alternate pipeline monitoring and control system. When I finished my modifications the employee responsible for integrating my changes into the master source code archive used my code as the baseline for all the versions there by removing 99% of the lost communication events.

On my last job, my department did not have any work for me to do so I requested a move to the web based application group. I in short order I learned their requirements tracking and specification and designed an upgrade to one of the Java packages.

I completely enjoy working either in a team as a team lead or on a small one man project. I prefer to be a major contributor in a new system whole life cycle and secondly to work on maintenance of an existing system. I also like to be where the software is directly controlling the hardware.

Bruce Wenholz’s Specialties:

I have several very strong areas. These are Rtos, embedded firmware/software, communications on small platforms, new real-time product design, fixing critical and fatal errors, reducing code size on small platforms that have almost no EPROM/flash code space left to add features. I am also very good at communication with upper management in terms that they understand.
