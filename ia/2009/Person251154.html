Gianluigi Zanettini’s Summary

Laureato in Informatica presso l’Università di Ferrara con tesi dal titolo “Realizzazione di una web application per la gestione dei servizi DNS, DHCP e RADIUS”, Gianluigi Zanettini è Responsabile Esecutivo presso il portale di informatica e tecnologia MegaLab.it, cui affianca la libera professione di consulente nel settore IT.

L’autore è specializzato in assistenza tecnico-sistemistica, project management, realizzazione di applicazioni e web application, e progettazione di reti di calcolatori per la PMI.

Si occupa di analizzare, smontare e rimontare sistemi Windows fin dal 1994, con particolare attenzione agli aspetti legati alle prestazioni, all’usabilità dell’interfaccia grafica ed alla sicurezza dell’ambiente operativo.


* Reti locali, centralizzate o peer in qualità di amministratore di sistema
* Assistenza tecnico-sistemistica su tutte le piatteforme Windows
* Deploy di webserver, ftpserver, mailserver, DNS con particolare attenzione a tutti gli aspetti di sicurezza correlati
* Progettazione ed amministrazione web, ed in particolare alle problematiche connesse alla gestione di siti Internet e portali ad alto traffico
* Redazione di testi e guide a carattere informatico
* Microsoft Office ed OpenOffice, in particolar modo di Microsoft Access e Microsoft Word
* Esperienza in ambiente 16 bit (Windows 3.11, Microsoft Dos 6.22)
* ASP, PHP e SQL
* DB Administration (Access/MySQL/SQLite/SQL Server)
* Esperienza con JSP e Java
* Visual Basic 6 e Visual Basic .NET in particolar modo nella realizzazione di gestionali ed ERP verso basi di dati
* C e C++ a livello scolastico
* Rudimenti di assembler e reverse engineering
