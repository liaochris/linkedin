Patrick Blume’s Summary

Patrick is an Enterprise Systems Architect, specialising in the Microsoft .Net technology arena. With over 13 years of experience in the field, Patrick has built a high level of expertise covering a range of leading edge technologies and solutions.

In such a rapidly developing technical environment, Patrick has enjoyed the challenge of designing and building solutions that utilise the latest technical opportunities and products while still delivering a solid, commercially viable outcome for the client.

Patrick’s most recent role was as Strategic Systems Architect at FCS Online, an Australian based database and web services company and has previously worked for Logica and MITS as Software Architect, Project Team Leader, Analyst and Developer.

Patrick is a Microsoft Certified Solution Developer, specialising in .Net technologies and SQL Server and has a First Class Honours degree in Mathematical and Computer Science from La Trobe University, Melbourne


In his spare time, Patrick is also a BodyPump and Freestyle Cycle Fitness Instructor.

Enterprise Software Architecture, Workflow Design, Application Design, Security Analysis, Performance Optimisation

    FCS Online

    (Privately Held; Information Technology and Services industry)

    January 2004 — March 2009 (5 years 3 months)

    Patrick is employed as Strategic Systems Architect leading a team of 10 developers, DBA’s and network staff at FCS Online, an Australian based database search services company. With real-time links to a number of major financial institutions, it is Patrick’s responsibility to design, build and deploy systems that are reliable, available and respond within tightly defined time frames.

    Patrick’s most recent projects at the company have involved the design of an online Identity Verification System, Identi-Check ®, that checks an applicant’s identity details against a number of databases containing tens of millions of records and produces a scoring report within 2 seconds. This project was implemented in C# using WCF and WWF to build a scalable multi-tier workflow system at the back end and an ASP.Net web service front end. This system is currently used by a number of major banks and telecommunications customers to verify new customers.

    Before that, Patrick designed and implemented a searching system that allows customer to search for debtors though databases of up to 100 million records and return results in an average of less than one second. This product, FCS Search, currently has over 3,000 customers, servicing over 80,000 requests per day.

    From an infrastructure perspective, Patrick has recently undertaken a complete restructuring of the FCS OnLine’s production environment. This included the design and implementation process of moving from a basic three tier architecture to a fully virtualised, load balanced, high availability environment using VMWare ESX Server, SQL Server 2005 Database Mirroring and IIS Web Server Load Balancing, based on Dell and EMC Fibre SAN hardware.
