Adam Ducker’s Summary

I am a 25 year old graduate of Computer Science with a minor in Digital Graphics. I have 3 years professional experience with Web Development and currently work as an independent contractor for companies on both the East and West coasts.

I built my first website about 1997 in some defunct WYSIWYG with no idea that ten years later it would be paying the bills. I spent the first half of that time learning to hand code HTML in notepad using tables for layout. I spent the next half decade unlearning all of it and studying XHTML and CSS for layout.

My specialty is integrating a number of server side languages with cross-browser compatible XHTML, CSS, and JavaScript. My current workload includes development on several different platforms including Windows, Linux, and Mac with languages including PHP, Java/JSP, and Ruby On Rails.

I currently live in Newark, DE.

Adam Ducker’s Specialties:

Mastery of CSS and standards conscious HTML, construction of dynamic web pages using both client and server side programming.
