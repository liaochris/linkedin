Finn Hausager’s Summary

I want to deliver the best possible solution.
This covers the basic functionallity as specified by the costumer, but also safe and reliable regarding EMC issues and durability ensuring long lifetime.
All this within a reasonable project time and cost.
Allthough my primary focus has been on communication electronic, i like to solve any problem using electronic circuits if possible.
I use signal integrity tools for digital circuits.
I use CST 3D solver for antennas and various s-parm modelling.
I use ADS for antenna tuning and other PSpice simulators.
