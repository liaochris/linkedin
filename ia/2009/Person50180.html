  • Senior Security Analyst / Senior Software Engineer

    Symantec Corp.

    (Public Company; SYMC; Computer Software industry)

    January 2002 — May 2009 (7 years 5 months)

    Performed hands-on vulnerability code assessment and managed vulnerability issues found in Symantec products as a member of the Product Security team. The team was responsible for the elimination of security vulnerabilities that would put customers at risk and potentially damage the company’s reputation as a security vendor.

    • Designed and implemented a software program in the Ruby programming language, to identify products using 3rd party components with newly discovered vulnerabilities. This resulted in savings of over 5 hours a week of manually monitoring security list for new vulnerabilities.

    • Collaborated with Purdue University to develop material that is used to train Symantec engineers how to write secure code. This has helped reduce the number of security vulnerabilities and bugs across the Symantec product line.

    • Implemented CISSP preparation training to help increase the security awareness of Symantec engineers. Security awareness helps engineers design and develop more secure products that results in a better stability and increased customer satisfaction.

    • Drafted the first version of Vulnerability Response process that is used within Symantec to resolve product vulnerabilities, protecting Symantec customers from security threats.

    • Contributed to the publication of the Guide for Vulnerability Management (now considered the industrial standard), as a member of the Organization for Internet Safety. This helped establish Symantec as a leader in the security arena.
