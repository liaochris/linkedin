Joel Helbling’s Summary

Founder of Adelphus Solutions, LLC, a web development firm in North East Ohio; software architect/team leader, process analyst, troubleshooter, trainer and technical writer. I have 12 years of experience in hardware and software systems engineering and development.

Current/recent projects:

* Live video streaming automation for schedule-driven events.

* Integration of 5+ retail DSS applications into a single functional system with a cohesive user experience using Java web services & XSLT.

* Constructing an ETL pipeline between a new POS system (OLTP) and the latest version of a mature exception-based reporting system (OLAP).

* Building a non-invasive many-noded network monitoring system for monitoring uptime of internet servers on multiple services and protocols, and assisting server admins with network connectivity troubleshooting information and "nines" accessibility verification. Will be eventually be positioned as a subscription service.

* Developing a web application framework designed for LAMP-based object-oriented web development.
