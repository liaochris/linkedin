Hoani Cross’s Summary

I started to work on web-oriented software development engineering but I rapidly shifted on Java EE oriented architectures. Open Source solutions early adopter, I always try to promote these solutions in my jobs while contributing to these products at the same time. Found of agile methodologies since 2000, I also tried to make companies I worked for adopt principles of agility. Today I mainly focus on software architecture and development, enterprise application integration and urbanization of information systems.
