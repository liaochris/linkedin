Chris Custine’s Summary

I have been working on large scale middleware integration, SOA, and web applications since 1995. I am currently working in the Apache community on several open source projects and I enjoy speaking at conferences and user groups about the projects I work on.

I have been involved with the Apache Software Foundation in various shapes and forms for several years. Officially I am involved with the following Apache projects:

- Committer on Apache ServiceMix, an open source ESB for integration.
- Committer and PMC member on the Apache Directory Server, a Java based LDAP v3 certified server implementation.

I am also involved as a contributor on several other Apache Software Foundation projects including Felix (OSGi container), ActiveMQ, Camel, and CXF.

I am currently employed by IONA Technologies and I work primarily on Fuse ESB, which is a commercially supported version of Apache ServiceMix. I also work with the rest of the Fuse product line including:

- Fuse Message Broker (Apache ActiveMQ)
- Fuse Mediation Framework (Apache Camel)
- Fuse Services Framework (Apache CXF)
