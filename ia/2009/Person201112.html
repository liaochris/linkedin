Robert van Drunen’s Summary

Architect and lead developer of a software system used by a travel agency to register customer data, flight registrations and generate financial reports.

Developer and technical project leader for a software system used by Rijkswaterstaat to monitor the acquisition of cadastral parcels. Given all the business rules that apply to this type of procedure, the system was complex and required an intimate knowledge of the way cadastral information is maintained and processed in the Netherlands.

All-round developer and consultant for a software system that is widely used in the public safety sector in the Netherlands.

All-round developer and technical project leader in a project which involved a system built using a combination of a central Oracle (Spatial) database, various programming languages such as Java and Delphi and a web interface built on open source-alternatives such as PHP, Apache and OpenLDAP. The system itself is being used by the KLIC corporation, one of the central organisations in the Dutch excavation world - processing more than a hundred thousand excavation applications a year.

Developer for a company that does, amongst others, a lot of work for Dutch Telecom companies. We maintain and develop a system that is used to keep track of the location of cables and network equipment in their infrastructure. Besides that we occasionally give training to end users and new developers.

Robert van Drunen’s Specialties:

(Spatial) Database technologies, .NET, Delphi, broad knowledge of software in the Geo-ICT sector, project leading, system architecture, open source solutions, web technology and -services.
