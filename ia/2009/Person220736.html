Shawn Parker’s Summary

Shawn has over 25 years software engineering and architecture experience using various Microsoft technologies and development tools including SharePoint 2007, Silverlight, Visual Studio .NET, ASP.NET, VB.NET, .NET Framework, Visual C/C++/C#, Visual Basic, WPF, XML, XSLT, OpenXML, COM, DCOM, COM+, ActiveX, .NET Remoting, Web-Services and WCF. Database experience includes SQL Server, MVDB, and Microsoft Access. Shawn is an experienced lead and solution architect. Shawn has designed, developed and implemented numerous enterprise applications. Shawn is able to create and maintain functional and efficient software solutions addressing business needs without sacrificing the ideals and company culture that make each company unique.
