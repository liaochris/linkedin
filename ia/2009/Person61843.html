Denny Mathew’s Summary

• Vehicle Automation using CAN protocol

Major components include PIC mid range series micro-controller, CAN controller and CAN Transceiver per node. One node controls head light and AC. Other node controls door lock.


• Automatic Toll Booth Authentication System

Major components include PIC16 series, RFID, IR sensor, Stepper Motor, LCD & 4x4 Keypad.
Concept of this project is to automate authentication process for vehicles passing over through toll gate. RFID module retrieves user toll card information and the micro-controller validates these credentials with stored database. Balance retrieval and recharge option implemented as add-on features.

• Simple Office Security System

Major components include LPC 2129 [ARM7TDMI CORE], RFID, GSM Modem & PIR sensor.
Objective of this project is to create simple Office room security system. PIR sensor is used to detect human presence. RFID interface retrieve user details which is then validated by the system. If an invalid card is presented, a SMS is sent to the Office in-charge. This system is used for a single room security, but it can be implemented for the multiple rooms.
