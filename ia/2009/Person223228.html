  • Founder

    Elemental Vision Mosaics (Self-employed)

    (Self-Employed; Myself Only; Building Materials industry)

    December 2006 — March 2008 (1 year 4 months)

    I had the idea of creating marble mosaics that looked like pictures for the housing industry while remodeling my bathroom some years back. By building a database of scans of marble tiles and using image recognition to match pieces of a digital picture a virtual mosaic could be constructed that approximated the image. The virtual mosaic could then be fed to an automated construction robot to build the physical mosaic. I started development using Java and MySQL with Eclipse as the development environment. As it became apparent that the costs involved were going to exceed my budget I repurposed the development effort as more of a learning experience. To that end I switched development tools to MS Developer Studio and MS SQL Server using C# and Linq.
