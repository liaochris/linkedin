  • Project co-founder and developer

    flashNavigator

    (Partnership; 1-10 employees; Computer Software industry)

    July 2006 — December 2007 (1 year 6 months)

    flashNavigator is a rapid development framework, used for publishing GIS and CAD data through Adobe Flash technology in web, desktop and recently mobile applications (Adobe Flash Lite).

    It has been in active development for over two years. At first it was being developed as a part of karteGradova.com project. Today is a product of Omnisdata company.

    Unlike other solutions it provides tiled vector to vector conversion of GIS and CAD file and database formats.

    It supports multiple vector and raster formats:
    ESRI Shapefile
    Mapinfo
    Oracle Spatial
    PostgreSQL
    OCAD
    GeoTIFF
    ...

  • Project co-founder and developer

    karteGradova.com

    (Partnership; 1-10 employees; Internet industry)

    April 2005 — November 2007 (2 years 8 months)

    karteGradova.com (means city maps in Croatian) is a unique mapping website that is powered by flashNavigator technology. It offers Web 2.0 features like map embedding, custom user location, etc.

    It's the leading mapping website in Croatia and it's used as location service by many users and other websites as well.

    Maps on this website where created by converting original OCAD format into Adobe Flash format using flashNavigator.

    Today karteGradova.com is a project of Omnisdata company. Development is continued toward map implementation of true GIS formats.
