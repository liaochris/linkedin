Benoit Trémouroux’s Summary

Experienced project manager working in information and communication fields.

I'm currently working on Uploadmanager : file transfer and storage for companies (http://www.uploadmanager.eu).

The initial idea is simple: all companies, nowadays, have needs as regards the transfer, sharing and storage of files. The use of the electronic mail can often be used for such means. However, the e-mail cannot be used for transferring heavy files in the most secure and traceable manner.

This is where Uploadmanager becomes handy. Uploadmanager 2.4 allows you to send files that are too heavy to attach to an e-mail and without all the inconveniences of other classic solutions:

• The delivery of a CD-ROM: an expensive solution that comes hand in hand with important delays caused by distance.
• The FTP server: often reserved to experimented users, it disposes of limited functionality.

Ergonomics and versatility: Uploadmanager leads the way.
In addition to giving a solution to the weaknesses of systems available on the market, Uploadmanager 2.4 puts forward a set of useful and original functionalities:

• User-friendly web interface that does not require any particular installation.
• Simple and intuitive creation with personalized access allowing a confidential exchange of files with different partners, clients, suppliers.
• The solution is available is French, English and Dutch.
• Complete confidential exchange via SSL certificate.
• File transfer via links.
• Files are available anytime and anywhere.
• Automatic and instantaneous email notification of the reception of a file.
• Accessible from any terminal connected to Internet, the system is compatible with Windows, Mac Os and Linux.
• Logins generated by the interface are also accessible via Ftp client.
• Possibility to personalise the interface and its easy implementation on the existing website.


For further information, visit our website : http://www.uploadmanager.eu
