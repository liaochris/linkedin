Mike Walsh’s Summary

A "SQL Server Geek" at heart, I enjoy helping out with all aspects of SQL Server projects and problems large or small.

I have been working with SQL for over 10 years in versions 6.5 --> 2008 in various capacities, mostly DBA/ DBA&Performance minded development.

I enjoy mentoring and learning through teaching and often develop custom coursework for helping individuals and companies better use SQL for their specific business needs.

Through StraightPath Solutions, I assist companies with various SQL Server challenges in my spare time. I love working with a customer who has had a poor experience with their databases and seeing that attitude turn around with excitement towards their database needs. I like to apply mentoring and training towards each consulting engagement with the majority of them being an opportunity to better equip the full time staff better handle their SQL environment. I also do work with Remote DBA arrangements where my services will be provided for a longer term basis when the budget or need isn't there for a 40 hour a week full time DBA.

Mike Walsh’s Specialties:

MIcrosoft SQL Server, SQL (TSQL, PL/SQL), SQL Server Performance Analysis, Tuning, SQL Server Database Administration, Backup and Recovery Strategies, ETL with various tools (DTS, SSIS, Oracle Warehouse Builder - 10g), Database Development, Database Design primarily with SQL Server (2000/2005/2008).
