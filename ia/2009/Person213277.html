Sekar Sonachalam’s Summary

Objective

Looking for an opportunity to work with SOC mobile/portable systems which are based on propriety or wince RTOS in the area of firmware development of SOC, application program for ARM processors, development and implementation of filters, fixed point arithmetic, implementation and development of audio image and video codecs, development of drivers, memory management.

Accomplishments

•Worked as Lead/Senior member in the Audio Core Technology team and provided the solutions to many popular mobile systems including the initial Apple IPOD versions, SanDisk Sansa, Samsung Yepp, Philips Flat, NVIDIA APx
•Developed various DSP filters using Matlab for audio post processing
•To achieve codec security, developed fuzzing algorithms for various audio and video formats for Wince5/6 based embedded system
•Familiar with platformbuilder and exposure to winCE5 and winCE6

•Solved hypersonic flows using 32/128/1024 nodes SIMD/MIMD system
•Trajectory optimization using extended Kalman filters.
•Developed an industry standard 3D Euler code called BHEEMA (Boltzmann Hypersonic Euler equation solver for Missile Aerodynamics) which is based on a new method called Kinetic Flux Vector Splitting Method (KFVS). This was the first 3D KFVS code for practical configurations which enabled a transformation of the algorithm from research to production code.
•Designed and developed a production standard code BHEEMA by using KFVS method for flow solvers

    I developed a pre and post processor using C, postscript and FORTRAN to analyze the results computed by the MIMD parallel system in a pesudo time.
