Adam Flater’s Summary

Adam Flater is a Software Architect at Universal Mind. His background is in working with Java and Microsoft technologies, building both web and desktop applications. He has been working in Flex user interface and server development since the beta release of Flex 2.0 in October of 2005. Adam was the lead developer on the ActionScript SDK for eBay's Web Services used in the eBay Desktop application. Adam is the founder of the Merapi Project, an open source project that bridges Java with Flex on AIR at the desktop. In 2007 Adam presented at Fig Leaf's Flex Maniacs in D.C., The eBay Developer's conference in Boston, 360Flex in Seattle, Adobe's OnAir Bus Tour in D.C., and Adobe's MAX Conference in Chicago.

Adam Flater’s Specialties:

Flex, AIR, Amazon S3, RIA, Distributed Team Development, Front End Architecture, Object Oriented ActionScript 3.0
