  • Toy'd

    (Computer Software industry)

    September 2005 — December 2007 (2 years 4 months)

    Toy'd is a student project developed by a small team of students at the {EPITECH.} European Institute of Technology.

    Toy'd is an acronym standing for Take Over Your Desktop. The project is composed of 2 parts: The Toy'd Window Manager and Aoof-Wm.

    The goal of the Toy'd Window Manager is to implement a fully portable, full-featured next-generation Window Manager, that would allow the standard user to take advantage of a common User Interface under Microsoft Windows or UNIX/GNU Linux systems.

    it relies on our multi-platform abstraction framework, Aoof-Wm (Advanced Object-Oriented Framework for Window Managers), which takes care of all platform and operating system specific functionalities.

    Toy'd can have an absolutely identical behavior on any host platform supported by Aoof-Wm.
