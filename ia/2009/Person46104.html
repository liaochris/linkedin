Cies Breijs’s Summary

Bachelor in Business Computer Science with experience in the open source (FLOSS) movement, web application development, and software auditing.

My first programming experience dates back to 1989, I was 7 years old writing my first lines using a LOGO-like language on an MSX. Two years later I attended a conference on an emerging new technology, the Internet, at the Erasmus University from which I would graduate 16 years later.

After being introduced to the open source (FLOSS) movement in 1997 I taught myself a variety of skills with help from the open source community. Since 2002 I'm a contributor to the KDE project, mainly through the acceptance of my open source pet project KTurtle (edu.kde.org/kturtle) into the edutainment module of KDE. KTurtle is a tool to teach children the basics of programming, in many ways similar to the LOGO-like language I coded my first lines in.

I became more aware and interested in the power of the new medium (Internet) and they way it helps evolving our global society in new directions.

From 2003 to 2007 I studied at the Erasmus University in Rotterdam and graduated with a Ba's in Business Computer Science. My main focus was on the economics of open source and the semantic web technology stack (RDF, RDFS, OWL and SPARQL).

From 2007 to 2009, while traveling Europe and Asia, I initiated my latest open source project, that now goes by the name of Truetopia (truetopiaproject.org). This project develops a web application that facilitates self-governing online communities.

Cies Breijs’s Specialties:

Web app development using Ruby (RoR/merb), Python and Java (Tomcat). Solid knowledge of web technologies: (X)HTML, CSS, SQL, XML, RDF(S), JavaScript (AJAX, mainly jQuery). Intermediate experience with Linux server administration: bash, Apache, MySQL, PostgreSQL, OpenVZ and Ruby/PHP web application stacks. Thorough C++ experience mainly using the Qt library for creating cross platform GUI applications.
