Shane Speering’s Summary

I am a Microsoft technologies developer with extensive industry experience in application security. During my tenure at b-sec I co-wrote the Business Application Security Assurance Framework. This framework provides a standard for assessing application code and identifying industry best practises in application security.

I have been coding in the .NET framework since the first beta release and have since obtained my .NET Microsoft Certified Application Developer (MCAD) certification.

I have performed a variety of security reviews, including recent penetration testing, code reviews and security policy assessments for financial institutions, state government agencies, health, legal, logistic sector organisations and critical infrastructure providers.

I have experience conducting Forensic data collection and evaluations using industry best practises. Collecting and managing evidence lifecycles to ensure admissibility in a court of law.

My in depth development knowledge coupled with my application security experience provides a highly desirable skill set as the focus in the security arena moves away from infrastructure and more to the application layers.
