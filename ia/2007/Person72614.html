Rick Hightower’s Summary

Rick Hightower, CTO of ArcMind, is a consultant, trainer and mentor who specializes in software development tools and processes, and developing enterprise applications using Spring, Hibernate, JSF, J2EE, XML, UML, SQL, and Oracle technologies. Rick’s publications include three books: the best seller Java Tools for eXtreme Programming., and Python Programming with the Java Class Libraries, and Jakarta Struts. Rick regularly writes articles.

Rick has 17 years of industry experience.
