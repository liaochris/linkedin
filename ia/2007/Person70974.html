Raphaël Pinson’s Summary

I have been working as a volunteer developer for Ubuntu and Kubuntu, making Debian software packages, triaging and fixing bugs, writing technical documentation, administrating machines.

I have also set my own project, Ichthux, with the same technical issues, in addition to project management.

In my current job, I supervise the technical side of the software deployment services for about 2000 Linux and Unix machines, including RedHat, Debian, OpenBSD and Solaris OSes. I have set a new deployment system, with a fully automated build farm (using buildd/wanna-build for the Debian side), an automated package repository (using reprepro for the Debian side) and an automated network installer (using FAI for the Debian side), and am now reorganizing and renewing the Cfengine (configuration deployment) system with tools developed internally.
