Willy Sudiarto Raharjo’s Summary

I have completed an education program suitable for disabled students in Netherland via OWG Project. OWG Project is a desktop-based educational application which is build using C# and .NET Framework. It was being made to help sight and hearing-impaired childrens to learn basic skills, such as playing with alphabet and words, basic mathematic operation, and many other. It has a TTS (Text-To-Speech) and highlighting feature, visual and audio feedback, and also teacher's section. It has a total of eleven modules, and each modules consists of six up to nine exercises. It's a complicated system, but i enjoyed working on this project.
