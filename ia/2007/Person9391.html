Barbara Ballard’s Summary

Wireless industry expert in product and user experience design. Evangelist for improved mobile product development process, incorporating device proliferation and user experience. Developer of mobile user interface patterns.

Experience includes phone user interfaces, Java ME, uiOne, mobile web, mobile payments, and location services.

Author, Designing the Mobile User Experience (Wiley 2007).

Professional services include mobile and wireless:
- user interface design consulting,
- usability consulting,
- graphic design,
- business strategy,
- user experience business process improvement,
- web development consulting, and
- application development consulting.
