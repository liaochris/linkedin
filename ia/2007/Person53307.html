Mahmoud Parsian’s Summary

I have been a software designer and developer for 30 years, starting with FORTRAN, Pascal, and C. For the past 12 years, I have been involved in Java technology (especially in the server-side). I have developed many server-side Java applications and frameworks, which have been deployed and used extensively in real-world. My most recent project was MyStuff with Ask.com (http://mystuff.ask.com). Currently, I am with LimeLife as a lead Server Architect (developing Java server-platform for running mobile applications). My current project is to build a scalable Java framework for building SMS applications.
