Austin Matthews’s Summary

I have 17 years of hands-on experience with UNIX, Windows, Linux, Mac OS and Networking technologies. I specialize in Oracle, Network Security, SMTP, DNS, Samba, Cisco, Snort IDS, EMC, Nagios, Backup/Restore, FTP, X11 and SSH.

I currently work for Iron Mountain as a System Engineer within the Digital Archive Operations group. I perform a variety of tasks including Linux/Solaris Administration, Storage Administration, Network Security, UNIX/Linux server builds, OS Patching, Oracle Installations, SQL DB Administration and Hardware upgrades.

Previous to working at Data Intensity as an Oracle/Unix Consultant, I worked for a Top Level Domain called Nu Domain as a System Administrator for UNIX, Linux, Windows and Mac OS. I maintained a large variety of systems at four world-wide locations on 3 separate continents in a 24/7/365 multi-tiered Internet production environment running MS SQL 2000, Proprietary Registration software and DNS on Solaris for 100K+ customers.

Before Nu Domain, I worked at Sun Microsystems in the Network Team as a Third Level (TSE) Network Engineer. I was the team lead engineer for Network Management, Linux, Windows OS, Samba and various older LAN/WAN technologies.

My database administration consists of basic SQL administration, tuning and weekly maintenance of Oracle, MS SQL 7 and MySQL databases. I have a working knowledge of AIX, C++, HP-UX, DB2, Oracle E-Business Suite and Oracle patching. I am capable of writing Java, Perl, XML and HTML scripts/code. I prefer the Bash shell, but I have been known to write a script for the Bourne or C-shell.

I currently hold two certifications from Sun Microsystems, one is for the System Administrator and the other is for the Network Administrator. I am a U.S. Army Certified UNIX Systems Administrator and I have held a Top Secret Clearance in the past, but I currently hold no military clearance.
