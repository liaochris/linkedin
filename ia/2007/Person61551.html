Michele Sciabarrà msciab@eprometeus.com’s Summary

I am specialized in Java Technology (Web, Gui, Mobile), Linux system administration, and I am a strong promoter of Open Source Software. I also have knowledge of Symbian C++ programming.

I started as a Java trainer and consultant, then I created a firm to develop web applications using Linux, Java and Open Source.

I specialized in in Mobile technologies, using the Symbian Mobile Operating System and the Java2 MicroEdition. I am a developer on both Symbian C++ and Java2 ME

I am currently interested in the growing usage of the video in the web, and in 3d virtual worlds.

As an author, I wrote two (italian) books: "Linux e Programmazione Web" (english: Linux and Web Programming), on Linux, Java, PHP, Perl and XML, and "il Software Open Source" (english: Open Source Software), both publishd by McGrawHill Libri Italia.

I created two popular italian web site, corsojava.it and corsolinux.it, with many tutorials (Java, XML, Php, Python). On corsojava.it I recently released an italian video tutorial on Java, in the form of a screencast, that is already quite popular.

I also created an english blog and tutorial on Symbian C++ programming, located at http://www.sciabarra.com.

I wrote many articles (more than 100), and taught a lot of classes (also, more than 100, I lost the count...)
