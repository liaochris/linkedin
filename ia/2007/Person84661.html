  • The Pennsylvania State University

    Masters, Computer Engineering, 1993 — 1995

    Thesis: New models for loop scheduling in distributed memory environments.

    Loops are the largest source of parallelism in many scientific and
    Engineering applications. These loops can be divided into two
    classifications: data independent and data dependent loops. Data
    independent, or DOALL loops allow for individual iterations of the loop
    to be assigned to processors in the system in any order.

    Traditional DOALL loop scheduling methods were designed for Shared
    Memory Environments. Iterations are allocated to processors in a
    decreasing manner. This decreasing chunk size pattern forces increased
    communication costs when transferred to distributed memory environments.

    Two methods have been proposed that increase chunk sizes and minimizes interprocessor communication. These methods can be implemented in parallel compilers. Experimental work on the nCUBE and IBM SP2 over applications have shown scalability over large number of processors and iterations.
