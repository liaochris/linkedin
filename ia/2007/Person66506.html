Paul June Domag’s Summary

Currently I am using Visual C# 2.0 as my main tool for developing both Web and Windows Forms. My preferred DBMS (Database Management System) is SQL 2005, though I've used several DBMS like Ingres, Oracle Express and MySql.

My past job requires me to extend applications such as AutoCAD which eventually grew into a hobby.
