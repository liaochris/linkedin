Tommy Hansen’s Summary

Currently Project Manager on the BGAN RAN project.
The BGAN RAN is the ground segment part of the infrastructure on the Inmarsat BGAN system providing over-satellite UMTS access at up to 432 kb/s.
I have been Project Manager on this system from initial project start up to now where full service is being provided for stationary, maritime, land-mobile and aeronautical use.
Project has involved as well advanced hardware, software as DSP development in addition to large subcontractor management.

Prior to this I have been working as R&D manager, designer and software developper on other satellite based communication systems.

Tommy Hansen’s Specialties:

Project Management, Subcontractor Management, large scale System Design, proposal and contract negotiation and Warranty, Support and Maintenance Management
