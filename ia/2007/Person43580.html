Jonathan Bachelor’s Summary

A life-long learner and technology enthusiast, I have been working with computers since 1983. My programming experiences have ranged from operating and maintaining a small, text-based Bulletin Board System on a 300 baud modem using an Apple IIc computer as a (woefully inadequate) server (1984), through developing n-tier applications using Visual Basic and Java (2000). My fascination with technology and computers carries on, thanks to an insatiable desire to learn, understand, and usefully employ the latest technological innovations in both the business and home environment, for both productivity and for entertainment.
