Baq Haidri’s Summary

I consider myself a very well-rounded Java/Web gal. My experience ranges from front end GUI design for both Java desktop and web applications all the way back to database schema design and all the layers in between. I've done rapid application prototyping, built high-throughput applications from scratch as well as been a team player for large production systems. I enjoy working directly with the users and am driven by the customer to provide the best solution possible in the fastest amount of time.
