Pandu Rudraraju’s Experience

  • Founder & CEO

    eBizPortals

    (Privately Held; 11-50 employees; Internet industry)

    1999 — 2005 (6 years)

Established partnerships with IBM and Oracle. Deployed the first simplified m-Commerce portal for AT&T Wireless. Developed Search Ahead vertical search solution that can dynamically search multiple websites concurrently in any given domain. Sold the IP assets to Fullpower.

  • Consultant

    Self-employed

  • VP of Engineering

    Interweave (Acquired by Cognos)

    (Privately Held; 11-50 employees; Computer Software industry)

    1996 — 1997 (1 year)

    Interweave was a venture funded startup that developed web based business intelligence software that simplified ad-hoc access to corporate databases. Managed engineering and worked closely with early customers. The company was acquired by Cognos.

  • Architect/Director of R&D

    Borland

    (Public Company; 1001-5000 employees; Computer Software industry)

    1989 — 1996 (7 years)

    As the principal architect, came up with the concept, architecture and design for core database technologies that included local database engines, query processors and transparent connectivity to server databases. Implemented the core foundation. Tuned the performance and helped Borland win many industry benchmarks. Built the core team. As Director, managed an engineering team of 35+ engineers and released five successive releases of the technology in various Borland products including Delphi, Paradox, dBase, C++, BDE and SQL Drivers. Represented Borland in SQL and X-Open standards committees and worked closely with partners like IBM and Novell. Gave talks at Borland Developer conferences. Conducted developer/partner conference focused on BDE. Recognized the need for an Application server to expand Borland development tools market and initiated the project in 1995.

    Carnegie Group

    (Privately Held; 51-200 employees; Computer Software industry)

    1986 — 1988 (2 years)

    Carnegie Group was founded by four Carnegie-Mellon University Professors and was one of the pioneers of AI/Expert Systems industry. Worked on integrating Lisp based AI/Expert Systems with Database Systems in association with Carnegie Group partner DEC.
