Joel Rosenberger’s Summary

My software development experience ranges from desktop applications, to server components, to web applications and services; mainly on application servers such as WebSphere Application Server and WebSphere Portal. My aim is to specialize in the development of Java applications and services, while providing added value in the areas of Websphere administration, project management, mentoring, customer support, .NET programming, and database development.

Challenges I would be particularly interested in include:
-Integrating systems, even if the systems in question would rather not work together.
-Providing access to data, even if the data lies across multiple databases or other data sources, or the data is poorly organized.
-Working with customers to brainstorm ideas, then turning scribbles on a napkin into reality in a short period of time.
-Figuring out how a new technology works, then through mentoring, helping others to use the technology effectively.
-Solving any problem for which others refuse to take ownership, or that has been dismissed as impossible.
-Leading software development teams through the life cycle, even if the team is relatively inexperienced, and needs help with configuration management, unit testing, architectural concepts, or otherwise.
