Uffe Fey’s Summary

Free agent slash freelance consultant slash contractor within the field of Microsoft .NET development, specialized in building web stuff but absolutely capable of doing any kind of software development.

I've been a contractor for 10 years and have 9 years of experience doing database driven web applications and services, including frontend development (GUI) and heavy backend systems. I've done everything from tiny no-name websites to big ass enterprise business solutions used by thousands of simultaneous users.
My area of work covers every range from doing estimates, analyzing, producing and documenting (see specialities below).

Building web 2.0 stuff is my passion at the moment and in that regard, I have big plans..

Uffe Fey’s Specialties:

Microsoft .NET 1.0, 1.1, 2.0 and 3.0 (C#), Windows Communication Foundation (WCF), web services, PDF creation, XML, XSL(T), XPath, ASP.NET, SQL Server 2000 + 2005, data modelling, SQL performance tuning, OO analysis and programming, application and framework architecture, documenting systems and code, performance tests of code, writing unit tests, writing secure code, GUI and front end development, XHTML, CSS (tableless layout), JavaScript, AJAX and pretty much everything else :)
