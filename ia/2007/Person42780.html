John Collins’s Summary

I am an open source evangelist and agile methods proponent, who’s aim is to promote the use of open source technologies and light-weight design methodologies in all organisations.

I am a strong advocate of test driven development, and always emphasize the importance of unit testing, code reviews, refactoring and documentation in all of my development work.

My main areas of expertise are in developing cross-browser scalable web interfaces, web application frameworks (MVC), web publishing solutions (CMS), automated build processes and version control, as well as other advanced database-driven web applications.

John Collins’s Specialties:

PHP, Perl, PEAR, DOM, Javascript, SVG, CSS, XML, SOAP, Java, Struts, Hibernate, Axis, Ruby on Rails, Ant, Maven, Phing, JUnit, PHPUnit, Linux, Samba, Subversion, Perforce, MySQL, MS SQL, DB2, Oracle, PostgreSQL, Apache, Tomcat, Eclipse, IntelliJ, lots of exposure to open source web applications like Moodle, SugarCRM, DSpace, Sakai, XRMS, Twiki and Mediawiki.
