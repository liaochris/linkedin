  • Inventor - Patent #6442247

    Genesys Telecommunications

    (Government Agency; Myself Only; Telecommunications industry)

    August 2002 — August 2002 (1 month)

    Sole inventor and awarded a patent for Method and apparatus for recording and automated playback of personal agent greetings in a communication-center environment. A system for presenting a recorded message on behalf of an agent receiving a call at an agent station in a call center, the agent station has a telephony switching apparatus enhanced by a computer-telephony integration (CTI) processor, an Interactive Voice Response (IVR) and a data repository having recorded messages stored on behalf of the agent and accessible to the CTI processor. The CTI processor routes incoming calls to the agent, uses data associated with the calls for selecting recorded messages, and causes a retrieved recorded message to be played to a caller upon the agent picking up the routed call. The system is useful with both conventional telephone systems and data network telephony, such as over the Internet.
