Ted Leung’s Summary

I am the development manager for OSAF's Cosmo project. Cosmo is a general sharing server that works with the Chandler desktop Personal Information Manager. It also contains an AJAXy web user interface to the data that is stored in it. In addition to Cosmo, my responsibilities also include community development for all the OSAF projects.

I am also a member of the Apache Software Foundation, and one of the original developers of the Xerces for Java XML parser. I have also served as the ASF's VP for XML. My first book, "Professional XML Development with Apache Tools", was published by Wrox in December 2003. I maintain a weblog at http://www.sauria.com/blog.

Ted Leung’s Specialties:

Open Source Software, Java, Python, XML, Web Services, Object-oriented databases
