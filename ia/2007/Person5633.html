  • Web and Systems Developer

    Becta

    (Government Agency; 201-500 employees; Internet industry)

    1999 — 2004 (5 years)

    I worked as a Web and Systems developer at Becta focussing on web application development and commercial application support. I worked on a major web portal produced for the government initiative 'The National Grid for Learning'. The portal was written in Java and utilised JSPs and Servlets to deliver the administration and public facing elements, built over MySQL and Oracle database systems using a bespoke JDBC persistence methodology. LDAP was used to deliver user registration services. I was heavily focussed on delivering the search element of the portal using Autonomy Ultraseek (formerly known as Inktomi Ultraseek and Verity Ultraseek).
