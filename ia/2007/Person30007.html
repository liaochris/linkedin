  • Sr. Software Engineer

    SmartStar

    (Privately Held; 51-200 employees; Computer Software industry)

    January 1991 — January 1994 (3 years 1 month)

    Team leader / engineer that created SmartReportPainter, an object-oriented GUI based DB report generator.

    Developed a "DOS" type windowing system that included a Dialog manager, Menu manager, Listbox manager, Edit box, Label, Radio button, Check box and a portability library.

    Senior engineer responsible for porting the SmartStar Vision product from the X Windows / Motif environment to other GUI's. Vision is an object-oriented GUI builder for SQL-based databases providing local and Client/Server access to databases such as Oracle, Sybase, Ingres, RDB and Sharebase, predating current object based systems on the PC.

    Vision used an internal class-based architecture, much like the X Intrinsics, but with the GUI abstracted out as a separate class to facilitate portability.

    I designed the GUI portability layer and did the port to Windows-NT (Win32) and provided assistance for the Macintosh port.
